A. Lesson — What are advanced relative clauses?

Relative clause

Rule / explanation: A relative clause adds information about a person, thing, place, time or entire statement.

Example / answer: The reform, which was introduced in 2025, remains controversial.

Defining clause

Rule / explanation: A defining clause identifies exactly which person or thing is meant and does not use commas.

Example / answer: Policies that reduce inequality tend to gain public support.

Non-defining clause

Rule / explanation: A non-defining clause adds supplementary information and is separated by commas.

Example / answer: The policy, which passed by a narrow majority, will take effect next year.

Advanced use

Rule / explanation: High-level relative clauses allow writers to compress information, link ideas and avoid repetitive sentences.

Example / answer: The report identifies several weaknesses, the most serious of which concerns regional funding.

B. Defining vs non-defining clauses

Essential information

Rule / explanation: A defining clause is necessary to identify the noun.

Example / answer: The voters who supported the reform were mainly urban.

Additional information

Rule / explanation: A non-defining clause adds information about an already identified noun.

Example / answer: The prime minister, who defended the reform, faced strong criticism.

Meaning changes

Rule / explanation: Commas can change the meaning by changing whether the group is restricted.

Example / answer: The ministers who opposed the bill resigned. / The ministers, who opposed the bill, resigned.

That in defining clauses

Rule / explanation: That can replace who or which in many defining clauses.

Example / answer: The evidence that the author presents is incomplete.

No that in non-defining clauses

Rule / explanation: Do not use that in a non-defining relative clause.

Example / answer: Correct: The report, which was published yesterday, caused controversy.

C. Who, whom, whose, which and that

Who

Rule / explanation: Use who for people as subject or, informally, as object.

Example / answer: The journalist who exposed the scandal received an award.

Whom

Rule / explanation: Use whom for people as object in formal English.

Example / answer: The minister whom the committee questioned later resigned.

Whose

Rule / explanation: Use whose to express possession for people, organisations and sometimes things.

Example / answer: The company whose practices were criticised issued an apology.

Which

Rule / explanation: Use which for things and animals.

Example / answer: The reform, which affects millions of workers, remains unpopular.

That

Rule / explanation: Use that for people or things in defining clauses, especially after superlatives and indefinite words.

Example / answer: This is the most convincing argument that the article presents.

D. Object pronoun omission

Omission in defining clauses

Rule / explanation: An object relative pronoun may be omitted in a defining clause.

Example / answer: The policy (that) the government introduced has failed.

No omission of subject

Rule / explanation: A subject relative pronoun cannot be omitted.

Example / answer: Wrong: The policy was introduced has failed. Correct: The policy that was introduced has failed.

No omission in non-defining clauses

Rule / explanation: Do not omit the relative pronoun in a non-defining clause.

Example / answer: The policy, which the government introduced last year, has failed.

Preposition restriction

Rule / explanation: Do not omit a relative pronoun when a preposition is placed before it.

Example / answer: The issue about which the article is written remains controversial.

E. Preposition + whom / which

Formal fronted preposition

Rule / explanation: In formal writing, place the preposition before whom or which.

Example / answer: The committee to which the minister reported demanded answers.

Informal stranded preposition

Rule / explanation: In less formal English, the preposition often remains at the end.

Example / answer: The committee which the minister reported to demanded answers.

No preposition + that

Rule / explanation: Do not place a preposition directly before that.

Example / answer: Wrong: the issue about that they argued. Correct: the issue about which they argued.

Whom after preposition

Rule / explanation: Use whom, not who, immediately after a fronted preposition.

Example / answer: The officials with whom the journalist spoke denied the allegation.

CPGE register

Rule / explanation: Fronted prepositions are useful in formal essays but should not be overused.

Example / answer: The principles on which the policy is based remain unclear.

F. Quantifier + of whom / of which

Some of whom

Rule / explanation: Use quantifier + of whom for people in a non-defining clause.

Example / answer: The committee interviewed ten experts, several of whom criticised the reform.

Many of which

Rule / explanation: Use quantifier + of which for things.

Example / answer: The report contains twenty recommendations, many of which require new funding.

None of whom

Rule / explanation: Use none of whom to exclude every person in the group.

Example / answer: The minister consulted three advisers, none of whom supported the proposal.

Most of which

Rule / explanation: Use most of which to comment on the majority of things previously mentioned.

Example / answer: The policy created several problems, most of which were predictable.

The most important of which

Rule / explanation: Use this structure to rank items in formal analysis.

Example / answer: The article raises three objections, the most important of which concerns legitimacy.

G. Which referring to a whole clause

Sentential which

Rule / explanation: Which can refer to the entire preceding clause, not just one noun.

Example / answer: The government withdrew the bill, which surprised many observers.

Comma required

Rule / explanation: A clause-level which comment is non-defining and follows a comma.

Example / answer: Turnout fell sharply, which weakened the government’s mandate.

Interpretation

Rule / explanation: The meaning is often this fact or this situation.

Example / answer: The reform increased costs, which was exactly what critics had predicted.

Avoid ambiguity

Rule / explanation: Place the which-clause carefully so the reference is clear.

Example / answer: Clear: The minister resigned, which ended the controversy.

Formal alternative

Rule / explanation: For greater formality, use a separate sentence with this, a result that or a development that.

Example / answer: The minister resigned. This development ended the controversy.

H. Relative adverbs: where, when and why

Where

Rule / explanation: Use where for places, institutions or situations that function metaphorically as places.

Example / answer: The country has reached a point where reform is unavoidable.

When

Rule / explanation: Use when for times, periods and occasions.

Example / answer: There are moments when compromise becomes impossible.

Why

Rule / explanation: Use why after reason in defining clauses.

Example / answer: The main reason why the policy failed was poor implementation.

Prepositional alternatives

Rule / explanation: Use in which, during which or for which for a more formal style.

Example / answer: The period during which the economy contracted was unusually long.

Avoid where for abstract nouns without a place-like meaning

Rule / explanation: Use in which or that when where sounds vague.

Example / answer: Better: a situation in which local authorities lacked resources.

I. Compound relative pronouns

Whoever

Rule / explanation: Whoever means any person who or the person who.

Example / answer: Whoever wins the election will face the same economic constraints.

Whatever

Rule / explanation: Whatever means anything that or regardless of what.

Example / answer: Whatever the government decides, the controversy will continue.

Whichever

Rule / explanation: Whichever refers to any choice from a limited set.

Example / answer: Whichever policy is adopted, implementation will remain difficult.

Wherever

Rule / explanation: Wherever means in any place where or regardless of where.

Example / answer: Wherever inequality rises, political distrust tends to follow.

Whenever

Rule / explanation: Whenever means at any time when or every time that.

Example / answer: Whenever public services decline, dissatisfaction increases.

J. Relative clauses after indefinite and superlative expressions

All that

Rule / explanation: That is especially common after all, everything, nothing and anything.

Example / answer: Everything that the government promised remains unfulfilled.

The only...that

Rule / explanation: Use that naturally after the only.

Example / answer: The only proposal that gained cross-party support concerned housing.

Superlative + that

Rule / explanation: That is common after superlatives.

Example / answer: This is the strongest argument that the author makes.

First / last + that

Rule / explanation: That is common after ordinal expressions.

Example / answer: The first issue that Parliament addressed was funding.

Who possible for people

Rule / explanation: Who remains possible when the antecedent clearly refers to a person.

Example / answer: She was the only minister who opposed the measure.

K. Advanced relative clauses in article commentary

The article, which...

Rule / explanation: Use a non-defining clause to add publication, tone or context.

Example / answer: The article, which appeared in a national newspaper, adopts a highly critical tone.

The author, whose...

Rule / explanation: Use whose to connect a writer to an argument, style or background.

Example / answer: The author, whose analysis focuses on inequality, questions the official figures.

Several claims, many of which...

Rule / explanation: Use quantifier relatives to evaluate groups of arguments.

Example / answer: The article makes several claims, many of which are poorly supported.

A point that...

Rule / explanation: Use a defining relative clause to identify a specific argument.

Example / answer: The author raises a point that deserves closer examination.

A conclusion which...

Rule / explanation: Use a clause to evaluate the effect or basis of a conclusion.

Example / answer: The text reaches a conclusion which the available evidence does not fully justify.

L. Relative clauses in CPGE essays

Compress information

Rule / explanation: Combine related statements without creating repetitive short sentences.

Example / answer: The reform, which was intended to reduce costs, ultimately increased them.

Define a concept

Rule / explanation: Use a defining relative clause to specify the meaning of an abstract noun.

Example / answer: A democracy is a system in which political power remains accountable.

Add analytical detail

Rule / explanation: Use a non-defining clause to enrich an argument without interrupting its structure.

Example / answer: Public trust, which is difficult to build, can collapse rapidly.

Present evidence

Rule / explanation: Use quantifier relatives to organise examples and data.

Example / answer: Three countries adopted the policy, two of which later abandoned it.

Formulate a problematique

Rule / explanation: Use relative structures to connect the issue to its wider implications.

Example / answer: The question is whether a system that prioritises efficiency can also guarantee fairness.

M. Punctuation and style

Commas

Rule / explanation: Use commas around non-defining relative clauses.

Example / answer: The reform, which was introduced last year, remains controversial.

No commas in defining clauses

Rule / explanation: Do not separate a defining clause from its noun.

Example / answer: The reforms that received adequate funding were more successful.

Two commas in mid-sentence

Rule / explanation: A non-defining clause inserted into a sentence normally needs a comma on both sides.

Example / answer: The minister, who had opposed the bill, later defended it.

Avoid overloading

Rule / explanation: Too many nested relative clauses make sentences difficult to follow.

Example / answer: Prefer two clear sentences to one sentence containing three embedded relatives.

Antecedent clarity

Rule / explanation: Place the relative clause immediately after the noun it modifies.

Example / answer: Clear: The committee rejected the proposal that the minister had submitted.

N. Typical mistakes by French speakers

That after a comma

Rule / explanation: That cannot introduce a non-defining relative clause.

Example / answer: Wrong: The policy, that was introduced last year... Correct: The policy, which was introduced last year...

Which for people

Rule / explanation: Use who or whom for people.

Example / answer: Wrong: the minister which resigned. Correct: the minister who resigned.

Missing commas

Rule / explanation: A non-defining relative clause must be marked by commas.

Example / answer: Correct: London, which is the UK capital, attracts millions of visitors.

Whose confusion

Rule / explanation: Whose means possession; who’s means who is or who has.

Example / answer: The author whose article we read... / Who’s responsible?

Comparing unlike antecedents

Rule / explanation: Make sure the relative pronoun clearly refers to the intended noun.

Example / answer: Avoid placing the clause too far from its antecedent.

Overusing whom

Rule / explanation: Whom is formal and mainly useful as an object or after a preposition.

Example / answer: Natural: the people who the journalist interviewed. Formal: the people whom the journalist interviewed.

O. Exercises

1. Add commas if needed

Rule / explanation: The reform which was introduced last year remains controversial.

Example / answer: The reform, which was introduced last year, remains controversial.

2. Choose who, whom or whose

Rule / explanation: The journalist ___ investigation exposed the scandal received an award.

Example / answer: The journalist whose investigation exposed the scandal received an award.

3. Omit the pronoun if possible

Rule / explanation: The policy that the government introduced has failed.

Example / answer: The policy the government introduced has failed.

4. Combine with of which

Rule / explanation: The report contains ten proposals. Most of them require new funding.

Example / answer: The report contains ten proposals, most of which require new funding.

5. Rewrite formally

Rule / explanation: The principles which the reform is based on are unclear.

Example / answer: The principles on which the reform is based are unclear.

6. Correct the sentence

Rule / explanation: The minister, that defended the bill, later resigned.

Example / answer: The minister, who defended the bill, later resigned.

7. Translate

Rule / explanation: Le rapport formule plusieurs recommandations, dont la plus importante concerne le financement régional.

Example / answer: The report makes several recommendations, the most important of which concerns regional funding.

8. Mini concours challenge

Rule / explanation: Write a paragraph using a defining clause, a non-defining clause, whose, which referring to a whole clause and many of which.

Example / answer: Example: Policies that ignore local conditions often fail. The reform, which was introduced without consultation, faced immediate resistance. The minister, whose authority was already weakened, refused to revise it, which intensified the controversy. Several objections were raised, many of which concerned regional funding.
