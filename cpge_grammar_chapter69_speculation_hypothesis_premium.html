Chapter 69 — Speculation and Hypothesis
B2 → C1 grammar for possible explanations, counterfactual reasoning and sophisticated analytical hypotheses.

Chapter objective

This premium grammar chapter develops advanced control of speculation and hypothesis: modal possibility, competing explanations, conditional reasoning, evidence-based hypotheses, counterfactuals and CPGE applications in essays, article commentary and khôlles.

The golden rule

A useful hypothesis links evidence to a possible explanation.
Evidence → cautious interpretation → alternative hypothesis → provisional conclusion
Present possibilitymay / might / could + base verb
Past possibilitymay / might / could have + past participle
Counterfactualif + past perfect → might / could / would have
Formal hypothesisassuming that / supposing / were to / had + inversion

A. Lesson — What are speculation and hypothesis?

Speculation

Rule / explanation: Speculation proposes a possible explanation or future development without claiming certainty.

Example / answer: The decline may reflect growing economic insecurity.

Hypothesis

Rule / explanation: A hypothesis is a reasoned proposition that can be examined, supported or challenged.

Example / answer: One hypothesis is that regional inequality has weakened trust in national institutions.

Evidence link

Rule / explanation: Strong speculation remains connected to observable facts, context or logical reasoning.

Example / answer: Given the timing of the announcement, the government may be responding to electoral pressure.

CPGE relevance

Rule / explanation: Speculation and hypothesis are central to article commentary, problematics, essay reasoning, oral debate and source comparison.

Example / answer: The author may be suggesting that the crisis is institutional rather than merely economic.

B. Modal verbs for present speculation

May

Rule / explanation: Use may for a plausible present possibility.

Example / answer: The reform may create additional administrative costs.

Might

Rule / explanation: Use might for a more tentative or remote possibility.

Example / answer: The disagreement might reflect deeper ideological divisions.

Could

Rule / explanation: Use could to introduce one possible explanation among several.

Example / answer: The discrepancy could result from incomplete data.

Must

Rule / explanation: Use must for a strong deduction supported by evidence.

Example / answer: The author must regard the issue as highly urgent.

Cannot

Rule / explanation: Use cannot to reject an explanation that conflicts with the evidence.

Example / answer: The trend cannot be explained by demographics alone.

C. Speculating about the past

May have + past participle

Rule / explanation: Use this for a possible past action or cause.

Example / answer: Officials may have underestimated public opposition.

Might have + past participle

Rule / explanation: Use this for a more tentative past possibility.

Example / answer: The journalist might have relied on an outdated source.

Could have + past participle

Rule / explanation: Use this for a possible past explanation or unrealised alternative.

Example / answer: The delay could have resulted from legal uncertainty.

Must have + past participle

Rule / explanation: Use this for a strong past deduction.

Example / answer: The government must have anticipated some resistance.

Cannot have + past participle

Rule / explanation: Use this for a past event judged impossible.

Example / answer: The committee cannot have reviewed the full report in one hour.

D. Speculating about the future

May / might / could

Rule / explanation: Use these modals for uncertain future outcomes.

Example / answer: The controversy may continue for several months.

Be likely to

Rule / explanation: Use this for a probable future development.

Example / answer: The proposal is likely to face parliamentary resistance.

Be unlikely to

Rule / explanation: Use this for a future outcome with low probability.

Example / answer: The measure is unlikely to resolve the structural problem.

There is a possibility that

Rule / explanation: Use this formal structure to introduce one future scenario.

Example / answer: There is a possibility that the coalition will call an early election.

It remains to be seen whether

Rule / explanation: Use this for an unresolved future question.

Example / answer: It remains to be seen whether the reform will restore public confidence.

E. If-clauses for hypotheses

Real hypothesis

Rule / explanation: Use if + present, with a future or modal result, for a realistic possibility.

Example / answer: If the government increases funding, waiting times may fall.

Unreal present hypothesis

Rule / explanation: Use if + past, with would/could/might, for an imagined or unlikely present situation.

Example / answer: If the policy were better funded, it might produce stronger results.

Unreal past hypothesis

Rule / explanation: Use if + past perfect, with would/could/might have, for a counterfactual past.

Example / answer: If ministers had acted earlier, the crisis might have been contained.

Mixed hypothesis

Rule / explanation: Use mixed conditionals when the condition and result belong to different time frames.

Example / answer: If the reform had been implemented properly, public trust might be higher now.

Were to

Rule / explanation: Use were to for a formal or remote future hypothesis.

Example / answer: If the coalition were to collapse, an early election would become likely.

F. Supposing, assuming and imagine

Supposing

Rule / explanation: Supposing introduces a hypothetical situation, often to test consequences.

Example / answer: Supposing the figures are inaccurate, how would this affect the conclusion?

Assuming

Rule / explanation: Assuming presents a condition provisionally accepted for reasoning.

Example / answer: Assuming that funding remains stable, the programme should continue.

Imagine

Rule / explanation: Imagine invites the listener or reader to consider a scenario.

Example / answer: Imagine that public trust continues to decline: what political consequences might follow?

Let us suppose

Rule / explanation: This formal phrase introduces a controlled thought experiment.

Example / answer: Let us suppose that regulation is removed entirely.

For the sake of argument

Rule / explanation: Use this to accept a premise temporarily without endorsing it.

Example / answer: For the sake of argument, let us assume that the forecast is correct.

G. What if and hypothetical questions

What if...?

Rule / explanation: Use what if to introduce a possible situation and explore its consequences.

Example / answer: What if the reform increases costs rather than reducing them?

What would happen if...?

Rule / explanation: Use this for explicit hypothetical consequence.

Example / answer: What would happen if public support fell below 20 per cent?

Could it be that...?

Rule / explanation: Use this to introduce a tentative explanation.

Example / answer: Could it be that the official figures exclude informal employment?

Might...?

Rule / explanation: Use might in formal speculative questions.

Example / answer: Might the decline be linked to changing media habits?

How would...?

Rule / explanation: Use how would to explore mechanisms or effects.

Example / answer: How would the economy react if interest rates rose sharply?

H. It is possible / conceivable / plausible that

Possible

Rule / explanation: Use possible for an open and realistic hypothesis.

Example / answer: It is possible that the sample is unrepresentative.

Conceivable

Rule / explanation: Use conceivable for an imaginable, often more remote possibility.

Example / answer: It is conceivable that the policy will have no lasting effect.

Plausible

Rule / explanation: Use plausible when an explanation seems reasonable and consistent with evidence.

Example / answer: It is plausible that regional underfunding caused the disparity.

Probable

Rule / explanation: Use probable when the hypothesis is more likely than not.

Example / answer: It is probable that the controversy will continue.

Unlikely

Rule / explanation: Use unlikely to reject a hypothesis cautiously rather than absolutely.

Example / answer: It is unlikely that one event alone explains the trend.

I. Seem, appear and look as though

Seem to

Rule / explanation: Use seem to for an interpretation based on available evidence.

Example / answer: The economy seems to be recovering slowly.

Appear to

Rule / explanation: Use appear to for a more formal cautious inference.

Example / answer: The reform appears to have widened regional inequalities.

It seems / appears that

Rule / explanation: Use an impersonal clause to reduce personal emphasis.

Example / answer: It appears that the government underestimated implementation costs.

Look as if / though

Rule / explanation: Use these mainly for observable appearance or informal inference.

Example / answer: It looks as though negotiations have broken down.

Seemingly

Rule / explanation: Use seemingly when something appears true, possibly in contrast with reality.

Example / answer: A seemingly minor change produced major consequences.

J. Hypotheses based on evidence

The evidence suggests that

Rule / explanation: Use this when evidence supports but does not prove a conclusion.

Example / answer: The evidence suggests that trust is declining.

The data are consistent with

Rule / explanation: Use this when findings fit a hypothesis without establishing it uniquely.

Example / answer: The results are consistent with the hypothesis that inequality reduces mobility.

This may indicate that

Rule / explanation: Use this for a cautious interpretation of a sign or pattern.

Example / answer: The lower turnout may indicate growing political disengagement.

One explanation is that

Rule / explanation: Use this to introduce a testable account.

Example / answer: One explanation is that local authorities lacked sufficient resources.

This raises the possibility that

Rule / explanation: Use this to derive a new hypothesis from prior evidence.

Example / answer: The regional gap raises the possibility that national averages are misleading.

K. Competing hypotheses

One possibility is that

Rule / explanation: Introduce the first plausible explanation.

Example / answer: One possibility is that voters rejected the policy itself.

Alternatively

Rule / explanation: Introduce a competing explanation.

Example / answer: Alternatively, turnout may have fallen because of political fatigue.

Another hypothesis is that

Rule / explanation: Add a distinct possible account.

Example / answer: Another hypothesis is that media coverage amplified dissatisfaction.

These explanations are not mutually exclusive

Rule / explanation: Use this when several causes may operate together.

Example / answer: Economic anxiety and institutional distrust may both have contributed.

Weighing alternatives

Rule / explanation: Compare each hypothesis against evidence rather than simply listing possibilities.

Example / answer: The demographic explanation is plausible, but it does not account for the timing.

L. Necessary and sufficient conditions

Necessary condition

Rule / explanation: A necessary condition must be present for an outcome, but may not be enough by itself.

Example / answer: Public support may be necessary for reform, but it is not sufficient.

Sufficient condition

Rule / explanation: A sufficient condition is enough to produce an outcome under the stated model.

Example / answer: A parliamentary majority is sufficient to pass the bill.

Only if

Rule / explanation: Only if introduces a necessary condition.

Example / answer: The reform can succeed only if local authorities receive adequate funding.

If

Rule / explanation: A simple if-clause often presents a sufficient condition in formal logic, though natural language is less exact.

Example / answer: If the vote passes, the bill will become law.

CPGE caution

Rule / explanation: Do not confuse correlation, necessity and sufficiency when constructing an argument.

Example / answer: Funding may be necessary without guaranteeing success.

M. Counterfactual hypothesis

Present counterfactual

Rule / explanation: Imagine a present situation contrary to fact.

Example / answer: If the country had a written constitution, the dispute might be resolved differently.

Past counterfactual

Rule / explanation: Imagine a different past event and result.

Example / answer: If the warning had been taken seriously, the crisis might have been avoided.

Could have

Rule / explanation: Use could have for an available but unrealised possibility.

Example / answer: The government could have phased in the reform.

Might have

Rule / explanation: Use might have for an uncertain alternative outcome.

Example / answer: A broader consultation might have reduced opposition.

Would have

Rule / explanation: Use would have for the expected result within the hypothetical model.

Example / answer: A delay would have increased administrative costs.

N. Hypothesis vs prediction

Hypothesis

Rule / explanation: A hypothesis explains why something happens or proposes a relationship.

Example / answer: The hypothesis is that economic insecurity weakens institutional trust.

Prediction

Rule / explanation: A prediction states what is expected to happen, often if the hypothesis is correct.

Example / answer: If the hypothesis is correct, distrust should be highest in the poorest regions.

Testing

Rule / explanation: Evidence may support, weaken or fail to distinguish between hypotheses.

Example / answer: The survey supports the hypothesis but does not prove it.

Falsifiability

Rule / explanation: A useful hypothesis should allow evidence that could count against it.

Example / answer: If no possible evidence could challenge it, the claim is analytically weak.

CPGE use

Rule / explanation: Distinguishing explanation from prediction strengthens scientific and social analysis.

Example / answer: The author predicts unrest but does not explain its underlying mechanism.

O. Correlation and causal speculation

Correlation

Rule / explanation: Two variables may change together without one causing the other.

Example / answer: Housing costs and political dissatisfaction are correlated.

Possible cause

Rule / explanation: Use may contribute to or could influence for cautious causal language.

Example / answer: High housing costs may contribute to lower social mobility.

Reverse causality

Rule / explanation: Consider whether the proposed effect could also influence the supposed cause.

Example / answer: Low trust may reduce participation, but low participation may also weaken trust.

Third variable

Rule / explanation: A separate factor may explain both observed trends.

Example / answer: Economic decline could influence both turnout and dissatisfaction.

No automatic causation

Rule / explanation: Avoid treating sequence or correlation as proof of cause.

Example / answer: The fact that support fell after the reform does not establish that the reform caused the fall.

P. Speculation in article summaries

Preserve the source’s caution

Rule / explanation: Report may, might and could without turning them into factual claims.

Example / answer: The article suggests that the policy may deepen inequality.

Distinguish fact from hypothesis

Rule / explanation: Separate reported events from the author’s explanation.

Example / answer: The figures show a decline; the journalist attributes it to weak public services.

Report alternatives

Rule / explanation: Use one explanation, alternatively and another possibility.

Example / answer: The author considers whether economic pressure or political fatigue is more important.

Avoid invented motives

Rule / explanation: Do not speculate about an author or politician without textual evidence.

Example / answer: Use the wording and structure of the source as support.

Concise synthesis

Rule / explanation: Group related hypotheses rather than listing every speculative detail.

Example / answer: The text presents the crisis as both economic and institutional.

Q. Speculation in CPGE commentary

Interpret language

Rule / explanation: Use cautious modals when inferring tone, intention or audience effect.

Example / answer: The headline may be designed to create urgency.

Contextualise

Rule / explanation: Present plausible links to wider events without claiming certainty.

Example / answer: The article could be responding to growing concern about regional inequality.

Explain omissions

Rule / explanation: Suggest why a source may exclude information, but mark the inference clearly.

Example / answer: The journalist may have omitted the data for reasons of space.

Test the author’s hypothesis

Rule / explanation: Ask whether the evidence actually supports the proposed explanation.

Example / answer: The argument is plausible, but the article provides little comparative evidence.

Offer alternatives

Rule / explanation: A strong commentary compares several explanations before judging them.

Example / answer: Economic anxiety may matter, but institutional distrust could be equally important.

R. Speculation in essays

Problematique

Rule / explanation: Use a hypothesis-driven question that reveals a genuine tension.

Example / answer: Could technological progress strengthen democracy while also increasing surveillance?

Introduction

Rule / explanation: State what may be happening and why the issue deserves examination.

Example / answer: Artificial intelligence may improve public services, but it could also deepen existing inequalities.

Development

Rule / explanation: Organise paragraphs around competing explanations, conditions or scenarios.

Example / answer: One hypothesis is that regulation restricts innovation; another is that it creates trust.

Counterargument

Rule / explanation: Use suppose, even if and assuming to test the limits of a claim.

Example / answer: Even if regulation increases costs, it may still be justified by greater safety.

Conclusion

Rule / explanation: State which hypothesis is best supported while preserving appropriate limits.

Example / answer: The evidence most strongly supports a mixed explanation involving both economic and political factors.

S. Speculation in khôlles and debate

Think aloud carefully

Rule / explanation: Use perhaps, may, might and one possibility is that while developing an interpretation.

Example / answer: Perhaps the author is contrasting national optimism with local frustration.

Build hypotheses

Rule / explanation: Move from evidence to interpretation explicitly.

Example / answer: Because the article repeats the word crisis, the writer may be seeking to create urgency.

Invite alternatives

Rule / explanation: Use could there be another explanation? or alternatively.

Example / answer: Alternatively, the decline could reflect changes in media consumption.

Avoid empty speculation

Rule / explanation: Do not use maybe as a substitute for knowledge or analysis.

Example / answer: Each possibility should have a reason behind it.

Conclude provisionally

Rule / explanation: Use on balance or the most plausible explanation appears to be.

Example / answer: On balance, the most plausible explanation is weak implementation rather than flawed design.

T. Formal hypothesis language

Let us assume that

Rule / explanation: Introduce a premise for analytical purposes.

Example / answer: Let us assume that the official figures are accurate.

On the assumption that

Rule / explanation: Make the underlying premise explicit.

Example / answer: On the assumption that funding remains stable, the programme is viable.

Were this to happen

Rule / explanation: Use inversion for a formal remote future hypothesis.

Example / answer: Were this to happen, public confidence would decline further.

Had the government acted

Rule / explanation: Use inverted past conditional for formal counterfactual analysis.

Example / answer: Had the government acted sooner, the damage might have been limited.

Should the policy fail

Rule / explanation: Use should-inversion for a formal future possibility.

Example / answer: Should the policy fail, the coalition would face intense pressure.

U. Common mistakes by French speakers

Maybe vs may be

Rule / explanation: Maybe is an adverb; may be is a modal verb plus be.

Example / answer: Maybe the policy will work. / The policy may be effective.

Double modal

Rule / explanation: Do not combine modal auxiliaries directly.

Example / answer: Wrong: might could happen. Correct: might happen / could happen.

If + will

Rule / explanation: Do not normally use will in the if-clause of a condition.

Example / answer: Wrong: If the government will act... Correct: If the government acts...

Past modal form

Rule / explanation: Use modal + have + past participle for past speculation.

Example / answer: Wrong: may went. Correct: may have gone.

Speculation as fact

Rule / explanation: Do not present motives, causes or outcomes as certain without evidence.

Example / answer: Use may, suggest or appear where appropriate.

Too many hypotheses

Rule / explanation: Listing possibilities without evaluating them creates weak analysis.

Example / answer: Rank explanations according to evidence and relevance.

V. Exercises

1. Present possibility

Rule / explanation: Perhaps the policy increases regional inequality.

Example / answer: The policy may / might increase regional inequality.

2. Past speculation

Rule / explanation: Perhaps officials underestimated public opposition.

Example / answer: Officials may / might have underestimated public opposition.

3. Formal hypothesis

Rule / explanation: Suppose the figures are wrong.

Example / answer: Let us suppose that the figures are inaccurate.

4. Competing explanation

Rule / explanation: The decline may reflect economic anxiety. Add another possibility.

Example / answer: Alternatively, it could result from political fatigue.

5. Counterfactual past

Rule / explanation: The government did not act early, so the crisis worsened.

Example / answer: If the government had acted earlier, the crisis might have been contained.

6. Correct the error

Rule / explanation: If Parliament will reject the bill, the coalition may collapse.

Example / answer: If Parliament rejects the bill, the coalition may collapse.

7. Translate

Rule / explanation: Une hypothèse plausible est que le manque de financement a aggravé les inégalités régionales.

Example / answer: A plausible hypothesis is that insufficient funding worsened regional inequalities.

8. Mini concours challenge

Rule / explanation: Write a paragraph using one hypothesis, one competing explanation, one counterfactual and one provisional conclusion.

Example / answer: Example: One hypothesis is that public trust declined because the reform was poorly implemented. Alternatively, dissatisfaction could reflect broader economic insecurity. Had local authorities received more funding, opposition might have been less intense. On balance, the evidence appears to support a combination of institutional and economic explanations.
