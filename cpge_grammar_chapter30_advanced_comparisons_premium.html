B1 → B2 grammar for nuanced comparison, contrast, data commentary and essay argumentation.

Chapter objective

This premium grammar chapter helps CPGE students move beyond basic comparatives and superlatives. It covers as...as, not as...as, the more... the more, modifiers such as far and slightly, comparison in data commentary, formal alternatives and common French-speaker mistakes.

The golden rule

Comparison is not only “more” and “the most”
as serious as / less convincing than / far more complex than / the more... the more...
Main useAdvanced comparisons help students compare countries, policies, articles, trends, arguments and historical periods precisely.
CPGE habitUse comparison to structure analysis: more convincing than, less effective than, as problematic as, far more nuanced than.
Key trapUse than after comparatives, not that: more complex than expected.

A. Lesson — Why advanced comparisons matter

Comparing arguments

French: comparer des arguments

Rule / explanation: Comparisons help evaluate which argument is stronger, weaker or more nuanced.

Example / answer: The second argument is more convincing than the first.

Comparing policies

French: comparer des politiques

Rule / explanation: Useful for discussing reforms and their effects.

Example / answer: This policy is less effective than previous measures.

Comparing data

French: comparer des données

Rule / explanation: Use comparison to describe gaps, trends and proportions.

Example / answer: Youth unemployment is significantly higher than the national average.

B. Equality and inequality

As...as

French: aussi... que

Rule / explanation: Use as + adjective/adverb + as to express equality.

Example / answer: The issue is as important as the author claims.

Not as...as

French: pas aussi... que

Rule / explanation: Use not as...as for weaker comparison.

Example / answer: The evidence is not as strong as the conclusion suggests.

Just as...as

French: tout aussi... que

Rule / explanation: Use just as...as to insist on equality.

Example / answer: Social inequality is just as important as economic growth.

C. Modifying comparatives

Much / far / considerably

French: beaucoup / nettement

Rule / explanation: Use these to strengthen a comparison.

Example / answer: The problem is far more complex than it seems.

Slightly / marginally

French: légèrement / marginalement

Rule / explanation: Use these for small differences.

Example / answer: Turnout was slightly lower than in the previous election.

Significantly

French: de manière significative

Rule / explanation: Useful for data and academic comparison.

Example / answer: Inflation was significantly higher in 2022.

D. Double comparisons

The more... the more...

French: plus... plus...

Rule / explanation: Use this structure to show two linked increases.

Example / answer: The more unequal a society becomes, the more fragile democracy may be.

The less... the more...

French: moins... plus...

Rule / explanation: Use it to show inverse relationships.

Example / answer: The less citizens trust institutions, the more likely they are to abstain.

The more... the less...

French: plus... moins...

Rule / explanation: Useful for nuanced cause-and-effect analysis.

Example / answer: The more polarised debate becomes, the less compromise seems possible.

E. Superlatives and ranking

The most / the least

French: le plus / le moins

Rule / explanation: Use for ranking within a group.

Example / answer: This is the most convincing part of the article.

One of the most

French: l’un des plus

Rule / explanation: Follow with a plural noun.

Example / answer: Housing is one of the most pressing issues today.

By far the most

French: de loin le plus

Rule / explanation: Strong superlative marker.

Example / answer: This is by far the strongest argument in the text.

F. Formal comparison structures

Compared with / compared to

French: comparé à

Rule / explanation: Both are used; compared with often stresses differences.

Example / answer: Compared with previous reforms, this measure is more ambitious.

In comparison with

French: en comparaison avec

Rule / explanation: Formal phrase for essays and data commentary.

Example / answer: In comparison with 2010, turnout was lower.

Unlike

French: contrairement à

Rule / explanation: Use unlike to introduce contrast.

Example / answer: Unlike the first article, the second focuses on economic causes.

G. Typical mistakes by French speakers

More... that

French: than obligatoire

Rule / explanation: Use than after comparatives.

Example / answer: Wrong: more important that. Correct: more important than.

As... than

French: as...as

Rule / explanation: Equality uses as...as.

Example / answer: Wrong: as important than. Correct: as important as.

One of the most + singular

French: pluriel obligatoire

Rule / explanation: Use a plural noun after one of the most.

Example / answer: Wrong: one of the most important issue. Correct: one of the most important issues.

Similar than

French: similar to

Rule / explanation: Use similar to.

Example / answer: Wrong: similar than. Correct: similar to.
