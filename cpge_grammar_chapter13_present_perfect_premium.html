Chapter 13 — Present Perfect
B1 grammar for current relevance, recent news, long-term trends and article analysis.

Chapter objective

This premium grammar chapter helps CPGE students master the Present Perfect. It covers have/has + past participle, unfinished time, present results, experience, recent developments, for and since, the contrast with the Past Simple, and common mistakes made by French speakers.

The golden rule

Subject + have / has + past participle
The crisis has changed public opinion.
Social media have transformed political debate.
Main useThe Present Perfect connects past actions or developments to the present.
CPGE habitUse it for current trends and recent relevance: inequality has increased, trust has declined, the article has shown.
Key trapDo not use the Present Perfect with a finished date: The crisis began in 2008, not The crisis has begun in 2008.

A. Lesson — What is the Present Perfect?

Present Perfect

French: present perfect

Rule / explanation: The Present Perfect links the past to the present. It is formed with have/has + past participle.

Example / answer: The crisis has changed public opinion.

Connection with the present

French: lien avec le présent

Rule / explanation: Use it when a past action has a present result or relevance.

Example / answer: The scandal has damaged trust in government.

Unfinished time

French: période non terminée

Rule / explanation: Use it with periods that are not finished: today, this week, this year, recently.

Example / answer: This year, inflation has affected many households.

Life experience

French: expérience

Rule / explanation: Use it to talk about experiences without saying exactly when.

Example / answer: Many voters have lost faith in traditional parties.

Recent change

French: changement récent

Rule / explanation: Use it for recent developments, often with just, already, yet or recently.

Example / answer: The government has just announced a new reform.

Repeated past to now

French: répétition jusqu’au présent

Rule / explanation: Use it for actions repeated from the past until now.

Example / answer: The issue has appeared in several recent articles.

For and since

French: for / since

Rule / explanation: Use for with duration, since with starting point.

Example / answer: The debate has lasted for years. / It has continued since 2016.

CPGE use

French: usage CPGE

Rule / explanation: The Present Perfect is essential for discussing current trends, recent news and long-term change.

Example / answer: Democracy has become more fragile in many countries.

B. Form — Affirmative sentences

I / You / We / They have + past participle

French: have + participe passé

Rule / explanation: Use have with I, you, we, they and plural subjects.

Example / answer: Many citizens have protested against the reform.

He / She / It has + past participle

French: has + participe passé

Rule / explanation: Use has with he, she, it and singular subjects.

Example / answer: The article has highlighted a major contradiction.

Regular past participles

French: participes réguliers

Rule / explanation: Regular verbs use -ed.

Example / answer: The crisis has affected low-income families.

Irregular past participles

French: participes irréguliers

Rule / explanation: Irregular participles must be memorised.

Example / answer: The government has chosen a controversial strategy.

Been

French: été

Rule / explanation: Past participle of be.

Example / answer: The debate has been intense.

Had

French: eu

Rule / explanation: Past participle of have.

Example / answer: The policy has had unexpected consequences.

Gone / been

French: gone / been

Rule / explanation: Gone = went and is still there; been = went and came back or experience.

Example / answer: The minister has gone to Brussels. / She has been to Brussels before.

Contracted forms

French: formes contractées

Rule / explanation: I've, you've, we've, they've, he's, she's, it's are common, especially in speech.

Example / answer: The issue has become urgent. / It’s become urgent.

C. Negative and question forms

Negative form

French: négation

Rule / explanation: Use have/has not + past participle.

Example / answer: The reform has not solved the problem.

Haven't / hasn't

French: formes contractées

Rule / explanation: Contractions are common in speech and informal writing.

Example / answer: The article hasn't addressed the economic context.

Question form

French: question

Rule / explanation: Put have/has before the subject.

Example / answer: Has the policy reduced inequality?

Wh- question

French: question en wh-

Rule / explanation: Wh-word + have/has + subject + past participle.

Example / answer: Why has public trust declined?

Short answers

French: réponses courtes

Rule / explanation: Use yes/no + subject + have/has.

Example / answer: Yes, it has. / No, they haven't.

No do

French: pas de do

Rule / explanation: Do not use do/does/did with the Present Perfect.

Example / answer: Wrong: Does the article has shown? Correct: Has the article shown?

Past participle required

French: participe passé requis

Rule / explanation: After have/has, use the past participle, not the base verb.

Example / answer: Wrong: The policy has fail. Correct: The policy has failed.

Question with ever

French: ever

Rule / explanation: Use ever in questions about life experience or historical experience.

Example / answer: Has democracy ever been under such pressure?

D. Time markers with the Present Perfect

Already

French: déjà

Rule / explanation: Use already for something completed earlier than expected.

Example / answer: The government has already announced new measures.

Yet

French: pas encore / déjà

Rule / explanation: Use yet in negatives and questions to mean up to now.

Example / answer: The bill has not passed yet. / Has the report been published yet?

Just

French: venir de

Rule / explanation: Use just for very recent events.

Example / answer: The newspaper has just released new figures.

Recently

French: récemment

Rule / explanation: Use recently for events in the near past with present relevance.

Example / answer: Several studies have recently challenged this assumption.

So far

French: jusqu’à présent

Rule / explanation: Use so far to mean up to now.

Example / answer: So far, the policy has failed to reassure voters.

Over the past decade

French: au cours de la dernière décennie

Rule / explanation: Use it for a period continuing to the present.

Example / answer: Over the past decade, rents have increased sharply.

For

French: depuis / pendant

Rule / explanation: Use for + duration.

Example / answer: The debate has lasted for several years.

Since

French: depuis

Rule / explanation: Use since + starting point.

Example / answer: The issue has become more visible since the pandemic.

E. Present Perfect vs Past Simple

Present Perfect

French: present perfect

Rule / explanation: Use it when the time is unfinished or the present result matters.

Example / answer: The government has introduced several reforms this year.

Past Simple

French: prétérit

Rule / explanation: Use it when the time is finished and specific.

Example / answer: The government introduced the reform in 2019.

No finished date with Present Perfect

French: pas de date terminée

Rule / explanation: Do not use the Present Perfect with yesterday, in 2016, last year if the time is finished.

Example / answer: Wrong: The crisis has begun in 2008. Correct: The crisis began in 2008.

Present result

French: résultat présent

Rule / explanation: Use Present Perfect when the result matters now.

Example / answer: The scandal has weakened the minister.

Finished event

French: événement terminé

Rule / explanation: Use Past Simple for a completed historical event.

Example / answer: The referendum took place in 2016.

News introduction

French: actualité récente

Rule / explanation: Headlines and introductions often use Present Perfect for recent news.

Example / answer: The government has announced a new climate plan.

Details after news

French: détails au prétérit

Rule / explanation: After introducing news with Present Perfect, details often use Past Simple.

Example / answer: The government has announced a plan. It presented the proposal yesterday.

CPGE contrast

French: contraste CPGE

Rule / explanation: Use the contrast to avoid major tense errors in article summaries.

Example / answer: The article has sparked debate, but it was published last Monday.

F. Common CPGE uses

Current trends

French: tendances actuelles

Rule / explanation: Use Present Perfect for trends continuing into the present.

Example / answer: Inequality has increased in many Western countries.

Recent political developments

French: actualité politique

Rule / explanation: Use it for current news relevance.

Example / answer: The Prime Minister has faced growing criticism.

Social change

French: changement social

Rule / explanation: Use it for changes over time.

Example / answer: Social media have transformed public debate.

Economic consequences

French: conséquences économiques

Rule / explanation: Use it for present consequences of past events.

Example / answer: Inflation has reduced purchasing power.

Environmental change

French: environnement

Rule / explanation: Use it for developments still relevant today.

Example / answer: Climate change has intensified extreme weather events.

Article summary

French: résumé d’article

Rule / explanation: Use it to explain what the article has done or shown.

Example / answer: The article has shown that the issue is more complex than expected.

Data commentary

French: commentaire de données

Rule / explanation: Use it for changes over an unfinished period.

Example / answer: Since 2010, youth unemployment has fallen in some regions.

Argumentation

French: argumentation

Rule / explanation: Use it to connect past developments to present debates.

Example / answer: Globalisation has reshaped national politics.

G. Typical mistakes by French speakers

Using Present Perfect with finished dates

French: date terminée

Rule / explanation: Use Past Simple with finished time markers.

Example / answer: Wrong: Brexit has happened in 2016. Correct: Brexit happened in 2016.

Forgetting have/has

French: oubli de have/has

Rule / explanation: Present Perfect needs have/has.

Example / answer: Wrong: The article shown. Correct: The article has shown.

Using the past simple form instead of participle

French: mauvais participe

Rule / explanation: Some irregular participles differ from past forms.

Example / answer: Wrong: has wrote. Correct: has written.

Using since with a duration

French: for / since

Rule / explanation: Use for with duration and since with starting point.

Example / answer: Wrong: since three years. Correct: for three years.

Using for with a starting point

French: for / since

Rule / explanation: Use since with a starting point.

Example / answer: Wrong: for 2016. Correct: since 2016.

Confusing been and gone

French: been / gone

Rule / explanation: Been often means experience/return; gone means still away.

Example / answer: She has been to London. / She has gone to London.

Using did in Present Perfect questions

French: did incorrect

Rule / explanation: Use have/has, not did.

Example / answer: Wrong: Did the article has shown? Correct: Has the article shown?

Overusing Present Perfect in historical context

French: abus du present perfect

Rule / explanation: For history, use Past Simple if the period is clearly finished.

Example / answer: Correct: The welfare state expanded after 1945.

H. Exercises

Exercise 1 — Complete the sentence

French: exercice

Rule / explanation: The crisis ___ (change) public opinion.

Example / answer: Answer: has changed

Exercise 2 — Complete the sentence

French: exercice

Rule / explanation: Several studies ___ (challenge) this assumption recently.

Example / answer: Answer: have challenged

Exercise 3 — Make it negative

French: exercice

Rule / explanation: The reform has solved the problem.

Example / answer: Answer: The reform has not solved the problem.

Exercise 4 — Make a question

French: exercice

Rule / explanation: The article has mentioned the economic context.

Example / answer: Answer: Has the article mentioned the economic context?

Exercise 5 — Choose for or since

French: exercice

Rule / explanation: The debate has lasted ___ 2016.

Example / answer: Answer: since

Exercise 6 — Choose for or since

French: exercice

Rule / explanation: The debate has lasted ___ several years.

Example / answer: Answer: for

Exercise 7 — Correct the mistake

French: exercice

Rule / explanation: The crisis has begun in 2008.

Example / answer: Answer: The crisis began in 2008.

Exercise 8 — Correct the mistake

French: exercice

Rule / explanation: The author has wrote a convincing article.

Example / answer: Answer: The author has written a convincing article.

Exercise 9 — Correct the mistake

French: exercice

Rule / explanation: Since three years, rents have increased.

Example / answer: Answer: For three years, rents have increased. / Rents have increased over the past three years.

Exercise 10 — Translate

Example / answer: Answer: Social media have transformed public debate.

Exercise 11 — Translate

Example / answer: Answer: The government has announced a new reform this week.
