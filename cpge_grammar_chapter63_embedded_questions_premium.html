The golden rule

Inside an embedded question: subject + verb order
Direct: Why did the reform fail?
Embedded: Could you explain why the reform failed?
Wh-questionframe + wh-word + subject + verb
Yes/no questionif / whether + subject + verb
Before infinitive or prepositionuse whether, not if

A. Lesson — What are embedded questions?

Embedded question

Rule / explanation: A question placed inside a larger statement, question or polite request.

Example / answer: Could you tell me why the reform was delayed?

Statement word order

Rule / explanation: Inside the embedded clause, use subject + verb order.

Example / answer: Direct: Why did it fail? Embedded: Do you know why it failed?

Function

Rule / explanation: Embedded questions express politeness, uncertainty, enquiry and academic problematics.

Example / answer: The article examines whether the reform can succeed.

CPGE relevance

Rule / explanation: They are essential for khôlles, interviews, summaries and essay problematics.

Example / answer: The central issue is whether growth can remain socially sustainable.

B. Direct vs embedded word order

Direct question

Rule / explanation: Direct questions normally use auxiliary–subject inversion.

Example / answer: Why did Parliament reject the bill?

Embedded clause

Rule / explanation: The embedded part uses statement order.

Example / answer: Could you explain why Parliament rejected the bill?

No double inversion

Rule / explanation: Invert only the introductory question, not the embedded clause.

Example / answer: Wrong: Do you know why did Parliament reject it?

Do-support disappears

Rule / explanation: Remove do, does and did from the embedded clause.

Example / answer: We need to know what the policy achieves.

Other auxiliaries remain

Rule / explanation: Perfect, modal and progressive auxiliaries stay in statement order.

Example / answer: Can you explain why the government has delayed action?

C. Wh-word embedded questions

What

Rule / explanation: Use what for things, actions or information.

Example / answer: The report does not explain what the government intends to do.

Why

Rule / explanation: Use why for cause or motivation.

Example / answer: Analysts are asking why confidence has declined.

How

Rule / explanation: Use how for manner, method or degree.

Example / answer: The article examines how social media shapes political debate.

When / where

Rule / explanation: Use these for time and place.

Example / answer: It remains unclear when the inquiry will publish its findings.

Who / which

Rule / explanation: Use who for people and which for a limited choice.

Example / answer: The committee must decide which proposal is preferable.

D. If and whether

Whether

Rule / explanation: Preferred in formal and academic embedded yes/no questions.

Example / answer: The article asks whether the reform is sustainable.

If

Rule / explanation: Common in ordinary embedded yes/no questions.

Example / answer: Do you know if the figures have been verified?

Whether or not

Rule / explanation: Use when both outcomes are explicitly relevant.

Example / answer: The committee must determine whether or not the data are reliable.

Whether...or

Rule / explanation: Use before stated alternatives.

Example / answer: The debate concerns whether regulation protects innovation or restricts it.

Formal preference

Rule / explanation: In CPGE writing, whether is usually the safer choice.

Example / answer: The central issue is whether growth benefits all social groups.

E. When only whether is possible

After a preposition

Rule / explanation: Use whether, not if, after a preposition.

Example / answer: The debate focuses on whether the plan is viable.

Before to-infinitive

Rule / explanation: Use whether before to + infinitive.

Example / answer: The government must decide whether to intervene.

As subject

Rule / explanation: Use whether when the clause is the subject.

Example / answer: Whether the policy will succeed remains uncertain.

Explicit alternatives

Rule / explanation: Use whether before alternatives.

Example / answer: They asked whether the failure came from funding or management.

Whether or not initially

Rule / explanation: Use whether when or not immediately follows.

Example / answer: Whether or not the bill passes, the debate will continue.

F. Polite question frames

Could you tell me...?

Rule / explanation: A standard polite request for information.

Example / answer: Could you tell me when the report will be published?

Do you know...?

Rule / explanation: A neutral way to ask whether someone has information.

Example / answer: Do you know why the vote was postponed?

Would you mind telling me...?

Rule / explanation: A highly polite form.

Example / answer: Would you mind telling me how the figures were calculated?

May I ask...?

Rule / explanation: A formal interview or oral-exam frame.

Example / answer: May I ask what evidence supports this conclusion?

I was wondering...

Rule / explanation: A soft and indirect request.

Example / answer: I was wondering whether you could clarify the author’s position.

G. Embedded questions as subjects

Subject clause

Rule / explanation: The whole embedded question can function as subject.

Example / answer: Why the policy failed remains unclear.

Whether-clause

Rule / explanation: Common for unresolved alternatives.

Example / answer: Whether the reform can succeed depends on funding.

Agreement

Rule / explanation: A clause subject normally takes a singular verb.

Example / answer: How the data were collected is important.

Introductory it

Rule / explanation: Use it to postpone a long subject clause.

Example / answer: It remains unclear why the policy failed.

Formal emphasis

Rule / explanation: Wh-clauses can create strong academic focus.

Example / answer: What the evidence demonstrates is that inequality persists.

H. Objects and complements

Object of a verb

Rule / explanation: Embedded questions follow know, ask, explain, determine, discover and consider.

Example / answer: The inquiry must determine who authorised the payment.

Complement after be

Rule / explanation: They can complete a noun or pronoun after be.

Example / answer: The question is whether the policy can work.

After adjectives

Rule / explanation: They follow unclear, uncertain, obvious and difficult.

Example / answer: It is unclear why the figures were withheld.

After nouns

Rule / explanation: They complete nouns such as issue, question and uncertainty.

Example / answer: The issue of whether regulation restricts innovation remains unresolved.

Problematique use

Rule / explanation: They frame academic enquiry elegantly.

Example / answer: The real question is how institutions can restore trust.

I. After reporting and thinking verbs

Ask

Rule / explanation: Introduces a reported request for information.

Example / answer: The journalist asks why ministers ignored the warning.

Wonder

Rule / explanation: Expresses uncertainty or curiosity.

Example / answer: Observers wonder whether the coalition can survive.

Consider

Rule / explanation: Introduces an issue for reflection.

Example / answer: The article considers how technology may reshape employment.

Determine

Rule / explanation: Introduces an evidence-based enquiry.

Example / answer: The committee must determine whether the law was broken.

Explain

Rule / explanation: Introduces mechanism or cause.

Example / answer: The report explains how the error occurred.

J. Embedded questions with infinitives

What to do

Rule / explanation: For choices or unresolved actions.

Example / answer: The government does not know what to do next.

How to

Rule / explanation: For method or procedure.

Example / answer: The report explains how to reduce delays.

Where / when to

Rule / explanation: For practical decisions.

Example / answer: Officials must decide where to allocate the funding.

Who to

Rule / explanation: For selecting a person.

Example / answer: The committee was uncertain who to question first.

Whether to

Rule / explanation: Use whether, never if, before an infinitive.

Example / answer: Parliament must decide whether to reopen the inquiry.

K. Negative embedded questions

Normal not placement

Rule / explanation: Use statement-order negation.

Example / answer: The inquiry asks why the government did not act sooner.

Whether not to

Rule / explanation: Use for a negative option.

Example / answer: The committee debated whether not to publish the names.

Why not

Rule / explanation: Keep why not together when non-action is questioned.

Example / answer: The article asks why not all regions benefited equally.

Scope

Rule / explanation: Place not carefully so the meaning is clear.

Example / answer: We need to know whether the policy is not merely costly but ineffective.

Critical force

Rule / explanation: Negative embedded questions may imply criticism.

Example / answer: The journalist asks why ministers have not acknowledged the problem.

L. Do you think / believe / suppose

Wh-word first

Rule / explanation: Place the wh-word at the beginning of the complete question.

Example / answer: Why do you think the reform failed?

Statement order after frame

Rule / explanation: Do not invert again after do you think.

Example / answer: Wrong: Why do you think did it fail?

Who as subject

Rule / explanation: No extra inversion follows.

Example / answer: Who do you think authorised the decision?

What as object

Rule / explanation: Use the full frame.

Example / answer: What do you believe the government should do?

Tentative speculation

Rule / explanation: Suppose and imagine soften speculation.

Example / answer: How do you suppose public opinion will react?

M. Article analysis

Central issue

Rule / explanation: Identify the text’s main enquiry.

Example / answer: The article asks whether the government can restore confidence.

Method

Rule / explanation: Use how to analyse construction of argument.

Example / answer: The text examines how statistics shape perception.

Missing explanation

Rule / explanation: Use why to expose a gap.

Example / answer: The author does not explain why rural areas suffered more.

Evidence

Rule / explanation: Use what to question support.

Example / answer: It is unclear what evidence justifies the conclusion.

Broader debate

Rule / explanation: Connect the article to wider issues.

Example / answer: The text raises the question of whether democracy can adapt to digital politics.

N. CPGE problematics

The question is whether...

Rule / explanation: Frames a genuine dilemma.

Example / answer: The question is whether regulation can protect consumers without restricting innovation.

The issue is how...

Rule / explanation: Focuses on method or reconciliation.

Example / answer: The issue is how governments can reduce inequality without weakening growth.

What is at stake is...

Rule / explanation: Identifies the deeper significance.

Example / answer: What is at stake is whether public trust can be rebuilt.

It remains to be seen whether...

Rule / explanation: For an unresolved future outcome.

Example / answer: It remains to be seen whether the reform will produce lasting change.

We may therefore ask...

Rule / explanation: A formal transition into a problématique.

Example / answer: We may therefore ask how far progress can remain compatible with accountability.

O. Oral interviews and khôlles

Clarification

Rule / explanation: Ask for a definition without sounding abrupt.

Example / answer: Could you clarify what you mean by social cohesion?

Follow-up

Rule / explanation: Refer to the previous answer.

Example / answer: I would like to know why you consider this reform ineffective.

Polite challenge

Rule / explanation: Question an assumption carefully.

Example / answer: I wonder whether the author underestimates regional differences.

Evidence request

Rule / explanation: Ask for support.

Example / answer: Could you explain what evidence supports that interpretation?

Nuance

Rule / explanation: Use whether or to what extent.

Example / answer: May I ask to what extent this trend is mainly economic?

P. Punctuation and style

Statement punctuation

Rule / explanation: No question mark when the entire sentence is a statement.

Example / answer: The article asks why the reform failed.

Question punctuation

Rule / explanation: Use a question mark when the whole sentence is a direct question.

Example / answer: Could you explain why the reform failed?

Capitalisation

Rule / explanation: Do not capitalise the embedded clause like a new sentence.

Example / answer: The issue is whether the policy can succeed.

Direct-question colon

Rule / explanation: A full statement may introduce a direct question.

Example / answer: One issue remains: can the policy succeed?

Quotation contrast

Rule / explanation: Distinguish reported and direct questions.

Example / answer: She asked why it failed. / She asked, “Why did it fail?”

Q. Common mistakes

Keeping inversion

Rule / explanation: Use statement order.

Example / answer: Wrong: Do you know where is the minister?

Keeping do-support

Rule / explanation: Remove did/does/do inside the embedded clause.

Example / answer: Wrong: I wonder why did the policy fail.

If + infinitive

Rule / explanation: Use whether before to + infinitive.

Example / answer: Wrong: They discussed if to intervene.

Wrong question mark

Rule / explanation: A statement with an embedded question has no question mark.

Example / answer: The report examines why costs increased.

Tense sequence

Rule / explanation: Use the tense required by meaning and reporting context.

Example / answer: She asked when the report would be published.

What vs which

Rule / explanation: Which = limited choice; what = open choice.

Example / answer: Which of these policies? / What policy?

R. Exercises

1. Correct the order

Rule / explanation: Could you tell me where is the meeting?

Example / answer: Could you tell me where the meeting is?

2. Remove do-support

Rule / explanation: I wonder why did the reform fail.

Example / answer: I wonder why the reform failed.

3. Embed a yes/no question

Rule / explanation: Will Parliament approve the bill?

Example / answer: It remains unclear whether Parliament will approve the bill.

4. Use whether to

Rule / explanation: The government must decide: should it intervene?

Example / answer: The government must decide whether to intervene.

5. Make it polite

Rule / explanation: Why was the report delayed?

Example / answer: Could you explain why the report was delayed?

6. Correct punctuation

Rule / explanation: The article examines why public trust has declined?

Example / answer: The article examines why public trust has declined.

7. Translate

Rule / explanation: La question est de savoir si la croissance peut rester compatible avec la justice sociale.

Example / answer: The question is whether economic growth can remain compatible with social justice.

8. Mini challenge

Rule / explanation: Use a wh-clause, a whether-clause, an infinitive and a polite frame.

Example / answer: The article examines why confidence has declined and asks whether reform can reverse the trend. The government must decide what to do next. Could the author explain how the measures will be funded?
