# Grammar Lab — installation

This version already includes the complete Grammar Lab.

Upload the **contents** of this folder to the root of the GitHub repository, replacing files when GitHub asks. The important new files are `grammar.html`, `grammar.css`, `grammar.js`, and `grammar-data.js`; all 86 chapter files and the site navigation are already updated.

Progress is stored locally in each visitor’s browser. No database or account is required.
