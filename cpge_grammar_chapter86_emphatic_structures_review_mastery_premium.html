Chapter objective

This premium grammar chapter reviews and consolidates emphatic structures: emphatic auxiliaries, it-clefts, pseudo-clefts, fronting, only, even, indeed, reflexive pronouns, contrastive patterns and strategic CPGE emphasis.

A. Lesson — What is grammatical emphasis?

Emphasis

Rule / explanation: Grammatical emphasis makes one element more prominent than the rest of the sentence.

Example / answer: What matters most is whether the reform is perceived as legitimate.

Neutral sentence

Rule / explanation: A neutral sentence presents information without special structural focus.

Example / answer: The government ignored the regional data.

Emphatic sentence

Rule / explanation: An emphatic structure reorganises or reinforces the sentence to highlight contrast, correction or importance.

Example / answer: It was the regional data that the government ignored.

CPGE relevance

Rule / explanation: Controlled emphasis strengthens essays, speeches, khôlles, source analysis and conclusions.

Example / answer: The structure directs the examiner towards the precise distinction on which the argument depends.

B. Choosing what to emphasise

Agent

Rule / explanation: Emphasise who performs the action.

Example / answer: It was local authorities that first identified the problem.

Object

Rule / explanation: Emphasise what receives the action.

Example / answer: It was the funding model that Parliament revised.

Time

Rule / explanation: Emphasise when something happened.

Example / answer: It was only after the election that the figures were released.

Place

Rule / explanation: Emphasise where something occurred.

Example / answer: It was in rural regions that waiting times rose most sharply.

Reason or condition

Rule / explanation: Emphasise why or under what condition something is true.

Example / answer: What makes the proposal convincing is its clear accountability mechanism.

C. Do, does and did for verbal emphasis

Present emphasis

Rule / explanation: Use do or does before the base verb in an affirmative clause.

Example / answer: The evidence does support a cautious conclusion.

Past emphasis

Rule / explanation: Use did before the base verb.

Example / answer: The minister did acknowledge the funding problem.

Contrastive correction

Rule / explanation: Use emphatic do to contradict a previous assumption.

Example / answer: The reform did reduce costs, even though it failed to improve access.

Polite insistence

Rule / explanation: Do may strengthen invitations, requests or imperatives.

Example / answer: Do consider the regional consequences.

Main verb form

Rule / explanation: After do, does or did, use the base verb.

Example / answer: Correct: The report does reveal, not does reveals.

D. Limits of emphatic do

Already auxiliary

Rule / explanation: Do not add do when the clause already contains be, have or a modal auxiliary.

Example / answer: Correct: The policy is effective, not does be effective.

Perfect form

Rule / explanation: Emphasise the existing auxiliary in speech or use another structure in writing.

Example / answer: The government has indeed revised the timetable.

Modal form

Rule / explanation: Stress or reinforce the modal rather than adding do.

Example / answer: The measure can genuinely improve access.

Register

Rule / explanation: Emphatic do is natural in speech and selective formal prose, but repetitive use sounds defensive.

Example / answer: Use it when a real contrast is present.

Not decorative

Rule / explanation: Do not insert it merely to make a sentence sound advanced.

Example / answer: The emphasis must respond to an implied or explicit alternative.

E. It-clefts — core pattern

Pattern

Rule / explanation: Use it + be + focused element + that/who clause.

Example / answer: It was the lack of funding that caused the delay.

People

Rule / explanation: Who is possible when the focused element is a person.

Example / answer: It was local residents who exposed the inconsistency.

Things and ideas

Rule / explanation: That is the safest general relative marker.

Example / answer: It is transparency that the system lacks.

Tense

Rule / explanation: The tense of be reflects the time or perspective of the emphasis.

Example / answer: It was in 2024 that Parliament revised the law.

Agreement

Rule / explanation: The verb inside the relative clause normally agrees with its own subject.

Example / answer: It was the regional authorities that were responsible.

F. It-clefts focusing the subject

Person as subject

Rule / explanation: Highlight the person or group performing the action.

Example / answer: It was the committee that rejected the proposal.

Institution as subject

Rule / explanation: Foreground institutional responsibility.

Example / answer: It was Parliament that introduced the safeguard.

Correction

Rule / explanation: Use a cleft to replace an incorrect assumption.

Example / answer: It was local journalists, not national newspapers, who first reported the story.

Contrast

Rule / explanation: The focused subject may be contrasted explicitly.

Example / answer: It was citizens rather than experts who demanded immediate action.

Analytical use

Rule / explanation: Subject clefts are useful when responsibility is central.

Example / answer: It is policymakers who must justify the unequal outcome.

G. It-clefts focusing objects and complements

Direct object

Rule / explanation: Highlight what was affected, changed or selected.

Example / answer: It was the funding formula that the government revised.

Prepositional complement

Rule / explanation: Focus an element introduced by a preposition.

Example / answer: It was on regional inequality that the report concentrated.

Place

Rule / explanation: Foreground location.

Example / answer: It was in northern regions that opposition was strongest.

Time

Rule / explanation: Foreground a date or stage.

Example / answer: It was only after publication that public concern intensified.

Reason

Rule / explanation: A because-clause may be focused, though this is less compact.

Example / answer: It was because the evidence was incomplete that the vote was postponed.

H. Pseudo-clefts with what

Pattern

Rule / explanation: Use what + clause + be + focused information.

Example / answer: What the policy lacks is a credible funding mechanism.

Subject focus

Rule / explanation: The what-clause functions as the subject of the sentence.

Example / answer: What concerns voters most is the absence of accountability.

Action focus

Rule / explanation: Use a bare infinitive or to-infinitive after be in some action-focused structures.

Example / answer: What the government must do is restore public trust.

Plural complement

Rule / explanation: The complement may be plural even though what is grammatically singular.

Example / answer: What the region needs is more doctors and better transport.

CPGE use

Rule / explanation: Pseudo-clefts are excellent for stating a key criterion or conclusion.

Example / answer: What ultimately determines legitimacy is whether citizens can challenge decisions.

I. Reversed pseudo-clefts

Pattern

Rule / explanation: Place the focused element before be and the what-clause after it.

Example / answer: A credible funding mechanism is what the policy lacks.

Contrastive force

Rule / explanation: The initial focus often corrects or narrows the discussion.

Example / answer: Fairness, not uniformity, is what the reform must guarantee.

Natural emphasis

Rule / explanation: The structure is useful in speech because it delays the explanatory clause.

Example / answer: Accountability is what voters are demanding.

Register

Rule / explanation: Reversed pseudo-clefts are formal enough for essays but also natural orally.

Example / answer: Implementation is what ultimately determines success.

Punctuation

Rule / explanation: A comma may separate a contrastive focused phrase.

Example / answer: Local flexibility, rather than complete autonomy, is what the system requires.

J. All-clefts

Pattern

Rule / explanation: Use all + clause + be + focused element.

Example / answer: All the committee requested was more time.

Meaning

Rule / explanation: The structure emphasises limitation: nothing more than the focused element.

Example / answer: All citizens want is a system they can trust.

Verb agreement

Rule / explanation: All may be treated according to the complement and natural usage.

Example / answer: All that remains is to publish the report.

Contrast

Rule / explanation: The structure can correct exaggeration.

Example / answer: All the proposal does is postpone the difficult decision.

Critical use

Rule / explanation: All-clefts often reveal how a source minimises or simplifies an issue.

Example / answer: All we are asking for is transparency presents the demand as modest.

K. The thing, the reason and the person clefts

The thing is...

Rule / explanation: Use the thing + relative clause + be to focus an issue in speech.

Example / answer: The thing that worries me is the lack of independent oversight.

The reason is...

Rule / explanation: Focus a cause or explanation.

Example / answer: The reason the policy failed is that local capacity was ignored.

The person is...

Rule / explanation: Focus an individual.

Example / answer: The person who first raised the concern was a regional auditor.

The place is...

Rule / explanation: Focus location.

Example / answer: The place where the impact is most visible is the north-east.

Register

Rule / explanation: These structures are natural and clear, though the thing is can sound conversational.

Example / answer: In formal essays, the central reason is may be preferable.

L. Fronting for emphasis

Object fronting

Rule / explanation: Move the object to the beginning to create contrast.

Example / answer: This explanation, the author never seriously considers.

Complement fronting

Rule / explanation: Front a complement to highlight evaluation.

Example / answer: Particularly troubling is the absence of regional data.

Adverbial fronting

Rule / explanation: Place a time, place or condition first.

Example / answer: In rural areas, the consequences were far more severe.

Literary or formal effect

Rule / explanation: Unusual fronting can sound elevated or dramatic.

Example / answer: Gone are the assumptions that once shaped the debate.

Control

Rule / explanation: Front only when the unusual order clearly serves focus or cohesion.

Example / answer: Excessive fronting makes prose artificial.

M. Fronted complements with be

Adjective first

Rule / explanation: Front an adjective complement, then use be + subject.

Example / answer: Most striking is the difference between urban and rural outcomes.

Participle first

Rule / explanation: Front a participial complement for formal description.

Example / answer: Hidden behind the national average are substantial regional inequalities.

Prepositional phrase first

Rule / explanation: Front a place or conceptual frame.

Example / answer: At the centre of the dispute is a question of responsibility.

Agreement

Rule / explanation: Make the verb agree with the postposed subject.

Example / answer: Particularly concerning are the gaps in the available data.

Essay use

Rule / explanation: This structure can introduce a paragraph’s key observation.

Example / answer: Equally important is the question of long-term legitimacy.

N. Emphasis with only

Only + focused element

Rule / explanation: Only restricts the meaning to one person, thing, time or condition.

Example / answer: Only local authorities had access to the complete data.

Only after

Rule / explanation: Fronted only after triggers inversion.

Example / answer: Only after publication did ministers respond.

Only by

Rule / explanation: Express the sole means of achieving a result.

Example / answer: Only by restoring transparency can the institution rebuild trust.

Only if

Rule / explanation: State a necessary condition.

Example / answer: The policy can succeed only if regional resources increase.

Scope

Rule / explanation: Place only immediately before the element it restricts.

Example / answer: Only the minister apologised differs from the minister only apologised.

O. Emphasis with even

Unexpected inclusion

Rule / explanation: Even highlights an element as surprising or extreme.

Example / answer: Even the policy’s supporters questioned the timetable.

Even + comparative

Rule / explanation: Strengthen a comparative.

Example / answer: The second proposal is even more expensive.

Even if

Rule / explanation: Express a concession in a hypothetical condition.

Example / answer: Even if costs fall, legitimacy may not improve.

Even though

Rule / explanation: Express strong factual concession.

Example / answer: Even though the data were incomplete, the report influenced Parliament.

Scope

Rule / explanation: Position even before the element that is surprising.

Example / answer: Even experts disagreed / Experts even disagreed on the basic definition.

P. Emphasis with indeed

Confirmation

Rule / explanation: Indeed confirms or reinforces a preceding statement.

Example / answer: The policy was costly; indeed, it exceeded every official forecast.

Strong agreement

Rule / explanation: In speech, indeed can confirm a statement formally.

Example / answer: This is indeed a serious concern.

Exemplification

Rule / explanation: Indeed may introduce stronger evidence for the previous claim.

Example / answer: Access deteriorated; indeed, three clinics closed within a year.

Position

Rule / explanation: It can appear initially, medially or after be.

Example / answer: The report is indeed methodologically limited.

Caution

Rule / explanation: Do not use indeed as a decorative synonym for therefore.

Example / answer: It reinforces; it does not automatically express consequence.

Q. Emphasis with very and own

The very

Rule / explanation: The very + noun means the exact or particular noun in question.

Example / answer: The very policy designed to restore trust ultimately weakened it.

Very as intensifier

Rule / explanation: Very strengthens gradable adjectives and adverbs.

Example / answer: The evidence is very limited.

Own

Rule / explanation: Own emphasises possession or personal responsibility.

Example / answer: The minister’s own advisers questioned the proposal.

Of one’s own

Rule / explanation: Emphasise independent possession or creation.

Example / answer: The region developed a funding model of its own.

Contrastive use

Rule / explanation: Very and own often expose irony or contradiction.

Example / answer: The institution violated its own transparency rules.

R. Reflexive pronouns for emphasis

Emphatic reflexive

Rule / explanation: Use myself, yourself, himself, herself, itself, ourselves or themselves to emphasise identity.

Example / answer: The minister herself authorised the release.

Position after noun

Rule / explanation: Place the reflexive immediately after the noun for strong identification.

Example / answer: The committee itself raised the objection.

End position

Rule / explanation: The reflexive may appear later for slightly less concentrated emphasis.

Example / answer: The minister authorised the release herself.

Not object reflexive

Rule / explanation: Distinguish emphasis from reflexive meaning.

Example / answer: She criticised herself means she was both agent and object.

Agreement

Rule / explanation: Match the reflexive form to the noun or pronoun.

Example / answer: The institutions themselves acknowledged the failure.

S. Emphatic comparison and contrast

Far from

Rule / explanation: Use far from + noun or gerund to reject an interpretation strongly.

Example / answer: Far from restoring trust, the reform intensified suspicion.

Nothing but

Rule / explanation: Present something as exclusively or merely one thing.

Example / answer: The proposal offers nothing but temporary relief.

Anything but

Rule / explanation: Mean definitely not.

Example / answer: The process was anything but transparent.

Not so much X as Y

Rule / explanation: Replace one interpretation with another more accurate one.

Example / answer: The crisis is not so much economic as institutional.

Rather than

Rule / explanation: Present the preferred or more accurate alternative.

Example / answer: The policy postponed the problem rather than solving it.

T. Emphasis through repetition and parallelism

Lexical repetition

Rule / explanation: Repeat a key word deliberately to establish thematic focus.

Example / answer: Trust is required for reform, trust for compliance and trust for legitimacy.

Anaphora

Rule / explanation: Repeat the opening of successive clauses.

Example / answer: We need transparency. We need accountability. We need action.

Parallel structure

Rule / explanation: Use matching grammatical forms for balanced emphasis.

Example / answer: The policy is costly, divisive and ineffective.

Rule of three

Rule / explanation: A three-part sequence often creates completeness and rhythm.

Example / answer: Fair, feasible and transparent.

CPGE caution

Rule / explanation: Strategic repetition creates force; accidental repetition weakens style.

Example / answer: Each repeated element should add a new dimension.

U. Emphasis in essays and khôlles

Thesis

Rule / explanation: Use a pseudo-cleft to foreground the decisive criterion.

Example / answer: What matters is not whether reform is popular, but whether it is legitimate.

Transition

Rule / explanation: Use fronting to introduce the next dimension.

Example / answer: More difficult to assess is the policy’s long-term social impact.

Counterargument

Rule / explanation: Use emphatic do or concession to acknowledge a valid point.

Example / answer: The reform does reduce costs; however, this gain remains unevenly distributed.

Conclusion

Rule / explanation: Use an only structure to state the necessary condition.

Example / answer: Only through transparent implementation can trust be restored.

Oral delivery

Rule / explanation: Let stress and intonation support the grammar instead of multiplying devices.

Example / answer: One well-placed cleft is stronger than several artificial structures.

V. Common mistakes, exercises and mastery

Do + inflected verb

Rule / explanation: After do, does or did, use the base form.

Example / answer: Correct: The article does reveal, not does reveals.

Wrong cleft agreement

Rule / explanation: Check agreement inside the relative clause.

Example / answer: It was the ministers who were responsible.

Overusing that

Rule / explanation: Who is natural for people, although that is possible in many contexts.

Example / answer: It was the journalist who exposed the inconsistency.

Unclear focus

Rule / explanation: A cleft is ineffective when the focused element is not genuinely contrastive.

Example / answer: Use the neutral sentence when no special focus is needed.

Only misplacement

Rule / explanation: Put only directly before the element it restricts.

Example / answer: She only criticised the policy differs from only she criticised the policy.

Exercise 1

Rule / explanation: Add emphatic do: The evidence supports the conclusion.

Example / answer: The evidence does support the conclusion.

Exercise 2

Rule / explanation: Create an it-cleft focusing the time: Parliament acted after the report was published.

Example / answer: It was after the report was published that Parliament acted.

Exercise 3

Rule / explanation: Create a pseudo-cleft: The policy lacks accountability.

Example / answer: What the policy lacks is accountability.

Exercise 4

Rule / explanation: Reverse the pseudo-cleft: What voters demand is transparency.

Example / answer: Transparency is what voters demand.

Exercise 5

Rule / explanation: Use far from: The reform did not restore trust; it weakened it.

Example / answer: Far from restoring trust, the reform weakened it.

Exercise 6

Rule / explanation: Correct: It was the regional authorities who was responsible.

Example / answer: It was the regional authorities who were responsible.

Exercise 7

Rule / explanation: Translate: Ce qui compte, ce n’est pas seulement le coût de la réforme, mais la manière dont ses effets sont répartis.

Example / answer: What matters is not merely the cost of the reform, but the way in which its effects are distributed.

Exercise 8

Rule / explanation: Translate: C’est précisément le manque de transparence qui a affaibli la confiance du public.

Example / answer: It was precisely the lack of transparency that weakened public trust.

Mini concours challenge

Rule / explanation: Write a paragraph using emphatic do, an it-cleft, a pseudo-cleft, fronting and only + inversion.

Example / answer: Model: The reform does provide a clearer national framework. It is its unequal implementation, however, that creates the central difficulty. What the policy fails to recognise is that equal rules do not automatically produce equal outcomes. Particularly troubling is the absence of targeted regional support. Only by adapting resources to local needs can the government reconcile national consistency with genuine fairness.
