The golden rule

Prepositions are pattern-based
depend on / responsible for / interested in
a rise in prices / a rise of 10% / prices rose by 10%
Main usePrepositions show relationships between time, place, movement, cause, method, quantity and ideas.
CPGE habitMemorise prepositions in chunks, not alone: focus on, deal with, access to, responsible for.
Key trapDo not translate French prepositions directly: depend on, not depend of; discuss inequality, not discuss about inequality.

A. Lesson — What are prepositions?

Prepositions

French: prépositions

Rule / explanation: Prepositions are short words that show relationships between people, things, ideas, time, place, cause or method.

Example / answer: The article focuses on inequality in modern Britain.

Preposition + noun

French: préposition + nom

Rule / explanation: A preposition is usually followed by a noun or noun phrase.

Example / answer: The debate about democracy remains central.

Preposition + gerund

French: préposition + gérondif

Rule / explanation: After a preposition, use a gerund, not an infinitive.

Example / answer: Before answering, students should define the key terms.

Preposition + pronoun

French: préposition + pronom complément

Rule / explanation: After a preposition, use an object pronoun.

Example / answer: The reform affects them, not they.

Fixed patterns

French: constructions fixes

Rule / explanation: Many verbs, adjectives and nouns require specific prepositions.

Example / answer: The author insists on the need for reform.

Meaning and precision

French: sens et précision

Rule / explanation: Changing the preposition can change the meaning.

Example / answer: A rise in prices / a rise of 10%.

CPGE use

French: usage CPGE

Rule / explanation: Prepositions are essential for precise essays, article summaries and data commentary.

Example / answer: The article deals with the consequences of political distrust.

High-frequency issue

French: problème fréquent

Rule / explanation: Prepositions are a major source of errors for French speakers because they rarely match French exactly.

Example / answer: depend on, responsible for, interested in, different from.

B. Prepositions of time

At

French: à

Rule / explanation: Use at for precise times and some fixed expressions.

Example / answer: The debate starts at 9 a.m.

On

French: le

Rule / explanation: Use on for days and dates.

Example / answer: The article was published on Monday.

In

French: en / dans

Rule / explanation: Use in for months, years, centuries and longer periods.

Example / answer: The crisis began in 2008.

During

French: pendant

Rule / explanation: Use during + noun to mean within a period.

Example / answer: Unemployment rose during the recession.

For

French: pendant / depuis

Rule / explanation: Use for + duration.

Example / answer: The debate has lasted for years.

Since

French: depuis

Rule / explanation: Use since + starting point.

Example / answer: Trust has declined since the scandal.

By

French: d’ici / avant

Rule / explanation: Use by to mean not later than a deadline.

Example / answer: The report must be submitted by Friday.

Until

French: jusqu’à

Rule / explanation: Use until to mean up to a point in time.

Example / answer: The negotiations continued until midnight.

C. Prepositions of place and movement

In

French: dans / en

Rule / explanation: Use in for inside an area, country, city, institution or enclosed space.

Example / answer: The article was published in the UK.

At

French: à

Rule / explanation: Use at for a point, event, institution or specific location.

Example / answer: The speech was delivered at the conference.

On

French: sur

Rule / explanation: Use on for surfaces, platforms, streets and media.

Example / answer: The figures were published on the website.

To

French: vers / à

Rule / explanation: Use to for movement towards a place, person or goal.

Example / answer: The minister travelled to Brussels.

Into

French: dans / à l’intérieur

Rule / explanation: Use into for movement from outside to inside.

Example / answer: The scandal pushed the party into crisis.

From

French: de / depuis

Rule / explanation: Use from for origin or starting point.

Example / answer: The data come from official sources.

Across

French: à travers

Rule / explanation: Use across for movement or distribution from one side to another.

Example / answer: The protests spread across the country.

Throughout

French: dans tout / tout au long de

Rule / explanation: Use throughout for something present in all parts of a place or period.

Example / answer: The issue appears throughout the article.

D. Prepositions in data and trends

Increase in

French: augmentation de

Rule / explanation: Use in for the thing that changes.

Example / answer: There has been an increase in unemployment.

Increase of

French: augmentation de

Rule / explanation: Use of for the amount of change.

Example / answer: There was an increase of 10%.

Rise in

French: hausse de

Rule / explanation: Use in for the affected variable.

Example / answer: The graph shows a rise in rents.

Fall in

French: baisse de

Rule / explanation: Use in for a decreasing variable.

Example / answer: There has been a fall in turnout.

By + amount

French: de / à hauteur de

Rule / explanation: Use by to show the size of the change.

Example / answer: Prices rose by 8%.

To + final level

French: à / jusqu’à

Rule / explanation: Use to for the final figure reached.

Example / answer: Inflation rose to 7%.

From... to...

French: de... à...

Rule / explanation: Use from and to for starting and ending points.

Example / answer: Unemployment fell from 9% to 6%.

Between... and...

French: entre... et...

Rule / explanation: Use for a range or period.

Example / answer: Between 2010 and 2020, inequality increased.

E. Verb + preposition patterns

depend on

French: dépendre de

Rule / explanation: Not depend of.

Example / answer: Democracy depends on public trust.

rely on

French: s’appuyer sur

Rule / explanation: To need or use something as support.

Example / answer: The article relies on recent data.

focus on

French: se concentrer sur

Rule / explanation: To concentrate on something.

Example / answer: The journalist focuses on housing.

deal with

French: traiter de / gérer

Rule / explanation: To discuss, handle or address.

Example / answer: The article deals with social inequality.

refer to

French: faire référence à

Rule / explanation: To mention or point to something.

Example / answer: The author refers to the 2008 crisis.

lead to

French: conduire à

Rule / explanation: To cause a result.

Example / answer: Rising rents may lead to social tensions.

result in

French: entraîner

Rule / explanation: To cause a consequence.

Example / answer: The policy resulted in higher costs.

contribute to

French: contribuer à

Rule / explanation: To help cause something.

Example / answer: Low wages contribute to poverty.

F. Adjective + preposition patterns

responsible for

French: responsable de

Rule / explanation: Used to express responsibility.

Example / answer: The government is responsible for public services.

interested in

French: intéressé par

Rule / explanation: Used to express interest.

Example / answer: The author is interested in political distrust.

aware of

French: conscient de

Rule / explanation: Used to express knowledge or consciousness.

Example / answer: Citizens are aware of the risks.

concerned about

French: préoccupé par

Rule / explanation: Used to express worry.

Example / answer: Voters are concerned about inflation.

different from

French: différent de

Rule / explanation: Standard comparison.

Example / answer: The British system is different from the American system.

similar to

French: semblable à

Rule / explanation: Used for resemblance.

Example / answer: This crisis is similar to previous ones.

related to

French: lié à

Rule / explanation: Used for connection.

Example / answer: The issue is related to inequality.

based on

French: fondé sur

Rule / explanation: Used for foundations or evidence.

Example / answer: The argument is based on recent figures.

G. Noun + preposition patterns

an increase in

French: augmentation de

Rule / explanation: Used for trends.

Example / answer: There has been an increase in poverty.

a decrease in

French: baisse de

Rule / explanation: Used for trends.

Example / answer: The report shows a decrease in trust.

a solution to

French: solution à

Rule / explanation: Not solution of.

Example / answer: Education is not the only solution to inequality.

a reason for

French: raison de

Rule / explanation: Used for cause or justification.

Example / answer: There are several reasons for this decline.

an answer to

French: réponse à

Rule / explanation: Used for response or solution.

Example / answer: The policy is not a convincing answer to the crisis.

a debate about / on

French: débat sur

Rule / explanation: Used for topic.

Example / answer: There is a debate about freedom of speech.

a threat to

French: menace pour

Rule / explanation: Used for danger.

Example / answer: Disinformation is a threat to democracy.

access to

French: accès à

Rule / explanation: Used for rights, services and resources.

Example / answer: Access to education remains unequal.

H. Typical mistakes by French speakers

Depend of

French: depend on

Rule / explanation: Use depend on, not depend of.

Example / answer: Wrong: Democracy depends of trust. Correct: Democracy depends on trust.

Discuss about

French: discuss

Rule / explanation: Discuss takes a direct object; talk about takes about.

Example / answer: Wrong: The article discusses about inequality. Correct: The article discusses inequality.

Responsible of

French: responsible for

Rule / explanation: Use responsible for.

Example / answer: Wrong: The state is responsible of schools. Correct: responsible for schools.

Interested by

French: interested in

Rule / explanation: Use interested in.

Example / answer: Wrong: The author is interested by poverty. Correct: interested in poverty.

Married with

French: married to

Rule / explanation: Use married to.

Example / answer: Wrong: She is married with him. Correct: married to him.

Explain me

French: explain to

Rule / explanation: Use explain something to someone.

Example / answer: Wrong: Explain me the issue. Correct: Explain the issue to me.

Preposition + infinitive

French: préposition + infinitif

Rule / explanation: After a preposition, use -ing.

Example / answer: Wrong: before to conclude. Correct: before concluding.

Confusing rise of and rise in

French: rise in / of

Rule / explanation: Use rise in for the variable, rise of for the amount.

Example / answer: A rise in prices / a rise of 10%.
