Chapter objective

This premium grammar chapter helps CPGE students master gerunds and infinitives. It covers verbs followed by -ing, verbs followed by to + verb, object + infinitive structures, adjective + infinitive phrases, preposition + gerund, meaning changes and common French-speaker mistakes.

The golden rule

Verb pattern matters
suggest + -ing: The article suggests reforming the system.
want + to + verb: The government wants to reduce inequality.
Main useGerunds and infinitives allow students to express aims, methods, recommendations, risks, decisions and abstract ideas.
CPGE habitMemorise high-value patterns: avoid -ing, suggest -ing, aim to, fail to, enable someone to, by -ing.
Key trapAfter a preposition, use a gerund: before concluding, not before to conclude.

A. Lesson — What are gerunds and infinitives?

Gerund

French: gérondif

Rule / explanation: A gerund is the -ing form of a verb used like a noun.

Example / answer: Regulating social media is difficult.

Infinitive

French: infinitif

Rule / explanation: An infinitive is to + base verb.

Example / answer: The government wants to reduce inequality.

Bare infinitive

French: infinitif sans to

Rule / explanation: A bare infinitive is the base verb without to.

Example / answer: The reform may improve public services.

Verb pattern

French: construction verbale

Rule / explanation: Some verbs are followed by a gerund, others by an infinitive.

Example / answer: The author suggests reforming the system. / The minister hopes to pass the bill.

Meaning change

French: changement de sens

Rule / explanation: Some verbs can take both forms, but the meaning changes.

Example / answer: They stopped discussing the issue. / They stopped to discuss the issue.

Subject gerund

French: gérondif sujet

Rule / explanation: A gerund can be the subject of a sentence.

Example / answer: Voting is a fundamental democratic right.

Purpose infinitive

French: infinitif de but

Rule / explanation: Use the infinitive to express purpose.

Example / answer: The government introduced the policy to reduce poverty.

CPGE use

French: usage CPGE

Rule / explanation: Gerunds and infinitives are essential for essay style, policy discussion and article summaries.

Example / answer: The article aims to explain why tackling inequality matters.

B. Verbs followed by gerunds

avoid + -ing

French: éviter de

Rule / explanation: Use avoid with a gerund.

Example / answer: Politicians should avoid making unrealistic promises.

consider + -ing

French: envisager de

Rule / explanation: Use consider with a gerund.

Example / answer: The government is considering raising taxes.

suggest + -ing

French: suggérer de

Rule / explanation: Suggest is followed by a gerund, not to + verb.

Example / answer: The article suggests reforming the welfare system.

deny + -ing

French: nier avoir / nier de

Rule / explanation: Use deny with a gerund.

Example / answer: The minister denied misleading Parliament.

admit + -ing

French: admettre avoir / admettre de

Rule / explanation: Use admit with a gerund.

Example / answer: The company admitted breaking environmental rules.

risk + -ing

French: risquer de

Rule / explanation: Use risk with a gerund.

Example / answer: The reform risks increasing inequality.

keep + -ing

French: continuer à

Rule / explanation: Use keep with a gerund for continuation.

Example / answer: The media keep focusing on scandals.

give up + -ing

French: renoncer à

Rule / explanation: Use give up with a gerund.

Example / answer: Some voters have given up trusting mainstream parties.

C. Verbs followed by infinitives

want to

French: vouloir

Rule / explanation: Use want + to + verb.

Example / answer: The government wants to restore public trust.

hope to

French: espérer

Rule / explanation: Use hope + to + verb.

Example / answer: The minister hopes to pass the bill.

decide to

French: décider de

Rule / explanation: Use decide + to + verb.

Example / answer: Parliament decided to investigate the scandal.

agree to

French: accepter de

Rule / explanation: Use agree + to + verb.

Example / answer: The parties agreed to cooperate.

refuse to

French: refuser de

Rule / explanation: Use refuse + to + verb.

Example / answer: The opposition refused to support the measure.

promise to

French: promettre de

Rule / explanation: Use promise + to + verb.

Example / answer: The candidate promised to lower taxes.

fail to

French: échouer à / ne pas

Rule / explanation: Use fail + to + verb.

Example / answer: The policy failed to reduce poverty.

tend to

French: avoir tendance à

Rule / explanation: Use tend + to + verb for general tendencies.

Example / answer: Inequality tends to undermine social cohesion.

D. Verbs followed by object + infinitive

ask someone to

French: demander à quelqu’un de

Rule / explanation: Use ask + object + to + verb.

Example / answer: The journalist asked the minister to clarify his position.

tell someone to

French: dire à quelqu’un de

Rule / explanation: Use tell + object + to + verb.

Example / answer: The teacher told students to justify their answers.

encourage someone to

French: encourager quelqu’un à

Rule / explanation: Use encourage + object + to + verb.

Example / answer: The article encourages readers to question simple narratives.

urge someone to

French: exhorter quelqu’un à

Rule / explanation: Use urge + object + to + verb.

Example / answer: Activists urged the government to act.

allow someone to

French: permettre à quelqu’un de

Rule / explanation: Use allow + object + to + verb.

Example / answer: The law allows citizens to challenge decisions.

enable someone to

French: permettre à quelqu’un de

Rule / explanation: Use enable + object + to + verb.

Example / answer: Education enables people to improve their prospects.

force someone to

French: forcer quelqu’un à

Rule / explanation: Use force + object + to + verb.

Example / answer: The crisis forced the government to change strategy.

persuade someone to

French: convaincre quelqu’un de

Rule / explanation: Use persuade + object + to + verb.

Example / answer: The campaign persuaded voters to support the reform.

E. Adjective + infinitive structures

It is difficult to

French: il est difficile de

Rule / explanation: Use adjective + infinitive after it is.

Example / answer: It is difficult to regulate online speech.

It is important to

French: il est important de

Rule / explanation: Useful for essays and recommendations.

Example / answer: It is important to protect fundamental rights.

It is necessary to

French: il est nécessaire de

Rule / explanation: Formal way to express necessity.

Example / answer: It is necessary to distinguish criticism from hate speech.

It is possible to

French: il est possible de

Rule / explanation: Use for possibility.

Example / answer: It is possible to reduce inequality through education.

It is likely to

French: il est probable que

Rule / explanation: Use likely + infinitive for probability.

Example / answer: The reform is likely to face opposition.

It is unlikely to

French: il est peu probable que

Rule / explanation: Use unlikely + infinitive for negative probability.

Example / answer: The measure is unlikely to satisfy all voters.

Too + adjective + to

French: trop... pour

Rule / explanation: Means excessively adjective for something to happen.

Example / answer: The proposal is too vague to be convincing.

Adjective enough to

French: assez... pour

Rule / explanation: Means sufficiently adjective for something to happen.

Example / answer: The evidence is strong enough to support the claim.

F. Preposition + gerund

after + -ing

French: après avoir / après

Rule / explanation: After a preposition, use a gerund.

Example / answer: After analysing the data, the author reaches a cautious conclusion.

before + -ing

French: avant de

Rule / explanation: Use before + gerund.

Example / answer: Before drawing conclusions, students should check the source.

without + -ing

French: sans

Rule / explanation: Use without + gerund.

Example / answer: The article criticises the reform without proposing an alternative.

by + -ing

French: en + participe présent

Rule / explanation: Use by + gerund to explain method.

Example / answer: The government hopes to reduce poverty by increasing benefits.

instead of + -ing

French: au lieu de

Rule / explanation: Use instead of + gerund for alternatives.

Example / answer: Instead of blaming voters, the author examines structural causes.

in favour of + -ing

French: en faveur de

Rule / explanation: Use in favour of + gerund.

Example / answer: The article argues in favour of regulating platforms.

responsible for + -ing

French: responsable de

Rule / explanation: Adjective/preposition combinations also take gerunds.

Example / answer: The company was responsible for polluting the river.

interested in + -ing

French: intéressé par

Rule / explanation: Use interested in + gerund.

Example / answer: The author is interested in explaining political distrust.

G. Verbs with gerund or infinitive and meaning change

stop doing

French: arrêter de faire

Rule / explanation: Stop an activity.

Example / answer: Politicians stopped discussing the issue.

stop to do

French: s’arrêter pour faire

Rule / explanation: Stop in order to do something else.

Example / answer: The minister stopped to answer a journalist’s question.

remember doing

French: se souvenir d’avoir fait

Rule / explanation: Remember a past action.

Example / answer: The journalist remembered interviewing the minister.

remember to do

French: penser à faire

Rule / explanation: Remember an obligation or future task.

Example / answer: Remember to quote the source.

try doing

French: essayer en faisant

Rule / explanation: Experiment with a method.

Example / answer: The campaign tried using social media to reach young voters.

try to do

French: essayer de faire

Rule / explanation: Make an effort to do something.

Example / answer: The government tried to reassure the public.

mean doing

French: signifier / impliquer

Rule / explanation: Involve or imply.

Example / answer: Cutting spending may mean reducing services.

mean to do

French: avoir l’intention de

Rule / explanation: Intend to do.

Example / answer: The author did not mean to oversimplify the issue.

H. Typical mistakes by French speakers

Suggest to do

French: suggest

Rule / explanation: Suggest is followed by a gerund or that-clause, not to + verb.

Example / answer: Wrong: The author suggests to reform the system. Correct: The author suggests reforming the system.

Avoid to do

French: avoid

Rule / explanation: Avoid takes a gerund.

Example / answer: Wrong: avoid to generalise. Correct: avoid generalising.

Want that someone does

French: want

Rule / explanation: Use want someone to do.

Example / answer: Wrong: The government wants that citizens trust it. Correct: The government wants citizens to trust it.

Preposition + infinitive

French: préposition + gérondif

Rule / explanation: After a preposition, use a gerund.

Example / answer: Wrong: before to conclude. Correct: before concluding.

Confusing stop doing and stop to do

French: stop

Rule / explanation: These have different meanings.

Example / answer: They stopped protesting = they ended the protest. They stopped to protest = they stopped another activity in order to protest.

Forgetting to after decide

French: decide

Rule / explanation: Decide takes an infinitive.

Example / answer: Wrong: They decided change the law. Correct: They decided to change the law.

Using infinitive as subject too often

French: sujet

Rule / explanation: A gerund often sounds more natural as subject.

Example / answer: Better: Voting matters, not To vote matters.

Translating French de mechanically

French: de français

Rule / explanation: French de may become to, -ing, or nothing depending on the verb pattern.

Example / answer: décider de = decide to; éviter de = avoid -ing; pouvoir = can + base verb.
