# Installation — Class & Colle Transfer Loop

This update adds one shared file, `site-core.js`, and links it from every HTML page.

It provides:
- one consistent menu on all main pages;
- automatic removal of the old About link;
- a visible reminder that English classes and regular colles are indispensable;
- a class-and-colle mission at the end of every page and chapter;
- a mission dashboard on the homepage;
- local progress only: nothing is sent to the teacher;
- an automatic warning when the daily news brief is stale.

## GitHub upload

Upload the complete contents of this folder to the root of the repository and replace files with the same names.
Do not upload the enclosing folder itself.

After committing, wait a few minutes and reload with Ctrl + F5.
