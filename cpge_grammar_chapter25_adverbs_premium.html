Chapter 25 — Adverbs and Adverb Placement
B1 → B2 grammar for nuance, precision, data commentary and academic argumentation.

Chapter objective

This premium grammar chapter helps CPGE students master adverbs and their position in English sentences. It covers frequency, manner, degree, sentence adverbs, data-commentary adverbs, word order and common French-speaker mistakes.

Level B1 → B2AdverbsWord OrderNuanceData CommentaryExercises + Answers

The golden rule

Adverb placement depends on function
Frequency: before main verb / after be
Manner: often after verb or object
Sentence adverb: often at the beginning
Main useAdverbs modify verbs, adjectives, other adverbs or whole sentences. They add nuance, precision and logical force.
CPGE habitUse adverbs to avoid simplistic claims: arguably, somewhat, significantly, gradually, clearly.
Key trapDo not confuse actually and currently: currently = now; actually = in fact.

A. Lesson — What are adverbs?

Adverbs

French: adverbes

Rule / explanation: Adverbs modify verbs, adjectives, other adverbs or whole sentences.

Example / answer: The article clearly explains the causes of the crisis.

Manner

French: manière

Rule / explanation: Adverbs of manner explain how something happens.

Example / answer: The government reacted slowly.

Frequency

French: fréquence

Rule / explanation: Adverbs of frequency explain how often something happens.

Example / answer: Politicians often promise reform.

Degree

French: degré

Rule / explanation: Adverbs of degree intensify or soften meaning.

Example / answer: The argument is highly convincing.

B. Frequency adverbs

Always / usually / often

French: toujours / habituellement / souvent

Rule / explanation: These usually go before the main verb but after be.

Example / answer: Voters often distrust institutions. / The issue is often misunderstood.

Sometimes / rarely / never

French: parfois / rarement / jamais

Rule / explanation: Use never without another negative.

Example / answer: The article never claims that reform is easy.

Hardly ever

French: presque jamais

Rule / explanation: Means almost never.

Example / answer: Voters hardly ever read full party manifestos.

C. Adverbs of manner and data commentary

Clearly / carefully

French: clairement / soigneusement

Rule / explanation: Useful to describe analysis and method.

Example / answer: The author carefully distinguishes freedom from impunity.

Gradually / steadily

French: progressivement / régulièrement

Rule / explanation: Useful for slow trends.

Example / answer: Public trust has gradually declined.

Sharply / dramatically

French: fortement / spectaculairement

Rule / explanation: Useful for strong or sudden changes.

Example / answer: Prices rose sharply in 2022.

Slightly / marginally

French: légèrement / marginalement

Rule / explanation: Useful for small changes.

Example / answer: Unemployment fell slightly.

D. Degree and nuance

Quite / rather / fairly

French: assez / plutôt

Rule / explanation: Useful for moderate judgement and careful criticism.

Example / answer: The article is fairly balanced, but the conclusion is rather simplistic.

Highly / deeply / extremely

French: très / profondément / extrêmement

Rule / explanation: Strong intensifiers; avoid overusing them.

Example / answer: The reform is highly controversial.

Somewhat

French: quelque peu

Rule / explanation: Formal softener meaning to some extent.

Example / answer: The proposal is somewhat unrealistic.

E. Sentence adverbs for CPGE argumentation

Arguably

French: on peut soutenir que

Rule / explanation: Presents a claim as defendable, not absolute.

Example / answer: Arguably, the housing crisis is now a generational issue.

Undoubtedly

French: sans aucun doute

Rule / explanation: Strong academic claim; use when evidence is solid.

Example / answer: Undoubtedly, education remains central to social mobility.

Apparently / presumably

French: apparemment / vraisemblablement

Rule / explanation: Useful for cautious interpretation.

Example / answer: Presumably, the author wanted to challenge conventional views.

Interestingly / importantly

French: fait intéressant / point important

Rule / explanation: Introduce an observation or key point.

Example / answer: Importantly, the article does not blame voters themselves.

F. Typical mistakes by French speakers

Never + another negative

French: double négation

Rule / explanation: Never already has negative meaning.

Example / answer: Wrong: The author does not never mention it. Correct: The author never mentions it.

Actually vs currently

French: actually / currently

Rule / explanation: Actually = in fact. Currently = at the moment.

Example / answer: Currently, the bill is being debated. Actually, the cost is higher than expected.

Eventually vs éventuellement

French: eventually

Rule / explanation: Eventually means in the end, not possibly.

Example / answer: Possibly = maybe. Eventually = in the end.

Hard vs hardly

French: dur / presque pas

Rule / explanation: Hard = with effort; hardly = almost not.

Example / answer: The government worked hard. / The policy hardly worked.

G. Exercises

Exercise 1 — Place the adverb

French: exercice

Rule / explanation: The author explains the problem. (clearly)

Example / answer: The author clearly explains the problem.

Exercise 2 — Choose the best adverb

French: exercice

Rule / explanation: Prices rose ___ in 2022. (sharply / hardly)

Example / answer: sharply

Exercise 3 — Correct

French: exercice

Rule / explanation: The author does not never mention poverty.

Example / answer: The author never mentions poverty.

Exercise 4 — Translate

French: exercice

Rule / explanation: L’article explique clairement le problème.

Example / answer: The article clearly explains the problem.

Exercise 5 — Mini concours challenge

French: exercice

Rule / explanation: Write 6 sentences using clearly, often, sharply, arguably, recently and somewhat.

Example / answer: Arguably, the reform is necessary, but the proposal is somewhat unrealistic.
