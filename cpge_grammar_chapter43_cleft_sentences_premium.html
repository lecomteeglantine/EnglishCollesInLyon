The golden rule

Cleft sentence = split sentence + focused idea
It is public trust that the crisis has damaged.
What the article shows is that inequality weakens democracy.
Main useClefts highlight the key element of a sentence: cause, responsibility, contrast, result or priority.
CPGE habitUse clefts to make article summaries and essay conclusions more precise and forceful.
Key trapClefts are powerful but heavy. Use them for important analytical moments only.

A. Lesson — What are cleft sentences?

Cleft sentence

French: phrase clivée

Rule / explanation: A cleft sentence divides one idea into two parts to highlight the most important element.

Example / answer: It is public trust that the scandal damaged most.

Focus

French: mise en relief

Rule / explanation: The focused element is the word or phrase the speaker wants to emphasise.

Example / answer: What the author questions is the legitimacy of the reform.

Rhetorical purpose

French: but rhétorique

Rule / explanation: Clefts help writers clarify contrast, responsibility, cause or priority.

Example / answer: It was not the policy itself that failed, but its implementation.

CPGE use

French: usage CPGE

Rule / explanation: Clefts are useful in essays, article summaries, khôlles and introductions.

Example / answer: What is at stake is not merely economic growth, but democratic accountability.

B. It-clefts: It is / was... that

Basic structure

French: structure de base

Rule / explanation: Use it is/was + focus + that/who + rest of sentence.

Example / answer: It is inequality that the author sees as the main threat.

Subject focus

French: sujet mis en relief

Rule / explanation: Use it-clefts to emphasise the person or thing doing the action.

Example / answer: It was the media that revealed the scandal.

Object focus

French: complément mis en relief

Rule / explanation: Use it-clefts to emphasise the object or affected element.

Example / answer: It is democratic trust that the crisis has damaged.

Time and place focus

French: temps / lieu mis en relief

Rule / explanation: Use it-clefts to emphasise when or where something happened.

Example / answer: It was after the pandemic that the problem became impossible to ignore.

C. Wh-clefts: What... is / was...

What... is...

French: ce qui... c’est

Rule / explanation: Use wh-clefts to highlight the main point, result or interpretation.

Example / answer: What the article shows is that inequality weakens democracy.

What matters is...

French: ce qui compte, c’est

Rule / explanation: Useful for conclusions and oral summaries.

Example / answer: What matters is whether institutions can regain legitimacy.

What is striking is...

French: ce qui frappe, c’est

Rule / explanation: Useful to introduce analysis after description.

Example / answer: What is striking is the gap between official promises and lived reality.

What the author fails to explain is...

French: ce que l’auteur n’explique pas

Rule / explanation: Useful for polite criticism.

Example / answer: What the author fails to explain is why the policy was underfunded.

D. Clefts for contrast and article summaries

Not X but Y

French: non pas X mais Y

Rule / explanation: Clefts can correct an assumption and redirect the argument.

Example / answer: It is not immigration that explains the crisis, but inequality.

It is implementation, not ambition...

French: contraste analytique

Rule / explanation: Use clefts to make analytical distinctions clearer.

Example / answer: It is implementation, not ambition, that remains the main weakness.

What the author argues is that...

French: résumé d’article emphatique

Rule / explanation: Emphatic version of article-summary reporting.

Example / answer: What the author argues is that housing has become a generational crisis.

What the article ultimately suggests is...

French: conclusion d’article

Rule / explanation: Useful for final interpretation.

Example / answer: What the article ultimately suggests is that trust cannot be restored without accountability.

E. Clefts in introductions and conclusions

What is at stake is...

French: ce qui est en jeu, c’est

Rule / explanation: High-value CPGE phrase for problematics.

Example / answer: What is at stake is the future of democratic representation.

The real question is whether...

French: la vraie question est de savoir si

Rule / explanation: Useful to frame a problem.

Example / answer: The real question is whether reform can restore public confidence.

It is precisely this tension that...

French: c’est précisément cette tension qui

Rule / explanation: Elegant way to introduce a contradiction.

Example / answer: It is precisely this tension that makes the debate so politically explosive.

What remains unclear is...

French: ce qui reste incertain, c’est

Rule / explanation: Useful for nuanced conclusions.

Example / answer: What remains unclear is whether the policy will be properly funded.

F. Typical mistakes by French speakers

Incomplete it-cleft

French: clivée incomplète

Rule / explanation: A full it-cleft needs it + be + focus + that/who.

Example / answer: Wrong: It the government that failed. Correct: It was the government that failed.

Overusing clefts

French: abus de clivées

Rule / explanation: Clefts are emphatic; too many make writing heavy.

Example / answer: Use clefts for key ideas, not every sentence.

Wrong tense in cleft

French: mauvais temps

Rule / explanation: Choose is/was according to meaning and context.

Example / answer: It was the media that revealed the scandal. / It is trust that matters now.

Confusing what and that

French: what / that

Rule / explanation: Wh-clefts usually begin with what, not that.

Example / answer: Wrong: That the article shows is... Correct: What the article shows is...
