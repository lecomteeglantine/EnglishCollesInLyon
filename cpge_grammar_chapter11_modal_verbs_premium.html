The golden rule

Modal + base verb
The government should act.
The policy may fail.
The law must be revised.
Main useModals express ability, possibility, permission, obligation, advice, probability, criticism and hypothetical meaning.
CPGE habitUse modals to nuance arguments: may and might for caution, must for necessity, should for proposals, would for hypotheses.
Key trapAfter a modal, use the base verb: must act, not must to act; should be revised, not should revised.

A. Lesson — What are modal verbs?

Modal verbs

French: verbes modaux

Rule / explanation: Modal verbs are auxiliary verbs used before a base verb to express ability, possibility, permission, obligation, advice, probability or criticism.

Example / answer: The government should address inequality.

Base verb after modal

French: base verbale après modal

Rule / explanation: After a modal verb, always use the base verb without to.

Example / answer: The state must protect citizens.

No -s after modals

French: pas de -s

Rule / explanation: Modal verbs do not take -s with he/she/it.

Example / answer: Wrong: The government musts act. Correct: The government must act.

No do in modal questions

French: pas de do

Rule / explanation: Questions with modals use inversion, not do.

Example / answer: Should governments regulate social media?

No do in modal negatives

French: négation sans do

Rule / explanation: Add not after the modal.

Example / answer: The article may not be reliable.

Modal meaning matters

French: sens du modal

Rule / explanation: Each modal changes the speaker’s attitude: certainty, duty, permission, possibility or advice.

Example / answer: The reform could work. / The reform must work.

CPGE use

French: usage CPGE

Rule / explanation: Modals are essential for nuanced essays, policy proposals, debate and press analysis.

Example / answer: Democracies must protect rights, but they may need to regulate platforms.

Modal + passive

French: modal + passif

Rule / explanation: Modals can be used with the passive: modal + be + past participle.

Example / answer: The policy should be revised.

B. Can, could and be able to

Can — ability

French: capacité présente

Rule / explanation: Use can to express present ability.

Example / answer: Citizens can challenge unfair laws.

Can — possibility

French: possibilité générale

Rule / explanation: Use can for general possibility.

Example / answer: Social media can amplify anger.

Can — permission

French: permission

Rule / explanation: Use can for permission, especially informal contexts.

Example / answer: Students can ask questions during the discussion.

Cannot / can't

French: impossibilité / interdiction

Rule / explanation: Use cannot or can't for impossibility or prohibition.

Example / answer: Democracy cannot survive without trust.

Could — past ability

French: capacité passée

Rule / explanation: Use could for general ability in the past.

Example / answer: In the past, newspapers could shape public opinion more directly.

Could — possibility

French: possibilité

Rule / explanation: Use could for a possible future or hypothetical situation.

Example / answer: The reform could reduce inequality.

Could — polite question

French: question polie

Rule / explanation: Use could for polite requests or questions.

Example / answer: Could you explain the author’s point?

Be able to

French: être capable de

Rule / explanation: Use be able to when can is impossible, especially future or perfect forms.

Example / answer: The government will be able to act if Parliament agrees.

C. May, might and probability

May — possibility

French: possibilité formelle

Rule / explanation: Use may for formal possibility.

Example / answer: The reform may increase social tensions.

Might — weaker possibility

French: possibilité plus faible

Rule / explanation: Use might for a more uncertain possibility.

Example / answer: The policy might fail.

May not

French: peut-être pas

Rule / explanation: Means possibly not.

Example / answer: The article may not convince all readers.

Might not

French: il se pourrait que non

Rule / explanation: Means it is possible that something will not happen.

Example / answer: The measure might not be enough.

May — permission

French: permission formelle

Rule / explanation: Use may for formal permission.

Example / answer: Students may use dictionaries during preparation.

Might for cautious claims

French: prudence argumentative

Rule / explanation: Useful in essays when evidence is limited.

Example / answer: One might argue that the crisis is structural.

May well

French: il se pourrait bien que

Rule / explanation: Means there is a good chance.

Example / answer: The scandal may well damage public trust.

Might have to

French: obligation possible

Rule / explanation: Use for possible future obligation.

Example / answer: Governments might have to regulate AI more strictly.

D. Must, have to and need to

Must — strong obligation

French: obligation forte

Rule / explanation: Use must for strong obligation or necessity, often from the speaker’s viewpoint.

Example / answer: Governments must protect civil liberties.

Have to — external obligation

French: obligation externe

Rule / explanation: Use have to for obligation imposed by circumstances, law or rules.

Example / answer: Citizens have to obey the law.

Need to

French: nécessité

Rule / explanation: Use need to for practical necessity.

Example / answer: The article needs to provide more evidence.

Must not

French: interdiction

Rule / explanation: Means prohibition.

Example / answer: Politicians must not ignore public anger.

Do not have to

French: absence d’obligation

Rule / explanation: Means absence of obligation.

Example / answer: Students do not have to agree with the author.

Do not need to

French: pas besoin de

Rule / explanation: Means something is unnecessary.

Example / answer: The essay does not need to be dramatic.

Must for deduction

French: déduction forte

Rule / explanation: Use must when something seems logically certain.

Example / answer: The figures must be wrong.

Have got to

French: obligation informelle

Rule / explanation: Informal equivalent of have to.

Example / answer: The government has got to respond quickly.

E. Should, ought to and had better

Should — advice

French: conseil

Rule / explanation: Use should to give advice or make recommendations.

Example / answer: The government should invest in education.

Should — expected situation

French: attente

Rule / explanation: Use should when something is expected or likely.

Example / answer: The report should be published tomorrow.

Should not

French: conseil négatif

Rule / explanation: Use should not for negative advice.

Example / answer: The author should not ignore economic factors.

Ought to

French: devrait

Rule / explanation: Similar to should, slightly more formal or moral.

Example / answer: Politicians ought to be transparent.

Had better

French: ferait mieux de

Rule / explanation: Stronger advice; often implies a warning.

Example / answer: The minister had better answer the accusations.

Should for essay proposals

French: recommandation

Rule / explanation: Useful for policy recommendations.

Example / answer: Democracies should strike a balance between freedom and security.

Should for criticism

French: critique

Rule / explanation: Use should to say what was or is appropriate.

Example / answer: The article should define its key terms more clearly.

Should + passive

French: should + passif

Rule / explanation: Common in formal proposals.

Example / answer: The policy should be reassessed.

F. Would and modal politeness

Would — hypothetical result

French: conditionnel

Rule / explanation: Use would with hypothetical situations.

Example / answer: The reform would be more effective with proper funding.

Would like

French: souhait poli

Rule / explanation: Polite expression of desire.

Example / answer: I would like to focus on the economic dimension.

Would rather

French: préférence

Rule / explanation: Preference.

Example / answer: Some voters would rather abstain than support either party.

Would you...?

French: question polie

Rule / explanation: Polite request or question.

Example / answer: Would you explain your point?

Would in indirect speech

French: discours rapporté

Rule / explanation: Use would as the past form of will in reported speech.

Example / answer: The minister said the reform would help families.

Would for repeated past behaviour

French: habitude passée

Rule / explanation: Use would for repeated actions in the past, especially narratives.

Example / answer: In the past, parties would rely on local newspapers.

Would for cautious claims

French: prudence

Rule / explanation: Useful to avoid being too direct.

Example / answer: It would be misleading to blame only social media.

Would + passive

French: conditionnel passif

Rule / explanation: Useful for hypothetical policy outcomes.

Example / answer: The proposal would be criticised by unions.

G. Modal structures for CPGE argumentation

must be addressed

French: doit être traité

Rule / explanation: Strong necessity in policy writing.

Example / answer: The housing crisis must be addressed urgently.

should be taken into account

French: devrait être pris en compte

Rule / explanation: Recommendation for balanced analysis.

Example / answer: The economic context should be taken into account.

may be explained by

French: peut s’expliquer par

Rule / explanation: Cautious explanation.

Example / answer: The result may be explained by voter frustration.

could lead to

French: pourrait conduire à

Rule / explanation: Possible consequence.

Example / answer: The reform could lead to greater inequality.

might suggest that

French: pourrait suggérer que

Rule / explanation: Cautious interpretation of evidence.

Example / answer: The figures might suggest a loss of trust.

cannot be ignored

French: ne peut être ignoré

Rule / explanation: Strong analytical statement.

Example / answer: The social dimension cannot be ignored.

should not be confused with

French: ne devrait pas être confondu avec

Rule / explanation: Useful distinction.

Example / answer: Freedom of speech should not be confused with impunity.

must be distinguished from

French: doit être distingué de

Rule / explanation: Formal analytical distinction.

Example / answer: Criticism must be distinguished from hate speech.

H. Typical mistakes by French speakers

Using to after a modal

French: to après modal

Rule / explanation: After a modal, do not use to.

Example / answer: Wrong: The government must to act. Correct: The government must act.

Adding -s to a modal

French: -s au modal

Rule / explanation: Modals do not take -s.

Example / answer: Wrong: She cans vote. Correct: She can vote.

Using do in modal questions

French: do inutile

Rule / explanation: Invert the modal and the subject.

Example / answer: Wrong: Do we should act? Correct: Should we act?

Using do in modal negatives

French: négation incorrecte

Rule / explanation: Add not after the modal.

Example / answer: Wrong: They do not must act. Correct: They must not act / do not have to act.

Confusing mustn't and don't have to

French: mustn't / don't have to

Rule / explanation: Mustn't = prohibited; don't have to = not necessary.

Example / answer: You mustn't lie. / You don't have to agree.

Confusing may and can

French: may / can

Rule / explanation: May is more formal for permission; can is common and direct.

Example / answer: May I ask a question? / Can students use notes?

Overusing can for probability

French: can abusif

Rule / explanation: Use may/might/could for uncertainty in essays.

Example / answer: Better: The policy may increase inequality.

Forgetting modal + be + past participle

French: modal passif

Rule / explanation: For passive meaning, use modal + be + past participle.

Example / answer: Wrong: The law should revised. Correct: The law should be revised.

I. Exercises

Exercise 1 — Choose the right modal

French: exercice

Rule / explanation: The government ___ protect vulnerable citizens. (must / can)

Example / answer: Answer: must

Exercise 2 — Choose the right modal

French: exercice

Rule / explanation: The reform ___ reduce inequality, but the evidence is uncertain. (might / must)

Example / answer: Answer: might

Exercise 3 — Correct the mistake

French: exercice

Rule / explanation: The state must to protect freedom of speech.

Example / answer: Answer: The state must protect freedom of speech.

Exercise 4 — Correct the mistake

French: exercice

Rule / explanation: Does the government should intervene?

Example / answer: Answer: Should the government intervene?

Exercise 5 — Correct the mistake

French: exercice

Rule / explanation: The policy should revised.

Example / answer: Answer: The policy should be revised.

Exercise 6 — Transform into a question

French: exercice

Rule / explanation: The government should regulate platforms.

Example / answer: Answer: Should the government regulate platforms?

Exercise 7 — Make it negative

French: exercice

Rule / explanation: Students have to agree with the author.

Example / answer: Answer: Students do not have to agree with the author.

Exercise 8 — Explain the difference

French: exercice

Rule / explanation: You must not use your phone. / You do not have to use your phone.

Example / answer: Answer: Must not = it is forbidden. Do not have to = it is not necessary.

Exercise 9 — Translate

French: exercice

Rule / explanation: Le gouvernement devrait investir dans l’éducation.

Example / answer: Answer: The government should invest in education.

Exercise 10 — Translate

French: exercice

Rule / explanation: Cette réforme pourrait aggraver les inégalités.

Example / answer: Answer: This reform could worsen inequality.

Exercise 11 — Translate

French: exercice

Rule / explanation: La liberté d’expression ne doit pas être confondue avec l’impunité.

Example / answer: Answer: Freedom of speech must not be confused with impunity.

Exercise 12 — Mini concours challenge

French: exercice

Rule / explanation: Write 6 sentences about social media using: can, may, might, must, should, would.

Example / answer: Possible answer: Social media can amplify anger. It may improve access to information. It might also increase polarisation. Governments must protect users. Platforms should be regulated carefully. A total ban would be unrealistic.
