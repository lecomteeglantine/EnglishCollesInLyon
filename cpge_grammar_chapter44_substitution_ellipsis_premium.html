Chapter 44 — Substitution and Ellipsis
B2 → C1 grammar for concise style, avoiding repetition and sounding natural in advanced English.

Chapter objective

This premium grammar chapter helps CPGE students master substitution and ellipsis. It covers one/ones, do so, so/not, auxiliary ellipsis, comparative ellipsis, coordinated structures and common French-speaker mistakes.

Level B2 → C1SubstitutionEllipsisConcise StyleAvoiding RepetitionExercises + Answers

The golden rule

Substitution replaces repeated words.
Ellipsis removes repeated words.
The first argument is convincing, but the second one is not.
Main useSubstitution and ellipsis make English lighter, clearer and less repetitive.
CPGE habitUse them to compare articles, policies and arguments without repeating the same nouns and verbs.
Key trapDo not delete words if the sentence becomes unclear. Concision must serve clarity.

A. Lesson — What are substitution and ellipsis?

Substitution

French: substitution

Rule / explanation: Substitution replaces a repeated word or phrase with a shorter form.

Example / answer: This argument is stronger than the previous one.

Ellipsis

French: ellipse

Rule / explanation: Ellipsis omits words that are understood from context.

Example / answer: The first article is persuasive; the second less so.

Why it matters

French: pourquoi c’est important

Rule / explanation: Advanced English avoids unnecessary repetition while keeping meaning clear.

Example / answer: The policy helped older voters more than younger ones.

B. One / ones

One

French: celui / celle

Rule / explanation: Use one to replace a singular countable noun.

Example / answer: This reform is more ambitious than the previous one.

Ones

French: ceux / celles

Rule / explanation: Use ones to replace a plural countable noun.

Example / answer: Local policies are often more effective than national ones.

Not with uncountable nouns

French: pas avec indénombrables

Rule / explanation: Do not use one/ones to replace uncountable nouns such as evidence or information.

Example / answer: Better: The evidence is stronger than before. Not: the evidence is a stronger one.

C. Do / does / did as substitution

Do so

French: le faire

Rule / explanation: Use do so to avoid repeating an action phrase.

Example / answer: The government promised to reform housing but failed to do so.

Did so

French: l’a fait

Rule / explanation: Use did so for a repeated past action.

Example / answer: Experts warned the government, and they did so repeatedly.

Formal use

French: usage formel

Rule / explanation: Do so is more formal than do it and is common in academic English.

Example / answer: If policymakers wish to restore trust, they must do so transparently.

D. So and not as substitutes

Think so

French: le penser

Rule / explanation: Use so after verbs such as think, hope, expect, suppose and believe.

Example / answer: Will the reform work? Many analysts think so.

Hope so / hope not

French: l’espérer / espérer que non

Rule / explanation: Use so/not to avoid repeating a full clause.

Example / answer: Will trust recover? The author hopes so, but the evidence suggests not.

So as adjective substitute

French: ainsi / autant

Rule / explanation: So can replace a repeated adjective or quality.

Example / answer: The first text is convincing; the second less so.

E. Auxiliary ellipsis

Auxiliary repeated

French: auxiliaire repris

Rule / explanation: In short answers and comparisons, repeat only the auxiliary.

Example / answer: The policy has failed, and the previous one has too.

Modal repeated

French: modal repris

Rule / explanation: Repeat only the modal when the main verb is clear.

Example / answer: The government could act, but it probably won’t.

Be repeated

French: be repris

Rule / explanation: Be can stand alone if the complement is understood.

Example / answer: The first argument is persuasive; the second is not.

F. Ellipsis in coordination and comparison

Shared subject

French: sujet partagé

Rule / explanation: When two verbs share the same subject, the subject is often omitted in the second part.

Example / answer: The article analyses the crisis and questions the government’s response.

Comparative ellipsis

French: ellipse comparative

Rule / explanation: Comparisons often omit repeated verbs or complements.

Example / answer: The second article is more nuanced than the first.

Than expected

French: que prévu

Rule / explanation: Than expected is an elliptical phrase meaning than people expected.

Example / answer: The crisis proved more severe than expected.

G. CPGE writing patterns

The former / the latter

French: le premier / le second

Rule / explanation: Use these to refer back to two previously mentioned items.

Example / answer: The article compares reform and revolution; the former seems realistic, the latter dangerous.

The same is true of...

French: il en va de même pour

Rule / explanation: Useful to extend an argument without repeating it.

Example / answer: The same is true of education policy.

Nor + auxiliary

French: non plus

Rule / explanation: Use nor + auxiliary + subject to add a similar negative idea.

Example / answer: The policy does not address housing, nor does it tackle wages.

H. Typical mistakes by French speakers

Repeating everything

French: répétition lourde

Rule / explanation: English often avoids repetition with one, so, do so or auxiliary ellipsis.

Example / answer: Heavy: the second argument is stronger than the first argument. Better: stronger than the first one.

One with uncountable nouns

French: one impossible

Rule / explanation: Do not use one for uncountable nouns.

Example / answer: Wrong: this evidence is a convincing one. Better: this evidence is convincing.

Unclear ellipsis

French: ellipse ambiguë

Rule / explanation: Do not omit words if the reader may misunderstand the comparison.

Example / answer: Clarity first: repeat the noun if needed.

I. Exercises

Exercise 1 — Avoid repetition

French: exercice

Rule / explanation: This policy is more effective than the previous policy.

Example / answer: This policy is more effective than the previous one.

Exercise 2 — Use do so

French: exercice

Rule / explanation: The government promised to act but failed to act.

Example / answer: The government promised to act but failed to do so.

Exercise 3 — Use so/not

French: exercice

Rule / explanation: Will trust recover? The author hopes that trust will recover.

Example / answer: Will trust recover? The author hopes so.

Exercise 4 — Correct

French: exercice

Rule / explanation: This information is a useful one.

Example / answer: This information is useful.

Exercise 5 — Translate

French: exercice

Rule / explanation: Le premier argument est convaincant, le second beaucoup moins.

Example / answer: The first argument is convincing; the second much less so.
