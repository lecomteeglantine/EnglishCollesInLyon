The golden rule

Determiner + noun
this argument / these figures / several factors / no evidence
A determiner makes the noun precise.
Main useDeterminers clarify whether a noun is specific, general, singular, plural, possessed or quantified.
CPGE habitUse determiners to avoid vague writing: this trend, several factors, each argument, the majority of voters.
Key trapIn formal English, avoid vague this. Write this trend, this issue or this argument.

A. Lesson — What are determiners?

Determiners

French: déterminants

Rule / explanation: Determiners are words placed before nouns to clarify quantity, possession, specificity or reference.

Example / answer: The article discusses several causes of political distrust.

Before a noun

French: avant le nom

Rule / explanation: A determiner usually comes before a noun, before any adjective.

Example / answer: These recent figures are worrying.

Specific reference

French: référence précise

Rule / explanation: Some determiners show that the reader knows which thing is meant.

Example / answer: The reform has been criticised by voters.

General reference

French: référence générale

Rule / explanation: Some determiners refer to things in a general or non-specific way.

Example / answer: A democracy needs public trust.

Quantity

French: quantité

Rule / explanation: Many determiners express quantity or amount.

Example / answer: Many voters feel ignored.

Possession

French: possession

Rule / explanation: Possessive determiners show who something belongs to.

Example / answer: The author supports his argument with statistics.

Academic precision

French: précision académique

Rule / explanation: Determiners make essays more precise and help avoid vague claims.

Example / answer: This argument is convincing, but that example is weak.

CPGE use

French: usage CPGE

Rule / explanation: Determiners are essential for article summaries, data commentary, essays and khôlles.

Example / answer: Several factors must be taken into account.

B. Articles as determiners

A / an

French: article indéfini

Rule / explanation: Use a/an with a singular countable noun when it is non-specific or mentioned for the first time.

Example / answer: The author presents a convincing argument.

The

French: article défini

Rule / explanation: Use the when the noun is specific, already known or unique in context.

Example / answer: The argument is supported by recent figures.

Zero article

French: article zéro

Rule / explanation: Use no article for general plural or uncountable nouns.

Example / answer: Democracy depends on trust.

A vs the

French: a vs the

Rule / explanation: A introduces something; the refers to something specific or already identified.

Example / answer: The article mentions a reform. The reform remains controversial.

An before vowel sound

French: an

Rule / explanation: Use an before a vowel sound, not just a vowel letter.

Example / answer: An economic crisis can transform politics.

The with institutions

French: institutions

Rule / explanation: Use the with specific institutions, but not always with abstract systems.

Example / answer: The government announced a reform. / Parliament debated the bill.

General concepts

French: concepts généraux

Rule / explanation: Abstract concepts often take zero article when used generally.

Example / answer: Freedom of speech must be protected.

CPGE habit

French: réflexe CPGE

Rule / explanation: Check whether your noun is countable, singular, plural, specific or general.

Example / answer: A policy can fail if the evidence is weak.

C. Demonstratives: this, that, these, those

This

French: ce / cette

Rule / explanation: Use this for singular nouns close in time, space or argument.

Example / answer: This article raises an important question.

That

French: ce / cette / cela

Rule / explanation: Use that for singular nouns more distant, already mentioned or contrasted.

Example / answer: That argument is less convincing.

These

French: ces

Rule / explanation: Use these for plural nouns close in time, space or argument.

Example / answer: These figures suggest a decline in trust.

Those

French: ces / ceux-là

Rule / explanation: Use those for plural nouns more distant or contrasted.

Example / answer: Those voters may feel excluded.

This vs it

French: this vs it

Rule / explanation: Use this to refer to a whole idea you want to comment on; use it for a specific noun already mentioned.

Example / answer: The policy failed. This explains public anger.

This + noun

French: this + nom

Rule / explanation: In academic writing, this should often be followed by a noun for clarity.

Example / answer: This trend may weaken democracy.

That of

French: celui/celle de

Rule / explanation: Use that of to avoid repetition in formal comparisons.

Example / answer: The British system differs from that of the United States.

Those who

French: ceux qui

Rule / explanation: Useful for referring to groups of people formally.

Example / answer: Those who feel ignored may abstain.

D. Possessive determiners

My / your / his / her / its / our / their

French: possessifs

Rule / explanation: Use possessive determiners before nouns.

Example / answer: The government defended its decision.

Its vs it’s

French: its / it’s

Rule / explanation: Its = possession; it’s = it is / it has.

Example / answer: The country must protect its institutions.

Their

French: leur / leurs

Rule / explanation: Use their for plural possessors and also as a gender-neutral singular option.

Example / answer: Each voter has their own priorities.

Possessive + noun

French: possessif + nom

Rule / explanation: Do not use an article before a possessive determiner.

Example / answer: Wrong: the their rights. Correct: their rights.

Institutional possession

French: possession institutionnelle

Rule / explanation: Possessives are common with governments, parties, countries and newspapers.

Example / answer: The newspaper changed its editorial line.

Avoid ambiguity

French: éviter l’ambiguïté

Rule / explanation: Make clear who the possessive refers to.

Example / answer: The party lost its credibility after the scandal.

His / her / their

French: his / her / their

Rule / explanation: Choose the determiner according to reference, but use their when gender is unknown or irrelevant.

Example / answer: A student should support their claims with evidence.

CPGE style

French: style CPGE

Rule / explanation: Possessive determiners help avoid repetitive noun phrases.

Example / answer: The article presents its argument clearly.

E. Quantifying determiners

Some

French: quelques / du

Rule / explanation: Use some for an unspecified positive quantity.

Example / answer: Some voters support the reform.

Any

French: aucun / des / n’importe quel

Rule / explanation: Use any in negatives, questions and non-specific contexts.

Example / answer: The article does not provide any evidence.

No

French: aucun

Rule / explanation: Use no before a noun to mean not any.

Example / answer: No policy can satisfy everyone.

Many

French: beaucoup de

Rule / explanation: Use many with plural countable nouns.

Example / answer: Many citizens distrust institutions.

Much

French: beaucoup de

Rule / explanation: Use much with uncountable nouns, often in negatives/questions.

Example / answer: The article does not provide much evidence.

A lot of

French: beaucoup de

Rule / explanation: Use with countable and uncountable nouns, but it is less formal.

Example / answer: A lot of voters are concerned about inflation.

Several

French: plusieurs

Rule / explanation: Means more than two but not many; useful in formal writing.

Example / answer: Several arguments support this view.

Few / little

French: peu de

Rule / explanation: Few + plural countable; little + uncountable. Both mean not much/many.

Example / answer: Few politicians mention poverty. / Little progress has been made.

F. Each, every, all, both, either, neither

Each

French: chaque

Rule / explanation: Focuses on individuals separately.

Example / answer: Each argument must be supported by evidence.

Every

French: chaque / tous

Rule / explanation: Focuses on all members of a group collectively.

Example / answer: Every democracy needs public trust.

All

French: tous / tout

Rule / explanation: Refers to the complete group or quantity.

Example / answer: All citizens should have access to education.

Both

French: les deux

Rule / explanation: Refers to two items together.

Example / answer: Both articles discuss inequality.

Either

French: l’un ou l’autre / non plus

Rule / explanation: Means one or the other of two options; also used in negatives.

Example / answer: Either solution could work.

Neither

French: ni l’un ni l’autre

Rule / explanation: Means not one and not the other.

Example / answer: Neither article provides enough evidence.

Either... or

French: soit... soit

Rule / explanation: Used to present alternatives.

Example / answer: The government must either fund the policy or abandon it.

Neither... nor

French: ni... ni

Rule / explanation: Used to reject two options.

Example / answer: The article is neither simplistic nor naïve.

G. Determiners in academic and essay writing

This argument

French: cet argument

Rule / explanation: Useful to refer back to a previous point clearly.

Example / answer: This argument is persuasive but incomplete.

This raises the question of

French: cela soulève la question de

Rule / explanation: Useful transition in essays and khôlles.

Example / answer: This raises the question of political legitimacy.

Such a

French: un tel

Rule / explanation: Means this type of; useful for generalisation.

Example / answer: Such a reform would be difficult to implement.

Such as

French: comme / tel que

Rule / explanation: Introduces examples, not a determiner alone but useful with nouns.

Example / answer: Issues such as housing and poverty dominate the debate.

A number of

French: un certain nombre de

Rule / explanation: Formal alternative to several/many.

Example / answer: A number of factors explain this trend.

The majority of

French: la majorité de

Rule / explanation: Formal expression for most.

Example / answer: The majority of voters supported the proposal.

A minority of

French: une minorité de

Rule / explanation: Formal expression for a smaller group.

Example / answer: A minority of respondents opposed the measure.

The extent of

French: l’ampleur de

Rule / explanation: Useful abstract noun phrase.

Example / answer: The article underestimates the extent of the crisis.

H. Typical mistakes by French speakers

This without a noun

French: this vague

Rule / explanation: In formal writing, avoid unclear this.

Example / answer: Weak: This is important. Better: This trend is important.

The with general concepts

French: the abusif

Rule / explanation: Do not use the for abstract ideas in general.

Example / answer: Wrong: The democracy needs trust. Correct: Democracy needs trust.

A with uncountable nouns

French: a + indénombrable

Rule / explanation: Do not use a/an with uncountable nouns.

Example / answer: Wrong: an evidence. Correct: evidence / a piece of evidence.

Much with plural countable nouns

French: much / many

Rule / explanation: Use many with plural countable nouns.

Example / answer: Wrong: much voters. Correct: many voters.

Many with uncountable nouns

French: many / much

Rule / explanation: Use much or a lot of with uncountable nouns.

Example / answer: Wrong: many information. Correct: much information / a lot of information.

Each with plural noun

French: each

Rule / explanation: Each is followed by a singular noun.

Example / answer: Wrong: each arguments. Correct: each argument.

All without the when needed

French: all / all the

Rule / explanation: Use all the when referring to a specific complete group.

Example / answer: All citizens = citizens in general. All the citizens = a specific group.

Its vs it’s

French: its / it’s

Rule / explanation: Do not confuse possessive its with it is.

Example / answer: Wrong: The country protects it’s institutions. Correct: its institutions.

I. Exercises

Example / answer: Answer: This article raises an important question.

Rule / explanation: The article does not provide ___ evidence. (many / much)

Example / answer: Answer: much
