# Australia Civilisation Atlas

Upload `colle-atlas-au.html` to the root of the `EnglishCollesInLyon` GitHub repository and replace the existing file with the same name.

## Included
- Back-to-basics opening with ten essential foundations
- 31-event expandable and filterable timeline
- Complete federal political and judicial system sections
- Interactive power map, legislative stepper and court-route explorer
- 15 historically significant leaders, activists and reformers
- 20 entries on First Peoples, settlement, land rights, migration and national identity
- States, territories, regions and Indo-Pacific relationships
- Society and contemporary debates, updated to 21 July 2026
- Article/source and freely reusable Wikimedia Commons image for every knowledge entry
- Search, essentials-only mode, reader mode, dark mode and accessibility controls
- Local revision list with responsive HTML export and import
