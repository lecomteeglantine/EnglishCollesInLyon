Chapter 40 — Used To, Would, Be Used To
B1 → B2 grammar for past habits, adaptation, contrast and historical change.

Chapter objective

This premium grammar chapter helps CPGE students master used to, would, be used to and get used to. It covers past habits, former states, repeated actions, adaptation, contrast with the Past Simple and common French-speaker mistakes.

Level B1 → B2Used ToWouldBe Used ToHistorical ChangeExercises + Answers

The golden rule

used to + base verb = past habit or former state
be used to + noun / -ing = be accustomed to
get used to + noun / -ing = become accustomed to
Main useThese structures help distinguish old habits, repeated past actions and adaptation to a situation.
CPGE habitUse them to analyse historical and social change: Voters used to trust institutions more, but they are now used to political instability.
Key trapUsed to do and be used to doing do not mean the same thing.

A. Lesson — What does used to mean?

Past habit

French: habitude passée

Rule / explanation: Use used to + base verb for habits that were true in the past but are no longer true.

Example / answer: Voters used to trust mainstream parties more.

Former state

French: ancien état

Rule / explanation: Used to can describe a past state that has changed.

Example / answer: Manufacturing used to be central to the local economy.

Contrast with now

French: contraste avec maintenant

Rule / explanation: Used to often implies that the present situation is different.

Example / answer: Britain used to rely more heavily on coal, but its energy mix has changed.

Analytical value

French: valeur analytique

Rule / explanation: It is useful for showing social, political, economic and cultural change.

Example / answer: The article shows that young people used to expect greater social mobility.

B. Used to: form and structure

Affirmative

French: forme affirmative

Rule / explanation: Use subject + used to + base verb.

Example / answer: Newspapers used to dominate public debate.

Negative

French: forme négative

Rule / explanation: Use did not use to + base verb.

Example / answer: Citizens did not use to rely so much on social media for news.

Question

French: question

Rule / explanation: Use did + subject + use to + base verb.

Example / answer: Did voters use to trust politicians more?

No present form

French: pas de présent

Rule / explanation: Used to is only for the past. For present habits, use usually or tend to.

Example / answer: Today, voters usually get their news online.

C. Would for repeated past actions

Repeated action

French: action répétée passée

Rule / explanation: Would can describe repeated actions in the past, especially in narratives.

Example / answer: During campaigns, candidates would travel across the country.

Not for states

French: pas pour les états

Rule / explanation: Use used to, not would, for past states such as be, have, believe or know.

Example / answer: Correct: The city used to be industrial. Not: The city would be industrial.

Narrative style

French: style narratif

Rule / explanation: Would is elegant when describing regular past behaviour after the context is clear.

Example / answer: In the 1980s, workers would gather outside the factory gates.

Used to first, would after

French: stratégie de rédaction

Rule / explanation: Introduce the past period with used to, then use would for repeated actions.

Example / answer: The area used to be industrial. Every morning, workers would fill the streets.

D. Used to vs Past Simple

Used to

French: ancienne habitude

Rule / explanation: Use it when you stress contrast between past and present.

Example / answer: The party used to attract working-class voters.

Past Simple

French: événement passé

Rule / explanation: Use it for a finished past event or a dated historical fact.

Example / answer: The party won the election in 1997.

Repeated past fact

French: fait répété passé

Rule / explanation: Past Simple can also describe repeated past events without emphasising contrast with now.

Example / answer: The newspaper published weekly reports during the inquiry.

Choosing the nuance

French: choisir la nuance

Rule / explanation: Use used to when change is your main point; use Past Simple when the event itself matters.

Example / answer: Britain used to have more manufacturing jobs. / Manufacturing jobs declined after 1980.

E. Be used to + noun / -ing

Meaning

French: être habitué à

Rule / explanation: Be used to means be accustomed to something.

Example / answer: Citizens are used to political scandals.

Be used to + noun

French: habitué à + nom

Rule / explanation: Use a noun after to.

Example / answer: Households are used to economic uncertainty.

Be used to + -ing

French: habitué à + gérondif

Rule / explanation: After be used to, use a gerund, not an infinitive.

Example / answer: Young people are used to reading news online.

Different tenses

French: différents temps

Rule / explanation: Be used to can be used in different tenses.

Example / answer: Voters were used to high turnout. / They will be used to coalition politics.

F. Get used to + noun / -ing

Meaning

French: s’habituer à

Rule / explanation: Get used to means become accustomed to something.

Example / answer: Citizens had to get used to digital public services.

Process of adaptation

French: processus d’adaptation

Rule / explanation: Use get used to when adaptation is gradual or difficult.

Example / answer: Workers are getting used to using AI tools.

Future adaptation

French: adaptation future

Rule / explanation: Use will get used to for future adjustment.

Example / answer: Students will get used to hybrid learning environments.

Negative adaptation

French: adaptation refusée

Rule / explanation: Use not get used to when someone cannot adapt.

Example / answer: Some communities never got used to the loss of local industries.

G. CPGE essay and article-summary patterns

Historical contrast

French: contraste historique

Rule / explanation: Use used to to show how a country, party or institution has changed.

Example / answer: The article argues that Britain used to see itself as politically stable.

Social change

French: changement social

Rule / explanation: Use used to for past expectations and be used to for present adaptation.

Example / answer: Young adults used to expect home ownership, but many are now used to renting for longer.

Economic transition

French: transition économique

Rule / explanation: Useful for deindustrialisation, digitalisation and labour-market change.

Example / answer: Workers who used to rely on manufacturing jobs are getting used to a service-based economy.

Political normalisation

French: normalisation politique

Rule / explanation: Be used to can describe what society now accepts as normal.

Example / answer: Voters have become used to permanent political crisis.

H. Typical mistakes by French speakers

Used to + -ing

French: confusion structure

Rule / explanation: Past habit uses used to + base verb, not -ing.

Example / answer: Wrong: People used to reading newspapers. Correct: People used to read newspapers.

Be used to + infinitive

French: infinitif incorrect

Rule / explanation: Be used to is followed by a noun or gerund.

Example / answer: Wrong: They are used to read online. Correct: They are used to reading online.

Did used to

French: did + base verb

Rule / explanation: After did, use use to, not used to.

Example / answer: Wrong: Did people used to trust parties? Correct: Did people use to trust parties?

Would for states

French: would incorrect avec état

Rule / explanation: Do not use would for past states.

Example / answer: Wrong: The city would be industrial. Correct: The city used to be industrial.

Exercise 4 — Correct

Rule / explanation: The city would be industrial.

Example / answer: The city used to be industrial.

French: exercice

Rule / explanation: Write 5 sentences using used to, would, be used to and get used to.
