Chapter 38 — Mixed Conditionals
B2 → C1 grammar for linking past causes to present results and present situations to past outcomes.

Chapter objective

This premium grammar chapter helps CPGE students master mixed conditionals. It covers past condition → present result, present condition → past result, regret, political analysis, historical consequences, modal alternatives and common French-speaker mistakes.

Level B2 → C1Mixed ConditionalsPast CausePresent ResultRegretExercises + Answers

The golden rule

Past condition → present result
If the government had acted earlier, the crisis would be less severe now.
Main useUse mixed conditionals when the condition and the result belong to different time periods.
CPGE habitUse them for historical and political analysis: If earlier reforms had worked, voters would trust institutions more today.
Key trapDo not force everything into a third conditional. Ask: is the result in the past, the present, or the future?

A. Lesson — What are mixed conditionals?

Mixed time reference

French: référence temporelle mixte

Rule / explanation: Mixed conditionals connect a condition in one time period to a result in another.

Example / answer: If the reform had succeeded, the country would be more stable today.

Past cause, present result

French: cause passée, résultat présent

Rule / explanation: Use Past Perfect in the if-clause and would + base verb in the result clause.

Example / answer: If wages had risen faster, families would be less vulnerable now.

Present condition, past result

French: condition présente, résultat passé

Rule / explanation: Use Past Simple in the if-clause and would have + past participle in the result clause.

Example / answer: If the minister were more competent, he would have handled the crisis better.

Analytical value

French: valeur analytique

Rule / explanation: Mixed conditionals express causal complexity and counterfactual reasoning.

Example / answer: If the country had invested more in education, inequality would be lower today.

B. Type 1 — Past condition → present result

Structure

French: structure

Rule / explanation: Use if + Past Perfect, would + base verb.

Example / answer: If the policy had been funded, it would be effective now.

Historical consequences

French: conséquences historiques

Rule / explanation: Use it to show how past choices shape present situations.

Example / answer: If previous governments had built more homes, rents would be lower today.

Regret and criticism

French: regret et critique

Rule / explanation: Useful for criticising past decisions and present consequences.

Example / answer: If leaders had listened to experts, public services would be stronger now.

Present result markers

French: marqueurs du présent

Rule / explanation: Words like now, today and currently often show the result is present.

Example / answer: If the warnings had been taken seriously, the situation would be different today.

C. Type 2 — Present condition → past result

Structure

French: structure

Rule / explanation: Use if + Past Simple, would have + past participle.

Example / answer: If the author were more balanced, he would have included opposing views.

Present character, past action

French: caractéristique présente, action passée

Rule / explanation: The present condition describes a stable quality that affected a past outcome.

Example / answer: If the minister were more cautious, she would not have made such a risky promise.

Hypothetical ability

French: capacité hypothétique

Rule / explanation: Use could have when the present condition would have made a past action possible.

Example / answer: If the state had stronger institutions, it could have resisted the crisis more effectively.

Critical evaluation

French: évaluation critique

Rule / explanation: Useful when evaluating articles, leaders or systems.

Example / answer: If the article were less ideological, it would have offered a more convincing analysis.

D. Modal alternatives

Would

French: résultat hypothétique

Rule / explanation: Would expresses the main hypothetical result.

Example / answer: If the reform had worked, voters would trust the government more.

Could

French: possibilité / capacité

Rule / explanation: Could expresses possible or potential results.

Example / answer: If the country had invested earlier, it could be more competitive now.

Might

French: possibilité prudente

Rule / explanation: Might expresses uncertainty and cautious analysis.

Example / answer: If the evidence had been stronger, the conclusion might be more persuasive.

Would not have

French: résultat passé négatif

Rule / explanation: Use for past results that would not have happened under a present condition.

Example / answer: If the system were fairer, voters would not have rejected it so strongly.

E. CPGE argumentation patterns

Political distrust

French: méfiance politique

Rule / explanation: Use mixed conditionals to link past failure to present distrust.

Example / answer: If politicians had kept their promises, citizens would be less distrustful today.

Social inequality

French: inégalités sociales

Rule / explanation: Useful for long-term cause and present result.

Example / answer: If education had been more equal, social mobility would be stronger now.

Climate policy

French: politique climatique

Rule / explanation: Use it to criticise delayed action.

Example / answer: If governments had acted earlier, climate adaptation would be less costly today.

Article evaluation

French: évaluation d’article

Rule / explanation: Use mixed conditionals to assess evidence and argument quality.

Example / answer: If the author had included more data, the argument would be more convincing.

F. Mixed Conditional vs Third Conditional

Third Conditional

French: conditionnel passé complet

Rule / explanation: Both condition and result are in the past.

Example / answer: If the government had acted earlier, the crisis would have been avoided.

Mixed Conditional

French: conditionnel mixte

Rule / explanation: The condition and result belong to different times.

Example / answer: If the government had acted earlier, the crisis would be less severe now.

Ask “when is the result?”

French: identifier le résultat

Rule / explanation: The result decides whether you need would have + past participle or would + base verb.

Example / answer: Past result: would have avoided. Present result: would be less severe.

Essay nuance

French: nuance argumentative

Rule / explanation: Mixed conditionals often sound more analytical because they show long-term consequences.

Example / answer: If institutions had been reformed earlier, democracy would be more resilient today.

G. Typical mistakes by French speakers

Wrong result time

French: mauvais temps du résultat

Rule / explanation: Use would + base verb for present result, not would have.

Example / answer: Wrong: If they had acted, the crisis would have be less severe now. Correct: would be less severe now.

Would in the if-clause

French: pas de would après if

Rule / explanation: Do not normally use would in the if-clause.

Example / answer: Wrong: If the government would have acted. Correct: If the government had acted.

Forgetting had

French: oubli de had

Rule / explanation: Past counterfactual condition needs Past Perfect.

Example / answer: Wrong: If leaders listened earlier. Correct: If leaders had listened earlier.

Confusing were and had been

French: condition présente vs passée

Rule / explanation: Use were for present hypothetical state; had been for past counterfactual state.

Example / answer: If he were competent, he would have reacted better. / If he had been competent then, he would be respected now.

H. Exercises

Exercise 1 — Complete

French: exercice

Rule / explanation: If the government ___ (act) earlier, the crisis ___ (be) less severe now.

Example / answer: had acted / would be

Exercise 2 — Complete

French: exercice

Rule / explanation: If the article ___ (be) more balanced, it ___ (convince) more readers.

Example / answer: were / would have convinced

Exercise 3 — Correct

French: exercice

Rule / explanation: If the reform would have worked, voters would trust the government now.

Example / answer: If the reform had worked, voters would trust the government now.

Exercise 4 — Choose the result

French: exercice

Rule / explanation: If wages had risen faster, families ___ less vulnerable today. (would be / would have been)

Example / answer: would be
