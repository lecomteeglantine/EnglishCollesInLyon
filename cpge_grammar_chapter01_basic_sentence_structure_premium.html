Chapter objective

This premium grammar chapter helps CPGE students build clear English sentences. It focuses on subject–verb order, complements, negatives, questions and the most common mistakes made by French speakers. The aim is simple: before writing sophisticated arguments, students must produce clean, reliable sentences.

A. Lesson — What is a basic English sentence?

Basic sentence structure

French: structure de phrase de base

Rule / explanation: A basic English sentence usually follows the order: Subject + Verb + Complement.

Example / answer: The government changed the law.

Subject

French: sujet

Rule / explanation: The person, thing or idea doing the action or being described.

Example / answer: Young voters support climate action.

Verb

French: verbe

Rule / explanation: The action or state in the sentence.

Example / answer: The economy slowed down.

Complement

French: complément

Rule / explanation: The information that completes the sentence: object, place, time, reason or description.

Example / answer: The reform affected low-income families.

Word order

French: ordre des mots

Rule / explanation: English word order is stricter than French word order. In a neutral sentence, do not move the verb before the subject.

Example / answer: Social media influences public opinion.

Affirmative sentence

French: phrase affirmative

Rule / explanation: A sentence that states something positively.

Example / answer: The article highlights growing inequality.

Negative sentence

French: phrase négative

Rule / explanation: A sentence that says something is not true, often with do not / does not / did not.

Example / answer: The policy does not solve the problem.

Question

French: question

Rule / explanation: A sentence used to ask for information, usually with an auxiliary before the subject.

Example / answer: Does the policy solve the problem?

B. Core patterns to memorise

S + V

French: S + V

Rule / explanation: Subject + verb. Used with intransitive verbs.

Example / answer: Democracy matters.

S + V + Object

French: S + V + COD

Rule / explanation: Subject + verb + object. The object receives the action.

Example / answer: The government introduced a reform.

S + V + Complement

French: S + V + attribut

Rule / explanation: Subject + verb + adjective/noun phrase after be, seem, become, remain.

Example / answer: The issue remains controversial.

S + V + Place

French: S + V + lieu

Rule / explanation: Place usually comes after the verb/object.

Example / answer: Inequality increased in many regions.

S + V + Time

French: S + V + temps

Rule / explanation: Time often comes at the end or beginning of the sentence.

Example / answer: The crisis began in 2008.

S + V + Reason

French: S + V + raison

Rule / explanation: Reason can be introduced by because, due to, owing to.

Example / answer: Voters abstained because they felt ignored.

There is / There are

French: il y a

Rule / explanation: Used to say that something exists.

Example / answer: There is a clear gap between rich and poor.

It is + adjective + to

French: il est + adjectif + de

Rule / explanation: Useful for general comments.

Example / answer: It is difficult to regulate online platforms.

C. Questions and negatives

Present Simple negative

French: négation au présent simple

Rule / explanation: Use do not / does not + base verb.

Example / answer: The article does not mention the economic context.

Past Simple negative

French: négation au prétérit

Rule / explanation: Use did not + base verb.

Example / answer: The reform did not reduce inequality.

Present Simple question

French: question au présent simple

Rule / explanation: Use do / does + subject + base verb.

Example / answer: Does the article criticise the government?

Past Simple question

French: question au prétérit

Rule / explanation: Use did + subject + base verb.

Example / answer: Did the policy affect young people?

Be negative

French: négation avec be

Rule / explanation: With be, add not directly after the verb.

Example / answer: The argument is not convincing.

Be question

French: question avec be

Rule / explanation: With be, put be before the subject.

Example / answer: Is the argument convincing?

Modal negative

French: négation avec modal

Rule / explanation: With modals, add not after the modal.

Example / answer: The state should not ignore poverty.

Modal question

French: question avec modal

Rule / explanation: With modals, put the modal before the subject.

Example / answer: Should governments regulate social media?

D. Typical mistakes by French speakers

Missing auxiliary in questions

French: auxiliaire manquant

Rule / explanation: French speakers often forget do / does / did.

Example / answer: Wrong: Why people vote? Correct: Why do people vote?

Verb before subject in normal sentences

French: inversion incorrecte

Rule / explanation: Do not copy French inversion into ordinary English sentences.

Example / answer: Wrong: In this article explains the journalist... Correct: In this article, the journalist explains...

Double subject

French: double sujet

Rule / explanation: Avoid repeating the subject with a pronoun.

Example / answer: Wrong: The government it failed. Correct: The government failed.

Adjective after noun

French: place de l’adjectif

Rule / explanation: Most adjectives come before the noun in English.

Example / answer: Wrong: a reform important. Correct: an important reform.

Forgetting third-person -s

French: -s de troisième personne

Rule / explanation: In the Present Simple, he/she/it takes -s.

Example / answer: Wrong: The article show. Correct: The article shows.

Using do with be

French: do avec be

Rule / explanation: Do not use do for questions or negatives with be.

Example / answer: Wrong: Does it is fair? Correct: Is it fair?

Object before verb

French: COD mal placé

Rule / explanation: English keeps the object after the verb.

Example / answer: Wrong: The reform inequality increased. Correct: The reform increased inequality.

Too long sentences

French: phrases trop longues

Rule / explanation: French-style long sentences often become unclear in English.

Example / answer: Use shorter sentences with clear subjects and verbs.

F. Exercises

Exercise 1 — Put the words in order

French: exercice

Rule / explanation: Rebuild the sentence: increased / rents / have / sharply / in big cities.

Example / answer: Answer: Rents have increased sharply in big cities.

Exercise 2 — Put the words in order

French: exercice

Rule / explanation: Rebuild the sentence: the article / inequality / discusses / economic.

Example / answer: Answer: The article discusses economic inequality.

Exercise 3 — Make it negative

French: exercice

Rule / explanation: The policy helps poor families.

Example / answer: Answer: The policy does not help poor families.

Exercise 4 — Make it negative

French: exercice

Rule / explanation: The reform reduced poverty.

Example / answer: Answer: The reform did not reduce poverty.

Exercise 5 — Make a question

Rule / explanation: The article criticises social media.

Example / answer: Answer: Does the article criticise social media?

Exercise 6 — Make a question

Rule / explanation: The government ignored the warning signs.

Example / answer: Answer: Did the government ignore the warning signs?

Exercise 7 — Correct the mistake

French: exercice

Rule / explanation: Why the journalist criticises the government?

Example / answer: Answer: Why does the journalist criticise the government?

Exercise 8 — Correct the mistake

French: exercice

Rule / explanation: The problem it affects young people.

Example / answer: Answer: The problem affects young people.

Exercise 9 — Correct the mistake

French: exercice
