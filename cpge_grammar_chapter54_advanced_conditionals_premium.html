The golden rule

Choose the conditional according to time and reality
If funding increases, the reform may succeed.
If funding increased, the reform might succeed.
Had funding increased, the reform might have succeeded.
Open conditionif / unless / provided that / as long as
Remote or unreal conditionif + past / if + past perfect / but for / without
Formal inversionshould / were / had + subject

A. Lesson — Advanced conditional meaning

Condition and result

Rule / explanation: A conditional sentence links a possible, hypothetical or counterfactual condition to its consequence.

Example / answer: If public trust declines further, turnout may fall.

Real vs unreal

Rule / explanation: Real conditionals describe open possibilities; unreal conditionals describe imagined, unlikely or impossible situations.

Example / answer: If funding increases, the reform may succeed. / If funding increased, the reform might succeed.

Time reference

Rule / explanation: The verb form signals whether the condition concerns general truth, the future, the present or the past.

Example / answer: If the government had acted earlier, the crisis would have been less severe.

CPGE relevance

Rule / explanation: Advanced conditionals are essential for hypothesis, policy evaluation, counterfactual analysis and nuanced argumentation.

Example / answer: Had the reform been properly funded, its outcome might have been very different.

B. The conditional system at a glance

Zero conditional

Rule / explanation: Use present + present for general truths, rules and predictable relationships.

Example / answer: If inequality rises, social mobility tends to decline.

First conditional

Rule / explanation: Use present + will/modal for a realistic future possibility.

Example / answer: If Parliament approves the bill, the law will take effect next year.

Second conditional

Rule / explanation: Use past + would/modal for an unlikely or hypothetical present or future situation.

Example / answer: If the policy were more flexible, local authorities could adapt it.

Third conditional

Rule / explanation: Use past perfect + would have + past participle for an unreal past.

Example / answer: If the minister had resigned earlier, the controversy might have ended sooner.

Mixed conditional

Rule / explanation: Combine time references when a past event has a present result or a present state affected a past event.

Example / answer: If the government had invested earlier, public services would be stronger today.

C. Unless, provided that and as long as

Unless

Rule / explanation: Unless means if not and introduces a negative condition.

Example / answer: The crisis will worsen unless the government acts.

Provided that

Rule / explanation: Provided that means only if and stresses a necessary condition.

Example / answer: The reform may succeed provided that sufficient funding is available.

Providing that

Rule / explanation: Providing that has the same meaning as provided that and is slightly less formal.

Example / answer: The agreement can continue providing that both sides respect the rules.

As long as

Rule / explanation: As long as means on condition that.

Example / answer: The policy is acceptable as long as civil liberties are protected.

On condition that

Rule / explanation: This formal expression states an explicit requirement.

Example / answer: The funds will be released on condition that the project remains transparent.

D. Even if, whether or not and no matter

Even if

Rule / explanation: Even if means that the result will remain true regardless of the condition.

Example / answer: Even if the economy recovers, public trust may remain weak.

Whether or not

Rule / explanation: Use whether or not when both possible outcomes lead to the same result.

Example / answer: The reform will affect households whether or not Parliament approves every measure.

No matter what

Rule / explanation: Use no matter what to express an unrestricted condition.

Example / answer: No matter what the government decides, the issue will remain controversial.

Whatever happens

Rule / explanation: Use whatever + clause as a concise concessive condition.

Example / answer: Whatever happens, institutional trust will take time to rebuild.

However difficult

Rule / explanation: Use however + adjective/adverb to mean no matter how.

Example / answer: However difficult the reform may be, delay is not a sustainable option.

E. Supposing, assuming and imagine

Supposing

Rule / explanation: Use supposing to introduce a hypothetical scenario, often in discussion.

Example / answer: Supposing the reform fails, what alternatives remain?

Assuming that

Rule / explanation: Use assuming that when reasoning from a provisional assumption.

Example / answer: Assuming that growth continues, unemployment should decline.

Imagine if

Rule / explanation: Use imagine if to invite consideration of a hypothetical situation.

Example / answer: Imagine if every region had equal access to public services.

What if

Rule / explanation: Use what if to introduce a possible problem, consequence or alternative.

Example / answer: What if the policy creates more inequality than it reduces?

Say

Rule / explanation: In informal analytical discussion, say can introduce an imagined case.

Example / answer: Say the government loses its majority: how would the reform proceed?

F. Otherwise and implied conditions

Otherwise

Rule / explanation: Otherwise means if not and introduces the consequence of failing to meet a condition.

Example / answer: Funding must increase; otherwise, the programme will collapse.

Or else

Rule / explanation: Or else is a stronger and less formal alternative to otherwise.

Example / answer: The rules must be clarified, or else legal disputes will continue.

With support

Rule / explanation: A prepositional phrase can imply an if-clause.

Example / answer: With adequate funding, the reform could succeed.

Without reform

Rule / explanation: Without + noun or gerund can express a negative condition.

Example / answer: Without stronger regulation, the risks will increase.

In that case

Rule / explanation: Use in that case to refer to a previously stated condition.

Example / answer: The coalition may collapse. In that case, an early election would become likely.

G. But for and if it were not for

But for + noun

Rule / explanation: But for means if it were not for or if it had not been for.

Example / answer: But for emergency funding, the programme would have collapsed.

If it were not for

Rule / explanation: Use this for a present condition preventing a different result.

Example / answer: If it were not for local volunteers, the service would be unable to operate.

If it had not been for

Rule / explanation: Use this for a past condition that prevented another past result.

Example / answer: If it had not been for international support, the agreement would have failed.

Were it not for

Rule / explanation: This is the formal inverted version of if it were not for.

Example / answer: Were it not for public pressure, the inquiry might never begin.

Had it not been for

Rule / explanation: This is the formal inverted version referring to the past.

Example / answer: Had it not been for the leaked report, the scandal might have remained hidden.

H. Inverted conditionals

Had + subject + past participle

Rule / explanation: Omit if and invert had for a formal third conditional.

Example / answer: Had the government acted earlier, the damage might have been limited.

Were + subject + to

Rule / explanation: Use were...to for a formal and remote future hypothesis.

Example / answer: Were the policy to fail, public trust would decline further.

Should + subject + base verb

Rule / explanation: Use should-inversion for a formal possible future condition.

Example / answer: Should Parliament reject the bill, the government will revise it.

Were it not for

Rule / explanation: Use this inversion for a present counterfactual condition.

Example / answer: Were it not for regional funding, many services would disappear.

Register

Rule / explanation: Inverted conditionals are formal and effective when used selectively.

Example / answer: Had the evidence been stronger, the author’s conclusion would have been more convincing.

I. Conditionals with modal verbs

May / might

Rule / explanation: Use may or might to express uncertain consequences.

Example / answer: If inflation continues, interest rates may rise.

Can / could

Rule / explanation: Use can or could for ability, possibility or policy options.

Example / answer: If local authorities receive more funding, they can expand services.

Should

Rule / explanation: Use should in the result clause for advice or expected action.

Example / answer: If the evidence remains weak, the author should qualify the conclusion.

Must

Rule / explanation: Use must for a necessary result or obligation.

Example / answer: If the reform is to succeed, the government must improve implementation.

Would have to

Rule / explanation: Use would have to for a hypothetical necessity.

Example / answer: If the system were decentralised, local authorities would have to assume greater responsibility.

J. Alternative conditional patterns

If...then

Rule / explanation: Then may be added to make the logical consequence explicit.

Example / answer: If regulation is weakened, then market concentration may increase.

If anything

Rule / explanation: If anything introduces a correction or stronger alternative.

Example / answer: The reform did not reduce inequality; if anything, it made it worse.

If so / if not

Rule / explanation: Use these substitutes to avoid repeating a full condition.

Example / answer: The economy may recover. If so, confidence could improve. If not, unrest may grow.

If necessary

Rule / explanation: This reduced form means if it is necessary.

Example / answer: The policy should be revised, if necessary.

If possible

Rule / explanation: This reduced form means if it is possible.

Example / answer: Local authorities should be consulted, if possible.

K. Conditionals in CPGE argumentation

Policy scenario

Rule / explanation: Use conditionals to examine the consequences of a proposed decision.

Example / answer: If the government caps prices, shortages may emerge.

Counterfactual history

Rule / explanation: Use third conditionals to evaluate alternative historical outcomes.

Example / answer: Had the authorities reacted sooner, the crisis might have been contained.

Qualified prediction

Rule / explanation: Use modal verbs to avoid presenting forecasts as certain.

Example / answer: If current trends continue, inequality could become a major electoral issue.

Necessary condition

Rule / explanation: Identify what must happen for an outcome to be possible.

Example / answer: The transition can succeed only if workers receive adequate support.

Comparative scenario

Rule / explanation: Use parallel conditionals to compare policy choices.

Example / answer: If taxes rise, public spending can increase; if they fall, services may face further cuts.

L. Conditionals in article commentary

The author assumes that...

Rule / explanation: Use this to identify an unstated condition behind an argument.

Example / answer: The author assumes that growth will automatically improve living standards.

This conclusion holds only if...

Rule / explanation: Use this to expose the conditions required for a claim to remain valid.

Example / answer: This conclusion holds only if the data are representative.

Had the author considered...

Rule / explanation: Use an inverted conditional for sophisticated criticism.

Example / answer: Had the author considered regional differences, the analysis would have been more nuanced.

If the figures are accurate...

Rule / explanation: Use an open conditional when the evidence has not been fully verified.

Example / answer: If the figures are accurate, the policy has produced substantial gains.

Even if the claim is accepted...

Rule / explanation: Use even if to show that a conclusion does not necessarily follow.

Example / answer: Even if the claim is accepted, it does not justify the proposed solution.

M. Typical mistakes by French speakers

Will in the if-clause

Rule / explanation: Do not normally use will in an ordinary future condition.

Example / answer: Wrong: If the government will act, confidence will return. Correct: If the government acts, confidence will return.

Would in both clauses

Rule / explanation: Do not normally use would in both parts of a second conditional.

Example / answer: Wrong: If the policy would change, voters would approve. Correct: If the policy changed, voters would approve.

Wrong past perfect

Rule / explanation: A third conditional requires had + past participle in the if-clause.

Example / answer: Wrong: If the minister would have resigned... Correct: If the minister had resigned...

Unless + negative

Rule / explanation: Unless already means if not.

Example / answer: Wrong: Unless the government does not act. Correct: Unless the government acts.

Confusing if and whether

Rule / explanation: Use whether, not if, after a preposition and before to-infinitives.

Example / answer: It depends on whether funding increases. / The question is whether to intervene.

Were agreement

Rule / explanation: Formal unreal conditionals traditionally use were with all subjects.

Example / answer: If I were / If the policy were more effective...

N. Exercises

1. Complete the first conditional

Rule / explanation: If Parliament ___ the bill, it will take effect next year. (approve)

Example / answer: If Parliament approves the bill, it will take effect next year.

2. Complete the second conditional

Rule / explanation: If the policy ___ more flexible, local authorities could adapt it. (be)

Example / answer: If the policy were more flexible, local authorities could adapt it.

3. Complete the third conditional

Rule / explanation: If the government ___ earlier, the damage might have been limited. (act)

Example / answer: If the government had acted earlier, the damage might have been limited.

4. Rewrite with unless

Rule / explanation: The crisis will worsen if the government does not act.

Example / answer: The crisis will worsen unless the government acts.

5. Rewrite with otherwise

Rule / explanation: Funding must increase. If it does not, the programme will collapse.

Example / answer: Funding must increase; otherwise, the programme will collapse.

6. Rewrite with inversion

Rule / explanation: If Parliament should reject the bill, the government will revise it.

Example / answer: Should Parliament reject the bill, the government will revise it.

7. Translate

Rule / explanation: Si la réforme avait été correctement financée, son résultat aurait pu être très différent.

Example / answer: Had the reform been properly funded, its outcome might have been very different.

8. Mini concours challenge

Rule / explanation: Write a paragraph using unless, even if, provided that, otherwise and one inverted conditional.

Example / answer: Example: The reform will fail unless implementation improves. Even if additional funds are approved, results will take time. The programme may survive provided that local authorities receive greater autonomy; otherwise, regional resistance will intensify. Had the government consulted them earlier, much of this opposition might have been avoided.
