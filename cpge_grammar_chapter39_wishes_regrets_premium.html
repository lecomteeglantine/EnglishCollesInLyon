Chapter 39 — Wishes and Regrets
B2 → C1 grammar for hypothetical thinking, criticism, regret and sophisticated argumentation.

Chapter objective

This premium grammar chapter helps CPGE students master wish, if only, would rather and regret structures. It covers present wishes, past regrets, irritation with would, formal critical language and common French-speaker mistakes.

Level B2 → C1WishIf OnlyRegretWould RatherExercises + Answers

The golden rule

Wish moves one step back in time
I wish the system were fairer.
I wish the government had acted earlier.
Main useUse wish structures to express regret, dissatisfaction, criticism or an unreal alternative.
CPGE habitUse them for nuanced critique: The author wishes policymakers had taken the issue more seriously.
Key trapFor past regret, use wish + Past Perfect: I wish they had acted, not I wish they acted.

A. Lesson — What do wish structures express?

Present dissatisfaction

French: insatisfaction présente

Rule / explanation: Use wish + Past Simple to say the present situation is not what you want.

Example / answer: Many citizens wish the system were fairer.

Past regret

French: regret passé

Rule / explanation: Use wish + Past Perfect to regret something that happened or did not happen.

Example / answer: The government probably wishes it had reacted sooner.

Irritation or desire for change

French: irritation / souhait de changement

Rule / explanation: Use wish + would to complain about repeated behaviour or hope for change.

Example / answer: Voters wish politicians would stop making empty promises.

Analytical value

French: valeur analytique

Rule / explanation: These structures help express counterfactual judgement elegantly.

Example / answer: The article suggests that Britain wishes it had prepared better for the crisis.

B. Wish + Past Simple: present wishes

Structure

French: structure

Rule / explanation: Use wish + Past Simple for an unreal present situation.

Example / answer: I wish the article gave more evidence.

Be → were

French: were hypothétique

Rule / explanation: In formal English, use were for all subjects after wish.

Example / answer: The author wishes the debate were less polarised.

Present political criticism

French: critique présente

Rule / explanation: Useful for criticising current systems or institutions.

Example / answer: Many voters wish Parliament worked more transparently.

Not the same as hope

French: wish vs hope

Rule / explanation: Wish describes an unreal situation; hope describes a possible future.

Example / answer: I hope the reform succeeds. / I wish the reform were more ambitious.

C. Wish + Past Perfect: past regrets

Structure

French: structure

Rule / explanation: Use wish + had + past participle for regret about the past.

Example / answer: The minister wishes she had listened to experts.

Missed opportunity

French: occasion manquée

Rule / explanation: Useful for saying a better decision was possible in the past.

Example / answer: The country wishes it had invested earlier in housing.

Political accountability

French: responsabilité politique

Rule / explanation: It allows elegant criticism without sounding too blunt.

Example / answer: Policymakers may wish they had taken warnings more seriously.

Article evaluation

French: évaluation d’article

Rule / explanation: Use it to criticise missing evidence, nuance or context.

Example / answer: One may wish the journalist had included more historical context.

D. Wish + would: irritation and desired change

Repeated behaviour

French: comportement répété

Rule / explanation: Use wish + would to complain about behaviour that someone could change.

Example / answer: Citizens wish politicians would keep their promises.

Future change

French: changement souhaité

Rule / explanation: It can express a strong wish for a future change.

Example / answer: Many campaigners wish the government would act faster.

Not for states

French: pas pour les états

Rule / explanation: Do not use would for states you cannot control directly.

Example / answer: Better: I wish the system were fairer. Not: I wish the system would be fairer.

Same subject warning

French: sujet identique

Rule / explanation: Avoid I wish I would for personal wishes; use could or Past Simple instead.

Example / answer: Better: I wish I could speak more fluently.

E. If only: stronger emotion

If only + Past Simple

French: si seulement + présent irréel

Rule / explanation: Use it for a stronger present wish.

Example / answer: If only the debate were less polarised.

If only + Past Perfect

French: si seulement + regret passé

Rule / explanation: Use it for stronger past regret.

Example / answer: If only the government had acted earlier.

If only + would

French: si seulement + changement souhaité

Rule / explanation: Use it for frustration with behaviour.

Example / answer: If only leaders would listen to younger voters.

Register warning

French: registre

Rule / explanation: If only is emotional; use it sparingly in formal essays.

Example / answer: Formal alternative: One may regret that the government acted too late.

F. Would rather and would prefer

Would rather + base verb

French: préférer faire

Rule / explanation: Use would rather + base verb when the subject chooses its own action.

Example / answer: Many voters would rather support a credible reform.

Would rather + subject + Past Simple

French: préférer que quelqu’un fasse

Rule / explanation: Use this when you want someone else to do something now or in the future.

Example / answer: Citizens would rather the government listened to them.

Would rather + subject + Past Perfect

French: préférer que quelqu’un ait fait

Rule / explanation: Use this to regret someone else’s past action.

Example / answer: Campaigners would rather the minister had consulted them earlier.

Would prefer

French: préférer

Rule / explanation: Would prefer is more neutral and often followed by to + verb.

Example / answer: The author would prefer to focus on structural causes.

G. CPGE argumentation patterns

One may wish...

French: on pourrait souhaiter...

Rule / explanation: Formal way to express criticism in essays.

Example / answer: One may wish the author had provided more data.

It is regrettable that...

French: il est regrettable que...

Rule / explanation: Formal alternative to wish for written analysis.

Example / answer: It is regrettable that the article ignores regional inequalities.

The author seems to regret...

French: l’auteur semble regretter...

Rule / explanation: Useful in article summaries.

Example / answer: The author seems to regret that policymakers acted too late.

In hindsight...

French: avec le recul...

Rule / explanation: Useful for discussing past mistakes with present knowledge.

Example / answer: In hindsight, the government should have anticipated the backlash.

H. Typical mistakes by French speakers

Wish + present

French: présent incorrect

Rule / explanation: Use Past Simple for present wishes.

Example / answer: Wrong: I wish the system is fairer. Correct: I wish the system were fairer.

Past regret without had

French: oubli du past perfect

Rule / explanation: Use Past Perfect for past regrets.

Example / answer: Wrong: I wish they acted sooner. Correct: I wish they had acted sooner.

I wish I would

French: même sujet

Rule / explanation: Avoid I wish I would for personal ability or situation.

Example / answer: Better: I wish I could analyse graphs more confidently.

Hope vs wish

French: espoir vs irréel

Rule / explanation: Use hope for possible future events, wish for unreal or regretted situations.

Example / answer: I hope the reform works. / I wish the reform were better designed.
