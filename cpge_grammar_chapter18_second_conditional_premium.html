The golden rule

If + Past Simple, would + base verb
If the government invested more, inequality would fall.
If the media were more reliable, citizens would trust them more.
Main useUse the Second Conditional for imaginary, unlikely or hypothetical present/future situations and their results.
CPGE habitUse it to discuss alternative solutions and thought experiments: If voting were compulsory, turnout might increase.
Key trapDo not use would after if: If the government acted, not If the government would act.

A. Lesson — What is the Second Conditional?

Second Conditional

French: deuxième conditionnel

Rule / explanation: Describes imaginary, unlikely or hypothetical present/future situations and their possible results.

Example / answer: If the government invested more in education, social mobility would improve.

Past form, present/future meaning

French: prétérit hypothétique

Rule / explanation: The verb after if is in the Past Simple, but the meaning is not past.

Example / answer: If voters trusted institutions, turnout would be higher.

Hypothetical result

French: résultat hypothétique

Rule / explanation: The result clause usually uses would + base verb.

Example / answer: If rents fell, young people would save more money.

CPGE use

French: usage CPGE

Rule / explanation: Useful for alternative policies, hypothetical reforms and debate questions.

Example / answer: If governments tackled inequality seriously, populism might lose ground.

B. Form and structure

If + Past Simple, would + base verb

French: forme principale

Rule / explanation: This is the most common Second Conditional pattern.

Example / answer: If the government raised taxes, it would face opposition.

Reversed order

French: ordre inversé

Rule / explanation: You can put the result clause first. No comma is needed then.

Example / answer: The government would face opposition if it raised taxes.

With could

French: avec could

Rule / explanation: Use could for possible ability, opportunity or consequence.

Example / answer: If schools received more funding, they could reduce inequality.

With might

French: avec might

Rule / explanation: Use might for a less certain hypothetical result.

Example / answer: If voting were compulsory, turnout might increase.

C. If I were and formal correctness

If I were

French: si j’étais

Rule / explanation: In formal English, use were for hypothetical situations.

Example / answer: If I were responsible for the policy, I would revise it.

If the system were fairer

French: si le système était plus juste

Rule / explanation: Were is preferred with all subjects in formal hypothetical clauses.

Example / answer: If the system were fairer, fewer people would feel excluded.

If it were not for

French: sans

Rule / explanation: Formal phrase meaning without.

Example / answer: If it were not for public pressure, the reform would not change.

As if / as though

French: comme si

Rule / explanation: Use Past Simple or were for unreal comparison.

Example / answer: The article speaks as if inequality were inevitable.

D. Second Conditional in essays and khôlles

Policy scenario

French: scénario politique

Rule / explanation: Use it to imagine alternative policies.

Example / answer: If the state invested more in housing, rents would become more affordable.

Criticism

French: critique

Rule / explanation: Use it to imply that current policy is insufficient.

Example / answer: If the government listened to experts, it would change course.

Economic reasoning

French: raisonnement économique

Rule / explanation: Use it for hypothetical economic consequences.

Example / answer: If wages rose faster than prices, purchasing power would improve.

Environmental policy

French: politique environnementale

Rule / explanation: Use it for ecological scenarios.

Example / answer: If governments banned fossil fuel subsidies, emissions might fall.

E. First vs Second Conditional

First Conditional

French: condition réelle

Rule / explanation: Use it for realistic future situations.

Example / answer: If the bill passes, it will change the law.

Second Conditional

French: condition hypothétique

Rule / explanation: Use it for imaginary or less likely situations.

Example / answer: If the bill passed, it would change the law.

Will vs would

French: will vs would

Rule / explanation: Will = realistic future result. Would = hypothetical result.

Example / answer: If it happens, it will matter. / If it happened, it would matter.

Can vs could

French: can vs could

Rule / explanation: Can = real possibility. Could = hypothetical possibility.

Example / answer: If citizens organised, they could influence policy.

F. Typical mistakes by French speakers

Would after if

French: would après if

Rule / explanation: Do not use would in the if-clause.

Example / answer: Wrong: If the government would act. Correct: If the government acted.

Present after if

French: présent après if

Rule / explanation: Use Past Simple after if for hypothetical meaning.

Example / answer: Wrong: If voters trust institutions, turnout would rise. Correct: If voters trusted institutions, turnout would rise.

Will in hypothetical results

French: will incorrect

Rule / explanation: Use would, not will, for imaginary situations.

Example / answer: Wrong: If I were Prime Minister, I will invest in schools. Correct: I would invest.

Second vs Third Conditional

French: 2e vs 3e conditionnel

Rule / explanation: Second imagines present/future; Third imagines past.

Example / answer: If the reform worked, it would help now. / If the reform had worked, it would have helped then.

G. Exercises

Exercise 1 — Complete

French: exercice

Rule / explanation: If the government ___ (invest) more in education, social mobility would improve.

Example / answer: Answer: invested

Exercise 2 — Complete

French: exercice

Rule / explanation: If rents fell, young people ___ (have) more disposable income.

Example / answer: Answer: would have

Exercise 3 — Correct

French: exercice

Rule / explanation: If the state would regulate platforms, public debate would improve.

Example / answer: Answer: If the state regulated platforms, public debate would improve.

Exercise 4 — Translate

French: exercice

Rule / explanation: Si j’étais Premier ministre, j’investirais dans l’éducation.

Example / answer: Answer: If I were Prime Minister, I would invest in education.

Exercise 5 — Translate

French: exercice

Rule / explanation: Si cette réforme fonctionnait, elle pourrait réduire les inégalités.

Example / answer: Answer: If this reform worked, it could reduce inequality.

Exercise 6 — Mini concours challenge

French: exercice

Rule / explanation: Write 5 Second Conditional sentences about democracy, media, climate, housing and education.

Example / answer: If citizens trusted institutions, turnout would be higher. If platforms were regulated, misinformation might spread less easily.
