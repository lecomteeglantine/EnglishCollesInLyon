# Ireland Civilisation Atlas

Upload `colle-atlas-ie.html` to the root of the `EnglishCollesInLyon` GitHub repository and replace the existing file with the same name.

## Included
- Back-to-basics opening with ten essential foundations
- 31-event expandable and filterable timeline
- Complete Republic of Ireland political and judicial system sections
- Interactive power map, legislative stepper and court-route explorer
- 15 historically significant leaders, organisers and peacemakers
- 20 entries on partition, the Troubles, the peace process and post-Brexit arrangements
- Island, counties, cities, languages, diaspora and cultural identity
- Society and contemporary debates, updated to 21 July 2026
- Article, video/source and freely reusable Wikimedia Commons image for every knowledge entry
- Search, essentials-only mode, reader mode, dark mode and accessibility controls
- Local revision list with responsive HTML export and import

## Important
External articles, source pages and Wikimedia Commons images require an internet connection. The explanatory text and revision pack structure work offline.
