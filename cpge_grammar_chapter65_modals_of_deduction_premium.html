Chapter 65 — Modals of Deduction
B2 → C1 grammar for certainty, possibility, evidence and nuanced academic interpretation.

Chapter objective

This premium chapter develops advanced control of modals of deduction: present and past inference, certainty, negative deduction, possibility, modal perfects, passive forms and evidence-based use in essays, article analysis and khôlles.

The golden rule

Present deduction: modal + base verb
Past deduction: modal + have + past participle
She must know. / She must have known.
Strong positive deductionmust
Strong negative deductioncan’t / cannot
Possibilitymay / might / could

A. Core principles

Deduction

Rule / explanation: A deduction is a logical conclusion based on evidence rather than direct knowledge.

Example / answer: The lights are on, so someone must be inside.

Certainty scale

Rule / explanation: Different modals express different levels of confidence.

Example / answer: The figures may be inaccurate. / The figures must be inaccurate.

Present vs past

Rule / explanation: Use a base verb for present deduction and have + past participle for past deduction.

Example / answer: She must know. / She must have known.

Academic value

Rule / explanation: These modals help qualify interpretations and avoid unsupported certainty.

Example / answer: The author may be underestimating the social impact of the reform.

B. Must: strong positive deduction

Strong inference

Rule / explanation: Use must when the evidence makes a conclusion highly likely.

Example / answer: The scandal must have had a major impact.

Present state

Rule / explanation: Use must + base verb for a current state.

Example / answer: The government must realise that the timetable is unrealistic.

Ongoing action

Rule / explanation: Use must be + verb-ing for an action in progress.

Example / answer: Ministers must be discussing the report now.

Not absolute proof

Rule / explanation: Must expresses a strong inference, not verified fact.

Example / answer: The committee must be considering further action.

CPGE use

Rule / explanation: Use must when textual evidence strongly supports an interpretation.

Example / answer: The author must regard inequality as a structural problem.

C. Can’t / cannot: strong negative deduction

Impossibility

Rule / explanation: Use can’t or cannot when evidence makes something highly unlikely or impossible.

Example / answer: The figures can’t be accurate.

Present state

Rule / explanation: Use can’t + base verb for a present negative deduction.

Example / answer: The minister can’t know the full consequences yet.

Ongoing action

Rule / explanation: Use can’t be + verb-ing for an impossible current action.

Example / answer: They can’t be using the final dataset.

Past impossibility

Rule / explanation: Use can’t have + past participle.

Example / answer: The government can’t have anticipated such a rapid backlash.

Formal register

Rule / explanation: Cannot is more formal than can’t.

Example / answer: The result cannot be explained by demographics alone.

D. May, might and could: possibility

Possible conclusion

Rule / explanation: Use may, might or could when a conclusion is possible but uncertain.

Example / answer: The decline may reflect economic insecurity.

Tentative nuance

Rule / explanation: Might often sounds slightly more tentative than may.

Example / answer: The reform might create regional disparities.

Ongoing possibility

Rule / explanation: Use may/might/could be + verb-ing.

Example / answer: The government may be preparing a revised proposal.

Past possibility

Rule / explanation: Use may/might/could have + past participle.

Example / answer: Officials may have underestimated opposition.

Formal caution

Rule / explanation: These forms are ideal when evidence is suggestive but incomplete.

Example / answer: The article could be overstating the scale of the crisis.

E. Must not vs can’t

Must not

Rule / explanation: Must not normally expresses prohibition.

Example / answer: Officials must not disclose confidential information.

Can’t

Rule / explanation: Can’t expresses negative deduction.

Example / answer: The figures can’t be correct.

Past form

Rule / explanation: Use can’t have, not mustn’t have, for past impossibility.

Example / answer: She can’t have approved the final version.

Learner safety

Rule / explanation: Prefer can’t/cannot for clear negative deduction.

Example / answer: The policy cannot have produced that result in two weeks.

Meaning check

Rule / explanation: Distinguish obligation from deduction through context.

Example / answer: You must leave = obligation. She must be tired = deduction.

F. Present deduction patterns

Modal + base verb

Rule / explanation: Use a modal followed by the base form.

Example / answer: The author may believe that regulation is ineffective.

Modal + be

Rule / explanation: Use modal + be before an adjective or noun.

Example / answer: The evidence must be incomplete.

There + modal + be

Rule / explanation: Use this for inferred existence.

Example / answer: There must be another explanation.

Modal + have

Rule / explanation: Use modal + have when have is the main verb.

Example / answer: The policy may have serious consequences.

State verbs

Rule / explanation: State verbs commonly follow deduction modals directly.

Example / answer: The minister must understand the political risk.

G. Past deduction: modal perfects

Must have

Rule / explanation: Strong positive deduction about a completed past event.

Example / answer: Officials must have ignored the warning.

May have

Rule / explanation: Possible past event.

Example / answer: The reform may have reduced costs in some regions.

Might have

Rule / explanation: More tentative past possibility.

Example / answer: The author might have overlooked a variable.

Could have

Rule / explanation: Possible explanation or unrealised possibility, depending on context.

Example / answer: The delay could have resulted from legal uncertainty.

Can’t have

Rule / explanation: Strong negative deduction about the past.

Example / answer: The minister can’t have read the report in ten minutes.

H. Ongoing and passive deductions

Must have been doing

Rule / explanation: Strong inference about an ongoing past action.

Example / answer: The government must have been negotiating behind the scenes.

May have been doing

Rule / explanation: Possible ongoing past action.

Example / answer: Officials may have been preparing a compromise.

Must have been done

Rule / explanation: Strong inference about a completed passive event.

Example / answer: The data must have been manipulated.

Could have been done

Rule / explanation: Possible passive explanation.

Example / answer: The discrepancy could have been caused by a technical error.

Can’t have been done

Rule / explanation: Past passive impossibility.

Example / answer: The report can’t have been approved before the review ended.

I. Should and ought to: expectation

Expected situation

Rule / explanation: Use should or ought to for a reasonable expectation.

Example / answer: The new measures should reduce waiting times.

Less certain than must

Rule / explanation: Should expresses expectation, not strong deduction.

Example / answer: The report should be available by Friday.

Negative expectation

Rule / explanation: Use shouldn’t for an expected negative.

Example / answer: The delay shouldn’t affect the outcome.

Past expectation

Rule / explanation: Use should have + past participle.

Example / answer: The committee should have received the data by now.

Possible criticism

Rule / explanation: Should have can also criticise a missed action.

Example / answer: Ministers should have consulted the regions.

J. Could have: two meanings

Past possibility

Rule / explanation: Something possibly happened.

Example / answer: The decline could have resulted from lower turnout.

Missed opportunity

Rule / explanation: Something was possible but did not happen.

Example / answer: The government could have consulted local authorities.

Conditional clue

Rule / explanation: Context clarifies unrealised possibility.

Example / answer: The reform could have succeeded if it had been funded.

Criticism

Rule / explanation: Could have often criticises a missed action.

Example / answer: Ministers could have acted sooner.

Policy analysis

Rule / explanation: Useful for discussing alternative choices.

Example / answer: Earlier intervention could have limited the crisis.

K. Evidence markers

Apparently

Rule / explanation: Signals inference or second-hand appearance.

Example / answer: Apparently, the committee is divided.

Seem / appear

Rule / explanation: Express inference without a modal.

Example / answer: The policy appears to have failed.

Judging by

Rule / explanation: Introduces the evidence behind a conclusion.

Example / answer: Judging by the turnout, interest must have been high.

Given

Rule / explanation: Introduces relevant circumstances.

Example / answer: Given the lack of funding, the programme may struggle.

On the evidence available

Rule / explanation: Formally limits the scope of a conclusion.

Example / answer: On the evidence available, the decline cannot be attributed to one factor.

L. Certainty scale

Near certainty

Rule / explanation: Must for positive deduction; can’t/cannot for negative deduction.

Example / answer: The figures must be incomplete. / They cannot be final.

Moderate possibility

Rule / explanation: May or could for a plausible conclusion.

Example / answer: The trend may reflect changing priorities.

Weak possibility

Rule / explanation: Might for cautious or remote possibility.

Example / answer: The reform might produce benefits later.

No fixed percentage

Rule / explanation: Modals do not equal exact mathematical probabilities.

Example / answer: Interpret certainty through context.

Academic calibration

Rule / explanation: Match modal strength to evidence strength.

Example / answer: Do not write must when the source merely suggests.

M. Article summaries and source analysis

Cautious interpretation

Rule / explanation: Use may, might and could when proof is incomplete.

Example / answer: The figures may indicate a widening regional divide.

Strong inference

Rule / explanation: Use must only when evidence is highly compelling.

Example / answer: The author must see the situation as critical.

Reject an explanation

Rule / explanation: Use cannot when evidence contradicts it.

Example / answer: The fall cannot be explained by demographics alone.

Earlier causes

Rule / explanation: Use modal perfects for possible past causes.

Example / answer: Officials may have underestimated public anger.

Source ownership

Rule / explanation: Clarify whether the deduction is the author’s or yours.

Example / answer: The journalist suggests that ministers may have ignored warnings.

N. CPGE oral commentary and essays

Interpret clues

Rule / explanation: Link every deduction to evidence.

Example / answer: The repeated references to urgency suggest that the author must see the situation as critical.

Offer alternatives

Rule / explanation: Use modals to compare explanations.

Example / answer: The decline may reflect anxiety, but it could also result from political fatigue.

Reject weak claims

Rule / explanation: Use cannot when an interpretation is contradicted.

Example / answer: This cannot simply be described as temporary.

Qualified thesis

Rule / explanation: Use modals to avoid false certainty.

Example / answer: Deregulation may increase competition but can weaken protection.

Balanced conclusion

Rule / explanation: Combine deduction with concession.

Example / answer: While the policy may reduce costs, it cannot solve structural inequality.

O. Common mistakes

Mustn’t for impossibility

Rule / explanation: Use can’t/cannot, not mustn’t, for negative deduction.

Example / answer: Wrong: The figures mustn’t be correct.

Missing have

Rule / explanation: Use modal + have + past participle for past deduction.

Example / answer: Wrong: She must knew. Correct: She must have known.

Past tense after modal

Rule / explanation: After a modal, use the base form.

Example / answer: Wrong: may went. Correct: may have gone.

Could of

Rule / explanation: Write could have, never could of.

Example / answer: The reduced pronunciation does not change the spelling.

Overusing must

Rule / explanation: Use may or might when uncertainty remains.

Example / answer: Modal strength must match the evidence.

P. Exercises

1. Strong present deduction

Rule / explanation: The lights are on. Someone ___ be inside.

Example / answer: Someone must be inside.

2. Strong negative deduction

Rule / explanation: The total exceeds the population. The figures ___ be correct.

Example / answer: The figures can’t / cannot be correct.

3. Past possibility

Rule / explanation: Perhaps officials underestimated the backlash.

Example / answer: Officials may / might / could have underestimated the backlash.

4. Past certainty

Rule / explanation: All evidence points to concealment.

Example / answer: Officials must have concealed the evidence.

5. Correct the error

Rule / explanation: The minister mustn’t have read the report; it was unavailable.

Example / answer: The minister can’t have read the report; it was unavailable.

6. Ongoing possibility

Rule / explanation: Perhaps the government is preparing a compromise.

Example / answer: The government may / might be preparing a compromise.

7. Translate

Rule / explanation: L’auteur a peut-être sous-estimé les conséquences sociales.

Example / answer: The author may / might have underestimated the social consequences.

8. Mini challenge

Rule / explanation: Use must, cannot, may have, might be and could have.

Example / answer: The fall in turnout must reflect deeper distrust. It cannot be explained by weather alone. Parties may have underestimated frustration, while younger voters might be disengaging. Earlier consultation could have limited the damage.
