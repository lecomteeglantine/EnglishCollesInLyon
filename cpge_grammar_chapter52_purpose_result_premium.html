A. Lesson — Expressing purpose and result

Purpose

Rule / explanation: Purpose explains why an action is performed or what objective it is intended to achieve.

Example / answer: The government introduced subsidies to encourage investment.

Result

Rule / explanation: Result explains the consequence produced by an action, quality or situation.

Example / answer: The subsidies were so generous that investment increased rapidly.

Purpose vs cause

Rule / explanation: Purpose refers to an intended objective; cause refers to an existing reason.

Example / answer: The policy was introduced to reduce emissions. / The policy failed because funding was insufficient.

CPGE relevance

Rule / explanation: Purpose and result structures help students explain policy objectives, assess outcomes and analyse argumentation.

Example / answer: The reform was designed to improve access, but it ultimately widened regional inequality.

B. To, in order to and so as to

To + infinitive

Rule / explanation: Use to + base verb for the most common and concise expression of purpose.

Example / answer: The government raised taxes to fund public services.

In order to

Rule / explanation: Use in order to when the purpose needs extra emphasis or clarity.

Example / answer: The rules were simplified in order to improve public understanding.

So as to

Rule / explanation: Use so as to in formal writing; it is less common in everyday speech.

Example / answer: The policy was revised so as to reduce administrative delays.

Negative purpose

Rule / explanation: Use in order not to or so as not to for a negative objective.

Example / answer: The minister delayed the announcement in order not to alarm investors.

Same subject

Rule / explanation: These infinitive structures normally require the same subject for both actions.

Example / answer: The government changed the law to protect consumers.

C. So that and in order that

So that + clause

Rule / explanation: Use so that when the purpose clause has its own subject.

Example / answer: The government increased funding so that schools could hire more staff.

Modal verbs

Rule / explanation: Purpose clauses often use can, could, will, would, may or might.

Example / answer: The rules were translated so that all citizens could understand them.

In order that

Rule / explanation: In order that is formal and less common than so that.

Example / answer: The policy was revised in order that local authorities might retain greater control.

Present and future purpose

Rule / explanation: Use can, will or may when the purpose relates to the present or future.

Example / answer: The platform stores the data securely so that users can access it later.

Past purpose

Rule / explanation: Use could, would or might for a purpose in a past context.

Example / answer: The law was amended so that smaller firms could compete.

D. For, for the purpose of and with a view to

For + noun

Rule / explanation: Use for before a noun to express intended use, not normally before a verb.

Example / answer: This funding is for regional development.

For + gerund

Rule / explanation: Use for + gerund to describe the function of an object or system.

Example / answer: This tool is used for measuring air quality.

For the purpose of

Rule / explanation: Use this formal phrase before a noun or gerund.

Example / answer: The data were collected for the purpose of assessing public opinion.

With a view to

Rule / explanation: Use with a view to + gerund for a planned future objective.

Example / answer: The law was revised with a view to reducing inequality.

With the aim of

Rule / explanation: Use with the aim of + gerund for a clearly stated objective.

Example / answer: The programme was launched with the aim of improving digital access.

E. So...that and such...that

So + adjective + that

Rule / explanation: Use so...that to show that a quality reaches a degree producing a result.

Example / answer: The crisis was so severe that Parliament intervened.

So + adverb + that

Rule / explanation: Use so before an adverb when manner produces the result.

Example / answer: Prices rose so rapidly that many households could not adapt.

Such + noun phrase + that

Rule / explanation: Use such...that with a noun phrase.

Example / answer: It was such a divisive reform that even supporters withdrew.

Such + adjective + plural noun

Rule / explanation: Use such before adjective + plural noun.

Example / answer: The policy created such serious problems that it had to be revised.

Word order

Rule / explanation: Use so before adjectives and adverbs; use such before noun phrases.

Example / answer: so expensive that / such an expensive policy that

F. Too...to and enough to

Too + adjective + to

Rule / explanation: Use too...to when the degree prevents the action.

Example / answer: The proposal was too costly to implement.

Adjective + enough to

Rule / explanation: Use enough after an adjective when the degree makes the action possible.

Example / answer: The evidence was strong enough to justify an inquiry.

Enough + noun + to

Rule / explanation: Use enough before a noun.

Example / answer: The government lacked enough support to pass the bill.

Too much / many to

Rule / explanation: Use too much or too many when quantity prevents an outcome.

Example / answer: There were too many objections to ignore.

For + person + to

Rule / explanation: Add for + person when the subject of the infinitive differs.

Example / answer: The regulations were too complex for small firms to follow.

G. Therefore, consequently and as a result

Therefore

Rule / explanation: Use therefore to introduce a formal logical result.

Example / answer: Funding was reduced; therefore, waiting times increased.

Consequently

Rule / explanation: Consequently presents a direct consequence in formal writing.

Example / answer: The policy lost public support. Consequently, it was withdrawn.

As a result

Rule / explanation: Use as a result to introduce an outcome in a clear, neutral style.

Example / answer: Energy prices rose sharply. As a result, production costs increased.

Thus

Rule / explanation: Thus is concise and common in analytical prose.

Example / answer: The reform centralised decision-making, thus reducing local autonomy.

Hence

Rule / explanation: Hence can introduce a concise consequence, often followed by a noun phrase.

Example / answer: The evidence was incomplete, hence the need for further investigation.

H. Result verbs and result phrases

Result in

Rule / explanation: Use result in + noun or gerund to introduce the consequence.

Example / answer: The cuts resulted in longer waiting times.

Lead to

Rule / explanation: Use lead to + noun or gerund for a developing consequence.

Example / answer: Poor regulation led to increased financial risk.

Give rise to

Rule / explanation: Use give rise to for the emergence of a new situation or problem.

Example / answer: The reform gave rise to widespread legal uncertainty.

Bring about

Rule / explanation: Use bring about for a deliberate or significant change.

Example / answer: The movement helped bring about major constitutional reform.

Have the effect of

Rule / explanation: Use have the effect of + gerund to describe an indirect outcome.

Example / answer: The new rule had the effect of discouraging small businesses.

I. Intended and unintended results

Intended outcome

Rule / explanation: Use was intended to, was designed to or aimed to for planned results.

Example / answer: The reform was intended to improve access to healthcare.

Actual outcome

Rule / explanation: Use actually, ultimately or in practice to contrast real results with intentions.

Example / answer: In practice, the reform increased regional inequality.

Unintended consequence

Rule / explanation: Use unintended consequence for an unforeseen result.

Example / answer: One unintended consequence was a rise in administrative costs.

Backfire

Rule / explanation: Use backfire when an action produces the opposite of its intended result.

Example / answer: The campaign backfired and strengthened the opposition.

Fail to achieve

Rule / explanation: Use fail to achieve when the intended result is not reached.

Example / answer: The measure failed to achieve its stated objective.

J. Purpose and result in article commentary

The policy aims to...

Rule / explanation: Use this to report a declared objective.

Example / answer: The policy aims to reduce emissions by 40 per cent.

The measure is designed to...

Rule / explanation: Use this to describe an intended function.

Example / answer: The measure is designed to protect vulnerable households.

The initiative resulted in...

Rule / explanation: Use this to report an observed consequence.

Example / answer: The initiative resulted in a substantial increase in participation.

The author seeks to...

Rule / explanation: Use this to analyse the writer’s rhetorical purpose.

Example / answer: The author seeks to challenge the government’s optimistic narrative.

The article is intended to...

Rule / explanation: Use this to discuss audience effect or communicative purpose.

Example / answer: The article is intended to alert readers to the risks of deregulation.

K. Purpose and result in CPGE essays

Policy objective

Rule / explanation: State what a policy is supposed to achieve.

Example / answer: The reform was introduced to increase efficiency and restore trust.

Assessment of effectiveness

Rule / explanation: Compare the intended purpose with the actual result.

Example / answer: Although the policy was designed to reduce costs, it ultimately increased them.

Causal chain

Rule / explanation: Link actions to immediate and long-term outcomes.

Example / answer: The cuts reduced staffing levels, resulting in longer delays and lower public confidence.

Rhetorical purpose

Rule / explanation: Analyse why an author uses a particular example or statistic.

Example / answer: The journalist cites these figures in order to expose the gap between official claims and lived experience.

Conclusion

Rule / explanation: Evaluate whether the stated purpose was achieved.

Example / answer: The reform may have met its short-term objective, but it failed to produce lasting structural change.

L. Typical mistakes by French speakers

For to + verb

Rule / explanation: Do not use for to to express purpose.

Example / answer: Wrong: The government acted for to reduce costs. Correct: The government acted to reduce costs.

So that + infinitive

Rule / explanation: So that must be followed by a clause, not an infinitive.

Example / answer: Wrong: so that reduce costs. Correct: so that it could reduce costs.

In order to with different subjects

Rule / explanation: Use so that when the purpose clause has a different subject.

Example / answer: Better: The government increased funding so that schools could recruit staff.

So / such confusion

Rule / explanation: Use so before adjectives and adverbs; use such before noun phrases.

Example / answer: so difficult that / such a difficult problem that

Enough word order

Rule / explanation: Enough follows adjectives but precedes nouns.

Example / answer: strong enough / enough evidence

Result in / result from

Rule / explanation: Result in introduces the outcome; result from introduces the cause.

Example / answer: The cuts resulted in delays. The delays resulted from the cuts.

M. Exercises

1. Express purpose with to

Rule / explanation: The government increased funding. It wanted to improve public services.

Example / answer: The government increased funding to improve public services.

2. Use so that

Rule / explanation: The rules were simplified. Citizens could understand them.

Example / answer: The rules were simplified so that citizens could understand them.

3. Correct the sentence

Rule / explanation: The law was changed for to protect consumers.

Example / answer: The law was changed to protect consumers.

4. Choose so or such

Rule / explanation: It was ___ a controversial decision that Parliament intervened.

Example / answer: It was such a controversial decision that Parliament intervened.

5. Rewrite with too...to

Rule / explanation: The proposal was so expensive that it could not be implemented.

Example / answer: The proposal was too expensive to implement.

6. Rewrite with enough

Rule / explanation: The evidence was sufficiently strong to justify an inquiry.

Example / answer: The evidence was strong enough to justify an inquiry.

7. Translate

Rule / explanation: La réforme a été conçue pour réduire les inégalités, mais elle a produit l’effet inverse.

Example / answer: The reform was designed to reduce inequality, but it produced the opposite effect.

8. Mini concours challenge

Rule / explanation: Write a paragraph using in order to, so that, result in, unintended consequence and ultimately.

Example / answer: Example: The policy was introduced in order to reduce housing costs. Subsidies were increased so that more families could buy homes. However, the measure resulted in higher demand, and one unintended consequence was a further rise in prices. Ultimately, the policy failed to improve affordability.
