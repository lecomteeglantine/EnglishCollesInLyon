Chapter objective

This premium grammar chapter helps CPGE students master the Present Perfect Continuous. It focuses on have/has been + V-ing, duration, ongoing activities, current trends, recent results, the contrast with the Present Perfect Simple, and classic French-speaker mistakes with depuis.

The golden rule

Subject + have / has been + verb-ing
They have been protesting for weeks.
The government has been losing support.
Main useUse this tense to emphasise duration, continuity, repeated activity or an ongoing process linking past and present.
CPGE habitUse it for current trends and long-running debates: rents have been rising, trust has been declining, politicians have been debating.
Key trapFrench present with depuis often becomes Present Perfect Continuous: Ils protestent depuis trois semaines = They have been protesting for three weeks.

A. Lesson — What is the Present Perfect Continuous?

Present Perfect Continuous

French: present perfect continu

Rule / explanation: The Present Perfect Continuous links the past to the present and emphasises duration, continuity or ongoing activity.

Example / answer: The government has been trying to restore public trust.

Form

French: forme

Rule / explanation: Use have/has been + verb-ing.

Example / answer: Voters have been demanding stronger climate policies.

Ongoing action

French: action toujours en cours

Rule / explanation: Use it for an action that started in the past and is still continuing.

Example / answer: The debate has been going on for years.

Recent activity with present result

French: activité récente avec résultat présent

Rule / explanation: Use it when a recent activity explains the present situation.

Example / answer: The minister looks tired because he has been answering questions all day.

Emphasis on duration

French: accent sur la durée

Rule / explanation: It often stresses how long something has been happening.

Example / answer: Activists have been campaigning for months.

Repeated activity until now

French: activité répétée jusqu’au présent

Rule / explanation: Use it for repeated actions over a period continuing to the present.

Example / answer: Journalists have been investigating the scandal for weeks.

Temporary process

French: processus temporaire

Rule / explanation: It often describes temporary or developing situations.

Example / answer: The party has been losing support since the scandal.

CPGE use

French: usage CPGE

Rule / explanation: It is useful for current affairs, social trends, political pressure and long-running debates.

Example / answer: Democracies have been struggling to regulate digital platforms.

B. Form — Affirmative sentences

I / You / We / They have been + V-ing

French: have been + V-ing

Rule / explanation: Use have been with I, you, we, they and plural subjects.

Example / answer: Many citizens have been protesting against the bill.

He / She / It has been + V-ing

French: has been + V-ing

Rule / explanation: Use has been with he, she, it and singular subjects.

Example / answer: The government has been facing criticism.

Regular -ing spelling

French: orthographe -ing

Rule / explanation: Use the same -ing rules as the Present Continuous.

Example / answer: debate → debating / stop → stopping / lie → lying

For + duration

French: for + durée

Rule / explanation: Use for to say how long the activity has continued.

Example / answer: The crisis has been unfolding for several months.

Since + starting point

French: since + point de départ

Rule / explanation: Use since to say when the activity began.

Example / answer: The issue has been gaining attention since 2020.

All day / all week / all year

French: toute la journée / semaine / année

Rule / explanation: Useful time markers for repeated or continuous activity.

Example / answer: The press has been covering the scandal all week.

Recently / lately

French: récemment / ces derniers temps

Rule / explanation: Useful for recent repeated activity.

Example / answer: Politicians have recently been discussing AI regulation.

Still continuing

French: continuité

Rule / explanation: The action may still be happening now or may have just stopped.

Example / answer: Students have been working on the article for an hour.

C. Negative and question forms

Negative form

French: négation

Rule / explanation: Use have/has not been + V-ing.

Example / answer: The government has not been listening to experts.

Haven't / hasn't been

French: formes contractées

Rule / explanation: Contractions are common in speech and informal writing.

Example / answer: The article hasn't been addressing the real issue.

Question form

French: question

Rule / explanation: Put have/has before the subject.

Example / answer: Have voters been losing trust in institutions?

Wh- question

French: question en wh-

Rule / explanation: Wh-word + have/has + subject + been + V-ing.

Example / answer: Why has inequality been increasing?

Short answers

French: réponses courtes

Rule / explanation: Use yes/no + subject + have/has.

Example / answer: Yes, it has. / No, they haven't.

No do

French: pas de do

Rule / explanation: Do not use do/does/did with this tense.

Example / answer: Wrong: Does the government has been trying? Correct: Has the government been trying?

Been is essential

French: been obligatoire

Rule / explanation: Do not forget been.

Example / answer: Wrong: They have protesting. Correct: They have been protesting.

Verb-ing is essential

French: V-ing obligatoire

Rule / explanation: After been, use verb-ing.

Example / answer: Wrong: The issue has been grow. Correct: The issue has been growing.

D. Present Perfect Simple vs Present Perfect Continuous

Simple focuses on result

French: résultat

Rule / explanation: Present Perfect Simple often stresses result or completion.

Example / answer: The government has passed the bill.

Continuous focuses on activity

French: activité / durée

Rule / explanation: Present Perfect Continuous stresses process, duration or effort.

Example / answer: The government has been debating the bill for weeks.

Completed vs ongoing

French: achevé vs en cours

Rule / explanation: Simple can imply completion; continuous often implies ongoing or recent activity.

Example / answer: The article has explained the issue. / The article has been exploring the issue.

How much / how many

French: quantité

Rule / explanation: Use Present Perfect Simple with quantities.

Example / answer: The journalist has written three articles.

How long

French: durée

Rule / explanation: Use Present Perfect Continuous to emphasise duration.

Example / answer: The journalist has been writing about the scandal for months.

State verbs

French: verbes d’état

Rule / explanation: Many state verbs normally use Present Perfect Simple, not Continuous.

Example / answer: Wrong: I have been knowing this. Correct: I have known this for years.

Temporary trend

French: tendance temporaire

Rule / explanation: Continuous is useful for trends currently developing.

Example / answer: Support for the party has been falling.

Permanent achievement

French: résultat final

Rule / explanation: Simple is better for completed achievements or final outcomes.

Example / answer: The reform has changed the law.

E. Common CPGE uses

Long-running debates

French: débats durables

Rule / explanation: Use it for debates that started earlier and continue now.

Example / answer: Politicians have been arguing about immigration for decades.

Political pressure

French: pression politique

Rule / explanation: Use it for ongoing pressure or criticism.

Example / answer: The opposition has been criticising the government for weeks.

Social trends

French: tendances sociales

Rule / explanation: Use it for social developments still unfolding.

Example / answer: Young people have been moving away from traditional media.

Economic change

French: économie

Rule / explanation: Use it for current economic processes.

Example / answer: Prices have been rising faster than wages.

Media coverage

French: couverture médiatique

Rule / explanation: Use it for repeated coverage over time.

Example / answer: Newspapers have been covering the scandal extensively.

Environmental change

French: environnement

Rule / explanation: Use it for continuing environmental processes.

Example / answer: Scientists have been warning about extreme weather for years.

Data commentary

French: commentaire de données

Rule / explanation: Use it to describe a trend that is still developing.

Example / answer: The unemployment rate has been falling gradually.

Article analysis

French: analyse d’article

Rule / explanation: Use it to describe what an article has been focusing on or a debate it reflects.

Example / answer: The article reflects a debate that has been intensifying since the pandemic.

F. Time markers and useful phrases

For several years

French: depuis plusieurs années

Rule / explanation: Duration continuing to now.

Example / answer: Experts have been warning about inequality for several years.

Since the pandemic

French: depuis la pandémie

Rule / explanation: Starting point continuing to now.

Example / answer: Remote work has been changing office culture since the pandemic.

Over the past few months

French: ces derniers mois

Rule / explanation: Recent period up to now.

Example / answer: The government has been losing credibility over the past few months.

Lately

French: ces derniers temps

Rule / explanation: Recently, with repeated or continuing activity.

Example / answer: The press has been focusing on the housing crisis lately.

Recently

French: récemment

Rule / explanation: Recent period with present relevance.

Example / answer: Researchers have recently been questioning the data.

All along

French: depuis le début / tout du long

Rule / explanation: Throughout the whole period.

Example / answer: Critics have been warning about this all along.

For too long

French: depuis trop longtemps

Rule / explanation: Useful in criticism and essays.

Example / answer: Politicians have been ignoring the issue for too long.

More and more

French: de plus en plus

Rule / explanation: Useful with continuous trends.

Example / answer: More and more voters have been turning to radical parties.

G. Typical mistakes by French speakers

Forgetting been

French: oubli de been

Rule / explanation: Present Perfect Continuous needs have/has been + V-ing.

Example / answer: Wrong: They have protesting. Correct: They have been protesting.

Using base verb after been

French: base verbale incorrecte

Rule / explanation: Use V-ing after been.

Example / answer: Wrong: The issue has been grow. Correct: The issue has been growing.

Using do in questions

French: do incorrect

Rule / explanation: Use have/has inversion, not do.

Example / answer: Wrong: Do they have been protesting? Correct: Have they been protesting?

Using continuous with state verbs

French: verbes d’état

Rule / explanation: Avoid continuous with know, believe, understand, own, belong, seem.

Example / answer: Wrong: Voters have been believing this. Correct: Voters have believed this.

Confusing for and since

French: for / since

Rule / explanation: For + duration; since + starting point.

Example / answer: for three years / since 2020

Using Past Simple for ongoing duration

French: prétérit abusif

Rule / explanation: Use Present Perfect Continuous for actions continuing to now.

Example / answer: Wrong: They protested for months and still protest. Correct: They have been protesting for months.

Overusing it for completed results

French: résultat achevé

Rule / explanation: Use Present Perfect Simple for completed results.

Example / answer: Better: The government has passed the law, not has been passing it.

Translating French present mechanically

French: présent français + depuis

Rule / explanation: French present with depuis often becomes Present Perfect Continuous.

Example / answer: Ils protestent depuis trois semaines = They have been protesting for three weeks.
