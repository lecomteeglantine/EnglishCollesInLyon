Chapter 77 — Discourse Markers
B2 → C1 language for structuring discourse, guiding interpretation and managing sophisticated written and oral argument.

Chapter objective

This premium grammar chapter develops advanced control of discourse markers: organisation, sequencing, reformulation, concession, emphasis, stance, source comparison, oral signposting and final synthesis.

Level B2 → C1 Discourse Markers Oral Signposting Written Organisation Source Synthesis Exercises + Answers

The golden rule

A discourse marker must perform a real function.
Choose the function first: organise, contrast, concede, reformulate, emphasise, summarise or interact.
Organisationto begin with, turning to, finally
Interpretationin other words, more precisely, indeed
Stanceadmittedly, arguably, in my view
Interactionwell, actually, let me begin by, to return to the question

A. Lesson — What are discourse markers?

Discourse marker

Rule / explanation: A discourse marker guides the reader or listener through the organisation, interpretation or interpersonal tone of a text.

Example / answer: Turning now to the second article, the focus shifts from economic cost to political legitimacy.

Beyond grammar

Rule / explanation: Discourse markers do not merely connect clauses; they also signal attitude, sequence, reformulation, emphasis and interaction.

Example / answer: To be fair, the government inherited an already unstable situation.

Written and spoken use

Rule / explanation: Some markers are typical of essays, while others are more natural in oral commentary and debate.

Example / answer: Furthermore is formal and written; well is primarily conversational.

CPGE relevance

Rule / explanation: Controlled discourse markers improve essays, syntheses, article commentary, khôlles and transitions between documents.

Example / answer: Taken together, the sources suggest that the crisis is both economic and institutional.

B. Organising an introduction

To begin with

Rule / explanation: Introduce the first broad point or stage of reasoning.

Example / answer: To begin with, the controversy must be placed in its political context.

At the outset

Rule / explanation: Use this formal marker to frame an initial clarification or limitation.

Example / answer: At the outset, it should be noted that the two surveys use different samples.

First and foremost

Rule / explanation: Give priority to the most important consideration.

Example / answer: First and foremost, the reform raises a question of democratic legitimacy.

The central issue is

Rule / explanation: Identify the key question directly.

Example / answer: The central issue is whether efficiency can be reconciled with fairness.

This raises the question of

Rule / explanation: Move from context to the analytical problem.

Example / answer: This raises the question of how public trust can be restored.

C. Sequencing an argument

First / firstly

Rule / explanation: Introduce the first main stage. Choose either first or firstly and remain consistent.

Example / answer: First, the article examines the economic causes of the crisis.

Secondly

Rule / explanation: Introduce the next substantial stage.

Example / answer: Secondly, it considers the political consequences.

Subsequently

Rule / explanation: Mark a later chronological or analytical development.

Example / answer: The proposal was subsequently amended.

Thereafter

Rule / explanation: Use this formal marker for events occurring after a stated point.

Example / answer: The committee published its findings and thereafter met only twice.

Finally

Rule / explanation: Introduce the last principal stage, not every concluding sentence.

Example / answer: Finally, the author proposes a new regulatory framework.

D. Moving to a new point

Turning to

Rule / explanation: Signal a controlled shift to a new topic, source or dimension.

Example / answer: Turning to the second document, its tone is considerably more cautious.

As for

Rule / explanation: Introduce a new item already present in the discussion; more common in speech.

Example / answer: As for the proposed solution, it remains financially unrealistic.

With regard to

Rule / explanation: Introduce a specific formal aspect.

Example / answer: With regard to regional access, the policy produced mixed results.

As far as X is concerned

Rule / explanation: Introduce one topic or stakeholder; natural in formal speech but heavy if repeated.

Example / answer: As far as rural communities are concerned, little has improved.

Moving on to

Rule / explanation: Use this natural oral marker to progress through a presentation.

Example / answer: Moving on to the broader context, the reform reflects a long-term centralisation trend.

E. Adding information

Furthermore

Rule / explanation: Add a substantial formal point.

Example / answer: Furthermore, the reform failed to improve rural access.

Moreover

Rule / explanation: Add a point that reinforces the previous claim.

Example / answer: Moreover, implementation costs exceeded the forecast.

In addition

Rule / explanation: Introduce neutral additional information.

Example / answer: In addition, local authorities received no new funding.

What is more

Rule / explanation: Add emphasis; more natural in speech or persuasive prose.

Example / answer: What is more, the consultation excluded the communities most affected.

Besides

Rule / explanation: Use with caution in academic writing; it is more conversational and can introduce an extra reason.

Example / answer: Besides, the proposed timetable is unrealistic.

F. Introducing contrast

However

Rule / explanation: Signal a direct contrast, qualification or limitation.

Example / answer: The reform reduced costs. However, access remained unequal.

Nevertheless

Rule / explanation: Maintain a conclusion despite a contrary fact.

Example / answer: The evidence is limited. Nevertheless, the trend is significant.

By contrast

Rule / explanation: Introduce a clear opposition between cases, sources or trends.

Example / answer: The first article is optimistic. By contrast, the second predicts prolonged instability.

On the other hand

Rule / explanation: Present another side of a genuine tension.

Example / answer: Regulation protects consumers. On the other hand, it may increase compliance costs.

Conversely

Rule / explanation: Introduce an opposite or reversed relationship.

Example / answer: Transparency may build trust; conversely, secrecy can intensify suspicion.

G. Conceding a point

Admittedly

Rule / explanation: Acknowledge a valid opposing point before qualifying it.

Example / answer: Admittedly, the reform reduced short-term expenditure.

Granted

Rule / explanation: Accept a point provisionally, often before a counterargument.

Example / answer: Granted, the proposal is ambitious; however, its funding remains uncertain.

To be sure

Rule / explanation: Acknowledge something true while preparing a limitation.

Example / answer: To be sure, inflation has fallen, but household costs remain high.

That said

Rule / explanation: Introduce a qualification after recognising a positive or valid point.

Example / answer: The programme is innovative. That said, implementation may be difficult.

All the same

Rule / explanation: Maintain a judgement despite a concession; more natural in speech.

Example / answer: The figures are incomplete. All the same, they reveal a worrying pattern.

H. Expressing cause and consequence

Therefore

Rule / explanation: Introduce a logical consequence or inference.

Example / answer: Funding was reduced; therefore, waiting times increased.

Consequently

Rule / explanation: Introduce a formal result.

Example / answer: Several clinics closed. Consequently, access deteriorated.

As a result

Rule / explanation: Present a factual consequence.

Example / answer: The timetable was delayed. As a result, expected savings did not materialise.

Thus

Rule / explanation: Express result or logical conclusion concisely.

Example / answer: The measure excluded rural areas, thus weakening its legitimacy.

Hence

Rule / explanation: Introduce a compact formal consequence or justification.

Example / answer: The evidence remains incomplete, hence the need for further research.

I. Giving examples

For example

Rule / explanation: Introduce an illustration of a general point.

Example / answer: Several regions lost services. For example, three rural clinics closed.

For instance

Rule / explanation: A close formal equivalent.

Example / answer: Some digital reforms improved access; for instance, remote consultations reduced travel.

A case in point

Rule / explanation: Introduce a particularly relevant example.

Example / answer: Scotland is a case in point.

To illustrate

Rule / explanation: Announce an example intended to clarify a claim.

Example / answer: To illustrate, rural waiting times rose by 18 per cent.

Consider

Rule / explanation: Invite the reader or listener to examine a revealing case.

Example / answer: Consider the effect of identical funding cuts on regions with unequal infrastructure.

J. Reformulating

In other words

Rule / explanation: Restate an idea more clearly or simply.

Example / answer: The policy lacked legitimacy; in other words, citizens did not regard it as justified.

That is to say

Rule / explanation: Provide explicit formal clarification.

Example / answer: The measure was regressive, that is to say, it affected poorer households more severely.

More precisely

Rule / explanation: Narrow, correct or sharpen the previous wording.

Example / answer: The problem concerns access, or more precisely, unequal regional access.

To put it differently

Rule / explanation: Offer another formulation, especially useful orally.

Example / answer: To put it differently, efficiency was achieved at the expense of fairness.

Rather

Rule / explanation: Correct a previous characterisation.

Example / answer: The reform did not remove inequality; rather, it redistributed it.

K. Clarifying and specifying

Namely

Rule / explanation: Specify the exact element previously announced.

Example / answer: One difficulty remains, namely the absence of long-term funding.

In particular

Rule / explanation: Highlight one especially relevant case or aspect.

Example / answer: Rural communities, in particular, experienced longer waiting times.

More specifically

Rule / explanation: Move from a general statement to a precise detail.

Example / answer: The policy affected access, more specifically in northern regions.

In this respect

Rule / explanation: Refer to the particular dimension just discussed.

Example / answer: In this respect, the second article is more convincing.

As regards

Rule / explanation: Introduce a specific formal topic.

Example / answer: As regards feasibility, the proposal remains questionable.

L. Emphasising

Indeed

Rule / explanation: Confirm, reinforce or exemplify the previous point.

Example / answer: The reform was costly; indeed, expenditure exceeded every official forecast.

Above all

Rule / explanation: Identify the most important point.

Example / answer: Above all, the crisis concerns public trust.

Crucially

Rule / explanation: Highlight a decisive factor.

Example / answer: Crucially, the policy offers no mechanism for local accountability.

Significantly

Rule / explanation: Draw attention to an important or revealing feature.

Example / answer: Significantly, the report excludes the poorest regions.

It should be stressed that

Rule / explanation: Use this formal phrase to give strong prominence to a point.

Example / answer: It should be stressed that correlation does not establish causation.

M. Limiting and qualifying

To some extent

Rule / explanation: Limit the force of a judgement.

Example / answer: To some extent, the government’s caution is understandable.

In broad terms

Rule / explanation: Present a general assessment while allowing detail and exceptions.

Example / answer: In broad terms, both sources describe a crisis of legitimacy.

Broadly speaking

Rule / explanation: Introduce a generalisation that does not claim universal validity.

Example / answer: Broadly speaking, younger voters are more sceptical of traditional institutions.

In most cases

Rule / explanation: Limit a claim to the majority rather than all cases.

Example / answer: In most cases, local authorities lacked sufficient resources.

At least

Rule / explanation: Introduce a minimum claim or limited perspective.

Example / answer: At least in the short term, the policy reduced expenditure.

N. Stating viewpoint and stance

In my view

Rule / explanation: Introduce a clear formal personal judgement.

Example / answer: In my view, the author underestimates regional inequality.

From this perspective

Rule / explanation: Present a conclusion based on the viewpoint just established.

Example / answer: From this perspective, stronger regulation appears justified.

Arguably

Rule / explanation: Introduce a defensible but debatable interpretation.

Example / answer: The article is arguably too optimistic.

Apparently

Rule / explanation: Signal appearance, inference or second-hand information.

Example / answer: Apparently, negotiations have resumed.

Presumably

Rule / explanation: Signal a reasonable assumption based on context.

Example / answer: Presumably, the committee will request further evidence.

O. Distancing and source attribution

According to

Rule / explanation: Attribute information or a view to a source.

Example / answer: According to the report, waiting times rose by 12 per cent.

Reportedly

Rule / explanation: Indicate that information has been reported but not independently confirmed.

Example / answer: Several ministers reportedly opposed the proposal.

Allegedly

Rule / explanation: Distance the speaker from an accusation or unproven claim.

Example / answer: Officials allegedly ignored repeated warnings.

The author claims that

Rule / explanation: Attribute a disputed or unverified proposition.

Example / answer: The author claims that deregulation automatically increases innovation.

The evidence suggests that

Rule / explanation: Connect a cautious conclusion to support.

Example / answer: The evidence suggests that implementation was uneven.

P. Summarising

In short

Rule / explanation: Give a concise restatement of the main point.

Example / answer: In short, the reform improved efficiency but weakened fairness.

In brief

Rule / explanation: Offer a short summary, often before a transition.

Example / answer: In brief, both sources identify the same problem but propose different solutions.

Overall

Rule / explanation: Present a general assessment after considering several aspects.

Example / answer: Overall, the article is persuasive but methodologically limited.

On the whole

Rule / explanation: Express a broad evaluation allowing some exceptions.

Example / answer: On the whole, the programme achieved its immediate objectives.

Taken together

Rule / explanation: Synthesize the combined significance of several sources or facts.

Example / answer: Taken together, the documents reveal a multidimensional crisis.

Q. Concluding

In conclusion

Rule / explanation: Introduce the formal final synthesis of an argument.

Example / answer: In conclusion, efficiency cannot guarantee legitimacy without fairness.

Ultimately

Rule / explanation: Introduce the deepest or final determining point.

Example / answer: Ultimately, the reform’s success depends on public trust.

On balance

Rule / explanation: Give a reasoned judgement after weighing competing factors.

Example / answer: On balance, revision is preferable to complete abandonment.

All things considered

Rule / explanation: Introduce a broad final judgement; more natural orally than in highly formal essays.

Example / answer: All things considered, the benefits appear to outweigh the costs.

The key point is that

Rule / explanation: End by foregrounding the central conclusion.

Example / answer: The key point is that implementation matters as much as policy design.

R. Interactive oral markers

Well

Rule / explanation: Use well to begin, delay or soften a spoken response; avoid it in formal essays.

Example / answer: Well, I would not entirely agree with that interpretation.

Actually

Rule / explanation: Correct, contrast or clarify previous information; it means in fact.

Example / answer: Actually, the second article is less optimistic than the first.

Of course

Rule / explanation: Refer to something treated as obvious or already accepted; avoid using it to dismiss complexity.

Example / answer: Of course, economic factors remain important.

You see

Rule / explanation: Explain or justify informally; generally unsuitable for formal CPGE delivery.

Example / answer: Prefer the reason is that.

I mean

Rule / explanation: Repair or clarify spoken language, but overuse weakens fluency.

Example / answer: The measure is unfair—or rather, I mean, uneven in its regional impact.

S. Managing oral argument

Let me begin by

Rule / explanation: Introduce the first stage of a spoken presentation.

Example / answer: Let me begin by summarising the author’s central argument.

I would now like to turn to

Rule / explanation: Signal a formal transition in a khôlle.

Example / answer: I would now like to turn to the broader historical context.

This leads me to

Rule / explanation: Connect one analytical point to the next.

Example / answer: This leads me to the question of political legitimacy.

Before moving on

Rule / explanation: Pause to complete or qualify the current point.

Example / answer: Before moving on, it is important to clarify the term accountability.

To return to the question

Rule / explanation: Refocus after an example or digression.

Example / answer: To return to the question, the reform can succeed only if it is perceived as fair.

T. Comparing sources in a synthesis

Both sources

Rule / explanation: Introduce a point of convergence.

Example / answer: Both sources identify declining trust as a central concern.

Similarly

Rule / explanation: Add a comparable observation.

Example / answer: Similarly, both authors question the official forecast.

Whereas

Rule / explanation: Contrast different emphases in one sentence.

Example / answer: Whereas the first source focuses on cost, the second examines legitimacy.

By contrast

Rule / explanation: Introduce a clear divergence.

Example / answer: By contrast, the writers disagree on the desirability of regulation.

Taken together

Rule / explanation: State the combined analytical meaning of the documents.

Example / answer: Taken together, the sources suggest that the crisis has both material and institutional causes.

U. Common mistakes

Marker overload

Rule / explanation: Do not begin every sentence with a connector or discourse marker.

Example / answer: Use markers only when they clarify structure or stance.

Wrong logic

Rule / explanation: Choose the relationship before choosing the expression.

Example / answer: However cannot express consequence; therefore cannot express contrast.

Register mismatch

Rule / explanation: Avoid conversational markers such as anyway, you know and like in formal essays.

Example / answer: Use nevertheless, in any case or for example where appropriate.

Mechanical sequence

Rule / explanation: First, secondly and thirdly do not create an argument by themselves.

Example / answer: The content must genuinely progress.

Actually

Rule / explanation: Actually means in fact, not actuellement.

Example / answer: Use currently for at present.

Eventually

Rule / explanation: Eventually means in the end, not éventuellement.

Example / answer: Use perhaps or possibly for possibility.

V. Exercises and corrections

1. Introduce a new source

Rule / explanation: Make this transition formal: Now I’m going to talk about the second article.

Example / answer: Turning now to the second article, its focus is markedly different.

2. Add a concession

Rule / explanation: The evidence is limited. The trend remains significant.

Example / answer: The evidence is limited. Nevertheless, the trend remains significant.

3. Reformulate

Rule / explanation: The policy lacked legitimacy. Citizens did not consider it justified.

Example / answer: The policy lacked legitimacy; in other words, citizens did not consider it justified.

4. Synthesize

Rule / explanation: The documents discuss economic insecurity and institutional distrust.

Example / answer: Taken together, the documents present the crisis as both economic and institutional.

5. Correct the marker

Rule / explanation: Funding fell. However, waiting times increased.

Example / answer: Funding fell. Consequently, waiting times increased.

6. Correct the false friend

Rule / explanation: Actually, the committee is examining the evidence.

Example / answer: Currently, the committee is examining the evidence.

7. Translate

Rule / explanation: Pour revenir à la question principale, la réforme ne peut réussir que si elle est perçue comme légitime.

Example / answer: To return to the central question, the reform can succeed only if it is perceived as legitimate.

8. Mini concours challenge

Rule / explanation: Write a short oral commentary using a transition, concession, reformulation, source comparison and final judgement.

Example / answer: Example: Turning now to the second source, its tone is considerably more cautious. Admittedly, both articles recognise the economic dimension of the crisis. In other words, neither author denies the importance of material insecurity. Whereas the first text presents deregulation as a solution, however, the second emphasises institutional accountability. On balance, the second interpretation appears more convincing because it better accounts for the persistence of public distrust.
