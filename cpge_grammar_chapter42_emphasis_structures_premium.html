The golden rule

Emphasis = focus + contrast + clarity
It is public trust that the crisis has damaged.
What matters is whether institutions can regain legitimacy.
Main useEmphasis structures help highlight the key idea, contrast or analytical point.
CPGE habitUse them to make thesis statements and conclusions more powerful without becoming theatrical.
Key trapDo not overuse emphasis. One strong emphatic structure is better than five heavy ones.

A. Lesson — What are emphasis structures?

Emphasis structures

French: structures emphatiques

Rule / explanation: Emphasis structures highlight a word, idea, contrast or argument more strongly than normal word order.

Example / answer: What the article reveals is a deeper crisis of trust.

Why use emphasis?

French: pourquoi insister ?

Rule / explanation: Use emphasis to make a thesis, contrast or key point clearer and more persuasive.

Example / answer: It is the lack of evidence that weakens the argument.

Formal style

French: style formel

Rule / explanation: Many emphasis structures are common in essays, speeches, journalism and academic analysis.

Example / answer: The real issue is not immigration but inequality.

CPGE relevance

French: usage CPGE

Rule / explanation: They help students build stronger introductions, conclusions and oral transitions.

Example / answer: What matters most is whether citizens still trust institutions.

B. It-clefts

It is / was... that

French: c’est... qui / que

Rule / explanation: Use an it-cleft to emphasise the focused element of the sentence.

Example / answer: It is inequality that the author sees as the root cause.

Emphasising the subject

French: insister sur le sujet

Rule / explanation: Use it-clefts to emphasise who or what does the action.

Example / answer: It was the media that exposed the scandal.

Emphasising the object

French: insister sur le complément

Rule / explanation: Use it-clefts to emphasise the thing affected by the action.

Example / answer: It is public trust that the crisis has damaged.

Emphasising time or place

French: insister sur le temps / lieu

Rule / explanation: Use it-clefts to emphasise when or where something happened.

Example / answer: It was after the referendum that the divisions became visible.

C. Wh-clefts

What... is / was...

French: ce qui... c’est

Rule / explanation: Use a wh-cleft to emphasise the most important point or result.

Example / answer: What the article criticises is the government’s lack of preparation.

What matters is...

French: ce qui compte, c’est

Rule / explanation: Very useful for essay conclusions and oral summaries.

Example / answer: What matters is not the promise itself but its credibility.

What is striking is...

French: ce qui frappe, c’est

Rule / explanation: Useful for introducing analysis after a summary.

Example / answer: What is striking is the gap between rhetoric and reality.

What the author shows is...

French: ce que montre l’auteur, c’est

Rule / explanation: Useful in article summaries.

Example / answer: What the author shows is that inequality damages democracy.

D. Fronting for emphasis

Fronting an object

French: mise en tête du complément

Rule / explanation: Move an object or complement to the front for emphasis.

Example / answer: This argument, the author develops very convincingly.

Fronting a contrast

French: mise en avant du contraste

Rule / explanation: Put the contrastive idea first to make opposition clearer.

Example / answer: This reform may be popular; effective, it is not.

As for / as regards

French: quant à / en ce qui concerne

Rule / explanation: Use these to shift focus to a topic.

Example / answer: As for the economic argument, it remains weak.

The problem is that...

French: le problème est que

Rule / explanation: A clear and effective emphasis pattern for essays.

Example / answer: The problem is that the article overlooks regional inequalities.

E. Do-emphasis

Do + base verb

French: do emphatique

Rule / explanation: Use do/does/did before the verb to add emphasis in affirmative sentences.

Example / answer: The article does raise an important question.

Does + base verb

French: does emphatique

Rule / explanation: Use does for third-person singular present.

Example / answer: The author does acknowledge the limits of his argument.

Did + base verb

French: did emphatique

Rule / explanation: Use did for past emphasis.

Example / answer: The policy did improve access to healthcare.

Careful tone

French: ton prudent

Rule / explanation: Do-emphasis can sound corrective or defensive, so use it carefully.

Example / answer: The reform did help some families, but it failed to address structural causes.

F. Negative emphasis

Not... but...

French: non pas... mais

Rule / explanation: Use this to correct or contrast ideas.

Example / answer: The crisis is not merely economic but deeply political.

Not only... but also

French: non seulement... mais aussi

Rule / explanation: Use this to add a second, stronger point.

Example / answer: The policy not only failed but also damaged public trust.

Far from + -ing

French: loin de

Rule / explanation: Use this formal phrase to emphasise the opposite of an expected result.

Example / answer: Far from solving the problem, the reform made it worse.

No longer

French: ne... plus

Rule / explanation: Use this to emphasise change over time.

Example / answer: The issue can no longer be ignored.

G. Intensifying phrases

Above all

French: avant tout

Rule / explanation: Highlights the most important point.

Example / answer: Above all, the article questions the legitimacy of the system.

More importantly

French: plus important encore

Rule / explanation: Introduces a stronger or more significant point.

Example / answer: More importantly, the data contradict the minister’s claim.

The real issue is...

French: le vrai problème est

Rule / explanation: Useful for moving from description to analysis.

Example / answer: The real issue is the erosion of democratic trust.

What is at stake is...

French: ce qui est en jeu, c’est

Rule / explanation: Very useful for high-level CPGE argumentation.

Example / answer: What is at stake is the future of political representation.

H. Emphasis in introductions and conclusions

This raises the question of...

French: cela soulève la question de

Rule / explanation: Use it to introduce a problematique elegantly.

Example / answer: This raises the question of whether democracy can survive without trust.

The key question is whether...

French: la question clé est de savoir si

Rule / explanation: Useful for framing an essay.

Example / answer: The key question is whether reform can restore legitimacy.

What remains unclear is...

French: ce qui reste incertain, c’est

Rule / explanation: Useful for nuanced conclusions.

Example / answer: What remains unclear is whether the policy will be properly funded.

Ultimately

French: en fin de compte

Rule / explanation: Use it to introduce a final assessment.

Example / answer: Ultimately, the debate is about power, trust and accountability.

I. Typical mistakes by French speakers

Overusing it is

French: abus de it is

Rule / explanation: Do not translate every French c’est with it is.

Example / answer: Weak: It is important to say that. Better: What matters is that.

Wrong cleft grammar

French: structure incomplète

Rule / explanation: Keep the cleft structure complete: it is/was + focus + that/who.

Example / answer: Wrong: It the government that failed. Correct: It was the government that failed.

Do + conjugated verb

French: do + base verb

Rule / explanation: After do/does/did, use the base verb.

Example / answer: Wrong: The article does raises. Correct: The article does raise.

Too much drama

French: surcharge emphatique

Rule / explanation: Emphasis is powerful but heavy if every sentence is emphatic.

Example / answer: Use emphasis for key points, not for every ordinary idea.
