Chapter objective

This premium grammar chapter helps CPGE students master the Third Conditional. It covers if + Past Perfect, would have + past participle, regret, criticism, missed opportunities, historical counterfactuals, could have, might have, and classic French-speaker mistakes.

The golden rule

If + Past Perfect, would have + past participle
If the government had acted earlier, the crisis would have been less severe.
Main useUse the Third Conditional for unreal past situations and imagined past results.
CPGE habitUse it for historical analysis, missed opportunities and nuanced criticism: If policymakers had listened to experts, the outcome might have been different.
Key trapDo not use would have after if: If the government had acted, not If the government would have acted.

A. Lesson — What is the Third Conditional?

Third Conditional

French: troisième conditionnel

Rule / explanation: The Third Conditional describes an unreal past situation and its imagined past result.

Example / answer: If the government had acted earlier, the crisis would have been less severe.

Unreal past

French: passé irréel

Rule / explanation: Use it for something that did not happen in the past.

Example / answer: If the minister had resigned sooner, the scandal might have faded.

Imagined past result

French: résultat passé imaginaire

Rule / explanation: The result is also imaginary because the past cannot be changed.

Example / answer: If the policy had worked, poverty would have decreased.

Past Perfect after if

French: past perfect après if

Rule / explanation: Use had + past participle after if.

Example / answer: If voters had trusted the system, turnout would have been higher.

Would have + past participle

French: would have + participe passé

Rule / explanation: Use would have + past participle in the result clause.

Example / answer: The reform would have succeeded if it had been properly funded.

Could have

French: could have

Rule / explanation: Use could have for possible past ability or opportunity.

Example / answer: If schools had received more funding, they could have reduced inequality.

Might have

French: might have

Rule / explanation: Use might have for uncertain past results.

Example / answer: If the article had included more data, it might have been more convincing.

CPGE use

French: usage CPGE

Rule / explanation: The Third Conditional is essential for regrets, criticism, missed opportunities, historical analysis and counterfactual reasoning.

Example / answer: If policymakers had listened to experts, the outcome might have been different.

B. Form and structure

If + Past Perfect, would have + past participle

French: forme principale

Rule / explanation: Most common Third Conditional structure.

Example / answer: If the state had invested earlier, the project would have succeeded.

Would have + past participle if + Past Perfect

French: ordre inversé

Rule / explanation: Same meaning with reversed order.

Example / answer: The project would have succeeded if the state had invested earlier.

If + Past Perfect, could have

French: avec could have

Rule / explanation: Use could have for possible ability or opportunity in the past.

Example / answer: If the government had acted sooner, it could have prevented the crisis.

If + Past Perfect, might have

French: avec might have

Rule / explanation: Use might have for uncertain past consequences.

Example / answer: If the reform had been clearer, voters might have supported it.

Negative condition

French: condition négative

Rule / explanation: Use had not / hadn't + past participle.

Example / answer: If the government had not ignored the warnings, the crisis would have been milder.

Negative result

French: résultat négatif

Rule / explanation: Use would not have / wouldn't have + past participle.

Example / answer: If the evidence had been stronger, the article would not have been criticised.

Question form

French: question

Rule / explanation: Use question word order in the result clause if it is a question.

Example / answer: What would have happened if the bill had passed?

Passive Third Conditional

French: passif

Rule / explanation: Use passive forms when the focus is on the action or object.

Example / answer: If the policy had been implemented earlier, it could have helped more families.

C. Third Conditional for regret and criticism

Regret

French: regret

Rule / explanation: Use it to express regret about the past.

Example / answer: If the government had acted earlier, fewer people would have suffered.

Missed opportunity

French: occasion manquée

Rule / explanation: Use it to describe a chance that was not taken.

Example / answer: If the country had invested in renewable energy, it could have reduced its dependence on fossil fuels.

Criticism

French: critique

Rule / explanation: Use it to criticise a past decision indirectly.

Example / answer: If the minister had been more transparent, the public might have trusted him.

Historical counterfactual

French: contrefactuel historique

Rule / explanation: Use it to imagine a different historical outcome.

Example / answer: If the referendum had failed, British politics would have followed a different path.

Policy failure

French: échec politique

Rule / explanation: Use it to analyse why a policy did not work.

Example / answer: If the reform had been properly funded, it might have succeeded.

Media analysis

French: analyse des médias

Rule / explanation: Use it to criticise past media coverage.

Example / answer: If the press had checked the figures, misinformation would not have spread so quickly.

Environmental regret

French: regret environnemental

Rule / explanation: Use it to discuss missed climate action.

Example / answer: If governments had reduced emissions earlier, some damage could have been avoided.

Academic caution

French: prudence académique

Rule / explanation: Use it carefully: counterfactual claims must be justified.

Example / answer: If the article had provided stronger evidence, its conclusion would have been more persuasive.

D. Would have, could have, might have

Would have

French: would have

Rule / explanation: Use it for the most direct imagined past result.

Example / answer: If wages had risen, consumption would have increased.

Could have

French: could have

Rule / explanation: Use it for possible ability, opportunity or alternative outcome.

Example / answer: If citizens had been better informed, they could have made a different choice.

Might have

French: might have

Rule / explanation: Use it for uncertainty or cautious speculation.

Example / answer: If the campaign had focused on housing, it might have attracted younger voters.

Would probably have

French: aurait probablement

Rule / explanation: Use it for a likely but still hypothetical past result.

Example / answer: If the scandal had continued, the minister would probably have resigned.

Could potentially have

French: aurait potentiellement pu

Rule / explanation: Use it for cautious formal speculation.

Example / answer: If it had been properly funded, the programme could potentially have improved mobility.

Might not have

French: n’aurait peut-être pas

Rule / explanation: Use it for uncertain negative past results.

Example / answer: If the law had changed, it might not have solved the deeper problem.

Would not necessarily have

French: n’aurait pas forcément

Rule / explanation: Useful for nuance.

Example / answer: If taxes had fallen, inequality would not necessarily have decreased.

Could have been + past participle

French: aurait pu être

Rule / explanation: Useful passive form for missed opportunities.

Example / answer: The crisis could have been avoided if warnings had been taken seriously.

E. Third Conditional in essays and khôlles

Historical analysis

French: analyse historique

Rule / explanation: Use it to discuss alternative historical outcomes.

Example / answer: If the welfare state had not expanded after 1945, British society would have been very different.

Political responsibility

French: responsabilité politique

Rule / explanation: Use it to evaluate past decisions.

Example / answer: If ministers had listened to local authorities, the policy might have worked better.

Economic consequences

French: conséquences économiques

Rule / explanation: Use it to discuss missed economic choices.

Example / answer: If wages had kept pace with inflation, households would have been less vulnerable.

Social policy

French: politique sociale

Rule / explanation: Use it to examine past reforms.

Example / answer: If the reform had targeted poorer families, it could have reduced inequality.

Media and misinformation

French: médias et désinformation

Rule / explanation: Use it to discuss preventable harm.

Example / answer: If platforms had acted earlier, misinformation might not have spread so widely.

Climate policy

French: politique climatique

Rule / explanation: Use it for delayed action and regret.

Example / answer: If governments had acted sooner, adaptation costs would have been lower.

Khôlle response

French: réponse en khôlle

Rule / explanation: Use it when asked to imagine a different past outcome.

Example / answer: If the author had included a counterargument, the article would have been more balanced.

Essay nuance

French: nuance

Rule / explanation: Use might/could instead of would when the causal link is uncertain.

Example / answer: If the law had been stricter, it might have reduced online abuse.

F. Third Conditional vs Second Conditional

Second Conditional

French: deuxième conditionnel

Rule / explanation: Imagines present or future situations.

Example / answer: If the policy worked, inequality would fall.

Third Conditional

French: troisième conditionnel

Rule / explanation: Imagines a different past.

Example / answer: If the policy had worked, inequality would have fallen.

Present/future vs past

French: présent/futur vs passé

Rule / explanation: Second = now or future; Third = past.

Example / answer: If voters trusted politicians, turnout would rise. / If voters had trusted politicians, turnout would have risen.

Would vs would have

French: would vs would have

Rule / explanation: Would = hypothetical present/future result; would have = hypothetical past result.

Example / answer: It would work now. / It would have worked then.

Could vs could have

French: could vs could have

Rule / explanation: Could = hypothetical possibility now/future; could have = missed possibility in the past.

Example / answer: They could act now. / They could have acted earlier.

Might vs might have

French: might vs might have

Rule / explanation: Might = possible hypothetical result; might have = possible past result.

Example / answer: It might help now. / It might have helped then.

CPGE distinction

French: distinction CPGE

Rule / explanation: Choose according to the time of the imagined situation.

Example / answer: If the article were more balanced, it would be stronger. / If the article had been more balanced, it would have been stronger.

Mixed risk

French: risque de mélange

Rule / explanation: Do not mix tenses unless using a real mixed conditional later.

Example / answer: For Chapter 19, keep the structure clean: if + had + participle, would have + participle.

G. Typical mistakes by French speakers

Would have after if

French: would have après if

Rule / explanation: Do not use would have in the if-clause of a standard Third Conditional.

Example / answer: Wrong: If the government would have acted. Correct: If the government had acted.

Past Simple instead of Past Perfect

French: prétérit au lieu de past perfect

Rule / explanation: Use had + past participle after if for unreal past.

Example / answer: Wrong: If the government acted earlier, it would have helped. Correct: If the government had acted earlier, it would have helped.

Forgetting have

French: oubli de have

Rule / explanation: The result clause needs would have / could have / might have.

Example / answer: Wrong: It would reduced inequality. Correct: It would have reduced inequality.

Wrong past participle

French: mauvais participe

Rule / explanation: Use the past participle after had and have.

Example / answer: Wrong: had wrote / would have went. Correct: had written / would have gone.

Confusing Second and Third Conditional

French: 2e vs 3e conditionnel

Rule / explanation: Second = present/future; Third = past.

Example / answer: If the reform worked, it would help now. / If the reform had worked, it would have helped then.

Overusing would have for certainty

French: certitude excessive

Rule / explanation: Use might have or could have when the result is uncertain.

Example / answer: Better: It might have reduced poverty, not always it would have reduced poverty.

Misplacing not

French: place de not

Rule / explanation: Put not after had or would have.

Example / answer: If the government had not ignored warnings... / It would not have failed.

Translating French mechanically

French: calque français

Rule / explanation: French plus-que-parfait + conditionnel passé maps to English Past Perfect + would have.

Example / answer: Si le gouvernement avait agi, la crise aurait diminué = If the government had acted, the crisis would have decreased.
