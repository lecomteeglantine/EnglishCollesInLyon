Chapter 75 — Cohesion and Coherence
B2 → C1 language for connected sentences, unified paragraphs and logically progressive CPGE argumentation.

Chapter objective

This premium grammar chapter develops advanced control of cohesion and coherence: paragraph unity, topic sentences, reference, repetition, substitution, ellipsis, logical connectors, information flow and coherent introductions, developments and conclusions.

The golden rule

Cohesion connects the language.
Coherence connects the ideas.
A strong paragraph needs both.
Cohesion toolsreference, repetition, substitution, ellipsis, connectors
Coherence toolsunity, relevance, logical order, evidence, progression
Paragraph patterntopic sentence → explanation → evidence → evaluation → link
Main warningconnectors cannot repair weak logic

A. Lesson — Cohesion vs coherence

Cohesion

Rule / explanation: Cohesion is the visible linguistic connection between sentences and clauses through reference, repetition, substitution and linking expressions.

Example / answer: The reform reduced costs. This reduction, however, created new regional inequalities.

Coherence

Rule / explanation: Coherence is the overall logical unity of a text: ideas are relevant, ordered and easy to follow.

Example / answer: The paragraph moves from the cause of the crisis to its consequences and then to possible solutions.

Difference

Rule / explanation: A text may contain many connectors and still be incoherent if the ideas do not form a logical progression.

Example / answer: Moreover cannot repair an unrelated or unsupported sentence.

CPGE relevance

Rule / explanation: Cohesion and coherence are essential for summaries, syntheses, essays, article commentary and oral presentations.

Example / answer: Each paragraph should advance one stage of the argument.

B. Topic sentences

Main function

Rule / explanation: A topic sentence states the central idea or analytical function of a paragraph.

Example / answer: The first major limitation of the reform is its uneven regional impact.

Position

Rule / explanation: It usually appears near the beginning of the paragraph.

Example / answer: This gives the reader an immediate framework.

Not a vague announcement

Rule / explanation: Avoid empty openings such as There are many things to say about this issue.

Example / answer: State the precise point directly.

Connection to thesis

Rule / explanation: Each topic sentence should contribute to answering the central question.

Example / answer: Economic efficiency alone cannot guarantee political legitimacy.

Paragraph control

Rule / explanation: Every following sentence should explain, support, illustrate or qualify the topic sentence.

Example / answer: Remove material that does not serve this function.

C. Paragraph unity

One central purpose

Rule / explanation: A paragraph should normally perform one main analytical task.

Example / answer: A paragraph on causes should not suddenly become a conclusion about solutions.

Relevant evidence

Rule / explanation: Examples and data must support the paragraph’s central claim.

Example / answer: A statistic on turnout is relevant only if the paragraph concerns participation.

Controlled development

Rule / explanation: Move from general claim to explanation, evidence and evaluation.

Example / answer: Claim → reason → example → implication.

Avoid digression

Rule / explanation: Interesting information should be removed if it does not advance the argument.

Example / answer: Relevance matters more than quantity.

Closing link

Rule / explanation: End by drawing out the significance of the paragraph.

Example / answer: This regional divide therefore weakens the reform’s claim to fairness.

D. Logical progression

General to specific

Rule / explanation: Begin with the broad claim, then narrow towards evidence or examples.

Example / answer: Institutional trust has declined. In particular, confidence in Parliament has fallen sharply.

Cause to consequence

Rule / explanation: Present the cause before its effect when this order clarifies reasoning.

Example / answer: Funding was reduced; consequently, waiting times increased.

Problem to solution

Rule / explanation: Define the problem before evaluating responses.

Example / answer: The policy first created an access gap, which later reforms attempted to address.

Chronological order

Rule / explanation: Use time order when explaining a sequence of events.

Example / answer: The bill was introduced, amended and finally adopted.

Increasing complexity

Rule / explanation: Move from the most accessible point to more complex qualifications.

Example / answer: The reform reduced costs, but its wider social consequences are more difficult to assess.

E. Reference words

Personal pronouns

Rule / explanation: Use pronouns only when their antecedent is clear.

Example / answer: The committee rejected the proposal because it lacked funding.

Demonstratives

Rule / explanation: This, that, these and those connect a new sentence to previous information.

Example / answer: The policy increased costs. This outcome surprised ministers.

This + noun

Rule / explanation: Use a precise noun after this to avoid vague reference.

Example / answer: This decline reflects a broader loss of trust.

Such

Rule / explanation: Use such + noun to refer to a previously described type or situation.

Example / answer: Such measures may discourage small businesses.

The former / the latter

Rule / explanation: Use these only when exactly two clearly identified items are involved.

Example / answer: Regulation and taxation were discussed; the former received greater support.

F. Avoiding vague reference

Vague this

Rule / explanation: A sentence beginning with This is may leave the reader unsure what this refers to.

Example / answer: Weak: This is a problem.

Precise reference

Rule / explanation: Add a noun that names the previous idea.

Example / answer: Better: This lack of transparency is a serious problem.

Ambiguous pronoun

Rule / explanation: When two nouns could be the antecedent, repeat the key noun.

Example / answer: The minister criticised the journalist after the report appeared; the minister later apologised.

Overloaded they

Rule / explanation: Do not use they for several different groups in one paragraph.

Example / answer: Distinguish ministers, voters and journalists.

CPGE repair

Rule / explanation: Clarity is more important than avoiding repetition at all costs.

Example / answer: Repeat the exact key term when necessary.

G. Lexical repetition

Useful repetition

Rule / explanation: Repeating a key term can maintain focus and prevent ambiguity.

Example / answer: The reform changed funding rules. The reform also reduced local autonomy.

Controlled variation

Rule / explanation: Alternate the key term with accurate related expressions.

Example / answer: The reform / the measure / the policy.

Avoid thesaurus drift

Rule / explanation: Do not replace a precise word with an inaccurate synonym merely to avoid repetition.

Example / answer: A law is not always a policy, and a problem is not always a crisis.

Keyword chains

Rule / explanation: Use related vocabulary from the same conceptual field.

Example / answer: Trust, legitimacy, accountability and transparency.

Strategic emphasis

Rule / explanation: Deliberate repetition can reinforce the central issue.

Example / answer: Without trust, implementation becomes harder; without trust, legitimacy weakens.

H. Synonymy and lexical chains

Synonymy

Rule / explanation: Near-synonyms create variety while preserving topic continuity.

Example / answer: The proposal was rejected. This measure had faced criticism for months.

Superordinate terms

Rule / explanation: A broader category can refer back to a specific item.

Example / answer: Housing costs rose sharply. This economic pressure affected young households.

Specific terms

Rule / explanation: Move from a broad category to a precise example.

Example / answer: Several institutions were criticised, particularly Parliament.

Collocation

Rule / explanation: Natural word partnerships strengthen coherence and accuracy.

Example / answer: Restore trust, address inequality, implement reform.

Semantic field

Rule / explanation: Related words help the reader recognise the paragraph’s topic.

Example / answer: Voting, representation, participation and legitimacy.

I. Substitution

One / ones

Rule / explanation: Use one or ones to avoid repeating a countable noun.

Example / answer: The rural programme was more effective than the urban one.

Do so

Rule / explanation: Use do so to replace a previously mentioned action.

Example / answer: Several regions introduced reforms, but others refused to do so.

So

Rule / explanation: Use so after think, believe, expect, hope and suppose to replace a clause.

Example / answer: Will the policy succeed? Many analysts believe so.

Not

Rule / explanation: Use not as negative clause substitution after certain verbs.

Example / answer: Will the vote be delayed? Officials hope not.

Limits

Rule / explanation: Do not use substitution if it creates ambiguity or an unnatural sentence.

Example / answer: Repeat the noun or clause when clarity requires it.

J. Ellipsis

Definition

Rule / explanation: Ellipsis omits words that are understood from context.

Example / answer: Some regions supported the plan; others did not.

Comparative ellipsis

Rule / explanation: Avoid repeating material after comparative structures.

Example / answer: Urban areas benefited more than rural areas did.

Coordinated clauses

Rule / explanation: Shared material can be omitted in the second clause.

Example / answer: The policy reduced costs and increased efficiency.

Formal caution

Rule / explanation: Excessive ellipsis can make academic prose compressed or unclear.

Example / answer: Keep enough structure for effortless reading.

Oral usefulness

Rule / explanation: Ellipsis is natural in spoken English and can prevent repetition.

Example / answer: The first source is optimistic; the second, far less so.

K. Addition connectors

Furthermore

Rule / explanation: Add a substantial related point in formal writing.

Example / answer: Furthermore, the policy failed to improve access.

Moreover

Rule / explanation: Add information that strengthens the preceding argument.

Example / answer: Moreover, implementation costs exceeded the forecast.

In addition

Rule / explanation: Use this neutral phrase for an extra point.

Example / answer: In addition, local authorities received no new funding.

Likewise

Rule / explanation: Add a parallel point or comparable example.

Example / answer: Rural services declined; likewise, local transport deteriorated.

Also

Rule / explanation: Also is correct and flexible but should be positioned naturally.

Example / answer: The reform also reduced local autonomy.

L. Contrast connectors

However

Rule / explanation: Introduce a contrast with the previous statement.

Example / answer: The reform reduced costs. However, access remained unequal.

Nevertheless

Rule / explanation: Maintain a point despite a difficulty or opposing fact.

Example / answer: The evidence is limited. Nevertheless, the trend is significant.

By contrast

Rule / explanation: Compare clearly opposed situations or sources.

Example / answer: The first article is optimistic. By contrast, the second predicts prolonged instability.

Whereas

Rule / explanation: Contrast two clauses inside one sentence.

Example / answer: Urban access improved, whereas rural access declined.

On the other hand

Rule / explanation: Present another side of a genuine tension.

Example / answer: Regulation protects consumers; on the other hand, it may increase compliance costs.

M. Cause and consequence connectors

Because / since

Rule / explanation: Introduce a cause inside a sentence.

Example / answer: The programme failed because funding was insufficient.

Therefore

Rule / explanation: Introduce a logical consequence.

Example / answer: Funding was insufficient; therefore, the programme failed.

Consequently

Rule / explanation: Use this formal connector for a resulting effect.

Example / answer: The budget was cut. Consequently, waiting times increased.

As a result

Rule / explanation: Use this for a clear factual consequence.

Example / answer: Several clinics closed. As a result, access deteriorated.

Thus

Rule / explanation: Use this concise formal connector for inference or result.

Example / answer: The policy excluded rural areas, thus weakening its legitimacy.

N. Concession connectors

Although

Rule / explanation: Introduce a fact that contrasts with the main clause.

Example / answer: Although costs fell, regional inequality increased.

Despite

Rule / explanation: Use before a noun phrase or gerund.

Example / answer: Despite reducing costs, the reform remained unpopular.

Admittedly

Rule / explanation: Acknowledge a valid opposing point.

Example / answer: Admittedly, the policy improved short-term efficiency.

Even though

Rule / explanation: Express a stronger or more surprising concession.

Example / answer: Even though the economy recovered, trust continued to decline.

While

Rule / explanation: Concede one point before qualifying it.

Example / answer: While the article raises a valid concern, its evidence is incomplete.

O. Exemplification

For example

Rule / explanation: Introduce one illustration of a general claim.

Example / answer: Several regions faced shortages. For example, rural Wales lost three clinics.

For instance

Rule / explanation: A close formal alternative to for example.

Example / answer: Some reforms improved access; for instance, digital consultations reduced travel.

Such as

Rule / explanation: Introduce examples inside a noun phrase.

Example / answer: Services such as transport and healthcare were affected.

This is illustrated by

Rule / explanation: Connect evidence explicitly to the claim.

Example / answer: This is illustrated by the sharp decline in rural provision.

A case in point

Rule / explanation: Use this formal phrase for a particularly relevant example.

Example / answer: Scotland is a case in point.

P. Reformulation and clarification

In other words

Rule / explanation: Restate an idea more clearly or simply.

Example / answer: The reform lacked legitimacy; in other words, citizens did not regard it as justified.

More precisely

Rule / explanation: Narrow or correct the previous formulation.

Example / answer: The problem concerns access, or more precisely, unequal regional access.

That is

Rule / explanation: Provide concise clarification.

Example / answer: The measure was regressive; that is, it burdened poorer households more heavily.

To put it differently

Rule / explanation: Offer another formulation, especially in oral explanation.

Example / answer: To put it differently, efficiency was achieved at the expense of fairness.

Namely

Rule / explanation: Specify the exact item or items previously announced.

Example / answer: One issue remains, namely the absence of long-term funding.

Q. Sequence and organisation

First / first of all

Rule / explanation: Introduce the first stage of analysis.

Example / answer: First, the article identifies the economic causes.

Secondly / next

Rule / explanation: Continue an ordered explanation.

Example / answer: Secondly, it examines the political consequences.

Finally

Rule / explanation: Introduce the last main stage, not every concluding remark.

Example / answer: Finally, the author proposes regulatory reform.

Subsequently

Rule / explanation: Use this for chronological progression.

Example / answer: The bill was amended and subsequently adopted.

Turning to

Rule / explanation: Use this as an oral or written transition to a new analytical section.

Example / answer: Turning to the second source, the tone is markedly more cautious.

R. Given-to-new information flow

Given information

Rule / explanation: Begin from information already established or accessible to the reader.

Example / answer: This decline has several important consequences.

New information

Rule / explanation: Place the main new contribution later in the sentence.

Example / answer: The most serious consequence is the loss of institutional legitimacy.

Sentence linkage

Rule / explanation: The beginning of a sentence should connect naturally with what precedes it.

Example / answer: The reform altered local funding. This change affected rural services.

Avoid abrupt subjects

Rule / explanation: Do not introduce a completely new topic without a transition.

Example / answer: Signal the shift or begin a new paragraph.

Coherent rhythm

Rule / explanation: Given-to-new order creates a sense of forward movement.

Example / answer: Known point → added detail → analytical implication.

S. Theme and rheme

Theme

Rule / explanation: The theme is the starting point of the sentence, usually linking to prior context.

Example / answer: Regional inequality remains a major concern.

Rheme

Rule / explanation: The rheme contains the new information added about the theme.

Example / answer: Regional inequality remains a major concern because funding is distributed unevenly.

Thematic continuity

Rule / explanation: Repeated or related themes help maintain paragraph focus.

Example / answer: The reform... This measure... Its implementation...

Thematic progression

Rule / explanation: The new information of one sentence can become the starting point of the next.

Example / answer: The policy created a funding gap. This gap affected rural clinics.

Avoid monotony

Rule / explanation: Continuity does not require every sentence to begin identically.

Example / answer: Vary structures while preserving the logical chain.

T. Paragraph transitions

Backward link

Rule / explanation: A transition should connect the new paragraph to the reasoning already established.

Example / answer: These economic limitations also have political consequences.

Forward signal

Rule / explanation: It should indicate the new analytical direction.

Example / answer: The discussion must now turn to legitimacy.

Contrastive transition

Rule / explanation: Use a sentence that explains why a different perspective is needed.

Example / answer: Economic efficiency, however, is only one criterion of success.

Cumulative transition

Rule / explanation: Show that the new paragraph adds another layer.

Example / answer: Beyond its financial impact, the reform also transformed local governance.

Avoid mechanical labels

Rule / explanation: Do not rely only on First part, second part or now I will speak about.

Example / answer: Express the logical relationship instead.

U. Coherence in introductions and conclusions

Introduction funnel

Rule / explanation: Move from context to specific tension and then to the key question.

Example / answer: Recent reforms have reduced costs but widened regional disparities. The central issue is therefore how efficiency can be reconciled with fairness.

Announcing direction

Rule / explanation: A plan should reveal the logic of the response, not merely list topics.

Example / answer: The analysis will first examine the limits of the reform before considering the conditions of a fairer alternative.

Conclusion synthesis

Rule / explanation: Bring together the main answer rather than repeating each paragraph.

Example / answer: The reform improved efficiency, but only at a substantial social and political cost.

Answer the question

Rule / explanation: The final judgement must respond directly to the problématique.

Example / answer: Efficiency can support legitimacy only when benefits and burdens are distributed fairly.

Opening

Rule / explanation: A final opening should grow naturally from the argument.

Example / answer: This raises the wider question of how public institutions can measure success.

V. Common mistakes and exercises

Connector overload

Rule / explanation: Adding a connector to every sentence makes the text mechanical.

Example / answer: Use connectors only when they clarify a real relationship.

False logic

Rule / explanation: Do not use therefore when the second statement is not a consequence.

Example / answer: Check the exact relation before choosing the connector.

Vague this

Rule / explanation: Add a noun after this when the reference is unclear.

Example / answer: This policy / this decline / this evidence.

Irrelevant evidence

Rule / explanation: A detailed example weakens coherence if it does not support the paragraph’s point.

Example / answer: Choose evidence by function, not by interest.

Exercise 1

Rule / explanation: Improve reference: The policy reduced funding. This created shortages.

Example / answer: The policy reduced funding. This reduction created shortages.

Exercise 2

Rule / explanation: Add a contrast connector: Costs fell. Access remained unequal.

Example / answer: Costs fell; however, access remained unequal.

Exercise 3

Rule / explanation: Improve the topic sentence: There are many problems with the reform.

Example / answer: The reform’s main weakness is its uneven regional impact.

Exercise 4

Rule / explanation: Create logical progression from claim, evidence and implication.

Example / answer: The reform widened regional gaps. Rural waiting times rose by 18 per cent. This disparity therefore undermines the policy’s claim to fairness.

Exercise 5

Rule / explanation: Translate: Cette baisse a entraîné une perte de confiance, qui a elle-même affaibli la légitimité du gouvernement.

Example / answer: This decline led to a loss of trust, which in turn weakened the government’s legitimacy.

Mini concours challenge

Rule / explanation: Write a coherent paragraph using a topic sentence, precise reference, evidence, concession and concluding link.

Example / answer: Example: The reform’s principal weakness is its uneven regional impact. Although national waiting times fell, this improvement was concentrated in urban areas. In rural regions, by contrast, clinic closures reduced access to care. These disparities suggest that the programme improved aggregate efficiency without ensuring fairness. The reform can therefore be considered only a qualified success.
