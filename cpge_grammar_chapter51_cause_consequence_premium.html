A. Lesson — Expressing cause and consequence

Cause

Rule / explanation: A cause explains why a situation happens.

Example / answer: The reform failed because it lacked funding.

Consequence

Rule / explanation: A consequence explains what happens as a result.

Example / answer: The reform lacked funding; therefore, it failed.

Logical relationship

Rule / explanation: Strong writing makes the direction of the relationship explicit.

Example / answer: Rising prices led to a decline in household consumption.

CPGE relevance

Rule / explanation: Cause and consequence are essential for analysing events, policies, trends and arguments.

Example / answer: The article links political distrust to repeated institutional failures.

B. Because, since and as

Because

Rule / explanation: Use because to give a direct and important reason.

Example / answer: The policy was rejected because it was too expensive.

Since

Rule / explanation: Use since when the reason is already known or less central.

Example / answer: Since public support was declining, the government changed its strategy.

As

Rule / explanation: As can introduce a background reason in formal writing.

Example / answer: As the evidence remained incomplete, the committee delayed its decision.

Because of

Rule / explanation: Use because of before a noun, pronoun or gerund.

Example / answer: The vote was postponed because of widespread opposition.

Clause order

Rule / explanation: A reason clause may come before or after the main clause.

Example / answer: Because the reform lacked funding, it failed.

C. Due to, owing to and as a result of

Due to

Rule / explanation: Use due to before a noun phrase, especially after be.

Example / answer: The delay was due to administrative errors.

Owing to

Rule / explanation: Owing to is formal and introduces a reason.

Example / answer: Owing to rising costs, the project was abandoned.

As a result of

Rule / explanation: Use as a result of before a noun phrase to highlight causation.

Example / answer: As a result of weak regulation, financial risks increased.

On account of

Rule / explanation: This formal expression means because of.

Example / answer: The proposal was rejected on account of its excessive cost.

Thanks to

Rule / explanation: Use thanks to for a positive or ironic cause.

Example / answer: Thanks to improved funding, waiting times fell.

D. Therefore, thus and consequently

Therefore

Rule / explanation: Use therefore to introduce a logical consequence.

Example / answer: The policy lacked funding; therefore, it could not succeed.

Thus

Rule / explanation: Thus is concise and common in formal analysis.

Example / answer: The reform reduced competition, thus increasing prices.

Consequently

Rule / explanation: Consequently presents a result in formal writing.

Example / answer: Public trust declined. Consequently, turnout fell.

Hence

Rule / explanation: Hence is formal and often followed by a noun phrase or concise conclusion.

Example / answer: The evidence was unreliable, hence the need for further research.

Punctuation

Rule / explanation: These connectors do not join two independent clauses with only a comma.

Example / answer: Correct: Costs rose; therefore, demand fell.

E. So, so that and such...that

So

Rule / explanation: So introduces a direct result in neutral or less formal English.

Example / answer: The policy was unpopular, so the government withdrew it.

So + adjective + that

Rule / explanation: Use so...that to express an intense cause leading to a result.

Example / answer: The crisis was so severe that Parliament intervened.

Such + noun + that

Rule / explanation: Use such...that with a noun phrase.

Example / answer: It was such a divisive proposal that even supporters withdrew.

So much / many / little / few that

Rule / explanation: Use these quantifier patterns to express degree and result.

Example / answer: The reform created so many problems that it had to be revised.

So that for purpose

Rule / explanation: Do not confuse result so...that with purpose so that.

Example / answer: The rules were simplified so that citizens could understand them.

F. Lead to, result in and cause

Lead to

Rule / explanation: Use lead to + noun or gerund for a consequence.

Example / answer: Poor planning led to higher costs.

Result in

Rule / explanation: Use result in + noun or gerund for an outcome.

Example / answer: The cuts resulted in longer waiting times.

Cause

Rule / explanation: Cause is direct and often strong.

Example / answer: The reform caused widespread confusion.

Trigger

Rule / explanation: Trigger suggests that one event starts another process.

Example / answer: The announcement triggered a wave of protests.

Contribute to

Rule / explanation: Use contribute to when a factor is only one cause among several.

Example / answer: Low wages contributed to growing social unrest.

G. Result from, stem from and be caused by

Result from

Rule / explanation: Use result from to introduce the cause of a situation.

Example / answer: The crisis resulted from years of underinvestment.

Stem from

Rule / explanation: Stem from is formal and common in analysis.

Example / answer: Public distrust stems from repeated political scandals.

Arise from

Rule / explanation: Arise from means originate in or develop from.

Example / answer: Many difficulties arise from unclear regulations.

Be caused by

Rule / explanation: Use the passive form when the result is more important than the agent.

Example / answer: The decline was caused by rising energy prices.

Be attributable to

Rule / explanation: This formal expression presents a probable or measurable cause.

Example / answer: The improvement is largely attributable to increased funding.

H. Advanced cause structures

The reason why...is that

Rule / explanation: Use this structure to identify a central explanation.

Example / answer: The reason why the reform failed is that it ignored local needs.

This is because

Rule / explanation: Use this to explain the previous statement.

Example / answer: Turnout declined. This is because many voters felt ignored.

Which is why

Rule / explanation: Use which is why to connect a fact to its consequence.

Example / answer: The evidence was incomplete, which is why the inquiry continued.

Given that

Rule / explanation: Use given that to introduce an accepted fact as a reason.

Example / answer: Given that costs are rising, further investment seems unlikely.

Insofar as

Rule / explanation: Use insofar as to limit a causal claim to a particular respect.

Example / answer: The policy succeeded insofar as it reduced short-term unemployment.

By virtue of

Rule / explanation: This formal phrase means because of a particular quality or status.

Example / answer: The institution has influence by virtue of its legal authority.

I. Causal language for nuanced analysis

Main cause

Rule / explanation: Use primary, principal or underlying cause for a major factor.

Example / answer: The principal cause of the crisis was chronic underinvestment.

Contributing factor

Rule / explanation: Use this when one factor plays a partial role.

Example / answer: Political instability was a significant contributing factor.

Immediate cause

Rule / explanation: Use immediate cause for the event directly preceding the result.

Example / answer: The immediate cause of the protests was the fuel-price increase.

Underlying cause

Rule / explanation: Use underlying cause for a deeper structural explanation.

Example / answer: The underlying cause was persistent regional inequality.

Correlation vs causation

Rule / explanation: Do not claim causation merely because two trends occur together.

Example / answer: The data show a correlation, but they do not prove that one trend caused the other.

J. Cause and consequence in data commentary

Increase due to

Rule / explanation: Use due to when data suggest a clear cause.

Example / answer: The rise in unemployment was partly due to factory closures.

Resulting in

Rule / explanation: Use a participle clause to present a consequence concisely.

Example / answer: Energy prices increased sharply, resulting in higher production costs.

Driven by

Rule / explanation: Use driven by to identify the force behind a trend.

Example / answer: The decline was driven by lower consumer demand.

Reflecting

Rule / explanation: Use reflecting to interpret what a trend indicates.

Example / answer: Turnout fell, reflecting growing political disengagement.

Associated with

Rule / explanation: Use associated with when causation is uncertain.

Example / answer: Higher inequality is associated with lower social mobility.

K. Cause and consequence in CPGE essays

Explain a phenomenon

Rule / explanation: Build a chain from structural cause to immediate effect.

Example / answer: Long-term underinvestment weakened services, which in turn reduced public confidence.

Avoid monocausal analysis

Rule / explanation: Complex issues usually have several interacting causes.

Example / answer: The crisis cannot be explained by inflation alone; weak institutions and poor planning also played a role.

Causal chain

Rule / explanation: Use several linked stages to show development.

Example / answer: Rising costs reduced investment, which slowed growth and ultimately increased unemployment.

Evaluation

Rule / explanation: Assess whether the alleged cause is convincing.

Example / answer: The author attributes the decline to regulation, but the evidence suggests that demand was the more important factor.

Conclusion

Rule / explanation: Summarise the most convincing causal relationship.

Example / answer: Ultimately, the failure resulted less from excessive ambition than from poor implementation.

L. Typical mistakes by French speakers

Because of + clause

Rule / explanation: Because of must be followed by a noun phrase or gerund, not a finite clause.

Example / answer: Wrong: because of the policy was costly. Correct: because the policy was costly / because of its cost.

Due to misuse

Rule / explanation: In careful formal writing, due to most naturally follows a form of be.

Example / answer: Natural: The delay was due to bad weather. Also common: Due to bad weather, the event was cancelled.

Lead to + infinitive

Rule / explanation: Lead to is followed by a noun or gerund.

Example / answer: Wrong: led to increase prices. Correct: led to higher prices / led to prices increasing.

Result in / result from

Rule / explanation: Result in introduces the consequence; result from introduces the cause.

Example / answer: Poor planning resulted in delays. The delays resulted from poor planning.

Therefore comma splice

Rule / explanation: Do not join two independent clauses with a comma before therefore.

Example / answer: Wrong: Costs rose, therefore demand fell. Correct: Costs rose; therefore, demand fell.

Overstating causation

Rule / explanation: Use contribute to, be associated with or may explain when evidence is uncertain.

Example / answer: Avoid claiming that one factor caused a complex phenomenon without proof.

M. Exercises

1. Combine with because

Rule / explanation: The reform failed. It lacked funding.

Example / answer: The reform failed because it lacked funding.

2. Rewrite with because of

Rule / explanation: The vote was postponed because the public strongly opposed the bill.

Example / answer: The vote was postponed because of strong public opposition.

3. Use therefore

Rule / explanation: Costs increased. Demand fell.

Example / answer: Costs increased; therefore, demand fell.

4. Choose result in or result from

Rule / explanation: Poor planning ___ long delays.

Example / answer: Poor planning resulted in long delays.

5. Choose result in or result from

Rule / explanation: The long delays ___ poor planning.

Example / answer: The long delays resulted from poor planning.

6. Correct the sentence

Rule / explanation: The reform led to increase unemployment.

Example / answer: The reform led to an increase in unemployment. / The reform led to unemployment increasing.

7. Translate

Rule / explanation: La méfiance du public découle de scandales politiques répétés.

Example / answer: Public distrust stems from repeated political scandals.

8. Mini concours challenge

Rule / explanation: Write a paragraph using because, contribute to, therefore, resulting in and stem from.

Example / answer: Example: The policy failed because it ignored local conditions. Weak coordination also contributed to its poor implementation. Costs therefore increased rapidly, resulting in public opposition. Much of this distrust stemmed from earlier government failures.
