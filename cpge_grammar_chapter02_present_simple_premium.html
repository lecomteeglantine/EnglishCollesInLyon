Chapter 2 — Present Simple
A2 foundation grammar for facts, habits, institutions, press summaries and essay claims.

Chapter objective

This premium grammar chapter helps CPGE students master the Present Simple, one of the most useful tenses for essays, khôlles and press article analysis. It covers affirmative forms, negatives, questions, third-person -s, common time markers and typical mistakes made by French speakers.

The golden rule

Subject + Present Simple verb
I / you / we / they work — he / she / it works
Main useFacts, habits, routines, stable situations, article summaries and general arguments.
CPGE habitUse the Present Simple to summarise what an article does: The article argues / shows / criticises / highlights...
Key trapAfter does, the verb returns to the base form: Does the article show...?, not Does the article shows...?

A. Lesson — What is the Present Simple?

Present Simple

French: présent simple

Rule / explanation: Used for facts, habits, routines, general truths and repeated situations.

Example / answer: The media shape public opinion.

General facts

French: vérités générales

Rule / explanation: Use it for things that are generally true.

Example / answer: Democracy requires informed citizens.

Habits and routines

French: habitudes

Rule / explanation: Use it for repeated actions or usual behaviour.

Example / answer: Many students read the press every week.

Permanent situations

French: situations permanentes

Rule / explanation: Use it for stable or long-term situations.

Example / answer: The UK has an uncodified constitution.

Opinions and beliefs

French: opinions et croyances

Rule / explanation: Use it with verbs such as think, believe, argue, claim, support.

Example / answer: The author argues that inequality is increasing.

State verbs

French: verbes d’état

Rule / explanation: Use it with be, have, know, need, seem, depend, belong.

Example / answer: The issue seems more complex than it first appears.

Present Simple vs now

French: présent général

Rule / explanation: The Present Simple does not mean “now”; it means general or repeated.

Example / answer: Young voters often care about climate change.

Press article use

French: usage presse

Rule / explanation: Use it to describe facts, institutions and arguments.

Example / answer: The article focuses on the cost-of-living crisis.

B. Form — Affirmative sentences

I / You / We / They

French: forme de base

Rule / explanation: Use the base verb.

Example / answer: They support the reform.

He / She / It

French: troisième personne du singulier

Rule / explanation: Add -s or -es to the verb.

Example / answer: The government supports the reform.

Regular -s

French: -s simple

Rule / explanation: Most verbs add -s.

Example / answer: The article explains the debate.

Verbs ending in -s, -sh, -ch, -x, -o

French: -es

Rule / explanation: Add -es.

Example / answer: The crisis goes beyond economics.

Consonant + y

French: y devient -ies

Rule / explanation: Change y to -ies.

Example / answer: The report identifies several causes.

Vowel + y

French: voyelle + y

Rule / explanation: Add -s only.

Example / answer: The survey shows what voters say.

Be

French: être

Rule / explanation: Forms: am / is / are.

Example / answer: The argument is convincing.

Have

French: avoir

Rule / explanation: Forms: have / has.

Example / answer: The country has a strong welfare tradition.

C. Negative and question forms

Do not / does not

French: négation

Rule / explanation: Use do not / does not + base verb.

Example / answer: The article does not mention the historical context.

Don't / doesn't

French: formes contractées

Rule / explanation: Contractions are common in speech, less formal in essays.

Example / answer: The policy doesn't address poverty.

No -s after does

French: pas de -s après does

Rule / explanation: After does / does not, use the base verb.

Example / answer: Wrong: Does the article shows? Correct: Does the article show?

Do questions

French: questions avec do

Rule / explanation: Use do + subject + base verb.

Example / answer: Do young people trust politicians?

Does questions

French: questions avec does

Rule / explanation: Use does + he/she/it + base verb.

Example / answer: Does the reform help low-income families?

Wh- questions

French: questions en wh-

Rule / explanation: Wh-word + do/does + subject + base verb.

Example / answer: Why does the author criticise the reform?

Be negatives

French: négation avec be

Rule / explanation: With be, use am not / is not / are not.

Example / answer: The figures are not convincing.

Be questions

French: questions avec be

Rule / explanation: With be, put be before the subject.

Example / answer: Is the article balanced?

D. Adverbs of frequency and time markers

always

French: toujours

Rule / explanation: 100% frequency; before the main verb, after be.

Example / answer: Politicians always promise change.

usually

French: habituellement

Rule / explanation: Most of the time.

Example / answer: Voters usually care about the economy.

often

French: souvent

Rule / explanation: Frequently.

Example / answer: The press often focuses on controversy.

sometimes

French: parfois

Rule / explanation: Occasionally.

Example / answer: Governments sometimes ignore early warnings.

rarely

French: rarement

Rule / explanation: Not often.

Example / answer: The article rarely mentions rural voters.

never

French: jamais

Rule / explanation: 0% frequency; do not use with another negative.

Example / answer: The author never claims that reform is easy.

every + time expression

French: chaque

Rule / explanation: Used for repeated actions.

Example / answer: The newspaper publishes political analysis every Sunday.

once / twice a week

French: une/deux fois par semaine

Rule / explanation: Used for frequency.

Example / answer: Students practise grammar twice a week.

E. Common CPGE uses

Describing institutions

French: institutions

Rule / explanation: Use the Present Simple for political systems and institutions.

Example / answer: Parliament debates and passes laws.

Summarising an article

French: résumé d’article

Rule / explanation: Use it to say what the article says or does.

Example / answer: The article examines the rise of populism.

Presenting an argument

French: argumentation

Rule / explanation: Use it for the author’s position.

Example / answer: The journalist argues that trust is declining.

Explaining causes

French: causes

Rule / explanation: Use it for general causal links.

Example / answer: Economic insecurity fuels resentment.

Comparing systems

French: comparaison

Rule / explanation: Use it for stable differences.

Example / answer: The US separates powers more strictly than the UK.

Describing trends generally

French: tendances

Rule / explanation: Use it with show, reveal, suggest, indicate.

Example / answer: The graph shows a steady rise in rents.

Discussing society

French: société

Rule / explanation: Use it for general social observations.

Example / answer: Social media amplifies political anger.

Making essay claims

French: dissertation

Rule / explanation: Use it for thesis statements.

Example / answer: Democracy depends on public trust.

F. Typical mistakes by French speakers

Forgetting the -s

French: oubli du -s

Rule / explanation: He/she/it needs -s in affirmative sentences.

Example / answer: Wrong: The article show. Correct: The article shows.

Adding -s after does

French: -s après does

Rule / explanation: Never add -s after does.

Example / answer: Wrong: Does the government supports reform? Correct: Does the government support reform?

Using Present Continuous for general facts

French: présent continu abusif

Rule / explanation: Do not use be + V-ing for permanent truths.

Example / answer: Wrong: The UK is having an uncodified constitution. Correct: The UK has an uncodified constitution.

Forgetting do in questions

French: oubli de do

Rule / explanation: Most Present Simple questions need do/does.

Example / answer: Wrong: Why people vote? Correct: Why do people vote?

Forgetting do in negatives

French: oubli de do négatif

Rule / explanation: Most Present Simple negatives need do/does not.

Example / answer: Wrong: The article not mention. Correct: The article does not mention.

Using be with ordinary verbs

French: be inutile

Rule / explanation: Do not use is before a normal verb in the Present Simple.

Example / answer: Wrong: The article is explain. Correct: The article explains.

Misplacing adverbs

French: place des adverbes

Rule / explanation: Frequency adverbs usually go before the main verb, after be.

Example / answer: Correct: The media often exaggerate scandals. / The media are often criticised.

Overusing simple present for now

French: confusion avec présent en cours

Rule / explanation: Use Present Continuous for actions happening now.

Example / answer: Wrong: The journalist writes now. Correct: The journalist is writing now.

G. Exercises

Exercise 1 — Add the correct form

French: exercice

Rule / explanation: The article ___ (show) that inequality is rising.

Example / answer: Answer: The article shows that inequality is rising.

Exercise 2 — Add the correct form

French: exercice

Rule / explanation: Many voters ___ (feel) ignored by political elites.

Example / answer: Answer: Many voters feel ignored by political elites.

Exercise 3 — Make it negative

French: exercice

Rule / explanation: The policy protects vulnerable families.

Example / answer: Answer: The policy does not protect vulnerable families.

Exercise 4 — Make it negative

French: exercice

Rule / explanation: The journalist supports the reform.

Example / answer: Answer: The journalist does not support the reform.

Exercise 5 — Make a question

French: exercice

Rule / explanation: The article focuses on climate change.

Example / answer: Answer: Does the article focus on climate change?

Exercise 6 — Make a question

French: exercice

Rule / explanation: Young people trust traditional media.

Example / answer: Answer: Do young people trust traditional media?

Exercise 7 — Correct the mistake

French: exercice

Rule / explanation: The government want to reduce inequality.

Example / answer: Answer: The government wants to reduce inequality.

Exercise 8 — Correct the mistake

French: exercice

Rule / explanation: Does the article criticises capitalism?

Example / answer: Answer: Does the article criticise capitalism?

Exercise 9 — Correct the mistake

French: exercice

Rule / explanation: The author not explains the causes of the crisis.

Example / answer: Answer: The author does not explain the causes of the crisis.

Exercise 10 — Translate

French: exercice

Rule / explanation: L’article montre que les loyers augmentent.

Example / answer: Answer: The article shows that rents are increasing. / The article shows that rents increase over time.

Exercise 11 — Translate

French: exercice

Rule / explanation: Le gouvernement ne comprend pas les jeunes électeurs.

Example / answer: Answer: The government does not understand young voters.

Exercise 12 — Mini concours challenge

French: exercice

Rule / explanation: Write 4 Present Simple sentences about social media: one fact, one negative, one question, one author argument.

Example / answer: Possible answer: Social media shapes public debate. It does not always improve democracy. Does it increase polarisation? The author argues that platforms reward outrage.
