Chapter 48 — Gradation and Intensification
B2 → C1 grammar for precise degree, nuanced judgement, data commentary and persuasive argumentation.

Chapter objective

This premium grammar chapter helps CPGE students express degree and intensity precisely. It covers intensifiers, downtoners, extreme adjectives, modified comparisons, data-commentary language and high-level structures for cautious or forceful argumentation.

The golden rule

Choose the degree that matches the evidence
slightly problematic → somewhat serious → highly controversial → utterly unacceptable
Main useGradation expresses how strong, weak, certain, serious or important something is.
CPGE habitUse precise degree markers instead of repeating very or making unsupported absolute claims.
Key trapIntensifier and adjective must be compatible: highly controversial, absolutely impossible, slightly lower.

A. Lesson — What are gradation and intensification?

Gradation

Rule / explanation: Gradation expresses different degrees of intensity, importance, certainty or change.

Example / answer: The reform is slightly controversial, highly divisive and potentially explosive.

Intensification

Rule / explanation: Intensifiers strengthen an adjective, adverb, verb or comparison.

Example / answer: The article is highly persuasive.

Downtoning

Rule / explanation: Downtoners reduce the force of a statement.

Example / answer: The evidence is somewhat limited.

CPGE relevance

Rule / explanation: These structures help avoid absolute claims and express precise analytical judgement.

Example / answer: The policy is deeply flawed but only partially ineffective.

B. Intensifiers with adjectives

Highly

Rule / explanation: Use highly with evaluative adjectives.

Example / answer: The reform is highly controversial.

Deeply

Rule / explanation: Use deeply with attitudes, emotions and serious states.

Example / answer: Citizens are deeply concerned.

Strongly

Rule / explanation: Use strongly with opinions, reactions and participles.

Example / answer: The proposal is strongly opposed by unions.

Utterly / completely

Rule / explanation: Use these with extreme or absolute adjectives.

Example / answer: The policy proved utterly ineffective.

C. Downtoners and cautious degree

Slightly

Rule / explanation: Use slightly for a small difference or change.

Example / answer: Turnout was slightly lower than expected.

Somewhat

Rule / explanation: Use somewhat for cautious criticism.

Example / answer: The conclusion is somewhat simplistic.

Fairly

Rule / explanation: Fairly expresses a moderate degree and is often neutral.

Example / answer: The argument is fairly convincing.

Relatively

Rule / explanation: Use relatively when judging something against a standard or context.

Example / answer: The reform remains relatively popular.

D. Extreme and non-gradable adjectives

Absolutely

Rule / explanation: Use absolutely with many non-gradable adjectives.

Example / answer: The evidence is absolutely essential.

Entirely

Rule / explanation: Use entirely with states that can be total.

Example / answer: The policy is entirely ineffective.

Virtually

Rule / explanation: Virtually means almost completely.

Example / answer: The opposition was virtually unanimous.

Avoid very + extreme adjective

Rule / explanation: Prefer a compatible intensifier.

Example / answer: Better: absolutely impossible, not very impossible.

E. Modifying comparatives

Much / far

Rule / explanation: Use much or far to express a large difference.

Example / answer: The second article is far more nuanced than the first.

Significantly

Rule / explanation: Use significantly in formal comparisons and data commentary.

Example / answer: Unemployment was significantly higher in rural areas.

Marginally

Rule / explanation: Use marginally for very small differences.

Example / answer: The new policy is marginally more effective.

By far

Rule / explanation: Use by far with superlatives or strong comparisons.

Example / answer: This is by far the most convincing argument.

F. Degree with verbs and nouns

Greatly

Rule / explanation: Greatly often modifies verbs of change, influence or appreciation.

Example / answer: The reform greatly increased public spending.

Severely

Rule / explanation: Severely modifies negative effects or restrictions.

Example / answer: The cuts severely weakened public services.

A considerable amount of

Rule / explanation: Use with uncountable nouns.

Example / answer: The reform generated a considerable amount of criticism.

A significant number of

Rule / explanation: Use with plural countable nouns.

Example / answer: A significant number of voters remain undecided.

G. Gradation in argumentation

Escalation

Rule / explanation: Move from a weaker point to a stronger one.

Example / answer: The reform is costly, deeply unpopular and potentially unsustainable.

Qualification

Rule / explanation: Use degree markers to avoid overstatement.

Example / answer: The policy was only partially successful.

Contrast in degree

Rule / explanation: Compare intensity precisely.

Example / answer: Economic damage was significant, but political damage was even greater.

Layered judgement

Rule / explanation: Combine intensification and downtoning for nuance.

Example / answer: The article is highly persuasive overall, though somewhat weak on historical context.

H. Intensification in data commentary

Sharp rise

Rule / explanation: Use sharp, steep or dramatic for large increases.

Example / answer: There was a sharp rise in unemployment.

Slight decline

Rule / explanation: Use slight, modest or marginal for small decreases.

Example / answer: The graph shows a slight decline in turnout.

Remain broadly stable

Rule / explanation: Broadly stable means mostly unchanged despite small variation.

Example / answer: Support remained broadly stable throughout the period.

Increase substantially

Rule / explanation: Substantially is useful for large measurable change.

Example / answer: Housing costs increased substantially after 2020.

I. High-level CPGE formulations

To a large extent

Rule / explanation: Use this to express strong but not total agreement.

Example / answer: To a large extent, the author’s criticism is justified.

To some extent

Rule / explanation: Use this for partial agreement.

Example / answer: To some extent, the reform addressed the immediate crisis.

By no means

Rule / explanation: Use this formal phrase to reject an absolute judgement.

Example / answer: The policy is by no means a complete failure.

All the more

Rule / explanation: Use all the more to intensify a point because of an additional reason.

Example / answer: The issue is all the more urgent because inequality is rising.

J. Typical mistakes by French speakers

Very + extreme adjective

Rule / explanation: Do not normally combine very with adjectives that already express an extreme.

Example / answer: Wrong: very impossible. Better: absolutely impossible.

Strongly + adjective

Rule / explanation: Strongly usually modifies verbs or participles, not ordinary adjectives.

Example / answer: Better: highly controversial / strongly opposed.

Much + positive adjective

Rule / explanation: Much usually modifies comparatives, not basic adjectives.

Example / answer: Wrong: much important. Correct: much more important.

Redundant intensification

Rule / explanation: Avoid stacking several intensifiers.

Example / answer: Heavy: absolutely completely impossible. Better: absolutely impossible.

K. Exercises

1. Choose the best intensifier

Rule / explanation: The reform is ___ controversial. (highly / strongly)

Example / answer: The reform is highly controversial.

2. Correct the sentence

Rule / explanation: The proposal is very impossible to implement.

Example / answer: The proposal is almost impossible to implement.

3. Downtone the claim

Rule / explanation: The article is simplistic.

Example / answer: The article is somewhat simplistic.

4. Intensify the comparison

Rule / explanation: The second text is more convincing than the first.

Example / answer: The second text is far more convincing than the first.

5. Rewrite formally

Rule / explanation: A lot of voters remain undecided.

Example / answer: A significant number of voters remain undecided.

6. Translate

Rule / explanation: La réforme n’est en aucun cas un échec total.

Example / answer: The reform is by no means a complete failure.
