Chapter objective

This premium grammar chapter helps CPGE students master parallel structure: keeping lists, comparisons, coordinated clauses and correlative constructions grammatically balanced. It also develops rhetorical rhythm for essays and khôlles.

A. Lesson — What is parallel structure?

Parallel structure

Rule / explanation: Parallel structure means using the same grammatical pattern for ideas of equal importance.

Example / answer: The policy aims to reduce poverty, improve housing and restore trust.

Why it matters

Rule / explanation: Parallelism makes writing clearer, more balanced and more persuasive.

Example / answer: The article is concise, well documented and carefully argued.

Grammatical balance

Rule / explanation: Items in a list or comparison should have matching forms.

Example / answer: The government promised to invest, to reform and to regulate.

CPGE relevance

Rule / explanation: Parallelism strengthens essays, article summaries, introductions and conclusions.

Example / answer: The debate concerns not only economic growth but also social cohesion.

B. Parallel nouns, verbs and adjectives

Parallel nouns

Rule / explanation: Use nouns with nouns.

Example / answer: The reform concerns housing, education and healthcare.

Parallel verbs

Rule / explanation: Use verbs in the same form.

Example / answer: The policy seeks to reduce inequality, create jobs and improve services.

Parallel adjectives

Rule / explanation: Use adjectives with adjectives.

Example / answer: The article is clear, balanced and persuasive.

Parallel clauses

Rule / explanation: Use clauses with matching structure.

Example / answer: The author argues that the policy is costly, that it is ineffective and that it lacks legitimacy.

C. Parallel infinitives and gerunds

Infinitive series

Rule / explanation: Repeat to before every verb, or use it once before the first verb.

Example / answer: The government wants to cut taxes, reduce spending and stimulate growth.

Repeated to for emphasis

Rule / explanation: Repeating to can make each aim sound more deliberate.

Example / answer: The reform seeks to protect workers, to regulate platforms and to restore confidence.

Gerund series

Rule / explanation: Use -ing forms consistently.

Example / answer: The policy focuses on reducing costs, improving access and supporting families.

Avoid mixing forms

Rule / explanation: Do not mix infinitives and gerunds in the same series without a clear reason.

Example / answer: Wrong: reducing costs, to improve access and supporting families.

D. Correlative conjunctions

Both...and

Rule / explanation: The two elements must have parallel grammatical form.

Example / answer: The reform affects both workers and employers.

Either...or

Rule / explanation: Keep the alternatives grammatically balanced.

Example / answer: The government must either raise taxes or reduce spending.

Neither...nor

Rule / explanation: Use matching structures after neither and nor.

Example / answer: The article neither explains the causes nor proposes solutions.

Not only...but also

Rule / explanation: Keep both parts parallel and balanced.

Example / answer: The policy not only increases costs but also weakens public trust.

E. Parallel comparison structures

As...as

Rule / explanation: The compared elements should have the same grammatical role.

Example / answer: The second article is as detailed as the first.

More...than

Rule / explanation: Compare equivalent structures.

Example / answer: The policy protects large firms more than it supports small businesses.

Rather than

Rule / explanation: Use parallel forms on both sides.

Example / answer: The government chose to delay reform rather than confront opposition.

Instead of

Rule / explanation: Use a noun or gerund after instead of.

Example / answer: Instead of reducing inequality, the policy widened the gap.

F. Parallelism in argumentation

Three-part argument

Rule / explanation: A three-part parallel structure creates rhythm and clarity.

Example / answer: The reform is expensive, ineffective and politically risky.

Cause sequence

Rule / explanation: Use matching noun phrases for causes.

Example / answer: The crisis resulted from weak regulation, poor planning and political delay.

Consequence sequence

Rule / explanation: Use matching verb phrases for consequences.

Example / answer: The policy increased costs, reduced access and undermined trust.

Recommendation sequence

Rule / explanation: Use matching structures for proposals.

Example / answer: Governments should invest in education, strengthen institutions and improve transparency.

G. Parallelism in introductions and conclusions

Thesis statement

Rule / explanation: Use balanced structures to present the main line of argument.

Example / answer: The issue is not merely economic but also social and political.

Problematique

Rule / explanation: Parallel structures clarify the tension at the centre of the essay.

Example / answer: The question is whether reform can be both effective and legitimate.

Conclusion

Rule / explanation: Use parallelism to create a memorable final sentence.

Example / answer: Democracy depends on trust, participation and accountability.

Oral delivery

Rule / explanation: Parallel rhythm makes khôlle presentations easier to follow.

Example / answer: The article informs, challenges and ultimately persuades.

H. Advanced rhetorical parallelism

Anaphora

Rule / explanation: Repeat the same opening structure for rhetorical emphasis.

Example / answer: We need better policy. We need stronger institutions. We need greater trust.

Antithesis

Rule / explanation: Balance contrasting ideas in parallel form.

Example / answer: The reform promised stability but produced uncertainty.

Climactic order

Rule / explanation: Arrange parallel elements from weaker to stronger.

Example / answer: The scandal damaged reputations, weakened institutions and threatened democracy.

Balanced sentence

Rule / explanation: Match two clauses of similar length and structure.

Example / answer: The government demanded patience; the public demanded results.

I. Typical mistakes by French speakers

Mixed forms

Rule / explanation: Do not mix nouns, infinitives and gerunds in one list.

Example / answer: Wrong: reducing costs, better access and to support families.

Broken correlative structure

Rule / explanation: Place both, either, neither and not only directly before parallel elements.

Example / answer: Better: The policy affects both workers and employers.

Unequal comparisons

Rule / explanation: Make sure the compared elements are logically equivalent.

Example / answer: Wrong: The economy of France is stronger than Germany. Better: stronger than Germany’s economy.

Missing repeated words

Rule / explanation: Sometimes repeating a preposition or article improves clarity.

Example / answer: The debate concerns the role of the state, of markets and of citizens.

Overloading

Rule / explanation: Long parallel lists lose impact.

Example / answer: Prefer three strong elements to six weak ones.

False balance

Rule / explanation: Parallel grammar does not guarantee logical balance.

Example / answer: Each item must also belong to the same category.

J. Exercises

1. Correct the list

Rule / explanation: The policy aims to reduce poverty, improving housing and greater equality.

Example / answer: The policy aims to reduce poverty, improve housing and promote greater equality.

2. Correct the infinitives

Rule / explanation: The government wants to cut taxes, reducing spending and to stimulate growth.

Example / answer: The government wants to cut taxes, reduce spending and stimulate growth.

3. Complete with both...and

Rule / explanation: The reform affects workers. It affects employers.

Example / answer: The reform affects both workers and employers.

4. Complete with neither...nor

Rule / explanation: The article does not explain the causes. It does not propose solutions.

Example / answer: The article neither explains the causes nor proposes solutions.

5. Correct the comparison

Rule / explanation: The economy of Britain is larger than Ireland.

Example / answer: Britain’s economy is larger than Ireland’s.

6. Rewrite with parallelism

Rule / explanation: The reform is expensive. It does not work. It is politically risky.

Example / answer: The reform is expensive, ineffective and politically risky.

7. Translate

Rule / explanation: La démocratie dépend de la confiance, de la participation et de la responsabilité.

Example / answer: Democracy depends on trust, participation and accountability.

8. Mini concours challenge

Rule / explanation: Write a three-sentence CPGE paragraph using one three-part list, one correlative structure and one antithesis.

Example / answer: Example: The policy is costly, divisive and ineffective. It harms both workers and employers. It promises stability but creates uncertainty.
