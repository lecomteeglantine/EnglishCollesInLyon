B1 → B2 grammar for academic precision, evidence, data, quantity and essay writing.

Chapter objective

This premium grammar chapter helps CPGE students master countable and uncountable nouns. It covers academic nouns such as evidence, information, research, advice, countable essay nouns, quantifiers, agreement, article-summary patterns and common French-speaker mistakes.

The golden rule

Countable: one argument / several arguments
Uncountable: evidence / information / research
Use a piece of evidence, not an evidence.
Main useThis distinction controls articles, plural forms, quantifiers and verb agreement.
CPGE habitMemorise common academic uncountables: evidence, information, research, advice, knowledge, progress.
Key trapDo not write informations, advices or an evidence.

A. Lesson — Countable vs uncountable nouns

Countable nouns

French: noms dénombrables

Rule / explanation: Countable nouns refer to things that can be counted as separate units.

Example / answer: The article presents three arguments.

Uncountable nouns

French: noms indénombrables

Rule / explanation: Uncountable nouns refer to substances, abstract ideas or masses that are not normally counted individually.

Example / answer: The article provides useful evidence.

Singular countable noun

French: nom dénombrable singulier

Rule / explanation: A singular countable noun usually needs a determiner such as a/an, the, this or my.

Example / answer: The author gives a convincing example.

Plural countable noun

French: nom dénombrable pluriel

Rule / explanation: Plural countable nouns can be used with numbers and many.

Example / answer: Many voters are concerned about inflation.

No plural -s for uncountable nouns

French: pas de pluriel

Rule / explanation: Uncountable nouns do not normally take plural -s.

Example / answer: Wrong: informations. Correct: information.

A piece of

French: un élément de

Rule / explanation: Use a piece of to count one item of an uncountable noun.

Example / answer: The journalist gives a piece of evidence.

General meaning

French: sens général

Rule / explanation: Plural countable nouns and uncountable nouns often take no article when used generally.

Example / answer: Politicians need evidence. / Democracies need institutions.

CPGE use

French: usage CPGE

Rule / explanation: This distinction is essential for essays, article summaries, statistics and academic precision.

Example / answer: The author lacks evidence but offers several examples.

B. Common uncountable nouns in academic English

Evidence

French: preuve / éléments probants

Rule / explanation: Uncountable; use evidence or a piece of evidence.

Example / answer: The article provides strong evidence.

Information

French: information

Rule / explanation: Uncountable; use information or pieces of information.

Example / answer: The report contains useful information.

Research

French: recherche

Rule / explanation: Uncountable when referring to academic work in general.

Example / answer: Recent research suggests that inequality has increased.

Advice

French: conseil

Rule / explanation: Uncountable; use advice or a piece of advice.

Example / answer: The examiner gave useful advice.

Knowledge

French: connaissance / savoir

Rule / explanation: Uncountable.

Example / answer: Education gives citizens knowledge and confidence.

Progress

French: progrès

Rule / explanation: Uncountable.

Example / answer: Little progress has been made.

Data

French: données

Rule / explanation: Usually plural in formal academic English, but often treated as mass in general use.

Example / answer: The data show a rise in inequality.

News

French: nouvelles / actualités

Rule / explanation: Uncountable despite final -s.

Example / answer: The news is worrying.

C. Countable nouns useful in CPGE essays

Argument

French: argument

Rule / explanation: Countable; use an argument, several arguments, many arguments.

Example / answer: The author develops three arguments.

Example

French: exemple

Rule / explanation: Countable.

Example / answer: The article gives a powerful example.

Factor

French: facteur

Rule / explanation: Countable; useful for causation.

Example / answer: Several factors explain this trend.

Policy

French: politique publique

Rule / explanation: Countable; plural policies.

Example / answer: The government introduced two new policies.

Measure

French: mesure

Rule / explanation: Countable; useful for reforms.

Example / answer: The measure was announced yesterday.

Consequence

French: conséquence

Rule / explanation: Countable; often plural.

Example / answer: The consequences are difficult to predict.

Issue

French: problème / enjeu

Rule / explanation: Countable; useful for article summaries.

Example / answer: This issue remains politically sensitive.

Trend

French: tendance

Rule / explanation: Countable; useful for data commentary.

Example / answer: This trend may continue.

D. Quantifiers with countable and uncountable nouns

Many + plural countable

French: beaucoup de

Rule / explanation: Use many with plural countable nouns.

Example / answer: Many citizens distrust institutions.

Much + uncountable

French: beaucoup de

Rule / explanation: Use much with uncountable nouns, especially in negatives and questions.

Example / answer: The article does not provide much evidence.

A lot of + both

French: beaucoup de

Rule / explanation: A lot of works with countable and uncountable nouns, but is less formal.

Example / answer: A lot of voters are worried. / A lot of evidence is missing.

Several + plural countable

French: plusieurs

Rule / explanation: Several is formal and useful in essays.

Example / answer: Several factors explain the crisis.

Few + plural countable

French: peu de

Rule / explanation: Few means not many; it often has a negative meaning.

Example / answer: Few politicians discuss poverty seriously.

Little + uncountable

French: peu de

Rule / explanation: Little means not much; it often has a negative meaning.

Example / answer: Little progress has been made.

A few

French: quelques

Rule / explanation: A few means some, enough to mention.

Example / answer: A few examples support the author’s claim.

A little

French: un peu de

Rule / explanation: A little means some, enough to mention.

Example / answer: The article offers a little evidence.

E. Some, any, no and enough

Some

French: du / des / quelques

Rule / explanation: Use some in affirmative sentences for an unspecified quantity.

Example / answer: The article gives some examples.

Any

French: du / des / aucun

Rule / explanation: Use any in negatives, questions and non-specific contexts.

Example / answer: The article does not give any evidence.

No

French: aucun

Rule / explanation: Use no before a noun to mean not any.

Example / answer: No solution is perfect.

Enough

French: assez de

Rule / explanation: Use enough before nouns to mean sufficient.

Example / answer: The author does not provide enough evidence.

Not enough

French: pas assez de

Rule / explanation: Useful for criticism.

Example / answer: There is not enough data to support the claim.

Some evidence

French: des éléments probants

Rule / explanation: Evidence is uncountable, so no plural -s.

Example / answer: There is some evidence that trust has declined.

Any arguments

French: des arguments

Rule / explanation: Argument is countable, so plural is possible after any.

Example / answer: Does the author provide any convincing arguments?

No clear answer

French: aucune réponse claire

Rule / explanation: Useful in nuanced conclusions.

Example / answer: There is no clear answer to this question.

F. Making uncountable nouns countable

A piece of evidence

French: un élément de preuve

Rule / explanation: Use a piece of to count evidence.

Example / answer: The article provides one piece of evidence.

A piece of information

French: une information

Rule / explanation: Use a piece of to count information.

Example / answer: This is a useful piece of information.

A piece of advice

French: un conseil

Rule / explanation: Use a piece of to count advice.

Example / answer: The teacher gave students a piece of advice.

An item of news

French: une nouvelle

Rule / explanation: Possible but less common; often use a news item.

Example / answer: This news item shocked the public.

A research paper

French: un article de recherche

Rule / explanation: Use paper/study/article to count research outputs.

Example / answer: The author cites three research papers.

A data point

French: une donnée

Rule / explanation: Use data point to refer to one item of data.

Example / answer: One data point is not enough.

A fact

French: un fait

Rule / explanation: Use fact when referring to a countable piece of reality.

Example / answer: The author mentions several facts.

A statistic

French: une statistique

Rule / explanation: Use statistic for one numerical fact.

Example / answer: This statistic supports the author’s claim.

G. Singular, plural and agreement

This evidence is

French: accord singulier

Rule / explanation: Uncountable nouns take singular agreement.

Example / answer: This evidence is convincing.

These examples are

French: accord pluriel

Rule / explanation: Plural countable nouns take plural agreement.

Example / answer: These examples are relevant.

The news is

French: news singulier

Rule / explanation: News is uncountable and singular in agreement.

Example / answer: The news is alarming.

The data show

French: data pluriel

Rule / explanation: In formal writing, data is often treated as plural.

Example / answer: The data show a steady decline.

Each + singular noun

French: each + singulier

Rule / explanation: Each is followed by a singular noun.

Example / answer: Each argument must be justified.

Every + singular noun

French: every + singulier

Rule / explanation: Every is followed by a singular noun.

Example / answer: Every example should be relevant.

Several + plural noun

French: several + pluriel

Rule / explanation: Several is followed by a plural countable noun.

Example / answer: Several studies support this view.

A number of + plural noun

French: un certain nombre de

Rule / explanation: A number of is followed by a plural countable noun.

Example / answer: A number of voters opposed the reform.

H. CPGE article-summary and essay patterns

The article provides evidence that

French: l’article fournit des preuves que

Rule / explanation: Useful for evaluating the strength of a text.

Example / answer: The article provides evidence that inequality has increased.

The author gives several examples of

French: l’auteur donne plusieurs exemples de

Rule / explanation: Useful for article summaries.

Example / answer: The author gives several examples of political distrust.

There is little evidence that

French: il y a peu de preuves que

Rule / explanation: Useful for criticism.

Example / answer: There is little evidence that the policy will work.

A number of factors explain

French: un certain nombre de facteurs expliquent

Rule / explanation: Useful for structured argumentation.

Example / answer: A number of factors explain the decline in trust.

Few examples are convincing

French: peu d’exemples sont convaincants

Rule / explanation: Useful for nuanced criticism.

Example / answer: Few examples are convincing enough.

Not enough data are provided

French: pas assez de données

Rule / explanation: Useful for data-based criticism.

Example / answer: Not enough data are provided to justify the conclusion.

The extent of the problem

French: l’ampleur du problème

Rule / explanation: Useful abstract noun phrase.

Example / answer: The article underestimates the extent of the problem.

The lack of evidence

French: le manque de preuves

Rule / explanation: Useful critical phrase.

Example / answer: The lack of evidence weakens the argument.

I. Typical mistakes by French speakers

Informations

French: information

Rule / explanation: Information is uncountable in English.

Example / answer: Wrong: informations. Correct: information / pieces of information.

Evidences

French: evidence

Rule / explanation: Evidence is uncountable in most academic English.

Example / answer: Wrong: evidences. Correct: evidence / pieces of evidence.

Advices

French: advice

Rule / explanation: Advice is uncountable.

Example / answer: Wrong: advices. Correct: advice / pieces of advice.

A research

French: research

Rule / explanation: Research is uncountable when general.

Example / answer: Wrong: a research. Correct: research / a study / a research paper.

Many evidence

French: many/much

Rule / explanation: Use much evidence or a lot of evidence.

Example / answer: Wrong: many evidence. Correct: much evidence.

Much arguments

French: much/many

Rule / explanation: Use many arguments.

Example / answer: Wrong: much arguments. Correct: many arguments.

An evidence

French: article with uncountable noun

Rule / explanation: Use a piece of evidence.

Example / answer: Wrong: an evidence. Correct: a piece of evidence.

The data shows

French: data agreement

Rule / explanation: In formal English, prefer the data show.

Example / answer: Formal: The data show a clear trend.
