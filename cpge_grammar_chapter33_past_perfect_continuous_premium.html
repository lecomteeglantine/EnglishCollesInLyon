The golden rule

had been + verb-ing
Public trust had been declining for years before the scandal broke.
Main useUse the Past Perfect Continuous to show that an action, process or trend had been continuing before another past moment.
CPGE habitUse it for long-term background causes: Inequality had been rising for years before the crisis became visible.
Key trapPast Perfect Simple = completed result. Past Perfect Continuous = duration, process, repeated activity.

A. Lesson — What is the Past Perfect Continuous?

Duration before a past point

French: durée avant un moment passé

Rule / explanation: It describes an action or situation that continued up to a past moment.

Example / answer: The government had been facing criticism for months before it changed course.

Past background process

French: processus d’arrière-plan

Rule / explanation: It highlights a process that explains a later past event.

Example / answer: Rents had been rising steadily before housing became a national issue.

Visible past result

French: résultat visible dans le passé

Rule / explanation: It can explain why a past situation looked or felt a certain way.

Example / answer: Voters were angry because wages had been stagnating for years.

Analytical value

French: valeur analytique

Rule / explanation: It shows that crises often result from long processes, not sudden accidents.

Example / answer: Trust had been eroding long before the election.

B. Form and structure

Affirmative

French: forme affirmative

Rule / explanation: Use had been + verb-ing for all subjects.

Example / answer: The economy had been slowing down.

Negative

French: forme négative

Rule / explanation: Use had not been / hadn’t been + verb-ing.

Example / answer: The government had not been listening to local communities.

Question

French: question

Rule / explanation: Put had before the subject.

Example / answer: Had voters been losing trust before the scandal?

Wh-question

French: question en wh-

Rule / explanation: Use wh-word + had + subject + been + V-ing.

Example / answer: Why had support been declining?

C. Past Perfect Continuous vs Past Perfect Simple

Process vs result

French: processus vs résultat

Rule / explanation: Continuous focuses on the process; simple focuses on completion or result.

Example / answer: Trust had been declining for years. / Trust had declined by 20%.

Duration

French: durée

Rule / explanation: Use Past Perfect Continuous with for, since and all morning/year/decade.

Example / answer: Inequality had been increasing since the financial crisis.

Completed number or amount

French: quantité terminée

Rule / explanation: Use Past Perfect Simple with completed quantities.

Example / answer: The author had analysed three reports before writing the article.

Repeated activity

French: activité répétée

Rule / explanation: Use continuous to stress repeated ongoing activity before a past moment.

Example / answer: Activists had been warning ministers for months.

D. Time markers and duration

For

French: depuis / pendant

Rule / explanation: Use for + duration.

Example / answer: The party had been losing support for several months.

Since

French: depuis

Rule / explanation: Use since + starting point.

Example / answer: Public services had been deteriorating since 2010.

All day / all year / for years

French: toute la journée / pendant des années

Rule / explanation: These phrases stress duration and repetition.

Example / answer: Journalists had been investigating the scandal for weeks.

Before / by the time

French: avant / au moment où

Rule / explanation: Use these to connect the ongoing past process to a later past moment.

Example / answer: By the time the report appeared, voters had been waiting for answers for months.

E. CPGE article-summary and essay patterns

Long-term crisis

French: crise de long terme

Rule / explanation: Use it to show that a crisis developed over time.

Example / answer: The article shows that the housing crisis had been worsening for years.

Political distrust

French: méfiance politique

Rule / explanation: Useful for explaining voter behaviour.

Example / answer: Voters had been losing faith in institutions long before the election.

Social trends

French: tendances sociales

Rule / explanation: Use it for trends that continued before a later past event.

Example / answer: Inequality had been rising steadily before the reform was introduced.

Media investigation

French: enquête médiatique

Rule / explanation: Useful for describing journalism and scandals.

Example / answer: Reporters had been investigating the case before the minister resigned.

F. Common verbs with Past Perfect Continuous

Had been rising / falling

French: augmentait / diminuait depuis

Rule / explanation: Excellent for data and trends.

Example / answer: Inflation had been rising before the policy changed.

Had been growing / declining

French: croissait / déclinait depuis

Rule / explanation: Useful for abstract trends.

Example / answer: Support for the party had been declining steadily.

Had been trying

French: essayait depuis

Rule / explanation: Shows repeated effort before a past result.

Example / answer: The government had been trying to reassure voters.

Had been warning

French: avertissait depuis

Rule / explanation: Useful for ignored expert advice.

Example / answer: Experts had been warning about the risks for years.

G. Typical mistakes by French speakers

Forgetting been

French: oubli de been

Rule / explanation: The structure needs had been + V-ing.

Example / answer: Wrong: Trust had declining. Correct: Trust had been declining.

Using it for completed results

French: résultat terminé

Rule / explanation: Use Past Perfect Simple for completed results or numbers.

Example / answer: Better: The author had read three reports, not had been reading three reports.

State verbs

French: verbes d’état

Rule / explanation: Avoid continuous with many state verbs such as know, believe, own.

Example / answer: Wrong: voters had been knowing. Correct: voters had known.

Present vs past link

French: présent vs passé

Rule / explanation: Present Perfect Continuous links past to now; Past Perfect Continuous links earlier past to later past.

Example / answer: Trust has been declining recently. / Trust had been declining before the election.
