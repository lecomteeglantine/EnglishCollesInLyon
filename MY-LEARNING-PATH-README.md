# My Learning Path

`learning-path.html` is the central progress dashboard. It reads existing browser-only progress from Grammar, Methodology, Pronunciation, Vocabulary and the Guided Learning Plan.

It also stores two dashboard-only values:

- `ecl_learning_path_settings_v1`: the optional date of the next colle;
- `ecl_learning_activity_v1`: recent pages visited on this device.

The shared header now links to **My Learning Path**. Progress export/import remains available from the dashboard and continues to include all localStorage keys.
