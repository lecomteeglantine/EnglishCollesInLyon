2. Import my progress

Use this on the new device. Choose the JSON file downloaded from the old device, then confirm the import.

How to transfer your progress
  • On the old device, open this page and download the backup.
  • Send the JSON file to yourself or save it on a USB drive or cloud storage.
  • On the new device, open this same page and select the backup file.
  • After the import, return to the site and refresh the page.
