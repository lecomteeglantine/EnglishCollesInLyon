Chapter 37 — First Conditional
B1 → B2 grammar for realistic future conditions, warnings, consequences and policy analysis.

Chapter objective

This premium grammar chapter helps CPGE students master the First Conditional. It covers if + Present Simple, will + verb, real future consequences, warnings, promises, unless, provided that, as long as, modal alternatives and common French-speaker mistakes.

The golden rule

If + Present Simple, will + base verb
If the government ignores the crisis, public trust will decline.
Main useUse the First Conditional for realistic or possible future situations and their likely consequences.
CPGE habitUse it for policy analysis: If rents keep rising, younger voters will demand stronger regulation.
Key trapDo not use will in the if-clause: If voters lose trust, not If voters will lose trust.

A. Lesson — What is the First Conditional?

Real future condition

French: condition réelle future

Rule / explanation: It describes a possible future condition and its likely result.

Example / answer: If inflation remains high, households will reduce spending.

Likely consequence

French: conséquence probable

Rule / explanation: The result clause expresses what will probably happen if the condition is met.

Example / answer: If the policy fails, the opposition will criticise the government.

Future warning

French: avertissement futur

Rule / explanation: The structure is useful for warnings and predictions.

Example / answer: If no action is taken, inequality will worsen.

CPGE relevance

French: pertinence CPGE

Rule / explanation: It is essential for discussing reforms, crises, climate, democracy and social policy.

Example / answer: If citizens feel ignored, they will turn away from mainstream parties.

B. Form and punctuation

If-clause first

French: subordonnée en premier

Rule / explanation: When the if-clause comes first, use a comma before the result clause.

Example / answer: If the reform is unpopular, the government will change its strategy.

Result clause first

French: principale en premier

Rule / explanation: When the result clause comes first, no comma is usually needed.

Example / answer: The government will change its strategy if the reform is unpopular.

Present Simple in if-clause

French: présent après if

Rule / explanation: Use Present Simple after if, even when the meaning is future.

Example / answer: If voters reject the bill, Parliament will reconsider it.

Will in result clause

French: will dans le résultat

Rule / explanation: Use will + base verb in the main clause to express the consequence.

Example / answer: If the article convinces readers, it will influence the debate.

C. Modal alternatives to will

If + present, may

French: possibilité

Rule / explanation: Use may when the consequence is possible but not certain.

Example / answer: If the crisis continues, voters may lose confidence.

If + present, might

French: possibilité plus prudente

Rule / explanation: Might is more tentative than may.

Example / answer: If the evidence is weak, the conclusion might be challenged.

If + present, can

French: possibilité générale

Rule / explanation: Can expresses ability, possibility or general consequence.

Example / answer: If social media spreads misinformation, it can damage democracy.

If + present, should / must

French: conseil / obligation

Rule / explanation: Use should for advice and must for necessity.

Example / answer: If the data are incomplete, students should be cautious.

D. Unless, provided that, as long as

Unless

French: sauf si / à moins que

Rule / explanation: Unless means if not. Do not usually add another negative.

Example / answer: Unless the government acts quickly, the crisis will deepen.

Provided that

French: à condition que

Rule / explanation: More formal than if; useful in academic and policy contexts.

Example / answer: The reform will work provided that it is properly funded.

As long as

French: tant que / à condition que

Rule / explanation: Means only if a condition continues to be true.

Example / answer: Democracy will remain stable as long as citizens trust institutions.

In case

French: au cas où

Rule / explanation: In case means to prepare for a possible situation, not exactly if.

Example / answer: The committee prepared alternatives in case the bill failed.

E. First Conditional in argumentation

Policy consequence

French: conséquence politique

Rule / explanation: Use it to explain the probable effect of a policy.

Example / answer: If the government cuts benefits, poverty will increase.

Climate warning

French: alerte climatique

Rule / explanation: Use it for realistic environmental consequences.

Example / answer: If emissions continue to rise, extreme weather will become more frequent.

Democratic risk

French: risque démocratique

Rule / explanation: Useful for social and political analysis.

Example / answer: If voters feel powerless, turnout will fall.

Article conclusion

French: conclusion d’article

Rule / explanation: Use it to sum up likely consequences.

Example / answer: If the author is right, the reform will not solve the deeper problem.

F. First Conditional vs Zero Conditional

Zero Conditional

French: vérité générale

Rule / explanation: Use if + Present Simple, Present Simple for general truths.

Example / answer: If institutions lose credibility, democracy suffers.

First Conditional

French: situation future probable

Rule / explanation: Use if + Present Simple, will + verb for specific future consequences.

Example / answer: If this reform fails, democracy will suffer.

General vs specific

French: général vs spécifique

Rule / explanation: Zero Conditional gives a general rule; First Conditional gives a likely future case.

Example / answer: If prices rise, families struggle. / If prices rise next year, families will struggle.

Essay use

French: usage dissertation

Rule / explanation: Use zero for principles, first for predictions.

Example / answer: If democracy excludes citizens, it weakens. If the current system excludes them, it will weaken further.

G. Typical mistakes by French speakers

Will after if

French: pas de will après if

Rule / explanation: Use Present Simple in the if-clause.

Example / answer: Wrong: If voters will reject the bill. Correct: If voters reject the bill.

Unless + negative

French: double négation

Rule / explanation: Unless already means if not.

Example / answer: Wrong: Unless the government does not act. Correct: Unless the government acts.

Using future in time clauses

French: après when / as soon as

Rule / explanation: Use Present Simple after when, as soon as, before, after and until when referring to the future.

Example / answer: Wrong: When voters will decide. Correct: When voters decide.

Confusing if and whether

French: si conditionnel vs si interrogatif

Rule / explanation: If can introduce a condition; whether reports uncertainty between options.

Example / answer: If the bill passes, it will matter. / The article asks whether the bill will pass.

H. Exercises

Exercise 1 — Complete

French: exercice

Rule / explanation: If inflation ___ (remain) high, households ___ (struggle).

Example / answer: remains / will struggle

Exercise 2 — Correct

French: exercice

Rule / explanation: If voters will lose trust, turnout will fall.

Example / answer: If voters lose trust, turnout will fall.

Exercise 3 — Rewrite with unless

French: exercice

Rule / explanation: If the government does not act, the crisis will worsen.

Example / answer: Unless the government acts, the crisis will worsen.

Exercise 4 — Correct

French: exercice

Rule / explanation: Unless the article does not provide evidence, it will be convincing.

Example / answer: If the article provides evidence, it will be convincing. / Unless the article lacks evidence, it will be convincing.

Exercise 5 — Translate

French: exercice

Rule / explanation: Si le gouvernement ignore la crise, la confiance diminuera.

Example / answer: If the government ignores the crisis, trust will decline.

Exercise 6 — Mini concours challenge

French: exercice

Rule / explanation: Write 5 First Conditional sentences about climate, housing, democracy, education and AI.

Example / answer: If AI transforms the labour market too quickly, governments will need stronger training policies.
