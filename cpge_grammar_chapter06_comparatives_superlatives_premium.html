A. Lesson — What are comparatives and superlatives?

Comparative

French: comparatif

Rule / explanation: A comparative compares two people, things, ideas or situations.

Example / answer: Housing is more expensive than it used to be.

Superlative

French: superlatif

Rule / explanation: A superlative identifies the highest or lowest degree within a group.

Example / answer: Housing is one of the most urgent issues today.

Comparison of superiority

French: comparatif de supériorité

Rule / explanation: Use -er or more to say that something has a higher degree.

Example / answer: This argument is stronger than the previous one.

Comparison of inferiority

French: comparatif d’infériorité

Rule / explanation: Use less to say that something has a lower degree.

Example / answer: This solution is less effective than expected.

Comparison of equality

French: comparatif d’égalité

Rule / explanation: Use as... as to say that two things are equal in degree.

Example / answer: The issue is as political as it is economic.

Superiority superlative

French: superlatif de supériorité

Rule / explanation: Use the -est or the most to identify the highest degree.

Example / answer: This is the most convincing argument in the article.

Inferiority superlative

French: superlatif d’infériorité

Rule / explanation: Use the least to identify the lowest degree.

Example / answer: This is the least realistic proposal.

CPGE use

French: usage CPGE

Rule / explanation: Comparatives and superlatives help students compare countries, policies, arguments, data and historical periods.

Example / answer: The US system is more polarised than the British system.

B. Form — Short and long adjectives

One-syllable adjectives

French: adjectifs courts

Rule / explanation: Add -er for the comparative and -est for the superlative.

Example / answer: small → smaller → the smallest

One-syllable ending in -e

French: adjectifs en -e

Rule / explanation: Add -r and -st.

Example / answer: large → larger → the largest

One vowel + one consonant

French: double consonne

Rule / explanation: Double the final consonant.

Example / answer: big → bigger → the biggest

Consonant + y

French: y devient i

Rule / explanation: Change y to -ier / -iest.

Example / answer: easy → easier → the easiest

Long adjectives

French: adjectifs longs

Rule / explanation: Use more / the most.

Example / answer: important → more important → the most important

Two-syllable adjectives

French: adjectifs de deux syllabes

Rule / explanation: Some take -er, others take more. When in doubt, use more.

Example / answer: clever → cleverer / more clever; complex → more complex

Less / the least

French: moins / le moins

Rule / explanation: Use less and the least for inferiority.

Example / answer: less convincing / the least convincing

Than

French: que comparatif

Rule / explanation: Use than after a comparative.

Example / answer: This reform is more ambitious than the previous one.

C. Irregular comparatives and key forms

Good / well

French: bon / bien

Rule / explanation: good or well → better → the best.

Example / answer: This source is better than the previous one.

Bad / badly

French: mauvais / mal

Rule / explanation: bad or badly → worse → the worst.

Example / answer: The situation is worse than expected.

Far

French: loin

Rule / explanation: far → farther/further → the farthest/furthest.

Example / answer: The consequences go further than the article suggests.

Little

French: peu

Rule / explanation: little → less → the least.

Example / answer: The policy gives less support to poorer families.

Much / many

French: beaucoup

Rule / explanation: much/many → more → the most.

Example / answer: More voters are turning away from mainstream parties.

Old

French: vieux / âgé

Rule / explanation: old → older/elder → the oldest/eldest.

Example / answer: Older voters were more likely to support the reform.

Late

French: tard / récent

Rule / explanation: late → later → the latest.

Example / answer: The latest figures confirm the trend.

Near

French: proche

Rule / explanation: near → nearer → the nearest.

Example / answer: The nearest example is the 2008 crisis.

D. Comparatives in academic argumentation

Much / far / a lot + comparative

French: beaucoup plus

Rule / explanation: Use these to make a comparison stronger.

Example / answer: The issue is far more complex than it appears.

Slightly / a little + comparative

French: légèrement plus

Rule / explanation: Use these to make a comparison weaker.

Example / answer: The second argument is slightly more convincing.

No longer

French: ne plus

Rule / explanation: Use it to show a change from past to present.

Example / answer: The system is no longer as stable as it once was.

Increasingly

French: de plus en plus

Rule / explanation: Use it to show a growing trend.

Example / answer: Politics is becoming increasingly polarised.

More and more

French: de plus en plus

Rule / explanation: Use it to describe a gradual increase.

Example / answer: More and more citizens distrust institutions.

Less and less

French: de moins en moins

Rule / explanation: Use it to describe a gradual decrease.

Example / answer: Traditional media are becoming less and less influential.

The more..., the more...

French: plus... plus...

Rule / explanation: Use it to show parallel development.

Example / answer: The more prices rise, the more voters feel insecure.

The less..., the more...

French: moins... plus...

Rule / explanation: Use it to show inverse development.

Example / answer: The less people trust institutions, the more vulnerable democracy becomes.

E. Common CPGE uses

Comparing countries

French: comparaison de pays

Rule / explanation: Use comparatives to compare political, social or economic systems.

Example / answer: The US is more polarised than Canada.

Comparing periods

French: comparaison historique

Rule / explanation: Use comparatives to compare past and present.

Example / answer: Politics is more fragmented today than it was twenty years ago.

Comparing policies

French: comparaison de politiques

Rule / explanation: Use comparatives to evaluate solutions.

Example / answer: Public investment may be more effective than tax cuts.

Comparing arguments

French: comparaison d’arguments

Rule / explanation: Use comparatives to assess reasoning.

Example / answer: The economic argument is stronger than the cultural one.

Commenting on data

French: commentaire de données

Rule / explanation: Use comparatives to interpret figures.

Example / answer: Youth unemployment is higher than the national average.

Nuancing claims

French: nuance

Rule / explanation: Use less, slightly, somewhat, far, much to avoid overstatement.

Example / answer: The reform is somewhat less radical than critics claim.

Essay introductions

French: introduction

Rule / explanation: Use comparatives to frame tensions.

Example / answer: The debate has become more urgent in recent years.

Conclusions

French: conclusion

Rule / explanation: Use superlatives to identify the main issue carefully.

Example / answer: The most important challenge is to rebuild public trust.

F. Typical mistakes by French speakers

Using more with -er

French: double comparatif

Rule / explanation: Do not combine more and -er.

Example / answer: Wrong: more easier. Correct: easier.

Forgetting than

French: oubli de than

Rule / explanation: Use than after a comparative.

Example / answer: Wrong: more important that. Correct: more important than.

Using that instead of than

French: that au lieu de than

Rule / explanation: French que becomes than after a comparative.

Example / answer: Wrong: richer that. Correct: richer than.

Forgetting the before superlatives

French: oubli de the

Rule / explanation: Most superlatives need the.

Example / answer: Wrong: most important issue. Correct: the most important issue.

Using as... than

French: as... as

Rule / explanation: Use as... as for equality, not than.

Example / answer: Wrong: as important than. Correct: as important as.

Bad word order

French: ordre des mots

Rule / explanation: Place the comparative before the noun when it modifies a noun.

Example / answer: Correct: a more convincing argument.

Confusing fewer and less

French: fewer / less

Rule / explanation: Use fewer with countable plural nouns; less with uncountable nouns.

Example / answer: fewer jobs, less money.

Overusing best

French: best abusif

Rule / explanation: Best is strong; in essays, justify it or use one of the most.

Example / answer: Safer: one of the most serious problems.

Example / answer: Answer: stronger

Example / answer: Answer: more serious

Example / answer: Answer: the most important

Example / answer: Answer: The reform is as controversial as the previous one.
