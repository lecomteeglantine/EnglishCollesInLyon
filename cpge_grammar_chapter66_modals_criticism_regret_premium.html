Chapter 66 — Modals of Criticism and Regret
B2 → C1 grammar for hindsight, missed opportunities, institutional evaluation and nuanced counterfactual analysis.

Chapter objective

This premium grammar chapter develops advanced control of past modal criticism and regret: should have, could have, might have, needn’t have, passive modal perfects, failed expectations and balanced CPGE evaluation.

Level B2 → C1 Should Have Could / Might Have Needn’t Have Hindsight Exercises + Answers

The golden rule

Past judgement = modal + have + past participle
should have = advisable but not done
shouldn’t have = done but inadvisable
could / might have = unrealised or possible alternative
Criticism of omissionshould have + past participle
Criticism of actionshouldn’t have + past participle
Missed possibilitycould have / might have + past participle
Unnecessary completed actionneedn’t have + past participle

A. Lesson — Criticism, regret and missed alternatives

Past modal criticism

Rule / explanation: English often uses a modal + have + past participle to judge a past action or imagine a different past choice.

Example / answer: The government should have consulted local authorities.

Regret

Rule / explanation: Regret focuses on a past action or omission that the speaker now wishes had been different.

Example / answer: Ministers should not have ignored the warning.

Missed opportunity

Rule / explanation: Could have and might have can describe an alternative that was possible but did not happen.

Example / answer: Earlier intervention could have prevented the crisis.

CPGE relevance

Rule / explanation: These structures are essential for evaluating policies, historical decisions, arguments and failures without resorting to simplistic blame.

Example / answer: The authorities might have limited the damage by acting sooner.

B. Should have + past participle

Past criticism

Rule / explanation: Use should have when an action was advisable, expected or morally preferable but did not happen.

Example / answer: The committee should have verified the figures.

Past regret

Rule / explanation: The speaker may regret their own past action or omission.

Example / answer: I should have checked the source more carefully.

Expected result

Rule / explanation: Should have can also mean that something was expected to happen, although the speaker is unsure whether it did.

Example / answer: The report should have arrived by now.

Context decides

Rule / explanation: The surrounding context distinguishes criticism from expectation.

Example / answer: Ministers should have acted sooner = criticism.

Negative form

Rule / explanation: Use should not have or shouldn’t have for an action that happened but was inadvisable.

Example / answer: The newspaper should not have published the names.

C. Ought to have + past participle

Formal equivalent

Rule / explanation: Ought to have is close to should have but often sounds more formal or moralising.

Example / answer: The government ought to have protected vulnerable households.

Moral duty

Rule / explanation: It is particularly useful for ethical judgement.

Example / answer: Officials ought to have disclosed the conflict of interest.

Negative form

Rule / explanation: Use ought not to have for past criticism.

Example / answer: The agency ought not to have ignored the safety concerns.

Register

Rule / explanation: In modern English, should have is more common, especially in speech.

Example / answer: Use ought to have selectively in formal analysis.

CPGE nuance

Rule / explanation: Ought to have can emphasise institutional responsibility.

Example / answer: Democratic institutions ought to have ensured greater transparency.

D. Could have + past participle

Missed opportunity

Rule / explanation: Could have describes a possible action that did not occur.

Example / answer: The government could have negotiated before imposing the measure.

Alternative outcome

Rule / explanation: It can introduce a plausible different result.

Example / answer: Earlier consultation could have reduced public opposition.

Mild criticism

Rule / explanation: Could have often criticises someone for failing to use an available option.

Example / answer: The author could have considered regional differences.

Past possibility

Rule / explanation: Could have can also mean that something possibly happened; context must clarify the meaning.

Example / answer: The discrepancy could have resulted from a technical error.

Conditional support

Rule / explanation: An if-clause often makes the unrealised meaning explicit.

Example / answer: The reform could have succeeded if it had been properly funded.

E. Might have + past participle

Tentative missed possibility

Rule / explanation: Might have can describe a weaker or more speculative alternative past.

Example / answer: A different communication strategy might have prevented the backlash.

Mild reproach

Rule / explanation: In speech, might have can express restrained criticism.

Example / answer: They might have warned residents earlier.

Past possibility

Rule / explanation: It may also mean that an event possibly occurred.

Example / answer: The minister might have misunderstood the figures.

Counterfactual result

Rule / explanation: In a conditional context, the result did not occur.

Example / answer: With greater investment, the programme might have succeeded.

Academic caution

Rule / explanation: Use might have when the alternative outcome remains uncertain.

Example / answer: The policy might have produced better results under different conditions.

F. Needn’t have + past participle

Unnecessary completed action

Rule / explanation: Needn’t have means that someone did something, but the action was unnecessary.

Example / answer: The committee needn’t have repeated the entire inquiry.

Action definitely happened

Rule / explanation: Unlike didn’t need to, needn’t have confirms that the action occurred.

Example / answer: They needn’t have purchased new equipment, but they did.

Mild criticism

Rule / explanation: The form may criticise wasted time, effort or resources.

Example / answer: The government needn’t have delayed the announcement.

British English

Rule / explanation: Needn’t have is especially common in British English.

Example / answer: The agency needn’t have commissioned another report.

Negative meaning

Rule / explanation: Despite the negative form, the action itself was completed.

Example / answer: She needn’t have apologised = she apologised, but it was unnecessary.

G. Didn’t need to vs needn’t have

Didn’t need to

Rule / explanation: This means there was no necessity; it does not always tell us whether the action happened.

Example / answer: They didn’t need to publish the data that day.

Needn’t have

Rule / explanation: This means the action definitely happened and was unnecessary.

Example / answer: They needn’t have published the data that day.

Context with completed action

Rule / explanation: A following clause may clarify whether the action occurred after didn’t need to.

Example / answer: They didn’t need to travel, so they stayed home.

Common confusion

Rule / explanation: Do not use needn’t have when you mean that an unnecessary action was avoided.

Example / answer: Correct: We didn’t need to attend, so we stayed home.

CPGE precision

Rule / explanation: The distinction matters when evaluating wasted action versus absent obligation.

Example / answer: The ministry needn’t have duplicated the study.

H. Was / were supposed to

Unfulfilled expectation

Rule / explanation: Was supposed to describes a planned, expected or required past action that may not have happened.

Example / answer: The report was supposed to be published in June.

Failure or delay

Rule / explanation: Context often implies that the expected action did not occur.

Example / answer: The committee was supposed to consult regional authorities.

Rules and duties

Rule / explanation: It can describe institutional obligation.

Example / answer: Officials were supposed to declare potential conflicts of interest.

Negative form

Rule / explanation: Wasn’t supposed to describes a prohibited or unexpected past action.

Example / answer: The document wasn’t supposed to be released.

Compared with should have

Rule / explanation: Was supposed to reports an expectation; should have directly evaluates what was right.

Example / answer: The report was supposed to appear. / It should have appeared.

I. Was / were meant to

Intended purpose

Rule / explanation: Was meant to expresses intention, design or expected function.

Example / answer: The reform was meant to reduce waiting times.

Failed purpose

Rule / explanation: It often highlights a gap between intention and result.

Example / answer: The policy was meant to simplify the system but made it more complex.

Personal expectation

Rule / explanation: It can describe an arrangement that did not happen.

Example / answer: The minister was meant to attend the hearing.

Negative form

Rule / explanation: Wasn’t meant to may express lack of intention or prohibition.

Example / answer: The preliminary figures weren’t meant to be published.

Analytical use

Rule / explanation: This form is particularly effective for contrasting declared aims with actual outcomes.

Example / answer: The measure was meant to improve access, yet rural inequalities widened.

J. Had better have

Rare past warning

Rule / explanation: Had better normally concerns present or future advice; had better have + past participle is rare and refers to an expected completed action.

Example / answer: You had better have submitted the form by Friday.

Strong expectation

Rule / explanation: The speaker expects that the action has already been completed.

Example / answer: The agency had better have checked the figures.

Threatening tone

Rule / explanation: This structure can sound forceful or confrontational.

Example / answer: Ministers had better have prepared a convincing explanation.

Not standard regret

Rule / explanation: Do not use had better have as the normal equivalent of should have.

Example / answer: For criticism, prefer should have.

Register warning

Rule / explanation: Avoid it in formal essays unless analysing spoken language or rhetoric.

Example / answer: It is more natural in dialogue than academic prose.

K. Modal perfects in the passive

Should have been + past participle

Rule / explanation: Use this to criticise a past passive action or omission.

Example / answer: The figures should have been verified independently.

Could have been + past participle

Rule / explanation: Use this for a missed passive possibility.

Example / answer: The crisis could have been contained earlier.

Might have been + past participle

Rule / explanation: Use this for a tentative alternative passive outcome.

Example / answer: The conflict might have been avoided through mediation.

Needn’t have been + past participle

Rule / explanation: Use this when a completed passive action was unnecessary.

Example / answer: The entire system needn’t have been redesigned.

Ought to have been + past participle

Rule / explanation: Use this for formal or moral criticism.

Example / answer: Affected communities ought to have been consulted.

L. Criticising actions vs omissions

Action criticised

Rule / explanation: Use shouldn’t have when something happened but was wrong or unwise.

Example / answer: The government shouldn’t have concealed the figures.

Omission criticised

Rule / explanation: Use should have when something did not happen but was advisable.

Example / answer: The government should have disclosed the figures.

Alternative available

Rule / explanation: Use could have when another option was genuinely possible.

Example / answer: Ministers could have delayed implementation.

Alternative uncertain

Rule / explanation: Use might have when the alternative result is speculative.

Example / answer: More dialogue might have reduced tensions.

Wasted action

Rule / explanation: Use needn’t have when an action occurred unnecessarily.

Example / answer: The committee needn’t have commissioned a second identical study.

M. Degrees of criticism

Strong criticism

Rule / explanation: Shouldn’t have and ought not to have directly condemn a past action.

Example / answer: Officials should not have ignored the legal advice.

Moderate criticism

Rule / explanation: Should have identifies a better course without necessarily sounding hostile.

Example / answer: The article should have considered alternative explanations.

Mild reproach

Rule / explanation: Could have or might have can sound less direct.

Example / answer: The journalist could have checked the source.

Institutional observation

Rule / explanation: Was supposed to or was meant to may report failure more neutrally.

Example / answer: The regulator was supposed to publish annual reviews.

Balanced analysis

Rule / explanation: Choose the form according to the strength of evidence and judgement.

Example / answer: Do not use moral condemnation for a merely debatable choice.

N. Regret about personal actions

I should have

Rule / explanation: Use this for personal regret over an omission or poor choice.

Example / answer: I should have prepared a clearer introduction.

I shouldn’t have

Rule / explanation: Use this when regretting an action that occurred.

Example / answer: I shouldn’t have interrupted the speaker.

I could have

Rule / explanation: Use this for a missed personal opportunity.

Example / answer: I could have developed the counterargument further.

If only

Rule / explanation: If only + past perfect expresses stronger emotional regret.

Example / answer: If only I had checked the figures.

Wish

Rule / explanation: Wish + past perfect is another standard regret structure.

Example / answer: I wish I had explained the context more clearly.

O. Regret without assigning blame

It would have been better to

Rule / explanation: Use this impersonal structure for measured evaluation.

Example / answer: It would have been better to consult the regions first.

A preferable approach would have been

Rule / explanation: This formal phrase proposes an alternative without direct accusation.

Example / answer: A preferable approach would have been to phase in the reform.

In retrospect

Rule / explanation: Use in retrospect to signal hindsight.

Example / answer: In retrospect, the timetable was excessively ambitious.

With hindsight

Rule / explanation: Use this to acknowledge that the judgement is made after the event.

Example / answer: With hindsight, earlier intervention would have been advisable.

Balanced tone

Rule / explanation: These alternatives are valuable when the evidence does not justify personal blame.

Example / answer: The process might have benefited from wider consultation.

P. Hindsight bias

Definition

Rule / explanation: Hindsight bias is the tendency to treat past outcomes as more predictable than they really were.

Example / answer: After the crisis, commentators claimed that failure had been inevitable.

Modal caution

Rule / explanation: Should have can sound unfair if the better option was not obvious at the time.

Example / answer: Ask whether decision-makers had the necessary information.

Evidence at the time

Rule / explanation: Evaluate choices according to what was known then, not only what is known now.

Example / answer: The warning signs may have appeared weaker before the collapse.

Alternative outcomes

Rule / explanation: Could have does not prove that the alternative would have succeeded.

Example / answer: Earlier action could have helped, but success was not guaranteed.

CPGE maturity

Rule / explanation: Sophisticated analysis distinguishes genuine negligence from retrospective certainty.

Example / answer: The authorities may have erred, but the outcome was not fully foreseeable.

Q. Historical and political analysis

Policy failure

Rule / explanation: Use should have to evaluate a neglected duty or prudent action.

Example / answer: The government should have anticipated regional resistance.

Missed reform

Rule / explanation: Use could have to discuss an available alternative.

Example / answer: Parliament could have adopted a more gradual approach.

Counterfactual history

Rule / explanation: Use might have for uncertain alternative outcomes.

Example / answer: A negotiated settlement might have prevented further conflict.

Unnecessary intervention

Rule / explanation: Use needn’t have for a completed but needless action.

Example / answer: The authorities needn’t have imposed such broad restrictions.

Declared aim vs result

Rule / explanation: Use was meant to for failed intentions.

Example / answer: The reform was meant to restore trust but instead deepened suspicion.

R. Article commentary and source criticism

Missing evidence

Rule / explanation: Use should have when the author fails to include necessary support.

Example / answer: The journalist should have provided more recent data.

Alternative interpretation

Rule / explanation: Use could have when another angle was available.

Example / answer: The article could have explored regional differences.

Tentative criticism

Rule / explanation: Use might have for a softer evaluation.

Example / answer: The author might have acknowledged the limits of the survey.

Unnecessary detail

Rule / explanation: Use needn’t have for material that adds little value.

Example / answer: The text needn’t have devoted so much space to personal anecdote.

Fairness

Rule / explanation: Support criticism with precise textual evidence.

Example / answer: Do not criticise a source merely because it reaches a different conclusion.

S. CPGE oral commentary

Evaluate decisions

Rule / explanation: Use modal perfects to move from description to reasoned judgement.

Example / answer: The government should have consulted the regions before implementation.

Offer alternatives

Rule / explanation: Use could have and might have to avoid binary thinking.

Example / answer: A gradual introduction could have reduced opposition.

Discuss consequences

Rule / explanation: Use passive modal perfects for outcomes.

Example / answer: The backlash might have been avoided through clearer communication.

Control tone

Rule / explanation: Prefer measured criticism to moral outrage.

Example / answer: The author could perhaps have considered the fiscal constraints more fully.

Link to evidence

Rule / explanation: Explain why the alternative was available or plausible.

Example / answer: Because several local reports had already warned of shortages, ministers should have acted sooner.

T. Formal essay patterns

It could be argued that...should have

Rule / explanation: Use this to present criticism cautiously.

Example / answer: It could be argued that Parliament should have demanded stronger safeguards.

X might have been avoided if...

Rule / explanation: Use this for counterfactual policy analysis.

Example / answer: The crisis might have been avoided if regulators had intervened earlier.

X was intended to..., yet...

Rule / explanation: Contrast intention and result.

Example / answer: The measure was intended to simplify access, yet it created new barriers.

A more effective approach would have been...

Rule / explanation: Present a formal alternative.

Example / answer: A more effective approach would have been to test the scheme locally.

Although X could have..., it is unclear whether...

Rule / explanation: Acknowledge an alternative without claiming certainty.

Example / answer: Although earlier action could have helped, it is unclear whether it would have prevented the crisis.

U. Common mistakes by French speakers

Should had

Rule / explanation: Use should have + past participle, never should had.

Example / answer: Wrong: should had acted. Correct: should have acted.

Past form after modal

Rule / explanation: After a modal, use have + past participle for past reference.

Example / answer: Wrong: could went. Correct: could have gone.

Needn’t have meaning

Rule / explanation: Remember that the action definitely happened.

Example / answer: They needn’t have paid = they paid, unnecessarily.

Could have ambiguity

Rule / explanation: Clarify whether it means possible past or missed opportunity.

Example / answer: Add context or an if-clause.

Was supposed to vs should have

Rule / explanation: The first reports expectation; the second evaluates what was right.

Example / answer: The report was supposed to appear. It should have appeared sooner.

Overconfident counterfactual

Rule / explanation: Do not present an alternative outcome as certain.

Example / answer: Prefer might have or could have when causation is uncertain.

V. Exercises

1. Criticise an omission

Rule / explanation: The government did not consult local authorities.

Example / answer: The government should have consulted local authorities.

2. Criticise an action

Rule / explanation: The newspaper published confidential names.

Example / answer: The newspaper should not have published confidential names.

3. Express a missed opportunity

Rule / explanation: Earlier intervention was possible, but it did not happen.

Example / answer: The authorities could have intervened earlier.

4. Express an uncertain alternative

Rule / explanation: Perhaps better communication would have reduced opposition.

Example / answer: Better communication might have reduced opposition.

5. Distinguish unnecessary action

Rule / explanation: They ordered a second report, but the first one was sufficient.

Example / answer: They needn’t have ordered a second report.

6. Use passive criticism

Rule / explanation: Nobody verified the figures independently.

Example / answer: The figures should have been verified independently.

7. Translate

Rule / explanation: La réforme aurait pu réussir si elle avait été correctement financée.

Example / answer: The reform could have succeeded if it had been properly funded.

8. Mini concours challenge

Rule / explanation: Write a paragraph using should have, could have, might have, needn’t have and was meant to.

Example / answer: Example: The reform was meant to simplify access, but implementation created additional barriers. The government should have tested the system locally before introducing it nationwide. A gradual rollout could have revealed the main weaknesses, while clearer communication might have reduced public opposition. Ministers also needn’t have replaced the entire digital platform when targeted improvements would have been sufficient.
