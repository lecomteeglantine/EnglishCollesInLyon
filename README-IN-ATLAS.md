# India Civilisation Atlas

Upload `colle-atlas-in.html` to the root of the `EnglishCollesInLyon` GitHub repository and replace the existing file with the same name.

## Included
- Back-to-basics opening with ten essential foundations
- 31-event expandable and filterable timeline
- Complete political, federal and judicial system sections
- Interactive power map, bill stepper and court-route explorer
- 15 historically significant leaders and reformers
- 20 colonial rule, independence, Partition and nation-building entries
- States, Union territories, languages, regions and identities
- Society and contemporary debates
- Article, video search and Wikimedia Commons image for every knowledge entry
- Search, essentials-only mode, reader mode, dark mode and accessibility controls
- Local revision list with responsive HTML export and import
