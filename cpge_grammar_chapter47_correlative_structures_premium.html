Paired conjunctions require parallel grammar

Main useCorrelative structures connect two equal elements and create balance.

Key trapThe elements after each half of the pair must have the same grammatical form.

A. Lesson — What are correlative structures?

Correlative conjunctions

Rule / explanation: Correlative structures connect two equal grammatical elements.

Example / answer: Both the policy and its implementation deserve criticism.

Paired structure

Rule / explanation: The two parts work together as a pair.

Example / answer: Either the government acts now or the crisis will worsen.

Parallel grammar

Rule / explanation: The elements after each half of the pair should have the same grammatical form.

Example / answer: The article is both informative and persuasive.

CPGE relevance

Rule / explanation: These structures are highly useful for balanced argumentation, contrast and synthesis.

Example / answer: The issue concerns not only economic growth but also democratic legitimacy.

B. Both...and

Basic use

Rule / explanation: Use both...and to include two elements.

Example / answer: The reform affects both workers and employers.

With nouns

Rule / explanation: Keep the noun phrases parallel.

Example / answer: Both housing policy and labour policy require reform.

With verbs

Rule / explanation: Use matching verb forms.

Example / answer: The article both explains the crisis and proposes solutions.

With adjectives

Rule / explanation: Use matching adjectives.

Example / answer: The argument is both nuanced and convincing.

Subject agreement

Rule / explanation: When both...and joins two subjects, the verb is usually plural.

Example / answer: Both the minister and the committee are responsible.

C. Either...or

Basic use

Rule / explanation: Use either...or to present two alternatives.

Example / answer: The government must either raise taxes or reduce spending.

Parallel alternatives

Rule / explanation: The elements after either and or should match.

Example / answer: The policy will either protect workers or benefit employers.

Subject agreement

Rule / explanation: With two subjects, the verb often agrees with the nearest subject in formal standard usage.

Example / answer: Either the ministers or the prime minister is responsible.

Negative possibility

Rule / explanation: Either...or can present two undesirable alternatives.

Example / answer: Either the reform fails, or public trust declines further.

CPGE use

Rule / explanation: Useful for dilemma-based problematics.

Example / answer: The country must either reform its institutions or accept further instability.

D. Neither...nor

Basic use

Rule / explanation: Use neither...nor to reject two possibilities.

Example / answer: The article neither explains the causes nor proposes solutions.

With nouns

Rule / explanation: Use parallel nouns or noun phrases.

Example / answer: Neither the government nor the opposition offers a convincing plan.

With adjectives

Rule / explanation: Use matching adjectives.

Example / answer: The policy is neither fair nor effective.

Subject agreement

Rule / explanation: The verb often agrees with the nearest subject.

Example / answer: Neither the experts nor the minister was prepared.

Avoid double negatives

Rule / explanation: Do not add another negative to a neither...nor structure.

Example / answer: Wrong: Neither the policy does not work nor it is fair.

E. Not only...but also

Basic use

Rule / explanation: Use not only...but also to add a second, often stronger point.

Example / answer: The reform not only increased costs but also weakened public trust.

With nouns

Rule / explanation: Keep noun phrases parallel.

Example / answer: The crisis affected not only households but also small businesses.

With verbs

Rule / explanation: Use matching verb forms.

Example / answer: The article not only criticises the policy but also questions its legitimacy.

Inversion at the beginning

Rule / explanation: When not only begins the sentence, invert the first clause.

Example / answer: Not only did the reform fail, but it also damaged trust.

Emphatic use

Rule / explanation: This structure is powerful and should be used selectively.

Example / answer: Not only is the argument clear, but it is also remarkably persuasive.

F. Whether...or

Alternative possibilities

Rule / explanation: Use whether...or to present two possible outcomes or viewpoints.

Example / answer: The debate concerns whether the reform will succeed or fail.

Whether or not

Rule / explanation: Use whether or not when both possibilities are included.

Example / answer: The policy will affect voters whether or not Parliament approves it.

Embedded question

Rule / explanation: Whether is common in indirect questions.

Example / answer: The article asks whether the government can restore trust.

Formal style

Rule / explanation: Whether is often more formal and precise than if in indirect alternatives.

Example / answer: It remains unclear whether the policy is sustainable.

G. As...as and so...that

As...as

Rule / explanation: Use as...as to compare equal qualities or degrees.

Example / answer: The second article is as detailed as the first.

Not as...as

Rule / explanation: Use not as...as to express inequality.

Example / answer: The evidence is not as strong as the conclusion suggests.

So...that

Rule / explanation: Use so + adjective/adverb + that to express result.

Example / answer: The crisis was so severe that the minister resigned.

Such...that

Rule / explanation: Use such + noun phrase + that to express result.

Example / answer: It was such a divisive policy that Parliament rejected it.

H. Rather...than and no sooner...than

Rather...than

Rule / explanation: Use rather...than to express preference or contrast.

Example / answer: The government chose delay rather than reform.

Would rather...than

Rule / explanation: Use would rather + base verb + than + base verb.

Example / answer: Voters would rather see action than hear promises.

No sooner...than

Rule / explanation: Use no sooner + inversion + than for one event immediately following another.

Example / answer: No sooner had the report appeared than the controversy began.

Hardly...when

Rule / explanation: Use hardly + inversion + when in a similar way.

Example / answer: Hardly had the vote ended when protests started.

I. Correlative structures in CPGE argumentation

Balanced thesis

Rule / explanation: Use paired structures to present a balanced argument.

Example / answer: The reform is both necessary and politically risky.

Dilemma

Rule / explanation: Use either...or to frame a difficult choice.

Example / answer: The government must either reform now or face greater instability later.

Double criticism

Rule / explanation: Use neither...nor to reject two weak options.

Example / answer: Neither austerity nor inaction offers a sustainable solution.

Escalating argument

Rule / explanation: Use not only...but also to strengthen the second point.

Example / answer: The crisis not only exposed policy failure but also revealed institutional weakness.

Nuanced problematique

Rule / explanation: Use whether...or to frame uncertainty.

Example / answer: The key question is whether democracy can adapt or whether distrust will deepen.

J. Typical mistakes by French speakers

Broken parallelism

Rule / explanation: Do not mix different grammatical forms after the two parts of the pair.

Example / answer: Wrong: both reducing costs and to improve access.

Wrong placement

Rule / explanation: Place the correlative words directly before the parallel elements.

Example / answer: Better: The policy affects both workers and employers.

Double negative

Rule / explanation: Do not combine neither...nor with another negative.

Example / answer: Wrong: Neither the policy does not work nor it is fair.

Agreement errors

Rule / explanation: Check the verb carefully when two subjects are linked.

Example / answer: Either the ministers or the prime minister is responsible.

Missing inversion

Rule / explanation: Use inversion when not only begins the sentence.

Example / answer: Not only did the policy fail, but it also damaged trust.

Overuse

Rule / explanation: These structures are effective, but too many can make prose mechanical.

Example / answer: Use them for key contrasts and balanced ideas.

K. Exercises

1. Complete with both...and

Rule / explanation: The reform affects workers. It affects employers.

Example / answer: The reform affects both workers and employers.

2. Complete with either...or

Rule / explanation: The government must raise taxes. Alternatively, it must reduce spending.

Example / answer: The government must either raise taxes or reduce spending.

3. Complete with neither...nor

Rule / explanation: The article does not explain the causes. It does not propose solutions.

Example / answer: The article neither explains the causes nor proposes solutions.

4. Rewrite with not only...but also

Rule / explanation: The policy increased costs. It weakened trust.

Example / answer: The policy not only increased costs but also weakened trust.

5. Correct the parallelism

Rule / explanation: The reform aims both to reduce poverty and improving housing.

Example / answer: The reform aims both to reduce poverty and to improve housing.

6. Correct the agreement

Rule / explanation: Either the ministers or the prime minister are responsible.

Example / answer: Either the ministers or the prime minister is responsible.

7. Translate

Rule / explanation: Ni le gouvernement ni l’opposition ne propose de solution convaincante.

Example / answer: Neither the government nor the opposition proposes a convincing solution.

8. Mini concours challenge

Rule / explanation: Write a short paragraph using both...and, either...or, neither...nor and not only...but also.

Example / answer: Example: The reform is both ambitious and controversial. The government must either revise it or defend it more clearly. Neither delay nor denial will solve the problem. Not only is public trust declining, but political polarisation is also increasing.
