The golden rule

A rhetorical question is not decoration: it makes an implied claim.
How can a democracy function without public trust?
Implied claim: it cannot function properly without it.
Purposechallenge / emphasise / frame / persuade / guide
Strong patternquestion → implied or explicit answer → analysis
Main riskleading, loaded or unanswered questions

A. Lesson — What is a rhetorical question?

Rhetorical question

Rule / explanation: A rhetorical question is asked to make a point, guide the audience or create emphasis rather than obtain information.

Example / answer: How can a democracy function without public trust?

Expected answer

Rule / explanation: The speaker usually assumes that the answer is obvious, shared or implied by the context.

Example / answer: Who could deny that climate change requires urgent action?

Argumentative function

Rule / explanation: It can challenge assumptions, frame a debate, expose contradictions or introduce a conclusion.

Example / answer: If the policy benefits only a minority, can it truly be called fair?

CPGE relevance

Rule / explanation: It is useful in oral commentary, debate and carefully controlled essay writing, but it must support analysis.

Example / answer: What is ultimately at stake if not the legitimacy of democratic institutions?

B. Rhetorical vs genuine questions

Genuine question

Rule / explanation: A genuine question requests information the speaker does not know.

Example / answer: What percentage of voters supported the reform?

Rhetorical question

Rule / explanation: A rhetorical question communicates a position through question form.

Example / answer: Can any reform succeed without adequate funding?

Context decides

Rule / explanation: The same form may be genuine or rhetorical depending on tone and context.

Example / answer: Do we really believe that inequality will disappear by itself?

Answer not required

Rule / explanation: A rhetorical question may be followed immediately by the speaker’s own answer.

Example / answer: Why did the policy fail? Because implementation was poorly planned.

Analytical caution

Rule / explanation: In academic work, make the implied answer clear through the surrounding argument.

Example / answer: The question should advance reasoning, not merely create drama.

C. Yes/no rhetorical questions

Positive question, negative implication

Rule / explanation: A positive rhetorical question often implies a negative answer.

Example / answer: Can economic growth alone solve every social problem?

Negative question, positive implication

Rule / explanation: A negative rhetorical question often implies agreement with a positive proposition.

Example / answer: Isn’t public accountability essential to democracy?

Modal questions

Rule / explanation: Can, could, should and would are common because they frame possibility, obligation and judgement.

Example / answer: Should citizens accept decisions made without consultation?

Do we really...?

Rule / explanation: Use really to challenge a belief or expose doubt.

Example / answer: Do we really expect local authorities to succeed without funding?

Is it not...?

Rule / explanation: This formal structure creates strong emphasis.

Example / answer: Is it not precisely in times of crisis that institutions must remain accountable?

D. Wh-rhetorical questions

Why

Rule / explanation: Why-questions can challenge the logic or legitimacy of an action.

Example / answer: Why should younger generations pay for decisions they did not make?

How

Rule / explanation: How-questions often imply impossibility, difficulty or contradiction.

Example / answer: How can trust be restored if transparency remains absent?

What

Rule / explanation: What-questions can identify stakes, alternatives or consequences.

Example / answer: What would democracy become without independent journalism?

Who

Rule / explanation: Who-questions often imply everyone, no one or an obvious group.

Example / answer: Who benefits from a system that excludes the most vulnerable?

Where

Rule / explanation: Where-questions can suggest a limit, destination or political consequence.

Example / answer: Where will continued institutional distrust eventually lead?

E. Negative rhetorical questions

Isn’t / aren’t

Rule / explanation: Use contracted negative forms in speech and less formal writing.

Example / answer: Isn’t the purpose of government to protect the public interest?

Is it not

Rule / explanation: Use the uncontracted form for a formal and deliberate tone.

Example / answer: Is it not reasonable to demand greater transparency?

Haven’t / didn’t

Rule / explanation: Perfect and past negative questions may remind the audience of known evidence.

Example / answer: Haven’t repeated crises already shown the limits of deregulation?

Expectation

Rule / explanation: Negative questions often assume the audience will agree.

Example / answer: Shouldn’t public policy be judged by its actual results?

Risk of bias

Rule / explanation: Because they contain an expected answer, negative rhetorical questions can sound manipulative.

Example / answer: Use them selectively and acknowledge complexity.

F. Rhetorical questions with modals

Can

Rule / explanation: Can often questions possibility or credibility.

Example / answer: Can such a limited measure produce lasting change?

Could

Rule / explanation: Could creates a more tentative or speculative challenge.

Example / answer: Could the decline in trust have deeper structural causes?

Should

Rule / explanation: Should questions duty, morality or appropriate policy.

Example / answer: Should private companies control essential public data?

Would

Rule / explanation: Would explores a hypothetical reaction or consequence.

Example / answer: Would voters accept the reform if its full cost were known?

Must

Rule / explanation: Must can express impatience or challenge necessity.

Example / answer: Must every crisis result in further restrictions on civil liberties?

G. Rhetorical questions and implied answers

Obviously yes

Rule / explanation: Some questions invite strong agreement.

Example / answer: Should democratic institutions be transparent?

Obviously no

Rule / explanation: Others reject a proposition as implausible or unacceptable.

Example / answer: Can inequality be eliminated through slogans alone?

Qualified answer

Rule / explanation: A sophisticated question may lead to a nuanced rather than absolute response.

Example / answer: Can regulation protect consumers without slowing innovation?

Question-answer sequence

Rule / explanation: State the question, then supply a reasoned answer.

Example / answer: Can the reform succeed? Only if implementation and funding improve.

Delayed answer

Rule / explanation: A paragraph may explore competing possibilities before resolving the question.

Example / answer: This creates structure and intellectual tension.

H. Hypophora: asking and answering

Hypophora

Rule / explanation: Hypophora is the technique of asking a question and immediately answering it.

Example / answer: Why has trust declined? Because institutions repeatedly failed to deliver.

Paragraph opening

Rule / explanation: Use hypophora to introduce the main issue of a paragraph.

Example / answer: What explains the regional divide? Several factors must be considered.

Oral clarity

Rule / explanation: In a khôlle, it can make a transition sound clear and purposeful.

Example / answer: What are the consequences of this shift? First, political polarisation is likely to increase.

Argument progression

Rule / explanation: The answer should develop analysis rather than merely repeat the question.

Example / answer: Why does this matter? Because legitimacy depends on perceived fairness.

Avoid overuse

Rule / explanation: Repeated hypophora can sound theatrical or mechanical.

Example / answer: Use it at major transitions only.

I. Questions as hooks and openings

Opening hook

Rule / explanation: A rhetorical question can immediately introduce a broad tension.

Example / answer: What happens when citizens stop believing their institutions?

Context first

Rule / explanation: In formal CPGE introductions, contextualisation should normally precede the main question.

Example / answer: Recent crises have weakened confidence in public institutions. How, then, can democratic legitimacy be restored?

Avoid clichés

Rule / explanation: Do not begin with vague questions such as What is democracy? unless the definition is central.

Example / answer: Prefer a precise contextual tension.

Link to problématique

Rule / explanation: The rhetorical question can lead naturally to the key question.

Example / answer: This raises a broader issue: can efficiency be pursued without sacrificing fairness?

Answer through structure

Rule / explanation: The essay or commentary must progressively answer the opening question.

Example / answer: Never leave a decorative question unresolved.

J. Rhetorical questions in conclusions

Return to the stakes

Rule / explanation: A final rhetorical question may broaden the implications of the analysis.

Example / answer: If trust cannot be restored, what future remains for representative democracy?

Opening

Rule / explanation: The question can create an opening towards a related debate.

Example / answer: Will future regulation protect citizens or merely create new forms of control?

Avoid new argument

Rule / explanation: Do not introduce an entirely new issue that the analysis has not prepared.

Example / answer: The final question should grow from the conclusion.

Controlled effect

Rule / explanation: A strong conclusion may use one carefully chosen rhetorical question.

Example / answer: Several consecutive questions weaken impact.

Alternative

Rule / explanation: A firm declarative sentence is often stronger than a vague question.

Example / answer: The future of the reform will depend on whether citizens perceive it as legitimate.

K. Speeches and journalism

Audience engagement

Rule / explanation: Questions invite listeners or readers to participate mentally in the argument.

Example / answer: What kind of society do we wish to build?

Moral framing

Rule / explanation: They can present an issue as a matter of values.

Example / answer: Can we accept prosperity built on exclusion?

Political challenge

Rule / explanation: Speakers use them to attack opponents or expose inconsistency.

Example / answer: How can the government promise equality while cutting essential services?

Headline style

Rule / explanation: Journalistic headlines often use questions to create curiosity or uncertainty.

Example / answer: Is the housing crisis becoming politically unsustainable?

Critical reading

Rule / explanation: A question-shaped headline may suggest a claim without fully asserting it.

Example / answer: Analyse what answer the text encourages the reader to accept.

L. Article commentary

Identify function

Rule / explanation: Explain whether the question challenges, dramatises, frames or guides the argument.

Example / answer: The opening question frames the reform as a moral rather than merely economic issue.

Analyse implied answer

Rule / explanation: State what conclusion the writer expects the reader to reach.

Example / answer: By asking whether citizens can still trust the system, the author implies that confidence has already been severely damaged.

Tone

Rule / explanation: Rhetorical questions may create urgency, indignation, irony or scepticism.

Example / answer: The repeated questions contribute to an increasingly accusatory tone.

Audience effect

Rule / explanation: Explain how the device positions the reader.

Example / answer: The reader is encouraged to see government inaction as indefensible.

Evaluation

Rule / explanation: Assess whether the technique strengthens the argument or oversimplifies it.

Example / answer: The question is effective rhetorically but reduces a complex debate to a binary choice.

M. CPGE oral commentary

Transition

Rule / explanation: Use one question to move from summary to commentary.

Example / answer: This leads us to ask whether the reform addresses the causes of the crisis or merely its symptoms.

Problematique

Rule / explanation: Frame the central tension as a genuine analytical question.

Example / answer: How can governments restore trust without creating unrealistic expectations?

Development

Rule / explanation: Use short question-answer sequences to signal structure.

Example / answer: Why is this distinction important? Because short-term efficiency may create long-term social costs.

Personal evaluation

Rule / explanation: Challenge the author’s assumptions with a measured question.

Example / answer: Can the author’s conclusion really be sustained on the basis of such limited evidence?

Delivery

Rule / explanation: Use natural intonation and pause briefly before the answer.

Example / answer: Do not turn every transition into a rhetorical performance.

N. Rhetorical questions in essays

Selective use

Rule / explanation: One or two well-placed questions may be effective; frequent use weakens formal authority.

Example / answer: Prefer developed claims to a chain of questions.

Follow with analysis

Rule / explanation: A rhetorical question should normally be answered or unpacked.

Example / answer: Can regulation encourage innovation? It can, provided that rules remain proportionate.

Avoid second person

Rule / explanation: Do not rely on questions addressed vaguely to you or the reader.

Example / answer: Better: Can such a system remain legitimate?

Formal integration

Rule / explanation: Use a colon, dash or transition phrase to integrate the question.

Example / answer: One issue remains: how can the policy be implemented fairly?

Declarative alternative

Rule / explanation: Convert the question into a thesis when certainty is more appropriate.

Example / answer: The central challenge is to reconcile efficiency with fairness.

O. Irony and rhetorical questions

Ironic question

Rule / explanation: An ironic question says one thing while implying a critical opposite.

Example / answer: What could possibly go wrong with removing all safeguards?

Context dependence

Rule / explanation: Irony depends heavily on tone and shared knowledge.

Example / answer: In writing, the intention must be clear enough to avoid misreading.

Understatement

Rule / explanation: A mild question can intensify criticism through understatement.

Example / answer: Was the government perhaps a little too optimistic?

Sarcasm risk

Rule / explanation: Sarcasm can sound aggressive or academically weak.

Example / answer: Use irony more freely in analysis of a source than in your own formal argument.

Interpretation

Rule / explanation: Explain both literal wording and implied meaning.

Example / answer: The question appears neutral but actually ridicules the policy’s lack of preparation.

P. Persuasion and framing

Shared values

Rule / explanation: Questions can activate values the audience is expected to share.

Example / answer: Who would oppose equal access to education?

False consensus

Rule / explanation: A writer may imply that all reasonable people agree.

Example / answer: Critical analysis should test whether the consensus is genuine.

Binary framing

Rule / explanation: Questions may reduce a complex issue to two alternatives.

Example / answer: Do we choose freedom or security?

Presupposition

Rule / explanation: A question can contain an assumption that has not been proven.

Example / answer: Why has the government failed again? presupposes that it has failed.

Persuasive scrutiny

Rule / explanation: Identify what the question assumes, excludes and encourages the audience to believe.

Example / answer: This is crucial in political rhetoric and media analysis.

Q. Loaded questions and presuppositions

Hidden assumption

Rule / explanation: The wording may treat a debatable claim as already established.

Example / answer: When will the government finally admit that the policy has failed?

Loaded question

Rule / explanation: A loaded question makes any direct answer appear to accept its premise.

Example / answer: Why does the opposition refuse to take the crisis seriously?

Neutral reformulation

Rule / explanation: Remove the assumption to create a fair analytical question.

Example / answer: How has the opposition responded to the crisis?

CPGE evaluation

Rule / explanation: Expose presuppositions when assessing objectivity.

Example / answer: The question frames disagreement as irresponsibility.

Responsible use

Rule / explanation: Your own problématique should reveal a tension without prejudging the answer.

Example / answer: How far can the policy be considered successful?

R. Grammar and punctuation

Question word order

Rule / explanation: Rhetorical questions follow the same grammar as genuine direct questions.

Example / answer: Why has public confidence declined?

Question mark

Rule / explanation: Use a question mark after a direct rhetorical question.

Example / answer: Can this situation continue?

Embedded issue

Rule / explanation: An embedded question takes no question mark when the whole sentence is a statement.

Example / answer: The issue is whether this situation can continue.

Colon

Rule / explanation: A complete statement may introduce the question with a colon.

Example / answer: One question remains: can the coalition survive?

Consistency

Rule / explanation: Keep capitalisation and punctuation consistent throughout the document.

Example / answer: Formal presentation strengthens clarity.

S. Intonation in spoken English

Falling wh-question

Rule / explanation: Wh-rhetorical questions often use falling intonation to sound assertive.

Example / answer: How can this be justified? ↘

Falling yes/no question

Rule / explanation: A falling tone can signal that the speaker is not genuinely requesting information.

Example / answer: Can anyone believe this? ↘

Rise-fall

Rule / explanation: A rise-fall may communicate surprise, indignation or strong emphasis.

Example / answer: Is that really the best solution? ↗↘

Pause

Rule / explanation: A short pause after the question gives weight to the answer.

Example / answer: Why does this matter? — Because trust determines legitimacy.

Natural delivery

Rule / explanation: Avoid exaggerated performance in a khôlle.

Example / answer: The device should clarify reasoning, not sound theatrical.

T. Typical mistakes

No inversion

Rule / explanation: Use normal direct-question word order.

Example / answer: Wrong: How this policy can succeed? Correct: How can this policy succeed?

Decoration only

Rule / explanation: Do not insert a question that the paragraph never answers.

Example / answer: Every rhetorical question must serve the argument.

Too many questions

Rule / explanation: A series of questions can sound emotional and unstructured.

Example / answer: Replace some with clear analytical statements.

Leading problématique

Rule / explanation: Do not formulate the key question so that only one answer seems possible.

Example / answer: Better: How far has the policy succeeded?

Vague opening

Rule / explanation: Avoid enormous abstract questions disconnected from the document.

Example / answer: Start from precise context and stakes.

Direct vs embedded

Rule / explanation: Use inversion only in direct questions.

Example / answer: Direct: Can it work? Embedded: The question is whether it can work.

U. Exercises

1. Identify the implied answer

Rule / explanation: Can a democracy survive without public trust?

Example / answer: Implied answer: probably not.

2. Transform

Rule / explanation: A reform cannot succeed without adequate funding.

Example / answer: How can a reform succeed without adequate funding?

3. Correct the grammar

Rule / explanation: How the government can justify this decision?

Example / answer: How can the government justify this decision?

4. Neutralise the loaded question

Rule / explanation: Why has the government failed the public again?

Example / answer: How effective has the government’s response been?

5. Add hypophora

Rule / explanation: Explain why the distinction matters.

Example / answer: Why does this distinction matter? Because short-term efficiency may create long-term social costs.

6. Convert to a formal statement

Rule / explanation: Can such a system remain legitimate?

Example / answer: The central issue is whether such a system can remain legitimate.

7. Translate

Rule / explanation: Comment une démocratie peut-elle fonctionner sans confiance publique ?

Example / answer: How can a democracy function without public trust?

8. Mini concours challenge

Rule / explanation: Use one rhetorical question, one hypophora and one embedded problématique.

Example / answer: Can the reform succeed without public support? Clearly, legal adoption is not enough. Why does legitimacy matter? Because citizens are more likely to accept difficult measures when procedures appear fair. The central question is therefore whether the government can rebuild trust while implementing unpopular change.
