Chapter objective

This premium grammar chapter helps CPGE students understand and use the imperative in English. It covers positive and negative imperatives, exam instructions, essay methodology, khôlle strategies, polite alternatives and common mistakes made by French speakers.

The golden rule

Base verb = positive imperative
Do not + base verb = negative imperative
Analyse the article. / Do not ignore the source.
Main useThe imperative gives instructions, advice, warnings, directions and exam tasks.
CPGE habitRecognise and use exam verbs precisely: analyse, compare, discuss, assess, justify, develop.
Key trapThe imperative uses the base verb: Compare the documents, not To compare the documents.

A. Lesson — What is the imperative?

Imperative

French: impératif

Rule / explanation: The imperative gives instructions, orders, advice, warnings or directions.

Example / answer: Consider the economic context before drawing conclusions.

Base verb

French: base verbale

Rule / explanation: Use the base verb without a subject.

Example / answer: Analyse the data carefully.

No subject

French: pas de sujet

Rule / explanation: The subject you is usually omitted.

Example / answer: Do not ignore the historical background.

Positive imperative

French: impératif affirmatif

Rule / explanation: Use the base verb to tell someone to do something.

Example / answer: Explain your point clearly.

Negative imperative

French: impératif négatif

Rule / explanation: Use do not / don’t + base verb.

Example / answer: Do not overgeneralise from one example.

Formal instruction

French: consigne formelle

Rule / explanation: Used in academic and exam instructions.

Example / answer: Compare the two documents and identify their main differences.

Advice and strategy

French: conseil / méthode

Rule / explanation: Useful for exam methodology and feedback.

Example / answer: Avoid vague claims and support your argument with evidence.

CPGE use

French: usage CPGE

Rule / explanation: Common in exam tasks, essay guidance, khôlles and debates.

Example / answer: Discuss the author’s argument and assess its limits.

B. Positive imperatives

Analyse

French: analyser

Rule / explanation: Ask for detailed study of a document, idea or figure.

Example / answer: Analyse the causes of political distrust.

Compare

French: comparer

Rule / explanation: Ask for similarities and differences.

Example / answer: Compare the British and American political systems.

Contrast

French: mettre en contraste

Rule / explanation: Emphasise differences.

Example / answer: Contrast the author’s view with the journalist’s conclusion.

Explain

French: expliquer

Rule / explanation: Ask for causes, meaning or reasoning.

Example / answer: Explain why the reform is controversial.

Identify

French: identifier

Rule / explanation: Ask students to find key elements.

Example / answer: Identify the main argument in the article.

Describe

French: décrire

Rule / explanation: Present features without necessarily judging.

Example / answer: Describe the trend shown in the graph.

Discuss

French: discuter

Rule / explanation: Ask for balanced argumentation.

Example / answer: Discuss the advantages and drawbacks of regulation.

Assess

French: évaluer

Rule / explanation: Ask for judgement based on evidence.

Example / answer: Assess the effectiveness of the policy.

Do not overstate

Do not ignore

Example / answer: Do not ignore the economic dimension.

Do not assume

French: ne pas supposer

Rule / explanation: Avoid unsupported claims.

Example / answer: Do not assume that all voters think the same way.

Do not translate word for word

Rule / explanation: Key warning for French speakers.

Example / answer: Do not translate French structures word for word.

Do not use vague language

Rule / explanation: Useful for essay writing.

Do not jump to conclusions

French: ne pas tirer de conclusions hâtives

Rule / explanation: Useful for graph and data analysis.

Example / answer: Do not jump to conclusions from one poll.

F. Let’s, let us and polite imperatives

Let’s

French: faisons / voyons

Rule / explanation: Suggestion including speaker and listener.

Example / answer: Let’s look at the graph first.

Let us

French: examinons

Rule / explanation: More formal version of let’s.

Example / answer: Let us consider the historical context.

Let’s not

French: ne faisons pas

Rule / explanation: Negative group suggestion.

Example / answer: Let’s not ignore the social consequences.

Please

French: s’il vous plaît

Rule / explanation: Makes an imperative more polite.

Example / answer: Please explain your reasoning.

Could you...?

French: pourriez-vous

Rule / explanation: More polite than an imperative.

Example / answer: Could you clarify your point?

Would you mind + -ing

French: cela vous dérangerait-il de

Rule / explanation: Very polite request.

Example / answer: Would you mind repeating the question?

I would suggest + -ing

French: je suggérerais de

Rule / explanation: Polite academic advice.

Example / answer: I would suggest focusing on the economic dimension.

It may be useful to

French: il peut être utile de

Rule / explanation: Formal recommendation.

Example / answer: It may be useful to compare the two documents.

H. Typical mistakes by French speakers

Using to after an imperative

French: to incorrect

Rule / explanation: The imperative uses the base verb, not to + verb.

Example / answer: Wrong: To explain your answer. Correct: Explain your answer.

Adding a subject

French: sujet inutile

Rule / explanation: Do not normally use you before an imperative.

Example / answer: Awkward: You analyse the text. Better: Analyse the text.

Using don't to for negatives

French: négation incorrecte

Rule / explanation: Negative imperative is do not + base verb.

Example / answer: Wrong: Do not to forget. Correct: Do not forget.

Confusing advice and obligation

French: conseil vs ordre

Rule / explanation: Use should for advice, imperative for direct instruction.

Example / answer: Instruction: Check the source. Advice: You should check the source.

Overusing imperatives in essays

French: impératif abusif

Rule / explanation: Essays normally explain and argue, not command the reader.

Example / answer: Avoid: Think about democracy. Better: Democracy should be understood as fragile.

Forgetting please in polite contexts

French: politesse

Rule / explanation: Direct imperatives can sound abrupt.

Example / answer: Better: Please clarify your point.

Using let’s for formal essay argument

French: let’s trop familier

Rule / explanation: Let us is more formal than let’s.

Example / answer: Better: Let us consider the economic context.

Translating il faut mechanically

French: traduction de il faut

Rule / explanation: Sometimes use imperative, should, need to or it is necessary to.

Example / answer: Nuance the argument. / The argument should be nuanced.
