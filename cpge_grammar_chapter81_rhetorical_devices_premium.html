A. Lesson — What is a rhetorical device?

Rhetorical device

Rule / explanation: A rhetorical device is a deliberate language pattern used to shape attention, emotion, memory or interpretation.

Example / answer: The speaker repeats We need change at the beginning of three successive sentences.

Form and effect

Rule / explanation: Identifying the name of a device is not enough; analysis must explain its function and audience effect.

Example / answer: The repetition creates rhythm and presents reform as urgent.

Context

Rule / explanation: The same device may persuade, clarify, entertain, criticise or manipulate depending on context.

Example / answer: A metaphor may illuminate a complex issue or oversimplify it.

CPGE relevance

Rule / explanation: Rhetorical devices are central to speeches, editorials, campaigns, headlines, literary extracts and khôlle commentary.

Example / answer: The article combines antithesis, metaphor and direct address to sharpen its criticism.

B. Repetition

Basic repetition

Rule / explanation: Repeating a word or phrase makes it more noticeable and memorable.

Example / answer: Trust is the issue. Trust is the condition. Trust is the solution.

Emphasis

Rule / explanation: Repetition may signal importance or emotional intensity.

Example / answer: The speaker repeats now to create urgency.

Cohesion

Rule / explanation: Repeated keywords can unify a passage around one central concept.

Example / answer: The recurring word responsibility links the argument’s different stages.

Manipulation risk

Rule / explanation: Repetition may make a claim feel true without providing new evidence.

Example / answer: The slogan is memorable, but its factual basis remains weak.

Analysis phrase

Rule / explanation: Explain what is repeated, where it appears and what it reinforces.

Example / answer: The repeated noun crisis frames every policy disagreement as an emergency.

C. Anaphora

Definition

Rule / explanation: Anaphora repeats the same word or phrase at the beginning of successive clauses or sentences.

Example / answer: We need transparency. We need accountability. We need reform.

Rhythm

Rule / explanation: The repeated opening creates a strong, predictable rhythm.

Example / answer: Each clause begins with They promised.

Collective mobilisation

Rule / explanation: Anaphora often supports political calls to action.

Example / answer: We will rebuild, we will recover, we will prevail.

Accumulation

Rule / explanation: Each repeated opening can add a new argument or consequence.

Example / answer: No trust in institutions, no trust in elections, no trust in reform.

Critical analysis

Rule / explanation: Assess whether the rising force is supported by evidence.

Example / answer: The anaphora amplifies urgency but simplifies the range of possible responses.

D. Epiphora and symploce

Epiphora

Rule / explanation: Epiphora repeats the same expression at the end of successive clauses.

Example / answer: They demanded justice, we promised justice, and the country still waits for justice.

Closing emphasis

Rule / explanation: End-position repetition gives the repeated term particular weight.

Example / answer: Each sentence ends with the word freedom.

Symploce

Rule / explanation: Symploce combines repetition at both the beginning and the end.

Example / answer: When trust disappears, institutions fail; when trust disappears, reforms fail.

Memorability

Rule / explanation: Repeated endings create a slogan-like or ceremonial quality.

Example / answer: The phrase for everyone closes each promise.

Analysis

Rule / explanation: Connect the structural pattern to its ideological focus.

Example / answer: The repeated ending places equality at the centre of every proposal.

E. Rule of three

Definition

Rule / explanation: The rule of three groups three parallel words, phrases or clauses.

Example / answer: The policy must be fair, feasible and transparent.

Completeness

Rule / explanation: Three elements often sound balanced, memorable and complete.

Example / answer: The speaker promises security, prosperity and freedom.

Escalation

Rule / explanation: The sequence may rise in importance.

Example / answer: The decision affected budgets, communities and lives.

Compression

Rule / explanation: A triad can summarise a complex programme efficiently.

Example / answer: Invest, reform, rebuild.

Critical analysis

Rule / explanation: Check whether the three-part formula hides missing dimensions.

Example / answer: The neat triad excludes the environmental consequences.

F. Parallelism

Definition

Rule / explanation: Parallelism repeats the same grammatical structure across coordinated elements.

Example / answer: To protect jobs, to restore trust and to strengthen communities.

Clarity

Rule / explanation: Parallel syntax helps the audience process several related points.

Example / answer: The proposal is economically risky, socially divisive and politically unstable.

Balance

Rule / explanation: Parallelism can make contrasting claims appear equally weighted.

Example / answer: The first source stresses cost; the second, legitimacy.

Oral force

Rule / explanation: Regular patterns improve rhythm and delivery.

Example / answer: We came to listen, we stayed to act.

Common error

Rule / explanation: Keep the grammatical forms genuinely parallel.

Example / answer: Correct: reducing costs, improving access and restoring trust.

G. Antithesis

Definition

Rule / explanation: Antithesis places contrasting ideas in a balanced structure.

Example / answer: The issue is not whether reform is costly, but whether inaction is costlier.

Sharp opposition

Rule / explanation: The structure clarifies a tension by placing alternatives side by side.

Example / answer: The policy protects institutions but abandons individuals.

Memorable balance

Rule / explanation: Similar syntax makes the contrast easy to remember.

Example / answer: We need less secrecy and more accountability.

False binary risk

Rule / explanation: Antithesis may oversimplify a spectrum into two opposing choices.

Example / answer: The speech presents regulation and freedom as incompatible.

Analysis

Rule / explanation: Explain both the contrast and the political framing it creates.

Example / answer: The antithesis recasts expenditure as investment.

H. Chiasmus

Definition

Rule / explanation: Chiasmus reverses the order of words or ideas in a mirrored ABBA pattern.

Example / answer: We shape our institutions, and our institutions shape us.

Symmetry

Rule / explanation: The reversal creates balance and memorability.

Example / answer: The people serve the system, and the system should serve the people.

Conceptual reversal

Rule / explanation: Chiasmus can expose mutual influence or contradiction.

Example / answer: Public trust enables reform; reform must also deserve public trust.

Epigrammatic force

Rule / explanation: The compact pattern can sound authoritative or final.

Example / answer: We do not live to work; we work to live.

Analysis

Rule / explanation: Do not merely call the sentence clever; explain the reversed relationship.

Example / answer: The chiasmus presents citizens and institutions as mutually responsible.

I. Climax and anticlimax

Climax

Rule / explanation: Climax arranges elements in increasing importance or intensity.

Example / answer: The reform damaged budgets, services and ultimately lives.

Emotional ascent

Rule / explanation: A gradual rise builds momentum towards the final term.

Example / answer: The speaker moves from concern to fear and finally to outrage.

Anticlimax

Rule / explanation: Anticlimax ends a rising sequence with something unexpectedly minor.

Example / answer: The scandal threatened the government, the economy and the minister’s weekend.

Humour or criticism

Rule / explanation: The sudden fall may mock inflated language or reveal misplaced priorities.

Example / answer: A grand speech ends with a complaint about paperwork.

Analysis

Rule / explanation: Identify the hierarchy and explain why the final position matters.

Example / answer: The climax turns an administrative issue into a moral emergency.

J. Metaphor

Definition

Rule / explanation: A metaphor describes one thing through another without using like or as.

Example / answer: Public trust is the foundation of democracy.

Framing

Rule / explanation: Metaphors select particular features and shape how an issue is understood.

Example / answer: The economy is presented as a patient requiring treatment.

Complexity made concrete

Rule / explanation: Abstract processes become easier to visualise.

Example / answer: The reform opens the door to greater inequality.

Ideological effect

Rule / explanation: A metaphor may imply danger, struggle, movement or natural inevitability.

Example / answer: A wave of migration suggests overwhelming force.

Critical analysis

Rule / explanation: Ask what the metaphor highlights, hides and encourages the audience to believe.

Example / answer: The war metaphor legitimises exceptional measures.

K. Extended metaphor

Definition

Rule / explanation: An extended metaphor develops the same comparison across several expressions.

Example / answer: The country is described as a ship, the government as its crew and the crisis as a storm.

Coherence

Rule / explanation: A sustained image can organise an entire paragraph or speech.

Example / answer: Economic recovery is presented as a journey with obstacles and a destination.

Narrative force

Rule / explanation: The audience is invited to imagine a continuing scenario.

Example / answer: The speaker asks citizens to stay the course.

Risk

Rule / explanation: A long metaphor can distort the issue by forcing every detail into one frame.

Example / answer: The disease metaphor treats dissent as contamination.

Analysis

Rule / explanation: Track the different elements of the image and their assigned roles.

Example / answer: The ship metaphor gives leaders control while citizens become passengers.

L. Simile

Definition

Rule / explanation: A simile makes an explicit comparison using like or as.

Example / answer: The policy spread like wildfire across the media.

Accessibility

Rule / explanation: Similes make unfamiliar or abstract ideas easier to picture.

Example / answer: The bureaucracy is described as slow as a blocked machine.

Evaluation

Rule / explanation: The chosen comparison carries positive or negative associations.

Example / answer: The proposal is like a sticking plaster on a deep wound.

Journalistic vividness

Rule / explanation: Similes create memorable description in commentary and reportage.

Example / answer: The market reacted like a startled animal.

Analysis

Rule / explanation: Explain the qualities transferred by the comparison.

Example / answer: The sticking-plaster simile presents the reform as superficial and temporary.

M. Personification

Definition

Rule / explanation: Personification gives human actions or qualities to an abstract idea, object or institution.

Example / answer: History will judge this decision.

Agency

Rule / explanation: An abstract force may appear to act independently.

Example / answer: The economy refuses to recover.

Naturalisation

Rule / explanation: Personification can conceal the human actors responsible for events.

Example / answer: The market demands cuts.

Emotional effect

Rule / explanation: Institutions may appear caring, hostile or vulnerable.

Example / answer: Democracy is described as fighting for survival.

Analysis

Rule / explanation: Ask whose agency disappears when an abstraction becomes the actor.

Example / answer: Saying the market punished workers hides the decisions of specific institutions.

N. Hyperbole

Definition

Rule / explanation: Hyperbole is deliberate exaggeration for emphasis or emotional effect.

Example / answer: This reform will change everything.

Urgency

Rule / explanation: Extreme language can present a problem as immediate and exceptional.

Example / answer: The country stands on the edge of disaster.

Humour

Rule / explanation: Obvious exaggeration may create comic effect.

Example / answer: I have explained this a thousand times.

Manipulation risk

Rule / explanation: Hyperbole can distort scale and suppress nuance.

Example / answer: A limited setback is called a national catastrophe.

Analysis

Rule / explanation: Compare the strength of the wording with the available evidence.

Example / answer: The apocalyptic language is not supported by the data.

O. Understatement and litotes

Understatement

Rule / explanation: Understatement presents something as less serious or intense than it is.

Example / answer: The collapse caused a few difficulties.

Litotes

Rule / explanation: Litotes expresses a positive idea by negating its opposite.

Example / answer: The outcome was not insignificant.

Irony

Rule / explanation: Understatement can create dry humour or criticism.

Example / answer: Calling a major scandal an unfortunate administrative episode.

Euphemistic effect

Rule / explanation: It may minimise harm or responsibility.

Example / answer: Job losses are described as a modest adjustment.

Analysis

Rule / explanation: Determine whether the reduced language is humorous, cautious or evasive.

Example / answer: The understatement distances the institution from the severity of the consequences.

P. Euphemism and dysphemism

Euphemism

Rule / explanation: A euphemism replaces a harsh or controversial expression with a milder one.

Example / answer: Job cuts become workforce restructuring.

Institutional function

Rule / explanation: Euphemism can reduce alarm and protect public image.

Example / answer: Civilian deaths become collateral damage.

Dysphemism

Rule / explanation: Dysphemism replaces neutral language with a harsher or more offensive expression.

Example / answer: A tax increase becomes a government raid.

Framing

Rule / explanation: Both devices influence emotional response before evidence is considered.

Example / answer: Reform and destruction may describe the same policy from opposing perspectives.

Analysis

Rule / explanation: Translate the term into neutral language to expose its persuasive force.

Example / answer: Workforce adjustment means employees will lose their jobs.

Q. Rhetorical questions

Definition

Rule / explanation: A rhetorical question is asked for effect rather than to obtain information.

Example / answer: How can democracy survive without trust?

Implied answer

Rule / explanation: The structure presents one answer as obvious.

Example / answer: Who could oppose fairness?

Challenge

Rule / explanation: It can expose contradiction or attack an opposing position.

Example / answer: If the policy works, why are waiting times rising?

Transition

Rule / explanation: A rhetorical question may introduce the next part of the reasoning.

Example / answer: What, then, would genuine reform require?

Risk

Rule / explanation: Questions may pressure the audience instead of proving the claim.

Example / answer: The speaker treats disagreement as unreasonable.

R. Direct address and apostrophe

Direct address

Rule / explanation: Using you engages the listener or reader personally.

Example / answer: Imagine that your access to care depended on your postcode.

Apostrophe

Rule / explanation: Apostrophe directly addresses an absent person, abstract idea or object.

Example / answer: O freedom, how easily your name is misused.

Immediacy

Rule / explanation: The device creates the impression of direct interaction.

Example / answer: You deserve a government that listens.

Collective pressure

Rule / explanation: Direct address may invite solidarity or impose responsibility.

Example / answer: You cannot remain silent.

Analysis

Rule / explanation: Identify who is addressed and how the relationship is constructed.

Example / answer: The second person turns readers from observers into moral participants.

S. Alliteration, assonance and sound patterning

Alliteration

Rule / explanation: Alliteration repeats initial consonant sounds.

Example / answer: A fairer, freer future.

Assonance

Rule / explanation: Assonance repeats vowel sounds.

Example / answer: The long rise in prices.

Consonance

Rule / explanation: Consonance repeats consonant sounds within or at the ends of words.

Example / answer: Blank and think.

Memorability

Rule / explanation: Sound patterns make slogans, headlines and phrases easier to remember.

Example / answer: Build back better.

Critical analysis

Rule / explanation: Do not overinterpret accidental sound repetition.

Example / answer: The pattern matters when it is noticeable, repeated and connected to emphasis.

T. Irony, sarcasm and parody

Irony

Rule / explanation: Irony creates a contrast between literal wording and intended meaning.

Example / answer: What a brilliant success, after the policy misses every target.

Sarcasm

Rule / explanation: Sarcasm is sharper and often intended to mock or wound.

Example / answer: Apparently, losing three ministers counts as stability.

Parody

Rule / explanation: Parody imitates a recognisable style for comic or critical effect.

Example / answer: A campaign slogan is rewritten to expose its contradictions.

Context dependence

Rule / explanation: Irony may be signalled by exaggeration, contradiction, quotation marks or tone.

Example / answer: The word efficient appears in quotation marks after evidence of waste.

Analysis

Rule / explanation: State both the literal meaning and the intended critical meaning.

Example / answer: The ironic praise exposes the gap between official claims and actual outcomes.

U. Allusion, quotation and intertextuality

Allusion

Rule / explanation: An allusion indirectly refers to a well-known event, text, person or cultural idea.

Example / answer: The headline describes the scandal as the government’s Watergate.

Quotation

Rule / explanation: A familiar quotation can borrow authority or shared cultural memory.

Example / answer: The speech echoes the phrase a house divided.

Intertextuality

Rule / explanation: A text may gain meaning through its relationship with earlier texts or genres.

Example / answer: The advertisement imitates a political manifesto.

Audience knowledge

Rule / explanation: The effect depends on whether the audience recognises the reference.

Example / answer: An obscure allusion may exclude some readers.

Analysis

Rule / explanation: Explain what associations the reference imports into the new context.

Example / answer: The Watergate allusion frames the scandal as historic and potentially fatal.

V. Exercises and corrections

1. Identify anaphora

Rule / explanation: We need honesty. We need transparency. We need change.

Example / answer: The repeated opening We need is anaphora, creating rhythm and urgency.

2. Identify antithesis

Rule / explanation: This is not a crisis of resources but a crisis of priorities.

Example / answer: The balanced not X but Y structure presents two opposing interpretations.

3. Analyse metaphor

Rule / explanation: Public trust is the oxygen of democracy.

Example / answer: The metaphor presents trust as essential to democratic survival.

4. Neutralise euphemism

Rule / explanation: The company announced a workforce adjustment.

Example / answer: The company announced job losses or redundancies.

5. Identify chiasmus

Rule / explanation: We shape technology, and technology shapes us.

Example / answer: The reversed structure emphasises mutual influence.

6. Evaluate hyperbole

Rule / explanation: The bill will destroy the nation.

Example / answer: The language creates fear but greatly exceeds what the claim itself proves.

7. Translate

Rule / explanation: La répétition donne du rythme au discours, mais elle ne remplace pas une démonstration fondée sur des preuves.

Example / answer: Repetition gives the speech rhythm, but it does not replace an evidence-based argument.

8. Mini concours challenge

Rule / explanation: Analyse a passage containing anaphora, metaphor, rhetorical question and antithesis.

Example / answer: Example: The repeated opening We cannot wait is anaphora, which creates urgency and collective pressure. The metaphor democracy is on life support presents the political system as critically ill and in need of immediate treatment. The rhetorical question How many warnings do we need? implies that further delay would be irrational. Finally, the antithesis not a question of cost but a question of survival transforms a policy debate into a moral emergency. These devices are powerful, although the dramatic framing may exaggerate the available evidence.
