# United States Civilisation Atlas

Upload `colle-atlas-us.html` to the root of the `EnglishCollesInLyon` GitHub repository and replace the existing file with the same name.

## Included
- Back-to-basics opening with ten essential foundations
- 33-event expandable and filterable timeline
- Complete political and judicial system sections
- Interactive power map, federal bill stepper and court-route explorer
- 16 historically significant presidents
- 20 continental expansion / empire / world-power entries and period explorer
- Federalism, state powers and four-region comparison
- Society and contemporary debates
- Article, video or multimedia source and freely reusable Wikimedia Commons image for every knowledge entry
- Search, essentials-only mode, reader mode, dark mode and accessibility controls
- Local revision list with responsive HTML export and import

## Important
External articles, videos and Wikimedia Commons images require an internet connection. The explanatory text and revision pack structure work offline.
