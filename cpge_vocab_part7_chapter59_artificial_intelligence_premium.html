A. Core AI concepts and technologies

artificial intelligence

IPA: /ˌɑːtɪˈfɪʃəl ɪnˈtelɪdʒəns/
Pronunciation: ar-ti-FISH-uhl in-TEL-i-juhns
French: intelligence artificielle

Definition: Computer systems able to perform tasks that normally require human intelligence.

Example: Artificial intelligence is reshaping science, business, education and politics.

AI

IPA: /ˌeɪ ˈaɪ/
Pronunciation: ay-eye
French: IA

Definition: Abbreviation for artificial intelligence.

Example: AI can support decision-making but also raise ethical concerns.

AI system

IPA: /ˌeɪ ˈaɪ ˈsɪstəm/
Pronunciation: ay-eye SIS-tuhm
French: système d’IA

Definition: A system using artificial intelligence to perform tasks or generate outputs.

Example: An AI system should be tested before being deployed.

algorithm

IPA: /ˈælɡərɪðəm/
Pronunciation: AL-guh-rith-uhm
French: algorithme

Definition: A set of instructions used by a computer to solve a problem or make a decision.

Example: Algorithms increasingly influence what people see online.

machine learning

IPA: /məˌʃiːn ˈlɜːnɪŋ/
Pronunciation: muh-SHEEN LUR-ning
French: apprentissage automatique

Definition: A type of AI in which systems learn patterns from data.

Example: Machine learning is used in finance, healthcare and advertising.

deep learning

IPA: /ˌdiːp ˈlɜːnɪŋ/
Pronunciation: deep LUR-ning
French: apprentissage profond

Definition: A type of machine learning using layered neural networks.

Example: Deep learning has improved image and speech recognition.

neural network

IPA: /ˌnjʊərəl ˈnetwɜːk/
Pronunciation: NYOOR-uhl NET-work
French: réseau de neurones

Definition: A computing system inspired by connections between neurons.

Example: A neural network can recognise patterns in large datasets.

artificial neural network

IPA: /ˌɑːtɪˈfɪʃəl ˌnjʊərəl ˈnetwɜːk/
Pronunciation: ar-ti-FISH-uhl NYOOR-uhl NET-work
French: réseau de neurones artificiels

Definition: A model made of connected processing units used in deep learning.

Example: Artificial neural networks power many AI applications.

model

IPA: /ˈmɒdəl/
Pronunciation: MOD-uhl
French: modèle

Definition: A computer system trained to make predictions or generate outputs.

Example: An AI model can be trained on millions of examples.

AI model

IPA: /ˌeɪ ˈaɪ ˈmɒdəl/
Pronunciation: ay-eye MOD-uhl
French: modèle d’IA

Definition: A model using artificial intelligence techniques.

Example: The AI model performed well on the test data.

Pronunciation: larj LANG-gwij MOD-uhl
French: grand modèle de langage

Pronunciation: JEN-uh-ruh-tiv ay-eye
French: IA générative

IPA: /ˈdeɪtəset/
Pronunciation: DAY-tuh-set
French: jeu de données

Definition: A structured collection of data.

Example: A biased dataset can lead to unfair results.

training data

IPA: /ˈtreɪnɪŋ ˌdeɪtə/
Pronunciation: TRAY-ning DAY-tuh
French: données d’entraînement

Definition: Data used to train an AI model.

Example: Training data shapes the behaviour of AI models.

test data

IPA: /ˈtest ˌdeɪtə/
Pronunciation: test DAY-tuh
French: données de test

Definition: Data used to evaluate a model after training.

French: données étiquetées

Pronunciation: un-LAY-buhld DAY-tuh
French: données non étiquetées

Definition: Data without predefined labels or categories.

big data

IPA: /ˌbɪɡ ˈdeɪtə/
Pronunciation: big DAY-tuh
French: données massives ; big data

Definition: Extremely large datasets analysed by computers.

Example: Big data has made modern AI possible.

data mining

IPA: /ˈdeɪtə ˌmaɪnɪŋ/
Pronunciation: DAY-tuh MY-ning
French: exploration de données

Definition: Finding patterns or information in large datasets.

IPA: /ˈdeɪtə ˌænəˈteɪʃən/
Pronunciation: DAY-tuh an-uh-TAY-shuhn
French: annotation de données

Definition: The process of labelling data for AI training.

Example: Data annotation is often repetitive and underpaid.

data privacy

IPA: /ˈdeɪtə ˈprɪvəsi/
Pronunciation: DAY-tuh PRIV-uh-see
French: confidentialité des données

Definition: Protection of personal data from misuse.

IPA: /ˌpɜːsənəl ˈdeɪtə/
Pronunciation: PUR-suh-nuhl DAY-tuh
French: données personnelles

Definition: Information relating to an identifiable person.

IPA: /ˌsensɪtɪv ˈdeɪtə/
Pronunciation: SEN-si-tiv DAY-tuh
French: données sensibles

Definition: Data requiring special protection, such as health or biometric data.

Example: Sensitive data must be handled carefully.

French: biais

Pronunciation: al-guh-RITH-mik BY-uhs
French: biais algorithmique

Definition: Unfair outcomes produced or amplified by algorithms.

Example: Algorithmic bias can affect hiring, policing and lending.

fairness

IPA: /ˈfeənəs/
Pronunciation: FAIR-nuhs
French: équité

Definition: The quality of treating people or groups justly.

Example: Fairness is difficult to define in AI systems.

discrimination

IPA: /dɪˌskrɪmɪˈneɪʃən/
Pronunciation: dis-krim-i-NAY-shuhn
French: discrimination

transparency

IPA: /trænˈspærənsi/

French: transparence

Definition: Openness about how a system works or makes decisions.

Example: Transparency helps users understand AI decisions.

Pronunciation: ik-splay-nuh-BIL-uh-tee
French: explicabilité
