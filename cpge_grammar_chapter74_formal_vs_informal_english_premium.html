The golden rule

Formal does not mean unnatural.
Informal does not mean incorrect.
Choose the register that fits the audience, purpose and medium.
Formal essayfull forms, precise vocabulary, explicit logic

A. Understanding register

Formal English

Rule / explanation: Used in academic, official and professional contexts.

Example / answer: The committee concluded that further investigation was necessary.

Informal English

Rule / explanation: Used in conversation, personal messages and relaxed interaction.

Example / answer: They figured out that they needed to look into it again.

Register choice

Rule / explanation: Audience, purpose, medium and relationship determine the appropriate register.

Example / answer: A khôlle should sound formal but natural.

CPGE relevance

Rule / explanation: Students must recognise register shifts and choose suitable language.

Example / answer: The journalist switches to informal wording to create proximity.

B. Contractions

Informal contractions

Rule / explanation: Contractions are normal in speech and informal writing.

Example / answer: The government didn’t expect such a strong reaction.

Formal full forms

Rule / explanation: Prefer full forms in formal essays.

Example / answer: The government did not expect such a strong reaction.

Oral flexibility

Rule / explanation: Contractions remain natural in formal spoken English.

Example / answer: I don’t entirely agree with the author.

Consistency

Rule / explanation: Do not alternate randomly between contracted and full forms.

Example / answer: Maintain a coherent register throughout the paragraph.

C. Formalising phrasal verbs

find out → determine

Rule / explanation: Use a precise single-word verb in formal analysis.

Example / answer: The inquiry sought to determine why the system failed.

look into → investigate

Rule / explanation: Investigate is more formal.

Example / answer: The committee investigated the allegations.

put off → postpone

Rule / explanation: Postpone suits formal prose.

Example / answer: The vote was postponed until September.

set up → establish

Rule / explanation: Establish is formal for institutions and systems.

Example / answer: The government established an independent commission.

deal with → address

Rule / explanation: Address often sounds more analytical.

Example / answer: The reform fails to address regional inequality.

D. More verb choices

bring up → raise

Rule / explanation: Use raise for an issue or objection.

Example / answer: The article raises an important constitutional question.

point out → note

Rule / explanation: Use note, observe, emphasise or highlight.

Example / answer: The author highlights the lack of transparency.

come up with → devise

Rule / explanation: Use devise, formulate or develop.

Example / answer: The committee devised a new funding model.

cut down on → reduce

Rule / explanation: Use reduce for quantity or impact.

Example / answer: The policy aims to reduce administrative costs.

go up → increase

Rule / explanation: Use increase, rise or grow.

Example / answer: Housing costs increased substantially.

E. Precise nouns

a lot of → considerable

Rule / explanation: Replace vague quantity with precise wording.

Example / answer: The reform generated considerable opposition.

things → factors

Rule / explanation: Choose the exact noun.

Example / answer: Several factors contributed to the decline.

problem → challenge

Rule / explanation: Use a noun that reflects the type of difficulty.

Example / answer: The policy faces an implementation challenge.

result → outcome

Rule / explanation: Distinguish neutral outcome from consequence.

Example / answer: The long-term outcome remains uncertain.

people → citizens / respondents

Rule / explanation: Name the relevant group.

Example / answer: Many respondents expressed concern.

F. Precise adjectives

good → effective

Rule / explanation: State the criterion being judged.

Example / answer: The measure was effective in reducing delays.

bad → inadequate

Rule / explanation: Select the precise weakness.

Example / answer: The evidence is inadequate.

big → significant

Rule / explanation: Use significant, major or substantial.

Example / answer: The scandal had significant consequences.

small → marginal

Rule / explanation: Use limited, minor or marginal.

Example / answer: The reform produced marginal improvements.

weird → unusual

Rule / explanation: Avoid conversational evaluation.

Example / answer: The pattern is statistically unusual.

G. Degree and intensity

really → highly

Rule / explanation: Use a more precise intensifier.

Example / answer: The proposal is highly controversial.

totally → entirely

Rule / explanation: Choose the formal equivalent.

Example / answer: The explanation is entirely unconvincing.

kind of → somewhat

Rule / explanation: Use somewhat to soften formal criticism.

Example / answer: The conclusion is somewhat overstated.

a bit → slightly

Rule / explanation: Use slightly for small degree.

Example / answer: Turnout was slightly lower than expected.

pretty → fairly

Rule / explanation: Pretty is informal; fairly suits formal British English.

Example / answer: The evidence is fairly convincing.

H. Personal reaction vs evaluation

I like → I find convincing

Rule / explanation: Replace taste with analysis.

Example / answer: I find the author’s argument convincing.

I do not like → I question

Rule / explanation: Express a reasoned objection.

Example / answer: I question the reliability of the survey.

interesting

Rule / explanation: Explain why the point matters.

Example / answer: The comparison is significant because it reveals a regional divide.

I feel that

Rule / explanation: Use in my view or I would argue.

Example / answer: I would argue that public trust is central.

shocking

Rule / explanation: Describe significance instead of emotion.

Example / answer: The figures reveal a severe and unexpected decline.

I. Directness and politeness

Direct request

Rule / explanation: Informal interaction often uses imperatives.

Example / answer: Send me the figures.

Polite request

Rule / explanation: Use could or would.

Example / answer: Could you please send me the figures?

Official request

Rule / explanation: Use a complete courteous formula.

Example / answer: I would be grateful if you could provide the report.

Formal disagreement

Rule / explanation: Soften personal confrontation.

Example / answer: I am not entirely convinced by this interpretation.

Clarification

Rule / explanation: Use a formal question frame.

Example / answer: Could you clarify what you mean by resilience?

J. Pronouns and distance

Vague you

Rule / explanation: Avoid addressing an undefined reader in formal prose.

Example / answer: Weak: You can see that the policy failed.

Impersonal alternative

Rule / explanation: State the evidence directly.

Example / answer: The evidence indicates that the policy failed.

Vague we

Rule / explanation: Avoid unsupported inclusive claims.

Example / answer: Weak: We all know that trust is declining.

One

Rule / explanation: One is formal but can sound stiff if repeated.

Example / answer: One might question the sample’s reliability.

Controlled I

Rule / explanation: First person can present a clear argumentative position.

Example / answer: I would argue that the conclusion is too broad.

K. Formal connectors

and → furthermore

Rule / explanation: Use for developed addition.

Example / answer: Furthermore, the policy failed to address access.

but → however

Rule / explanation: Use for explicit contrast.

Example / answer: However, the evidence remains limited.

so → therefore

Rule / explanation: Use for logical consequence.

Example / answer: Funding was reduced; therefore, waiting times increased.

plus → moreover

Rule / explanation: Replace conversational addition.

Example / answer: Moreover, local authorities received no extra funding.

at the end of the day → ultimately

Rule / explanation: Avoid conversational clichés.

Example / answer: Ultimately, legitimacy depends on public trust.

L. Questions and exclamations

Informal rhetoric

Rule / explanation: Conversation and journalism may use emotional questions.

Example / answer: How on earth could this happen?

Formal question

Rule / explanation: Remove emotional intensification.

Example / answer: How could the regulatory failure have occurred?

Exclamation marks

Rule / explanation: Avoid them in academic essays.

Example / answer: The figures are significant.

Question chains

Rule / explanation: Several consecutive questions sound dramatic and unstructured.

Example / answer: Use one key question followed by analysis.

Embedded form

Rule / explanation: Formal prose often embeds the issue.

Example / answer: The central issue is how the failure occurred.

M. Slang, idioms and clichés

Slang

Rule / explanation: Avoid slang in essays and formal presentations.

Example / answer: Weak: The policy was a total mess.

Formal replacement

Rule / explanation: Describe the weakness analytically.

Example / answer: The policy was poorly designed and inconsistently implemented.

Idioms

Rule / explanation: Idioms may reduce precision.

Example / answer: Use limited impact rather than a drop in the ocean.

Loaded labels

Rule / explanation: Analyse them when they occur in a source.

Example / answer: The phrase tax grab creates a hostile tone.

Rare expressions

Rule / explanation: Do not force unfamiliar idioms.

Example / answer: Prefer accurate standard English.

N. Ellipsis and fragments

Conversation fragments

Rule / explanation: Informal speech often omits recoverable words.

Example / answer: Sounds good. Probably not. Any questions?

Formal completeness

Rule / explanation: Academic prose normally uses complete sentences.

Example / answer: This interpretation appears plausible.

Headlines

Rule / explanation: Headlines often omit articles and auxiliaries.

Example / answer: Government under pressure after scandal.

Notes and slides

Rule / explanation: Fragments are acceptable in headings or notes.

Example / answer: Main cause: inadequate funding.

Analysis

Rule / explanation: Recognise compression without copying it into essays.

Example / answer: The headline creates urgency through ellipsis.

O. Formal email language

Greeting

Rule / explanation: Use Dear + title/name.

Example / answer: Dear Professor Williams,

Purpose

Rule / explanation: State the reason for writing clearly.

Example / answer: I am writing to request information about the examination.

Request

Rule / explanation: Use a polite modal structure.

Example / answer: I would be grateful if you could confirm the deadline.

Closing

Rule / explanation: Use an appropriate formal closing.

Example / answer: Yours sincerely,

Avoid chat style

Rule / explanation: Do not use emojis, abbreviations or abrupt one-line requests.

Example / answer: Maintain a courteous and complete message.

P. Formal spoken English

Natural formality

Rule / explanation: Spoken academic English may use contractions and simpler syntax.

Example / answer: I don’t entirely agree, because the evidence is limited.

Signposting

Rule / explanation: Guide the listener through the structure.

Example / answer: I would now like to turn to the broader context.

Repair

Rule / explanation: Reformulate naturally when needed.

Example / answer: More precisely, the issue concerns access rather than funding alone.

Avoid stiffness

Rule / explanation: Do not recite excessively written sentences.

Example / answer: Use clear clauses and controlled pauses.

Engagement

Rule / explanation: Formal speech can still sound lively.

Example / answer: That is an important point, but another interpretation is possible.

Q. Journalism and register shifts

Mixed register

Rule / explanation: Articles often combine neutral reporting with informal headlines or quotations.

Example / answer: The body is neutral, while the headline uses policy chaos.

Accessibility

Rule / explanation: Informal wording can make complex issues accessible.

Example / answer: The journalist uses everyday vocabulary for a broad audience.

Proximity

Rule / explanation: Questions and pronouns can create closeness.

Example / answer: What does this mean for your household?

Bias

Rule / explanation: Loaded informal labels may shape judgement.

Example / answer: The phrase bureaucratic nightmare frames regulation negatively.

Shift effect

Rule / explanation: Explain why the register changes.

Example / answer: The informal ending makes the warning more immediate.

R. Common French-speaker mistakes

according to me

Rule / explanation: Use in my view or I would argue.

Example / answer: In my view, the policy is disproportionate.

actually

Rule / explanation: Actually means in fact, not actuellement.

Example / answer: Currently means at present.

eventually

Rule / explanation: Eventually means in the end, not éventuellement.

Example / answer: Possibly or perhaps express possibility.

assist to

Rule / explanation: Attend means be present; assist means help.

Example / answer: She attended the hearing.

demand

Rule / explanation: Demand is strong; request is more neutral.

Example / answer: The committee requested further evidence.

pass an exam

Rule / explanation: Take or sit means participate; pass means succeed.

Example / answer: She sat the exam and passed it.

S. Exercises

1. Formalise the verb

Rule / explanation: The committee looked into the problem.

Example / answer: The committee investigated the problem.

2. Remove informal intensity

Rule / explanation: The policy was really bad.

Example / answer: The policy was highly ineffective.

3. Replace vague language

Rule / explanation: A lot of things caused the crisis.

Example / answer: Several factors contributed to the crisis.

4. Formalise the opinion

Rule / explanation: I don’t like the author’s argument.

Example / answer: I am not convinced by the author’s argument.

5. Formalise the connector

Rule / explanation: The budget was cut, so waiting times increased.

Example / answer: The budget was reduced; consequently, waiting times increased.

6. Adapt for speech

Rule / explanation: It is not possible for me to concur entirely.

Example / answer: I don’t entirely agree.

7. Translate

Rule / explanation: Le journaliste adopte un ton plus familier afin de se rapprocher du lecteur.

Example / answer: The journalist adopts a more informal tone to create proximity with the reader.

8. Mini challenge

Rule / explanation: Write one formal written version and one natural formal oral version.

Example / answer: Written: The government did not assess the consequences adequately. Oral: The government didn’t assess the consequences carefully enough.

Mini recap
